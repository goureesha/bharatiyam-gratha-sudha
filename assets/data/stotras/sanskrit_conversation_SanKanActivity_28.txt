ಏತಾವತ್ ಪರ್ಯಂತಂ ಕುತ್ರ ಗತವಾನ್? = ಇಷ್ಟು ಹೊತ್ತು ಎಲ್ಲಿ ಹೋಗಿದ್ದೆ?
ಕುತ್ರಾಪಿ ನ, ಅತ್ರೈವ ಗತವಾನ್ = ಎಲ್ಲೂ ಇಲ್ಲ, ಇಲ್ಲೇ ಹೋಗಿದ್ದೆ.
ಪಠನು ಕಿಮಪಿ ನಾಸ್ತಿ ವಾ? = ಓದುವುದು ಏನೂ ಇಲ್ಲವಾ?
ವಕ್ತಾ ಪ್ರಷ್ಟಾ ಕೋಶ್ಪಿ ನಾಸ್ತಿ ವಾ? =  ಹೇಳುವವರು ಕೇಳುವವರು ಯಾರೂ ಇಲ್ಲವಾ?
ಸಹಸ್ರವಾರಂ ಉಕ್ತಂ, ಶ್ರುತವಾನ್ ವಾ? = ಸಾವಿರ ಸಾರಿ ಹೇಳಿದೆ, ಕೇಳಿದೆಯಾ?
ಮಮ ಗೃಹಪಾಠಃ ಬಹು ಅಸ್ತಿ = ನನಗೆ ಹೋಂವರ್ಕ್ ತುಂಬಾ ಇದೆ.
ಅದ್ಯ ಏವ ಶುಲ್ಕಂ ದಾತವ್ಯಂ ಅಸ್ತಿ  = ಇವತ್ತೇ ಫೀಸ್ ಕೊಡಬೇಕು.
ಅಂತಿಮ ದಿನಾಂಕಃ ಕದಾ? = ಕೊನೇ ದಿನಾಂಕ ಯಾವಾಗ?
ಅದ್ಯ ಏವ ದಾತವ್ಯಂ ವಾ? = ಇವತ್ತೇ ಕೊಡಬೇಕಂತಾ?
ಕಿಂ? ಅದ್ಯ ದಾತವ್ಯಂ ವಾ? = ಏನು? ಇವತ್ತು ಕೊಡಬೇಕಂತಾ?
ಅಂಬ, ಏಕಂ ನವೀನಂ ಯುತಕಂ ಆವಶ್ಯಕಮ್ = ಅಮ್ಮ, ನಂಗೊಂದು ಹೊಸ ಅಂಗಿ ಬೇಕು.
ಗೋವಿಂದ, ಆಪಣಂ ಗತ್ವಾ ಆಗಚ್ಛತಿ ವಾ? = ಗೋವಿಂದ, ಅಂಗಡಿಗೆ ಹೋಗಿ ಬರುತ್ತೀಯಾ?
ಅಂಬ, ವೇಣೀಬಂಧಂ ಕರೋತು, ಶಾಲಾಯಾಃ ವಿಲಂಬಃ ಭವತಿ =  ಅಮ್ಮ, ಜಡೆ ಹಾಕು, ಶಾಲೆಗೆ ಹೊತ್ತಾಗಿ ಬಿಡುತ್ತೆ.
ಸಮಯಃ ಕಃ ಇತಿ ಜಾನಾತಿ ವಾ? = ಘಂಟೆ ಎಷ್ಟು ಅಂತ ಗೊತ್ತಾ?
ಇದಾನಿಂ ವಾ ಗೃಹ ಆಗಮನಮ್? = ಈಗಲಾ ಮನೆಗೆ ಬರುವುದು?
ಪಿತರಂ ಏಕವಾರಂ ಆಹ್ವಯತು = ತಂದೆಯವರನ್ನು ಒಮ್ಮೆ ಕರೆ.
ಭವತಃ ಕೃತೇ ಕಥನಾಪೇಕ್ಷಯಾ ಸ್ವಯಂ ಕರಣಂ ವರಮ್  = ನಿನಗೆ ಹೇಳೋದಕ್ಕಿಂತಲೂ ಮಾಡಿಕೊಳ್ಳೋದು ವಾಸಿ.
ಭವತಿ ಅನಂತರಂ ಉಪವಿಶತು = ನೀನು ಅನಂತರ ಕುಳಿತುಕೋ
ತಸ್ಮೈ ಚತ್ವಾರಿ, ಮಮ ತು ತ್ರೀಣಿ ಏವ = ಅವನಿಗೆ ನಾಲ್ಕು, ನಂಗೆ ಮಾತ್ರ ಮೂರು.
ಸರ್ವಂ ಭವಾನ್ ಏವ ಭಕ್ಷಿತವಾನ್ ವಾ? = ಎಲ್ಲವನ್ನೂ ನೀನೇ ತಿಂದು ಬಿಟ್ಟೆಯಾ?
ಏಕಂ ಕಾರ್ಯಂ ಕತಿವಾರಂ ವಕ್ತವ್ಯಮ್? = ಒಂದು ಕೆಲಸವನ್ನು ಎಷ್ಟು ಸಾರಿ ಹೇಳಬೇಕು?
ಏಕಕಾರ್ಯಸ್ಯ ಕೃತೇ ಕತಿವಾರಂ ಆಹ್ವಾನಮ್? = ಒಂದು ಕೆಲಸಕ್ಕೆ ಎಷ್ಟು ಸಾರಿ ಕರೆಯಬೇಕು?
ಕುತ್ರಾಪಿ ಸ್ಥಾಪಯತಿ, ಅನಂತರಂ ಮಾಂ ಪೃಚ್ಛತಿ = ಎಲ್ಲಾದರೂ ಇಟ್ಕೋತೀಯಾ, ಅನಂತರ ನನ್ನ ಕೇಳ್ತೀಯಾ!
ಭೋಜನಸಮಯೇ ಆಹ್ವಯಾಮಿ, ಗಚ್ಛತು ಇದಾನೀಮ್ = ಊಟದ ಹೊತ್ತಿಗೆ ಕರೀತೀನಿ, ಹೋಗು ಈಗ.
'ತೇನ ಸಹ ನ ಗಚ್ಛತು' ಇತಿ ದಶವಾರಂ ಉಕ್ತವತೀ = ಅವನ ಸಂಗಡ ಹೋಗಬೇಡ ಅಂತ ಹತ್ತು ಸಾರಿ ಹೇಳಿದೆ.
ಪಾಶ್ರ್ವಗೃಹಂ ಗತ್ವಾ ಪತ್ರಿಕಾಂ ಆನಯತು = ಪಕ್ಕದ ಮನೆಗೆ ಹೋಗಿ ಪೇಪರ್ ತಗೊಂಬಾ.
ಅಹಂ ಪುಟೀಕರ್ತುಂ, ಏತೇ ವ್ಯಸ್ತಂ ಕರ್ತುಮ್ = ನಾನು ಮಡಿಚಿಡೋಕೆ, ಇವರು ಕಿತ್ತೆಸೆಯೋದಕ್ಕೆ.
ಅಂಬಾ! ಕಿಂಚಿತ್ ಸೀವನಂ ಕೃತ್ವಾ ದದಾತು = ಅಮ್ಮ, ಸ್ವಲ್ಪ ಹೊಲಿದು ಕೊಡು.
ಏತತ್ ಬಹಿಃ ಕ್ಷಿಪ್ತ್ವಾ ಆಗಚ್ಛತು = ಇದನ್ನು ಹೊರಗೆಸೆದು ಬಾ.
ಶಾಕಂ ಕಿಂಚಿತ್ ಕರ್ತಯಿತ್ವಾ ದದಾತು = ಸ್ವಲ್ಪ ತರಕಾರಿ ಹೆಚ್ಚಿಕೊಡು.
ಏಕವಾರಂ ಏವ ಸರ್ವಂ ವದತು, ಭವತೀ = ಒಂದೇ ಸಾರಿಗೆ ಎಲ್ಲಾ ಹೇಳಮ್ಮಾ.
ಭವಾನ್ ಧೀಮಾನ್/ಕುಶಲಃ ಕಿಲ? = ನೀನು ಜಾಣ ಮರಿ ಅಲ್ವಾ?
ಅಂಬಾ, ಅತ್ರ ಕಿಂಚಿತ್ ವೇದನಾ ಅಸ್ತಿ = ಅಮ್ಮ, ಇಲ್ಲಿ ಸ್ವಲ್ಪ ನೋಯುತ್ತೆ.
ನಿದ್ರಾಂ ಕರೋತು, ಸರ್ವಂ ಸಮ್ಯಕ್ ಭವತಿ = ನಿದ್ದೆ ಮಾಡು, ಎಲ್ಲವೂ ಸರಿ ಹೋಗುತ್ತೆ.
ರಾತ್ರೌ ಕಷಾಯಂ ಕರೋಮಿ, ಪೀತ್ವಾ ಶಯನಂ  ಕರೋತು = ರಾತ್ರಿ ಕಷಾಯ ಮಾಡ್ತೀನಿ ಕುಡಿದು ಮಲಗು.
ಕಿಂಚಿತ್ ಪಠತು ನಾಮ = ಸ್ವಲ್ಪ ಓದಿಕೋ, ನೋಡೋಣ.
ಕಿಂ, ಇದಾನೀಂ ಏವ ನಿದ್ರಾ ವಾ? = ಏನು ಈಗಲೇ ನಿದ್ರೇನಾ?
ಗಣಿತೇ ದಶ ಅಂಕಾಃ ಏವ ಇತಿ ಸ್ಮರತಿ ವಾ? = ಗಣಿತದಲ್ಲಿ ಹತ್ತೇ ಅಂಕ ಅಂತ ನೆನಪಿದೆಯಾ?
ಅಂಬ, ಅನ್ಯತ್ರ ಪಠಿತುಂ ಕಥಯತು = ಅಮ್ಮ, ಬೇರೆಡೆ ಓದ್ಕೋಳೋಕೆ ಹೇಳು.
ಏತಾವತ್ ಧನಂ ನ ಪರ್ಯಾಪ್ತಮ್ = ಇಷ್ಟು ಹಣ ಸಾಕಾಗೋಲ್ಲ.
ಶಿರಸಿ ತೈಲಸಮುಕ್ಷಣಂ ಕರೋತು = ತಲೆಗೆ ಎಣ್ಣೆ ಹಾಕು.
ನಿದ್ರಯಾ ಆಂದೋಲನಂ ಕರೋತಿ, ಪಶ್ಯತು = ತೂಕಡಿಸ್ತಿದ್ದಾನೆ, ನೋಡು.
ಶಿಶುಃ ರೋದಿತಿ, ಪಶ್ಯತು = ಮಗು ಅಳ್ತಿದೆ, ನೋಡು.
ಸಂಮಾರ್ಜನಂ ಕೃತವತೀ ವಾ? = ಗುಡಿಸಿದೆಯಾ?
ಇದಾನೀಂ ಅಪಿ ರಂಗವಲ್ಲೀಂ ನ ಲಿಖಿತವತೀ ವಾ? = ಇನ್ನೂ ರಂಗೋಲಿ ಬರೆದಿಲ್ಲವಾ?
ಕತಿವಾರಂ ವಕ್ತವ್ಯಮ್? = ಎಷ್ಟು ಸಾರಿ ಹೇಳಬೇಕು?
ಕುತ್ರ ಗತಃ? ಗೃಹೇ ನಾಸ್ತಿ ವಾ? = ಎಲ್ಲಿ ಹೋದ? ಮನೆಯಲ್ಲಿ ಇಲ್ಲವಾ?
ಪ್ರತ್ಯುತ್ತರಂ ಮಾಂ ವದತು? = ತಿರುಗಿ  ಉತ್ತರ ಕೊಡಬೇಡ.
ಉಕ್ತಂ  ನ ಶ್ರುತವಾನ್ ವಾ? = ಹೇಳಿದ್ದು ಕೇಳಲಿಲ್ಲವಾ?
ಅದ್ಯತನಪತ್ರಿಕಾಂ ದದಾತು = ಇವತ್ತಿನ ಪೇಪರ್ ಕೊಡು.
ಸಃ ಆಗತಃ ಏಷಃ ಪ್ರಸ್ಥಿತಃ = ಅವನು ಬಂದ, ಇವ ಹೊರಟ.
ಕಟಂ ಪ್ರಸಾರಯತು = ಚಾಪೆ ಹಾಕು
ರಾಜಕಃ ವಸ್ತ್ರಂ ನೀತವಾನ್ ವಾ? = ಅಗಸರವ ಬಟ್ಟೆ ಒಯ್ದನೇ?
ರಾಜಕಾಯ ವಸ್ತ್ರಾಣಿ ದತ್ತವತೀ ವಾ? = ಅಗಸನಿಗೆ ಬಟ್ಟೆ ಕೊಟ್ಟೇನಮ್ಮಾ?
ಆದ್ರ್ರವಸ್ತ್ರಾಣಿ ಆತಪೇ ಪ್ರಸಾರಯತು = ಒದ್ದೆಬಟ್ಟೆ ಬಿಸಿಲಿನಲ್ಲಿ ಒಣಗಿಹಾಕು
ಗತ್ವಾ ಶಯನಂ ಕರೋತು = ಹೋಗಿ ಮಲಗಿಕೋ.
ಏಕಂ ಆಸಂದಂ ಆನಯತು = ಒಂದು ಕುರ್ಚಿ ತಗೊಂಡು ಬಾ.
ಇದಾನೀಂ ಅಪಿ ನಿದ್ರಾ ನ ಆಗತಾ ವಾ? = ಇನ್ನೂ ನಿದ್ದೆ ಬಂದಿಲ್ಲವಾ?
ಪಶ್ಯತು ಕಃ ಶಬ್ದಂ ಕರೋತಿ? = ಯಾರು ಶಬ್ದ ಮಾಡ್ತಿದ್ದಾರೆ, ನೋಡು.
ಕಿಮಿದಂ? ಸರ್ವತ್ರ ಅವಕರಃ ! = ಏನಿದು? ಎಲ್ಲೆಡೆ ಕಸ !
ದ್ವಾರಂ ಪಿದಧಾತು (ಪಿಧೀಯತಾಂ) = ಬಾಗಿಲು ಹಾಕು.
ಏತತ್ ಕಿಂಚಿತ್ ಗೃಹ್ಣಾತು = ಇದನ್ನು ಸ್ವಲ್ಪ ಹಿಡಿ.
ಮಧ್ಯೇ ವಾರ್ತಾಲಾಪಂ ಮಾ ಕರೋತು = ಮಧ್ಯೇ ಮಾತನಾಡಬೇಡ.
ಕೋಲಾಹಲಂ ಮಾ ಕರೋತು = ಗದ್ದಲ ಮಾಡಬೇಡ.
ದೀಪಂ ಜ್ವಾಲಯತು = ದೀಪ ಹಾಕು.
ದೀಪಂ ನಿರ್ವಾಪಯತು = ದೀಪ ಆರಿಸು.
ವ್ಯಜನಂ ಚಾಲಯತು = ಫ್ಯಾನ್ ಹಾಕು.
ಕರದೀಪಂ ಗೃಹೀತ್ವಾ ಗಚ್ಛತಂ, ಬಹಿಃ ಅಂಧಕಾರಃ = ಟಾರ್ಚ್ ಹಿಡಿದು ಹೋಗು, ಹೊರಗಡೆ ಕತ್ತಲೆ ಇದೆ.
ಆವಹಿತಮನಸಾ ಕರ್ತವ್ಯಮ್ = ಮನಸಿಟ್ಟು ಮಾಡಬೇಕು.
ತಂ ಯಾನಂ ಆರೋಪ್ಯ ಆಗಚ್ಛತು = ಅವನನ್ನು ಬಸ್ ಹತ್ತಿಸಿ ಬಾ.
ಪಾಣಿಪಾದಂ ಪ್ರಕ್ಷಾಲಯತು = ಕೈಕಾಲು ತೊಳೆ.
ಕುಡ್ಮಾನ್ ಯೋಜಯತು = ಗುಂಡಿ ಹಾಕಿಕೋ.
ಆರೋಗ್ಯಮ್ ನಿರ್ಲಕ್ಷ್ಯ ಮಾ ಪ ಪಟಾತು = ಆರೋಗ್ಯ ಹಾಳು ಮಾಡಿಕೊಂಡು ಓದಬೇಡ.
ಇತಃ ಪರಂ ಏವಂ ನ ಕರ್ತವ್ಯಮ್! = ಇನ್ನು ಮೇಲೆ ಹೀಗೆ ಮಾಡಬಾರದು!
ದ್ರಾವಿಡ ಪ್ರಾಣಾಯಾಮೇನ ಮಾ ವದತು = ಸುತ್ತಿ ಬಳಸಿ ಮಾತನಾಡಬೇಡ.
ಸಮ್ಯಕ್ ದಂತ ಧಾವನಂ ಕರೋತು = ಸರಿಯಾಗಿ ಹಲ್ಲು ಉಜ್ಜು
ವಿನಾ ಕಾರಣಂ ಕುಪ್ಯತಿ = ಸುಮ್ನೆ ಕೋಪಗೊಳ್ಳುತ್ತೀಯ
ವೃಥಾ ಕಾಲಹರಣಂ ಕರೋತಿ = ವ್ಯರ್ಥ ಕಾಲ ಕಳೀತೀಯ.
ಮಾಂ ವೃಥಾ ಮಾ ಕೋಪಯತು = ನನ್ನನ್ನ ಸುಮ್ಮನೆ ರೇಗಿಸ್ಬೇಡ
ಹಠಂ ಮಾ ಕರೋತು = ಹಠ ಮಾಡಬೇಡ.
ಕಿಮರ್ಥಂ ಭವಾನ್ ಕ್ಷುಭ್ಯತಿ? = ಯಾಕೆ ನೀನು ಸಿಡುಕುತ್ತೀಯ?
ಪ್ರಥಮಂ ಶಿರೋಮಾರ್ಜನಂ ಕರೋತು = ಮೊದಲು ತಲೆ ಒರೆಸಿಕೋ.
ಆದ್ರ್ರಂ ವಸ್ತ್ರಂ ನ ಧಾರಯತು = ಒದ್ದೆ ಬಟ್ಟೆ ಹಾಕ್ಕೋಬೇಡ.
ಶೀಘ್ರಂ ಸ್ನಾನಂ ಕೃತ್ವಾ ಆಗಚ್ಛತು = ಬೇಗ ಸ್ನಾನ ಮಾಡಿ ಬಾ.
ಭಾಂಡೇ ಜಲಂ ಅಸ್ತಿ ವಾ, ಪಶ್ಯತು? = ಹಂಡೆಯಲ್ಲಿ ನೀರಿದೆಯಾ ನೋಡು?
ಆಕಾಶವಾಣೀಂ ಚಾಲಯತು, ವಿವಿಧಭಾರತಿ ಯೋಜಯತು = ರೇಡಿಯೋ ಹಾಕು, ವಿವಿಧಭಾರತಿ ಹಾಕು
ವಾರ್ತಾ ಸಮಾಪ್ತಾ ವಾ? = ವಾರ್ತೆ ಮುಗಿಯಿತೇ?
