ಮಾತಾ/ಜನನೀ = ತಾಯಿ
ಪುತ್ರಃ = ಮಗ
ಪುತ್ರೀ = ಮಗಳು
ಭಗಿನೀ = ಸೋದರಿ
ಭ್ರಾತಾ = ಸೋದರ
ಅಗ್ರಜಾ = ಅಕ್ಕ
ಅನುಜಾ = ತಂಗಿ
ಅಗ್ರಜಃ = ಅಣ್ಣ
ಅನುಜಃ = ತಮ್ಮ
ಜ್ಯೇಷ್ಠಃ = ಹಿರಿಯವನು
ಕನಿಷ್ಠಃ = ಕಿರಿಯವನು
ಪೌತ್ರಃ = ಮಗನ ಮಗ
ಪೌತ್ರೀ = ಮಗನ ಮಗಳು
ದೌಹಿತ್ರಃ = ಮಗಳ ಮಗ
ದೌಹಿತ್ರೀ = ಮಗಳ ಮಗಳು
ಶ್ವಶುರಃ = ಮಾವ (ಗಂಡ/ಹೆಂಡತಿಯ ತಂದೆ)
ಶ್ವಶ್ರೂಃ = ಅತ್ತೆ (ಗಂಡ/ಹೆಂಡತಿಯ ತಾಯಿ)
ಜಾಮಾತಾ = ಅಳಿಯ
ಸ್ನುಷಾ = ಸೊಸೆ
ಭ್ರಾತೃಜಾ = ಸೋದರನ ಮಗಳು
ಕುಟುಂಬಃ = ಕುಟುಂಬದವರು
ಪಿತಾಮಹಃ = ತಂದೆಯ ತಂದೆ
ಪಿತಾಮಹೀ = ತಂದೆಯ ತಾಯಿ
ಮಾತಾಮಹಃ = ತಾಯಿಯ ತಂದೆ
ಮಾತಾಮಹೀ = ತಾಯಿಯ ತಾಯಿ
ಪಿತೃವ್ಯಃ = ಚಿಕ್ಕಪ್ಪ/ದೊಡ್ಡಪ್ಪ
ಮಾತುಲಃ = ಸೋದರ ಮಾವ
ಮಾತುಲಾನೀ = ಸೋದರತ್ತೆ
ಭಾಗಿನೇಯಃ = ಸೋದರಿಯ ಮಗ
ಭಾಗಿನೇಯೀ = ಸೋದರಿಯ ಮಗಳು
ಭ್ರಾತೃಜಾಯಾ = ಸೋದರನ ಪತ್ನಿ
ದೇವರಃ = ಪತಿಯ ಸೋದರ
