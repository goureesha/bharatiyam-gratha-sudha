|| ನೈವೇದ್ಯ  =  ಸಂಸ್ಕೃತ ಭಾಷೆಯಲ್ಲಿ ||

ಬಾಳೆಹಣ್ಣು = ಕದಳೀ ಫಲಂ
ಮಾವಿನ ಹಣ್ಣು = ಆಮ್ರ ಫಲಂ
ಕಿತ್ತಳೆ = ನಾರಂಗ ಫಲಂ
ತೆಂಗಿನಕಾಯಿ =  ನಾರಿಕೇಳ ಫಲಂ
ಸೀಬೆ =  ಬಹುಬೀಜ ಫಲಂ
ದಾಳಿಂಬೆ = ದಾಡಿಮ ಫಲಂ
ಹಲಸಿನ ಹಣ್ಣು = ಪನಸ ಫಲಂ
ಎಲಚಿ = ಬದರ ಫಲಂ
ಖರ್ಜೂರ = ಖರ್ಜೂರ ಫಲಂ
ಮೂಸಂಬಿ = ಜಂಬೀರ ಫಲಂ
ದ್ರಾಕ್ಷಿ = ದ್ರಾಕ್ಷಾ ಫಲಂ
ನೇರಳೆ = ಜಂಬೂ ಫಲಂ
ಚಿಕ್ಕೂ/ಸಪೋಟ = ಕೃಷ್ಣಬೀಜ ಫಲಮ್
ಸೌತೆಕಾಯಿ = ಉರ್ವಾರುಕ ಫಲಂ
ಚಕ್ಕೋತ = ಲಿಕುಚ ಫಲಂ
ಕಬ್ಬು = ಇಕ್ಷು ಖಂಡಮ್
ಕಡಲೆಬೇಳೆ = ಚಣಕ ದಳ
ತಂಬಿಟ್ಟು = ತಂಡುಲ ಚೂರ್ಣ
ಹೆಸರುಬೇಳೆ = ಮುದ್ಗದಳ
ಕಡಲೆಪುರಿ = ಧಾನಾ
ಮಂಡಕ್ಕಿ = ಭ್ರಷ್ಟಯವಃ
ಒಬ್ಬಟ್ಟು = ಅಪೂಪ (ಪೋಲಿಕಾ)
ಲಾಡು = ಲಡ್ಡು ಕಮ್
ಒಡೆ = ಮಾಷಾ ಪೂಪ
ರೊಟ್ಟಿ = ಕಾಂದವ
ವಾಂಗಿಬಾತು = ವೃಂತಕಾನ್ನಂ
ತಂಬಿಟ್ಟು = ತಂಡುಲ ಚೂರ್ಣ ತಂಡುಲ ಪಿಷ್ಠ
ಪಲ್ಯ = ವ್ಯಂಜನಂ
ಪಾಯಸ = ಪಾಯಸಂ
ಗೋಡಂಬಿ = ಭಾಲ್ಲಾತಕಃ
ಸಿಹಿ ಗಳಿಗೆ = ಮಧುರ
ಬೆಲ್ಲದ ಪಾನಕ = ಗುಡ ಮಿಶ್ರ ಜಲಂ
ಪಾನಕ = ಶರ್ಕರಾಮಿಶ್ರ ಜಲಂ
ಅನಾನಸ್ = ಅನಾನಸಂ
ಮೊಸರು ವಡೆ = ದದಿ ಮಾಷಾಪೂಪ
ವೀಳೇದೆಲೆ = ನಾಗವಲ್ಲೀ
ಅಡಿಕೆ = ಕ್ರಮುಖ
ಕುಂಬಳಕಾಯಿ = ಕೂಷ್ಮಾಂಡ
ಹಾಲು = ಕ್ಷೀರ
ಎಳಗಾಯಿ = ಶಲಾಟು
ಕಡುಬು = ಮೋದಕ
ಅತ್ತಿಹಣ್ಣು = ಔದುಂಬರ ಫಲ
ಚಟ್ನಿ = ತಮನಂ, ನಿಷ್ಠಾನಂ
ಅವಲಕ್ಕಿ = ಪೃಥುಕಃ, ಚಿಪಿಟಕಃ
ಗುಗ್ಗುರಿ = ಅಭ್ಯೂಷಃ
ಎಲಚಿ = ಬದರಿ
ಜಾಮೂನ್/ಜಿಲೇಬಿ = ವಿಜಿಲಂ
ಜಾಂಗೀರ್/ಪಲಾವ್ = ಪ್ರಯಸ್ತಂ
ಬೆಣ್ಣೆ = ನವನೀತಂ
ನೆಲ್ಲಿಕಾಯಿ = ಆಮಲಕ
ಹುರಿಗಡಲೆ = ಬಾಲ ಭೋಜ್ಯ
ಪೊಂಗಲ್ = ಮುದ್ಗೋದನಂ
ತೊವ್ವೆ = ಸೂಪಂ
ಮೊಸರನ್ನ = ದಧ್ಯನ್ನ.
