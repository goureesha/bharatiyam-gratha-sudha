ಭವತಃ ವೇತನಶ್ರೇಣೀ ಕಾ? = ನಿಮ್ಮ ಸಂಬಳ ಶ್ರೇಣಿ ಎಷ್ಟು?
ಇದಾನೀಂ ಸರ್ವತ್ರ ಸಮಾನಾ ಕಿಲ? = ಈಗ ಎಲ್ಲಾ ಕಡೆಯೂ ಒಂದೇ ರೀತಿ ಅಲ್ಲವೇ?
ಪ್ರಾಚಾರ್ಯಸ್ಯ ಆದೇಶಂ ದೃಷ್ಟವಾನ್ ವಾ? = ಪ್ರಿಂಸಿಪಾಲರ ಮೆಮೋ ನೋಡಿದಿರಾ?
ತತ್ ತು ಸಾಮಾನ್ಯಂ = ಅದು ಮಾಮೂಲಿ, ಬಿಡಿ
ಅಧಿವೇತನಂ ಲಬ್ಧಂ ವಾ? = ಇನ್ಕ್ರಿಮೆಂಟ್ ಸಿಕ್ಕಿತೇ?
ಲಿಪಿಕಂ ದೃಷ್ಟವಾನ್ ವಾ? = ಕ್ಲರ್ಕ್‍ರನ್ನು ನೋಡಿದಿರಾ?
ಏವಂ ಚೇತ್ ಕಥಂ ಜೀವಾಮಃ? = ಹೀಗಾದರೆ ಹೇಗೆ ಬದುಕೋದು?
'ಮಹಾನ್ ಕೋಲಾಹಲಃ' ಇತಿ ಶ್ರುತವಾನ್ = ಬಹಳ ಗದ್ದಲ ಎಂದು ಕೇಳಿದೆ
ಪತ್ರಿಕಾ ಪಠಿತಂ ವಾ? ವೇತನಂ ವರ್ಧಿತಂ = ಪತ್ರಿಕೆ ಓದಿದಿರಾ? ಸಂಬಳ ಹೆಚ್ಚಿಸಲಾಗಿದೆ
ಕದಾ ಆರಭ್ಯ ಅನ್ವಯಃ? = ಯಾವಾಗಿನಿಂದ ಅನ್ವಯ ಆಗುತ್ತದೆ?
ಇದಾನೀಂ ಕಕ್ಷ್ಯಾ ಅಸ್ತಿ ವಾ? = ಈಗ ತರಗತಿ ಇದೆಯಾ?
ಅದ್ಯ ಕಕ್ಷ್ಯಾಂ ನ ಸ್ವೀಕರೋಮಿ ಇತಿ ಸೂಚಯತು = ಇಂದು ತರಗತಿ ತಗೊಳಲ್ಲ ಅಂತ ತಿಳಿಸಿಬಿಡಿ
ಅಸ್ಮಿನ್ ಮಾಸೇ ಕತಿ ವಿರಾಮಾಃ? = ಈ ತಿಂಗಳಲ್ಲಿ ಎಷ್ಟು ರಜೆ?
ಪರಶ್ವಃ ವಿರಾಮಃ ಅಸ್ತಿ ವಾ? = ನಾಡಿದ್ದು ರಜೆ ಇದೆಯಂತಾ?
ಪ್ರಶ್ನಪತ್ರಿಕಾ ಸಜ್ಜೀಕೃತಾ ವಾ? = ಪ್ರಶ್ನಪತ್ರಿಕೆ ಸಿದ್ಧವಾಯಿತಾ?
ಅಸ್ಮಿನ್ ವರ್ಷೇ ಫಲಿತಾಂಶಃ ಕಥಂ? = ಈ ವರ್ಷ ಫಲಿತಾಂಶ ಹೇಗಿದೆ?
ಏತಾವಂತಃ ಅಂಕಾಃ ಕಥಂ ಲಬ್ಧಾಃ ಇತಿ? = ಇಷ್ಟೆಲ್ಲ ಅಂಕಗಳು ಹೇಗೆ ಸಿಕ್ಕವು ಅಂತ?
ಪರೀಕ್ಷಕಾಣಾಂ ಔದಾರ್ಯಂ = ಪರೀಕ್ಷಕರ ಔದಾರ್ಯ (ಗ್ರೇಸ್) ಬಿಡಿ
ಪರೀಕ್ಷಾ ಅನ್ಯಾ, ಯೋಗ್ಯತಾ ಅನ್ಯಾ = ಪರೀಕ್ಷೆಯೇ ಬೇರೆ, ಯೋಗ್ಯತೆಯೇ ಬೇರೆ
ಮೌಲ್ಯಮಾಪನಾರ್ಥಂ ಗಚ್ಛತಿ ವಾ? ಕುತ್ರ? = ಮೌಲ್ಯಮಾಪನಕ್ಕೆ ಹೋಗ್ತೀರಾ? ಎಲ್ಲಿ?
ಅಸ್ವಸ್ಥಃ ಅಪಿ ಆಗತವಾನ್ = ಹುಷಾರಿಲ್ಲದಿದ್ದರೂ ಬಂದೆ
ಇದಾನೀಂತನ ಬಾಲಾಃ ತು .. = ಈಗಿನ ಕಾಲದ ಹುಡುಗರು ಅಂದರೆ..
ಅತ್ರ ಏಕವಾರಂ ಆಗಚ್ಛತು = ಇಲ್ಲಿ ಬಾ ಒಮ್ಮೆ
ಗಣಿತಸ್ಯ ಅಧ್ಯಾಪಕಃ ಅಸ್ತಿ ವಾ? ಪಶ್ಯತು = ಗಣಿತದ ಅಧ್ಯಾಪಕರು ಇದ್ದಾರೆಯೇ? ನೋಡು
ತೇ ತು ಬಾಲಾಃ ಕಿಲ? = ಎಷ್ಟಾದರೂ ಹುಡುಗರಲ್ಲವೇ?
ಕಿಂ ಭೋಃ ಸಮ್ಯಕ್ ಪಠತಿ ಕಿಲ? = ಏನಯ್ಯಾ ಚೆನ್ನಾಗಿ ಓದ್ತಾ ಇದೀಯಾ ತಾನೆ?
ಸಂಶಯಃ ಅಸ್ತಿ ಚೇತ್ ಪೃಚ್ಛಂತು = ಸಂಶಯವಿದ್ದರೆ ಕೇಳಿ
ಜ್ಞಾತಂ ಕಿಲ? = ತಿಳಿಯಿತಾ?
ಪುನಃ ಏಕವಾರಂ ವದತು = ಇನ್ನೊಮ್ಮೆ ಹೇಳು
ಏಕಂ ಅಪಿ ಗಣಿತಂ ನ ಕೃತವಾನ್ ವಾ? = ಒಂದು ಲೆಕ್ಕವನ್ನೂ ಮಾಡಿಲ್ಲವಾ?
ಏವಂ ಚೇತ್ ಪರೀಕ್ಷಾಯಾಂ ಕಿಂ ಕರೋತಿ? = ಹೀಗೆ ಮಾಡಿದರೆ ಪರೀಕ್ಷೆಯಲ್ಲಿ ಏನು ಮಾಡುತ್ತೀಯಾ?
ಸೇವಕಂ ಕಿಂಚಿತ್ ಆಹ್ವಯತು = ಪ್ಯೂನನ್ನು ಒಮ್ಮೆ ಕರೆ
ಘಂಟಾ ತಾಡಿತಾ ವಾ? = ಬೆಲ್ಲ್ ಆಯಿತಾ?
ಟಿಪ್ಪಣೀಂ ಲಿಖತು = ನೋಟ್ಸ್ ಬರೆದುಕೊ
ಏಕೋಽಪಿ ನ ಜಾನಾತಿ ವಾ? = ಒಬ್ಬರಿಗೂ ಗೊತ್ತಿಲ್ಲವಾ?
ಭವಾನ್ ಜ್ಞಾತವಾನ್ ವಾ? = ನಿನಗೆ ಅರ್ಥವಾಯಿತಾ?
ವದತು ಕಿಂಚಿತ್ = ಹೇಳು ನೋಡೋಣ
ಅದ್ಯ ಏತಾವದೇವ ಪರ್ಯಾಪ್ತಮ್ = ಇವತ್ತಿಗೆ ಇಷ್ಟೇ ಸಾಕು
ಅಸ್ಯ ಅನುಚ್ಛೇದಸ್ಯ ಅನಂತರಂ ಸಮಾಪಯಾಮಃ = ಈ ಪ್ಯಾರ ಮಾಡಿಬಿಟ್ಟು ನಿಲ್ಲಿಸೋಣ
ಶ್ವಃ ಏತತ್ ಸಮ್ಯಕ್ ಪಠಿತ್ವಾ ಆಗಂತವ್ಯಮ್ = ನಾಳೆ ಇದನ್ನು ಚೆನ್ನಾಗಿ ಓದಿಕೊಂಡು ಬರಬೇಕು
ಗೃಹೇ ಕಿಮಪಿ ಪಠಂತಿ ವಾ? = ಮನೆಯಲ್ಲಿ ಏನಾದರೂ ಓದ್ತೀರಾ?
ಕಿಂ ಭೋಃ ! ಸಃ ಕೋಲಾಹಲಃ? = ಏನಯ್ಯಾ ಅದು ಗದ್ದಲ?
ಹ್ಯಃ ಕಿಯತ್ ಪರ್ಯಂತಂ ಪಾಠಿತವಾನ್? = ನಿನ್ನೆ ಇಲ್ಲಿಯವರೆಗೆ ಪಾಠ ಮಾಡಿದ್ದೆ?
