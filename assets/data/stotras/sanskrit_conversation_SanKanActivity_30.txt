ಮಮ ಲೇಖನೀಂ ಸ್ವೀಕೃತವಾನ್ ವಾ? = ನನ್ನ ಪೆನ್ ತೆಗೆದುಕೊಂಡೆಯಾ?
ಪಿತಾ ಅಸ್ತಿ ತೂಷ್ಣೀಂ ಉಪವಿಶತು = ತಂದೆಯವರಿದ್ದಾರೆ, ಸುಮ್ಮನಿರು
ಮನಸಿ ಪಠತು ಕಿಂಚಿತ್ = ಮನಸ್ಸಿನಲ್ಲಿ ಓದಿಕೊ ಸ್ವಲ್ಪ
ಗಣಿತಂ ಕಿಂಚಿತ್ ಪಾಠಯತಿ ವಾ? = ಸ್ವಲ್ಪ ಗಣಿತ ಹೇಳಿಕೊಡ್ತಿಯಾ?
ಅಸ್ಮಾಕಂ ಶಿಕ್ಷಕಃ ಏವಂ ಏವ ಪಾಠಿತವಾನ್ = ನಮ್ಮ ಮೇಷ್ಟ್ರು ಹೀಗೆಯೇ ಹೇಳಿಕೊಟ್ಟಿದ್ದು
ಭವತಃ ಲೇಖನೀ ಕುತ್ರ? = ನಿನ್ನ ಪೆನ್ನು ಎಲ್ಲಿ?
ಮಮ ಛತ್ರಂ ಭವಾನ್ ಕಿಮರ್ಥಂ ಸ್ವೀಕೃತವಾನ್ = ನನ್ನ ಕೊಡೆಯನ್ನು ನೀನೇಕೆ ತೆಗೆದುಕೊಂಡೆ?
ತಸ್ಮೈ ಕಿಮರ್ಥಂ ದತ್ತವಾನ್? = ಅವನಿಗೆ ಯಾಕೆ ಕೊಟ್ಟೆ?
ಇದಾನೀಂ ಅನುಭವತು = ಈಗ ಅನುಭವಿಸು
ಮಾಸ್ತು, ಪಿತರಂ ಸೂಚಯಾಮಿ = ಬೇಡ! ಅಪ್ಪನಿಗೆ ಹೇಳುತ್ತೇನೆ
ಭವದೀಯಂ ಸರ್ವಂ ಜಾನಾಮಿ ಅಹಂ = ನಿಂದೆಲ್ಲಾ ಗೊತ್ತು ನನಗೆ
ಪಠನಂ ನಾಸ್ತಿ ಕಿಮಪಿ ನಾಸ್ತಿ, ಕೇವಲಂ ಅಟತಿ = ಓದಲ್ಲ ಏನಿಲ್ಲ, ಬರೀ ತಿರುಗುತ್ತಾನೆ
ಭವತೀ ಬಹು ಪಠತಿ, ಜಾನಾಮಿ = ನೀನು ತುಂಬಾ ಓದುತ್ತೀಯ, ನನಗೆ ಗೊತ್ತು ಕಣೆ
ಅದ್ಯ ಭವತಃ ಸ್ನೇಹಿತಃ ಮಾರ್ಗೇ ಮಿಲಿತಃ = ಇಂದು ನಿನ್ನ ಗೆಳೆಯ ದಾರಿಯಲ್ಲಿ ಸಿಕ್ಕಿದ್ದ
ಅದ್ಯ ಭವತ್ಯಾಃ ಸಖೀ ಮಿಲಿತವತೀ = ಇವತ್ತು ನಿನ್ನ ಗೆಳತಿ ಸಿಕ್ಕಿದ್ದಳು
ಕಿಮಪಿ ಉಕ್ತವಾನ್ ವಾ? = ಏನಾದರೂ ಹೇಳಿದನಾ?
ಪರೀಕ್ಷಾ ಕದಾ ಇತಿ ಸ್ಮರತಿ ಕಿಲ? = ಪರೀಕ್ಷೆ ಯಾವಾಗ ಅಂತ ನೆನಪಿದೆ ತಾನೆ?
ರಮೇಶಃ ಭವಂತಂ ಆಹ್ವಯತಿ = ರಮೇಶನು ನಿನ್ನನ್ನು ಕರೆಯುತ್ತಿದಾನೆ
ಪಶ್ಯತಿ ನಾಸಿಕಾ ಸ್ರವತಿ = ನೋಡು ಮೂಗು ಸುರಿಯುತ್ತಿದೆ
ನಾಸಿಕಾಂ ಸ್ವಚ್ಛಂ ಕೃತ್ವಾ ಆಗಚ್ಛತು = ಮೂಗು ಒರೆಸಿಕೊಂಡು ಬಾ
ವಕ್ತವ್ಯಾಂ ಆಸೀತ್, ಕರೋಮಿ ಸ್ಮ = ಹೇಳಬೇಕಿತ್ತು, ಮಾಡುತ್ತಿದ್ದೆ
ಅಂಕನ್ಯಾ ಮಾಸ್ತು, ಲೇಖನ್ಯಾ ಲಿಖತು = ಪೆನ್ಸಿಲ್ಲಿಂದ ಬೇಡ, ಪೆನ್ನಿಂದ ಬರೆ
ತಿಷ್ಠತು ಯುತಕಂ ಪರಿವತ್ರ್ಯ ಆಗಚ್ಛಾಮಿ = ಇರು, ಬಟ್ಟೆ ಬದಲಾಯಿಸಿ ಬರುತ್ತೇನೆ
ಏತತ್ ಯುತಕಂ ಬಹು ಸಂಪೃಕ್ತಂ = ಈ ಬಟ್ಟೆ ತುಂಬಾ ಬಿಗಿ
ತತ್ ಯುತಕಂ ಏವಂ ನಾಸ್ತಿ = ಆ ಬಟ್ಟೆ ಹೀಗಿಲ್ಲ
