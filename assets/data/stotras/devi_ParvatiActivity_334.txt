|| ಲಲಿತಾಸಹಸ್ರನಾಮ ಅರ್ಥ - 1 ||

1. ಶ್ರೀಮಾತಾ = ಎಲ್ಲರ ಉತ್ಪತ್ತಿಗೆ ಕಾರಣಳು
2. ಶ್ರೀಮಹಾರಾಜ್ಞೀ = ಸಕಲ ಪ್ರಪಂಚವನ್ನು ಪಾಲಿಸುವವಳು
3. ಶ್ರೀಮತ್ಸಿಂಹಾಸನೇಶ್ವರೀ = ಉತ್ತಮವಾದ ಆಸನದಲ್ಲಿ ಕುಳಿತಿರುವವಳು
4. ಚಿದಗ್ನಿಕುಂಡಸಂಭೂತಾ = ಚಿತ್ (ಬ್ರಹ್ಮ) ಎಂಬ ಅಗ್ನಿಕುಂಡದಿಂದ ಹುಟ್ಟಿರುವವಳು
5. ದೇವಕಾರ್ಯಸಮುದ್ಯತಾ = ದೇವತೆಗಳ ಕಾರ್ಯಕ್ಕಾಗಿ ಆವಿರ್ಭೂತಳಾದವಳು
6. ಉದ್ಯದ್ಭಾನುಸಹಸ್ರಾಭಾ = ಉದಯಿಸುತ್ತಿರುವ ಸಾವಿರಾರು ರಕ್ತ ವರ್ಣದ ಸೂರ್ಯರಂತೆ ಪ್ರಕಾಶ ಉಳ್ಳವಳು
7. ಚತುರ್ಬಾಹು ಸಮನ್ವಿತಾ = ನಾಲ್ಕು ಬಾಹುಗಳನ್ನು ಹೊಂದಿದವಳು
8. ರಾಗಸ್ವರೂಪಾಶಾಢ್ಯಾ = ರಾಗವೆಂಬ ಪಾಶವನ್ನು ಹಿಡಿದಿರುವವಳು
9. ಕ್ರೋಧಾಕಾರಾಂಕುಶೋಜ್ಜ್ವಲಾ = ಕ್ರೋಧ-ವಿಷಯಜ್ಞಾನಗಳೆಂಬ ಅಂಕುಶದಿಂದ ಉಜ್ಜ್ವಲಳಾಗಿರುವವಳು
10. ಮನೋರೂಪೇಕ್ಷುಕೋದಂಡಾ = ಮನೋರೂಪವಾದ ಕಬ್ಬಿನ ಬಿಲ್ಲನ್ನು ಹಿಡಿದಿರುವವಳು
11. ಪಂಚತನ್ಮಾತ್ರಸಾಯಕಾ = ಪಂಚ ತನ್ಮಾತ್ರಗಳನ್ನೇ ಬಾಣಗಳಾಗಿ ಉಳ್ಳವಳು (ಮನಸ್ಸೆಂಬ ಬಿಲ್ಲಿಗೆ ಪಂಚೇಂದ್ರಿಯಗಳೇ ಬಾಣಗಳು, ಕ್ಷೋಭಣ, ದ್ರಾವಣ, ಆಕರ್ಷಣ, ವಶ್ಯ, ಉನ್ಮಾದ (ಜ್ಞಾನಾರ್ಣವ), ಮದನ, ಉನ್ಮಾದನ, ಮೋಹನ, ದೀಪನ, ಶೋಷಣ (ತಂತ್ರರಾಜ), ಹರ್ಷಣ, ರೋಚನ, ಮೋಹನ, ಶೋಷಣ, ಮಾರಣ (ಕಾಲಿಕಾಪುರಾಣ))
12. ನಿಜಾರುಣಪ್ರಭಾಪೂರಮಜ್ಜದ್ ಬ್ರಹ್ಮಾಂಡಮಂಡಲಾ = ತನ್ನ ಅರುಣ ಪ್ರಭೆಯ ಪ್ರವಾಹದಲ್ಲಿ ಬೆಳಗುತ್ತಿರುವ ಬ್ರಹ್ಮಾಂಡ ಮಂಡಲಗಳುಳ್ಳವಳು
13. ಚಂಪಕಾಶೋಕಪುನ್ನಾಗ ಸೌಗಂಧಿಕಲಸತ್ಕಚಾ = ಚಂಪಕ, ಅಶೋಕ, ಪುನ್ನಾಗ, ಸೌಗಂಧಿಕ - ಈ ಪುಷ್ಪಗಳಿಂದ ಶೋಭಿಸುವ ಕೂದಲುಳ್ಳವಳು
14. ಕುರುವಿಂದಮಣಿಶ್ರೇಣೀ ಕನತ್ಕೋಟೀರಮಂಡಿತಾ = ಕುರುವಿಂದ ಮಣಿಗಳ ಶ್ರೇಣಿಯಿರುವ ಕಿರೀಟದಿಂದ ಅಲಂಕೃತಳಾದವಳು
15. ಅಷ್ಟಮೀಚಂದ್ರ ವಿಭ್ರಾಜದಲಿಕಸ್ಥಲ ಶೋಭಿತಾ = ಅಷ್ಟಮೀ ಚಂದ್ರನಂತೆ ವಿರಾಜಿಸುತ್ತಿರುವ ಲಲಾಟದಿಂದ ಶೋಭಿಸುತ್ತಿರುವವಳು
16. ಮುಖಚಂದ್ರ ಕಲಂಕಾಭಮೃಗನಾಭಿ ವಿಶೇಷಕಾ = ಮುಖವೆಂಬ ಚಂದ್ರನಿಗೆ ಕಳಂಕಪ್ರಾಯವಾಗಿರುವ ಕಸ್ತೂರೀತಿಲಕವನ್ನು ಧರಿಸಿರುವವಳು
17. ವದನಸ್ಮರಮಾಂಗಲ್ಯ ಗೃಹತೋರಣಚಲ್ಲಿಕಾ = ತನ್ನ ಮುಖವೆಂಬ ಕಾಮರಾಜನ ಮಂಗಳಗೃಹಕ್ಕೆ ತೋರಣವಾಗಿರುವ ಹುಬ್ಬುಗಳುಳ್ಳವಳು
18. ವಕ್ತ್ರಲಕ್ಷ್ಮೀಪರೀವಾಹ ಚಲನ್ಮೀನಾಭಲೋಚನಾ = ತನ್ನ ಮುಖ ಶ್ರೀ ಎಂಬ ಜಲಪ್ರವಾಹದಲ್ಲಿ ಮೀನುಗಳಂತೆ ಕಣ್ಣುಗಳುಳ್ಳವಳು
19. ನವಚಂಪಕಪುಷ್ಪಾಭ ನಾಸಾದಂಡವಿರಾಜಿತಾ = ಆಗತಾನೇ ಅರಳಿರುವ ಸಂಪಿಗೆಯ ಹೂವಿನಂತಿರುವ ಮೂಗಿನಿಂದ ವಿರಾಜಿಸುತ್ತಿರುವವಳು
20. ತಾರಾಕಾಂತಿತಿರಸ್ಕಾರಿ ನಾಸಾಭರಣಭಾಸುರಾ = ನಕ್ಷತ್ರ ಕಾಂತಿಯನ್ನು ತಿರಸ್ಕರಿಸುವ ಮೂಗುತಿಯಿಂದ ಬೆಳಗುತ್ತಿರುವವಳು
21. ಕದಂಬಮಂಜರೀಕ್ಲೃಪ್ತ ಕರ್ಣಪೂರಮನೋಹರಾ = ಕಿವಿಯ ಮೇಲ್ಭಾಗದಲ್ಲಿ ಅಲಂಕರಿಸಲ್ಪಟ್ಟ ಕದಂಬಗಳ ಪುಷ್ಪಗುಚ್ಛದಿಂದ ರಮಣೀಯವಾಗಿರುವವಳು
22. ತಾಟಂಕಯುಗಲೀಭೂತ ತಪನೋಡುಪಮಂಡಲಾ = ಸೂರ್ಯ-ಚಂದ್ರರ ಮಂಡಲಗಳನ್ನೇ ಕಿವಿಯ ಎರಡು ಆಭರಣಗಳಾಗಿ ಉಳ್ಳವಳು
23. ಪದ್ಮರಾಗಶಿಲಾದರ್ಶ ಪರಿಭಾವಿಕಪೋಲಭೂಃ = ಪದ್ಮರಾಗವೆಂಬ ಶಿಲೆಯ ಕನ್ನಡಿಯನ್ನು ಕೂಡ ತಿರಸ್ಕರಿಸುವ ಕಪೋಲ ಉಳ್ಳವಳು
24. ನವವಿದ್ರುಮಬಿಂಬ ಶ್ರೀನ್ಯಕ್ಕಾರಿರದನಚ್ಛದಾ = ಹೊಸದಾದ ಹವಳ ಮತ್ತು ಪಕ್ವವಾದ ಬಿಂಬಫಲ - ಇವುಗಳ ಕಾಂತಿಯನ್ನು ತಿರಸ್ಕರಿಸುವ ತುಟಿಗಳನ್ನು ಉಳ್ಳವಳು
25. ಶುದ್ಧವಿದ್ಯಾಂಕುರಾಕಾರ ದ್ವಿಜಪಂಕ್ತಿದ್ವಯೋಜ್ಜ್ವಲಾ = ಶುದ್ಧ ವಿದ್ಯೆಯ ಅಂಕುರದಂತೆ ಆಕಾರವಿರುವ ದಂತಪಂಕ್ತಿಗಳಿಂದ ಉಜ್ಜ್ವಲಳಾಗಿರುವವಳು
26. ಕರ್ಪೂರವೀಟಿಕಾಮೋದ ಸಮಾಕರ್ಷದ್ದಿಗಂತರಾ = ಮುಖಕಮಲದಲ್ಲಿ ದಿಕ್ಕುಗಳನ್ನೆಲ್ಲ ವ್ಯಾಪಿಸಿರುವ ಆಕರ್ಷಕವಾದ ತಾಂಬೂಲದ ಪರಿಮಳವನ್ನು ಉಳ್ಳವಳು
27. ನಿಜಸಲ್ಲಾಪಮಾಧುರ್ಯ ವಿನಿರ್ಭತ್ರ್ಸಿತಕಚ್ಛಪೀ = ತನ್ನ ಮಾತಿನ ಮಾಧುರ್ಯವು ಕಚ್ಛಪವೀಣೆಯನ್ನೂ ತಿರಸ್ಕರಿಸುವಂಥದ್ದು
28. ಮಂದಸ್ಮಿತಪ್ರಭಾಪೂರ ಮಜ್ಜತ್ಕಾಮೇಶಮಾನಸಾ = ಕಾಮೇಶನ ಮನಸ್ಸು ಮುಳುಗುವ, ಮಂದ ಹಾಸದ ಲಾವಣ್ಯ ಪ್ರವಾಹ ಉಳ್ಳವಳು
29. ಅನಾಕಲಿತಸಾದೃಶ್ಯ ಚುಬುಕಶ್ರೀವಿರಾಜಿತಾ = ಅಸದೃಶವಾದ ಗಲ್ಲದ ಕಾಂತಿಯಿಂದ ವಿರಾಜಿಸುತ್ತಿರುವವಳು
30. ಕಾಮೇಶಬದ್ಧಮಾಂಗಲ್ಯ ಸೂತ್ರಶೋಭಿತಕಂಧರಾ = ತನ್ನ ಕಾಮೇಶನಿಂದ ಕಟ್ಟಲ್ಪಟ್ಟ ಮಾಂಗಲ್ಯಸೂತ್ರದಿಂದ ಶೋಭಿಸುವ ಕಂಠ ಇರುವವಳು
31. ಕನಕಾಂಗದಕೇಯೂರ ಕಮನೀಯಭುಜಾನ್ವಿತಾ = ಚಿನ್ನದ ಅಂಗದ ಕೇಯೂರಗಳಿಂದ, ರಮಣೀಯವಾದ ಬಾಹುಗಳನ್ನು ಉಳ್ಳವಳು
32. ರತ್ನಗ್ರೈವೇಯ ಚಿಂತಾಕಲೋಲಮುಕ್ತಾಫಲಾನ್ವಿತಾ = ರತ್ನ ಖಚಿತವಾದ ಗ್ರೈವೇಯಹಾರದ ಜೋಲಾಡುವ ಮುತ್ತಿನಿಂದ ಶೋಭಿಸುತ್ತಿರುವವಳು
33. ಕಾಮೇಶ್ವರ ಪ್ರೇಮರತ್ನಮಣಿ ಪ್ರತಿಪಣಸ್ತನೀ = ಕಾಮೇಶ್ವರನ ಪ್ರೇಮವೆಂಬ ರತ್ನಮಣಿಗೆ ತನ್ನ ಸ್ತನದ್ವಯವೆಂಬ ಬೆಲೆಯನ್ನು ಕೊಟ್ಟಿರುವವಳು
34. ನಾಭ್ಯಾಲವಾಲರೋಮಾಲಿ ಲತಾಫಲಕುಚದ್ವಯೀ = ನಾಭಿಯಿಂದ ಹುಟ್ಟಿರುವ ರೋಮಲತೆಯಲ್ಲಿ ತನ್ನ ಕುಚದ್ವಯವನ್ನೇ ಫಲವಾಗಿ ಉಳ್ಳವಳು
35. ಲಕ್ಷ್ಯರೋಮಲತಾಧಾರತಾಸಮುನ್ನೇಯ ಮಧ್ಯಮಾ = ರೋಮಲತೆಗೆ ಆಧಾರವಿರಬೇಕಾದ್ದರಿಂದ ಊಹಿಸಲ್ಪಟ್ಟ ಕಟಿ ಪ್ರದೇಶವುಳ್ಳವಳು
36. ಸ್ತನಭಾರದಲನ್ಮಧ್ಯ ಪಟ್ಟಬಂಧವಲಿತ್ರಯಾ = ಸ್ತನಭಾರದಿಂದ ಬಗ್ಗಿದ ಕಟಿಪ್ರದೇಶಕ್ಕೆ ಆಧಾರವಾಗಿರುವ ವಲಿತ್ರಯವನ್ನೇ ಕನಕಪಟ್ಟಿಯಾಗಿ ಉಳ್ಳವಳು
37. ಅರುಣಾರುಣಕಸುಂಭ ವಸ್ತ್ರಭಾಸ್ವತ್ಕಟೀತಟೀ = ರಕ್ತವರ್ಣದ ವಸ್ತ್ರದಿಂದ ಶೋಭಿಸುವ ಕಟಿಪ್ರದೇಶ ಉಳ್ಳವಳು
38. ರತ್ನಕಿಂಕಿಣಿಕಾ ರಮ್ಯರಶನಾದಾಮಭೂಷಿತಾ = ರತ್ನಮಯವಾದ ಗೆಜ್ಜೆಗಳಿಂದ ಅಲಂಕರಿಸಲ್ಪಟ್ಟ ಸುವರ್ಣಮೇಖಲೆಯಿಂದ ಭೂಷಿತಳಾದವಳು
39. ಕಾಮೇಶಜ್ಞಾತಸೌಭಾಗ್ಯ ಮಾರ್ದವೋರುದ್ವಯಾನ್ವಿತಾ = ಕಾಮೇಶ್ವರನಿಂದ ತಿಳಿಯಲ್ಪಟ್ಟ ಲಾವಣ್ಯ ಕೋಮಲತೆಗಳಿಂದ ಕೂಡಿದ ತೊಡೆಗಳನ್ನು ಉಳ್ಳವಳು
40. ಮಾಣಿಕ್ಯಮಕುಟಾಕಾರ ಜಾನುದ್ವಯವಿರಾಜಿತಾ = ಮಾಣಿಕ್ಯ ಮಕುಟದಂತೆ ಆಕಾರವುಳ್ಳ ಮೊಣಕಾಲುಗಳಿಂದ ವಿರಾಜಿಸುತ್ತಿರುವವಳು
41. ಇಂದ್ರಗೋಪಪರಿಕ್ಷಿಪ್ತ ಸ್ಮರತೂಣಾಭಜಂಘಿಕಾ = ಇಂದ್ರಗೋಪಗಳಿಂದ ಮಾಡಲ್ಪಟ್ಟ ಮನ್ಮಥನ ಬತ್ತಳಿಕೆಯಂತಿರುವ ಕಾಲುಗಳನ್ನು ಉಳ್ಳವಳು
42. ಗೂಢಗಲ್ಫಾ = ಮಾಂಸಲವಾದ ಹಿಮ್ಮಡಿಯುಳ್ಳವಳು
43. ಕೂರ್ಮಪೃಷ್ಠಜಯಿಷ್ಣು ಪ್ರಪದಾನ್ವಿತಾ = ಕೂರ್ಮಪೃಷ್ಠದಂತೆ ವೃತ್ತ ಖಂಡಾಕಾರವಾಗಿರುವ ಪಾದಾಗ್ರ ಉಳ್ಳವಳು
44. ನಖದೀಧಿತಿಸಂಛನ್ನ ನಮಜ್ಜನತಮೋಗುಣಾ = ನಮಿಸುವವರ ತಮೋಗುಣವನ್ನು ನಾಶ ಮಾಡುವ ನಖ ಕಿರಣಗಳಿಂದ ಕೂಡಿರುವವಳು
45. ಪದದ್ವಯಪ್ರಭಾಜಾಲ ಪರಾಕೃತಸರೋರುಹಾ = ತನ್ನ ಪಾದದ್ವಯಗದ ಪ್ರಭಾಸಮೂಹದಿಂದ ಕಮಲವೂ ಹೀನವಾಗಿ ಕಾಣುವಂತೆ ಮಾಡುವವಳು
46. ಶಿಂಜಾನಮಣಿಮಂಜೀರ ಮಂಡಿತಶ್ರೀಪದಾಂಬುಜಾ = ಝಣತ್ಕರಿಸುವ ಮಣಿಯುಕ್ತ ಕಾಲ್ಗೆಜ್ಜೆಗಳಿಂದ ಭೂಷಿತವಾದ ಸಿರಿಯಡಿದಾವರೆಗಳನ್ನು ಉಳ್ಳವಳು
47. ಮರಾಲೀಮಂದಗಮನಾ = ಹಂಸಿಯಂತೆ ಮಂದಗತಿ ಉಳ್ಳವಳು
48. ಮಹಾಲಾವಣ್ಯಶೇವಧಿಃ = ಮಹಾಲಾವಣ್ಯದ ನಿಧಿಯಾಗಿರುವವಳು
49. ಸರ್ವಾರುಣಾ = ಎಲ್ಲವೂ ಅರುಣವಾಗಿರುವವಳು
50. ಅನವದ್ಯಾಂಗೀ = ದೋಷರಹಿತವಾದ ಅಂಗಗಳನ್ನು ಉಳ್ಳವಳು
51. ಸರ್ವಾಭರಣಭೂಷಿತಾ = ಸರ್ವಾಭರಣಗಳಿಂದ ಭೂಷಿತಳಾಗಿರುವವಳು
52. ಶಿವಕಾಮೇಶ್ವರಾಂಕಸ್ಥಾ = ಶಿವಕಾಮೇಶ್ವರನ ಅಂಕದಲ್ಲಿ ಕುಳಿತಿರುವವಳು
53. ಶಿವಾ = ಶಿವೆಯು, ಪರಶಿವನ ಇಚ್ಛೆಯಾಗಿರುವವಳು
54. ಸ್ವಾಧೀನವಲ್ಲಭಾ = ತನಗೆ ಅಧೀನನಾಗಿರುವ ಕಮೇಶ್ವರನನ್ನು ವಲ್ಲಭನನ್ನಾಗಿ ಹೊಂದಿರುವವಳು
55. ಸಮೇರುಮಧ್ಯಶೃಂಗಸ್ಥಾ = ಸಮೇರು ಪರ್ವತದ ಮಧ್ಯ ಶೃಂಗದಲ್ಲಿ ಇರುವವಳು
56. ಶ್ರೀಮನ್ನಗರನಾಯಿಕಾ = ಶ್ರೀಮನ್ನಗರದ ನಾಯಿಕೆ
57. ಚಿಂತಾಮಣಿಗೃಹಾಂತಸ್ಥಾ = ಚಿಂತಾಮಣಿಗಳಿಂದ ಕಟ್ಟಿದ ಗೃಹದಲ್ಲಿ ವಾಸಿಸುವವಳು
58. ಪಂಚಬ್ರಹ್ಮಾಸನಸ್ಥಿತಾ = ಐದು ಬ್ರಹ್ಮರೇ ಆಸನವಾಗಿರುವ ಮಂಚದಲ್ಲಿ ಸ್ಥಿತಳಾಗಿರುವವಳು
59. ಮಹಾಪದ್ಮಾಟವೀಸಂಸ್ಥಾ = ಮಹಾಪದ್ಮವನದಲ್ಲಿ ಇರುವವಳು
60. ಕದಂಬವನವಾಸಿನೀ = ಕದಂಬವನದಲ್ಲಿ ವಾಸಿಸುವವಳು
61. ಸುಧಾಸಾಗರಮಧ್ಯಸ್ಥಾ = ಸುಧಾಸಾಗರದ ಮಧ್ಯದಲ್ಲಿ ಇರುವವಳು
62. ಕಾಮಾಕ್ಷೀ = ಕಮನೀಯವಾದ ಕಣ್ಣುಗಳಿರುವವಳು
63. ಕಾಮದಾಯಿನೀ = ಮನೋರಥಗಳನ್ನು ಅನುಗ್ರಹಿಸುವವಳು
64. ದೇವರ್ಷಿಗಣಸಂಘಾತ ಸ್ತೂಯಮಾನಾತ್ಮವೈಭವಾ = ದೇವರ್ಷಿಗಳ ಸಮುದಾಯದಿಂದ ಹೊಗಳಲ್ಪಟ್ಟ ಆತ್ಮವೈಭವ ಉಳ್ಳವಳು
65. ಭಂಡಾಸುರವಧೋದ್ಯುಕ್ತ ಶಕ್ತಿಸೇನಾಸಮನ್ವಿತಾ = ಭಂಡಾಸುರರನ್ನು ವಧಿಸಲು ಉದ್ಯುಕ್ತವಾದ ಶಕ್ತಿಗಳ ಸೇನೆಯಿಂದ ಕೂಡಿರುವವಳು
66. ಸಂಪತ್ಕರೀಸಮಾರೂಢ ಸಿಂಧುರವ್ರಜಸೇವಿತಾ = ಸಂಪತ್ಕರಿಯ ನಾಯಕತ್ವದಲ್ಲಿರುವ ಆನೆಗಳ ಗುಂಪುಗಳಿಂದ ಸೇವಿತಳಾಗಿರುವವಳು
67. ಅಶ್ವಾರೂಢಾಧಿಷ್ಠಿತಾಶ್ವ ಕೋಟಿಕೋಟಿಭಿರಾವೃತಾ = ಅಶ್ವಾರೂಢಳ ನಾಯಕತ್ವದಲ್ಲಿರುವ ಕೋಟಿ ಕೋಟಿ ಅಶ್ವಗಳಿಂದ ಆವೃತಳಾಗಿರುವವಳು
68. ಚಕ್ರರಾಜರಥಾರೂಢ ಸರ್ವಾಯುಧಪರಿಷ್ಕೃತಾ = ಚಕ್ರರಾಜವೆಂಬ ರಥದಲ್ಲಿ ಆರೂಢವಾದ ಸರ್ವಾಯುಧಗಳಿಂದ ಅಲಂಕೃತವಾಗಿರುವವಳು
69. ಗೇಯಚಕ್ರರಥಾರೂಢ ಮಂತ್ರಿಣೀಪರಿಸೇವಿತಾ = ಗೇಯಚಕ್ರದಲ್ಲಿ ಆರೂಢಳಾದ ಮಂತ್ರಿಣಿಯಿಂದ (ಶ್ಯಾಮಲಾದೇವಿಯಿಂದ) ಪರಿಸೇವಿತಳಾಗಿರುವವಳು
70. ಕಿರಿಚಕ್ರರಥಾರೂಢ ದಂಡನಾಥಾಪುರಸ್ಕೃತಾ = ಕಿರಿಚಕ್ರವೆಂಬ ರಥದಲ್ಲಿ ಆರೂಢಳಾದ ದಂಡನಾಥಳಿಂದ (ವಾರಾಹೀ ದೇವಿಯಿಂದ) ಪುರಸ್ಕೃತಳಾದವಳು
71. ಜ್ವಾಲಾಮಾಲಿನಿಕಾಕ್ಷಿಪ್ತ ವಹ್ನಿಪ್ರಾಕಾರಮಧ್ಯಗಾ = ಜ್ವಾಲಾಮಾಲಿನಿಯಿಂದ ನಿರ್ಮಿತವಾದ ವಹ್ನಿಪ್ರಾಕಾರದ ಮಧ್ಯದಲ್ಲಿ ಇರುವವಳು
72. ಭಂಡಸೈನ್ಯವಧೋದ್ಯುಕ್ತ ಶಕ್ತಿವಿಕ್ರಮಹರ್ಷಿತಾ = ಭಂಡಾಸುರನ ಸೈನ್ಯದ ವಧೆಯಲ್ಲಿ ಉದ್ಯುಕ್ತರಾದ (ನಕುಲ್ಯಾದಿ) ಶಕ್ತಿಗಳ ಪರಾಕ್ರಮದಿಂದ ಹರ್ಷಿತಳಾಗಿರುವವಳು
73. ನಿತ್ಯಾಪರಾಕ್ರಮಾಟೋಪ ನಿರೀಕ್ಷಣಸಮುತ್ಸುಕಾ = ನಿತ್ಯರ ಪರಾಕ್ರಮದ ಆಟೋಪವನ್ನು (ವಿಸ್ತಾರವನ್ನು) ಉತ್ಸಾಹದಿಂದ ನೋಡುವವಳು
74. ಭಂಡಪುತ್ರವಧೋದ್ಯುಕ್ತ ಬಾಲಾವಿಕ್ರಮನಂದಿತಾ = ಭಂಡಾಸುರನ ಪುತ್ರರ ವಧೆಯಲ್ಲಿ ಉದ್ಯುಕ್ತಳಾದ ಬಾಲೆಯ ಪರಾಕ್ರಮದಿಂದ ಆನಂದಗೊಳ್ಳುವವಳು
75. ಮಂತ್ರಿಣ್ಯಂಬಾವಿರಚಿತ ವಿಷಂಗವಧತೋಷಿತಾ = ಮಂತ್ರಿಣಿಯಾದ ಶ್ಯಾಮಲಾ ದೇವಿಯಿಂದ ಮಾಡಲ್ಪಟ್ಟ ವಿಷಂಗನ ವಧೆಯಿಂದ ತೋಷಿತಳಾಗಿರುವವಳು
76. ವಿಶುಕ್ರಪ್ರಾಣಹರಣ ವಾರಾಹೀವೀರ್ಯನಂದಿತಾ = ವಿಶುಕ್ರನ ಪ್ರಾಣ ಹರಣ ಮಾಡಿದ ವಾರಾಹಿಯ ವೀರ್ಯದಿಂದ ಆನಂದಿತಳಾದವಳು
77. ಕಾಮೇಶ್ವರಮುಖಾಲೋಕ ಕಲ್ಪಿತಶ್ರೀಗಣೇಶ್ವರಾ = ಕಾಮೇಶ್ವರನ ಮುಖಾವಲೋಕನದಿಂದ ಶ್ರೀ ಗಣೇಶ್ವರನನ್ನು ಕಲ್ಪಿಸಿರುವವಳು
78. ಮಹಾಗಣೇಶನಿರ್ಭಿನ್ನ ವಿಘ್ನಯಂತ್ರಪ್ರಹರ್ಷಿತಾ = ಮಹಾಗಣೇಶನು ವಿಘ್ನಯಂತ್ರವನ್ನು ನಿಃಶೇಷವಾಗಿ ಭೇದಿಸಿದಾಗ ಹರ್ಷಿತಳಾದವಳು
79. ಭಂಡಾಸುರೇಂದ್ರನಿರ್ಮುಕ್ತ ಶಸ್ತ್ರಪ್ರತ್ಯಸ್ತ್ರವರ್ಷಿಣೀ = ಭಂಡಾಸುರೇಂದ್ರನಿಂದ ಬಿಡಲ್ಪಟ್ಟ ಶಸ್ತ್ರಗಳಿಗೆ ಪ್ರತಿಯಾಗಿ ಅಸ್ತ್ರಗಳ ಮಳೆಗರೆದವಳು
80. ಕರಾಂಗುಲಿನಖೋತ್ಪನ್ನ ನಾರಾಯಣದಶಾಕೃತಿಃ = ಅವಳ ಬೆರಳುಗಳ ನಖಗಳಿಂದ ದಶಾಕೃತಿಗಳುಳ್ಳ ನಾರಾಯಣನು ಉತ್ಪನ್ನನಾದನು
81. ಮಹಾಪಾಶುಪತಾಸ್ತ್ರಾಗ್ನಿ ನಿರ್ದಗ್ಧಾಸುರಸೈನಿಕಾ = ಮಹಾ ಪಾಶುಪತಾಸ್ತ್ರದ ಅಗ್ನಿಯಿಂದ ಅಸುರಸೈನ್ಯವನ್ನು ದಹನ ಮಾಡಿದವಳು
82. ಕಾಮೇಶ್ವರಾಸ್ತ್ರನಿರ್ದಗ್ಧ ಸಭಂಡಾಸುರಶೂನ್ಯಕಾ = ಕಾಮೇಶ್ವರಾಸ್ತ್ರದಿಂದ ಭಂಡಾಸುರನೊಂದಿಗೆ ಶೂನ್ಯಕ ನಗರವನ್ನು ದಹಿಸಿರುವವಳು
83. ಬ್ರಹ್ಮೋಪೇಂದ್ರ ಮಹೇಂದ್ರಾದಿ ದೇವಸಂಸ್ತುತವೈಭವಾ = ಬ್ರಹ್ಮ-ವಿಷ್ಣು-ಮಹೇಂದ್ರ ಇವರೇ ಮೊದಲಾದ ದೇವತೆಗಳಿಂದ ಸ್ತುತಿಸಲ್ಪಟ್ಟ ವೈಭವ ಉಳ್ಳವಳು
84. ಹರನೇತ್ರಾಗ್ನಿಸಂದಗ್ಧ ಕಾಮಸಂಜೀವನೌಷಧಿಃ = ಹರನೇತ್ರಾಗ್ನಿಯಿಂದ ಭಸ್ಮೀಭೂತನಾದ ಮನ್ಮಥನಿಗೆ ಸಂಜೀವನೌಷಧಿ ಆಗಿರುವವಳು
85. ಶ್ರೀಮದ್ವಾಗ್ಭವಕೂಟೈಕ ಸ್ವರೂಪಮುಖಪಂಕಜಾ = ಮುಖ ಪಂಕಜವು ಶ್ರೀಮದ್ವಾಗ್‍ಭವಕೂಟವನ್ನು ಮುಖ್ಯ ಸ್ವರೂಪವಾಗಿ ಹೊಂದಿರುವವಳು
86. ಕಂಠಾಧಃಕಟಿಪರ್ಯಂತ ಮಧ್ಯಕೂಟಸ್ವರೂಪಿಣೀ = ಕಂಠದಿಂದ ಕಟಿ ಪ್ರದೇಶದವರೆಗೆ ಮಧ್ಯಕೂಟವನ್ನು ಸ್ವರೂಪವಾಗಿ ಹೊಂದಿರುವವಳು
87. ಶಕ್ತಿಕೂಟೈಕತಾಪನ್ನ ಕಟ್ಯಧೋಭಾಗಧಾರಿಣೀ = ಕಟಿ ಪ್ರದೇಶದ ಅಧೋ ಭಾಗವನ್ನು ಶಕ್ತಿಕೂಟವಾಗಿ ಹೊಂದಿರುವವಳು
88. ಮೂಲಮಂತ್ರಾತ್ಮಿಕಾ = ಮೂಲಮಂತ್ರದ ಸ್ವರೂಪಳಾಗಿರುವವಳು
89. ಮೂಲಕೂಟತ್ರಯಕಲೇವರಾ = ಮೂಲಮಂತ್ರದ ಮೂರು ಕೂಟಗಳನ್ನು ಶರೀರವಾಗಿ ಉಳ್ಳವಳು
90. ಕುಲಾಮೃತೈಕರಸಿಕಾ = ಕುಲಾಮೃತದ ರಸಿಕಳಾಗಿರುವವಳು
91. ಕುಲಸಂಕೇತಪಾಲಿನೀ = ಕುಲಸಂಕೇತಗಳನ್ನು ಪಾಲಿಸುವವಳು
92. ಕುಲಾಂಗನಾ = ಪಾತಿವ್ರತ್ಯಾದಿ ಗುಣರಾಶಿಗಳಿಂದ ಕೂಡಿದ ವಂಶದವಳು, ಕುಲಾಂಗನೆ
93. ಕುಲಾಂತಸ್ಥಾ = ಕುಲಶಾಸ್ತ್ರದಲ್ಲಿ ಜ್ಞೇಯವಾಗಿರುವವಳು
94. ಕೌಲಿನೀ = ಕುಲಕ್ಕೆ ಸೇರಿರುವವಳು, ಅಧಃ ಸ್ಥಿತವಾದ ಕೆಂಪು ವರ್ಣದ ಸಹಸ್ರದಳ ಕಮಲದ ಕರ್ಣಿಕೆಯಲ್ಲಿ ಇರುವವಳು
95. ಕುಲಯೋಗಿನೀ = ಆಕಾಶದಲ್ಲಿ ಚಕ್ರವನ್ನು ಮನಸ್ಸಿನಿಂದ ಬರೆದು ಮಾಡುವ ಪೂಜಾದಿಗಳ ಸಂಬಂಧವಿರುವವಳು
96. ಅಕುಲಾ = ಅಕುಲವೆಂಬ ಸಹಸ್ರಾರ ಪದ್ಮದಲ್ಲಿ ಇರುವವಳು
97. ಸಮಯಾಂತಸ್ಥಾ = ಸಮಯದಲ್ಲಿ ಇರುವವಳು
98. ಸಮಯಾಚಾರತತ್ಪರಾ = (ರುದ್ರಯಾಮಲದಲ್ಲಿ ಉಪದಿಷ್ಟವಾದ) ಸಮಯಾಚಾರದಲ್ಲಿ ತತ್ಪರಳಾಗಿರುವವಳು
99. ಮೂಲಾಧಾರೈಕನಿಲಯಾ = ಮೂಲಾಧಾರದಲ್ಲಿಯೇ ಮುಖ್ಯವಾಗಿ ನೆಲೆಸಿರುವವಳು
100. ಬ್ರಹ್ಮಗ್ರಂಥಿವಿಭೇದಿನೀ = ಬ್ರಹ್ಮಗ್ರಂಥಿಯನ್ನು ಭೇದಿಸಿ..
101. ಮಣಿಪೂರಾಂತರುದಿತಾ = ಮಣಿಪೂರ(ಚಕ್ರ)ದಲ್ಲಿ ಉದಯಿಸುವವಳು
102. ವಿಷ್ಣುಗ್ರಂಥಿವಿಭೇದಿನೀ = ವಿಷ್ಣುಗ್ರಂಥಿಯನ್ನು ಭೇದಿಸಿ..
103. ಆಜ್ಞಾಚಕ್ರಾಂತರಾಲಸ್ಥಾ = ಆಜ್ಞಾಚಕ್ರದ ಮಧ್ಯದಲ್ಲಿ ನೆಲೆಸಿರುವವಳು
104. ರುದ್ರಗ್ರಂಥಿವಿಭೇದಿನೀ = ರುದ್ರಗ್ರಂಥಿಯನ್ನು ಭೇದಿಸಿ
105. ಸಹಸ್ರಾರಾಂಬುಜರೂಢಾ = ಸಹಸ್ರದಳ ಪದ್ಮದಲ್ಲಿ ಆರೂಢಳಾಗಿ..
106. ಸುಧಾಸಾರಾಭಿವರ್ಷಿಣೀ = (ಅಲ್ಲಿಂದ) ಸುಧೆಯ ಧಾರಾಸಂಪಾತವನ್ನು ಅಭಿವರ್ಷಿಸುವವಳು
107. ತಟಿಲ್ಲತಾಸಮರುಚಿಃ = ವಿದ್ಯುಲ್ಲೇಖೆಯಂತೆ ಕಾಂತಿಯುತಳಾಗಿರುವವಳು
108. ಷಟ್‍ಚಕ್ರೋಪರಿಸಂಸ್ಥಿತಾ = ಷಟ್‍ಚಕ್ರಗಳ ಮೇಲೆ ಸ್ಥಿತಳಾಗಿರುವವಳು
109. ಮಹಾಸಕ್ತಿಃ = (ಸಹಸ್ರಾರದಲ್ಲಿ) ಶಿವನೊಂದಿಗೆ/ಮಹದಲ್ಲಿ ಸೇರಲು ಆಸಕ್ತಿಯುಳ್ಳವಳು
110. ಕುಂಡಲಿನೀ = (ಭುಜಂಗಿಯಂತೆ) ಕುಂಡಲಾಕಾರವಾಗಿ ಇರುವವಳು
111. ಬಿಸತಂತುತನೀಯಸೀ = ಕಮಲ ನಾಳದಂತೆ ಕೃಶಾಕೃತಿಯುಳ್ಳವಳು
112. ಭವಾನೀ = ಭವ/ಸಂಸಾರಕ್ಕೆ ಪ್ರಾಣವನ್ನು ಕೊಡುವವಳು
113. ಭಾವನಾಗಮ್ಯಾ = ಭಾವನೆಯ ಮೂಲಕ ಹೊಂದಲ್ಪಡುವವಳು
114. ಭವಾರಣ್ಯಕುಠಾರಿಕಾ = ಸಂಸಾರವೆಂಬ ಅರಣ್ಯವನ್ನು ಕತ್ತರಿಸುವವಳು
115. ಭದ್ರಪ್ರಿಯಾ = ಮಂಗಳವನ್ನು ಪ್ರೀತಿಸುವವಳು
116. ಭದ್ರಮೂರ್ತಿಃ = ಭವ್ಯವಾದ ಸ್ವರೂಪವುಳ್ಳವಳು
117. ಭಕ್ತಸೌಭಾಗ್ಯದಾಯಿನೀ = ಭಕ್ತರಿಗೆ ಸೌಭಾಗ್ಯವನ್ನು ಅನುಗ್ರಹಿಸುವವಳು
118. ಭಕ್ತಿಪ್ರಿಯಾ = (ಮುಖ್ಯ ಮತ್ತು ಗೌಣೀ) ಭಕ್ತಿಯನ್ನು ಪ್ರೀತಿಸುವವಳು
119. ಭಕ್ತಿಗಮ್ಯಾ = ಭಕ್ತಿಯ ಮೂಲಕ ಹೊಂದಲ್ಪಡುವವಳು
120. ಭಕ್ತಿವಶ್ಯಾ = ಭಕ್ತಿಗೆ ವಶಳಾಗಿರುವವಳು
121. ಭಯಾಪಹಾ = ಭಯವನ್ನು ನಾಶಮಾಡುವವಳು
122. ಶಾಂಭವೀ = ಶಂಭುವಿನ ಪತ್ನೀ
123. ಶಾರದಾರಾಧ್ಯಾ = ಶಾರದೆಯಿಂದ ಆರಾಧಿಸಲ್ಪಡುವವಳು
124. ಶರ್ವಾಣೀ = ಶರ್ವನ ಪತ್ನೀ
125. ಶರ್ಮದಾಯಿನೀ = ಸುಖವನ್ನು ಅನುಗ್ರಹಿಸುವವಳು
126. ಶಾಂಕರೀ = ಶಂಕರನ ಪತ್ನೀ (ಶಂಕರ=ಸುಖವನ್ನು ಮಾಡುವವನು)
127. ಶ್ರೀಕರೀ = ಶ್ರೀಯನ್ನು (ಐಶ್ವರ್ಯವನ್ನು) ಉಂಟುಮಾಡುವವಳು
128. ಸಾಧ್ವೀ = ಪತಿವ್ರತೆ
129. ಶರಚ್ಚಂದ್ರ ನಿಭಾನನಾ = ಶರಚ್ಚಂದ್ರನಂತೆ ಮುಖವುಳ್ಳವಳು
130. ಶಾತೋದರೀ = ತೆಳುವಾದ ನಡುವುಳ್ಳವಳು
131. ಶಾಂತಿಮತೀ = ಶಾಂತ ಸ್ವಭಾವವುಳ್ಳವಳು
132. ನಿರಾಧಾರಾ = ಆಧಾರ ರಹಿತಳಾದವಳು
133. ನಿರಂಜನಾ = ಅವಿದ್ಯಾಸಂಪರ್ಕ ಇಲ್ಲದವಳು, ಆವರಣಶಕ್ತಿಯಿಂದ ಭೇದ ಬುದ್ಧಿ ಇಲ್ಲದವಳು
134. ನಿರ್ಲೇಪಾ = ಕರ್ಮದಿಂದ ಉಂಟಾಗುವ ದೋಷಗಳ ಅಂಟು/ಲೇಪ ಇಲ್ಲದವಳು
135. ನಿರ್ಮಲಾ = ನಿತ್ಯ ಶುದ್ಧಳು
136. ನಿತ್ಯಾ = ಕಾಲತ್ರಯದಲ್ಲಿಯೂ ಇರುವವಳು
137. ನಿರಾಕಾರಾ = ಆಕಾರವಿಲ್ಲದವಳು
138. ನಿರಾಕುಲಾ = ಭ್ರಮೆ/ಸಂಶಯ ಇಲ್ಲದವಳು
139. ನಿರ್ಗುಣಾ = ಶರೀರ ಧರ್ಮಗಳಾದ ಗುಣಗಳು ಇಲ್ಲದವಳು (ಚಿತ್ ಸ್ವರೂಪಳು)
140. ನಿಷ್ಕಲಾ = ಕಲಾ ಶೂನ್ಯಳು, ಅವಯವ ರಹಿತಳು
141. ಶಾಂತಾ = ಶಮ ಸ್ವಭಾವದವಳು
142. ನಿಷ್ಕಾಮಾ = ಕಾಮನೆಯಿಲ್ಲದವಳು
143. ನಿರುಪಪ್ಲವಾ = ನಾಶವಿಲ್ಲದವಳು
144. ನಿತ್ಯಮುಕ್ತಾ = ನಿತ್ಯವೂ ಮುಕ್ತಸ್ವರೂಪಳಾಗಿರುವವಳು
145. ನಿರ್ವಿಕಾರಾ = ವಿಕಾರ ರಹಿತಳು
146. ನಿಷ್ಪ್ರಪಂಚಾ = ಪ್ರಪಂಚ/ವಿಸ್ತಾರ/ವಿಕಾಸ ಶೂನ್ಯಳು
147. ನಿರಾಶ್ರಯಾ = ಆಶ್ರಯ ರಹಿತಳು
148. ನಿತ್ಯಶುದ್ಧಾ = ನಿತ್ಯವೂ ಶುದ್ಧಳಾಗಿರುವವಳು
149. ನಿತ್ಯಬುದ್ಧಾ = ಚಿದ್ರೂಪಳು
150. ನಿರವದ್ಯಾ = ದೋಷರಹಿತಳು
151. ನಿರಂತರಾ = ಅವಧಿ/ಅಂತರ್ಧಾನ/ಭೇದ ರಹಿತಳು
152. ನಿಷ್ಕಾರಣಾ = ಕಾರಣ ಶೂನ್ಯಳು
153. ನಿಷ್ಕಲಂಕಾ = ಕಳಂಕ ರಹಿತಳು
154. ನಿರುಪಾಧಿಃ = ಅವಿದ್ಯೆಯೆಂಬ ಉಪಾಧಿ ಇರದವಳು
155. ನಿರೀಶ್ವರಾ = ಅವಳಲ್ಲದೆ ಈಶ್ವರನಿಲ್ಲ, ನಿರೀಶ್ವರಿಯು
156. ನೀರಾಗಾ = ಇಚ್ಛಾರಹಿತಳು
157. ರಾಗಮಥನೀ = ರಾಗವನ್ನು ಮಥಿಸುವವಳು/ನಾಶ ಮಾಡುವವಳು (ವೈರಾಗ್ಯ-ದಾನ ಮಾಡುವವಳು)
158. ನಿರ್ಮದಾ = ಮದ ರಹಿತಳು
159. ಮದನಾಶಿನೀ = ಮದವನ್ನು ನಾಶ ಮಾಡುವವಳು
160. ನಿಶ್ಚಿಂತಾ = ಚಿಂತಾಶೂನ್ಯಳು
161. ನಿರಹಂಕಾರಾ = (ತ್ರಿವಿಧ) ಅಹಂಕಾರ ರಹಿತಳು
162. ನಿರ್ಮೋಹಾ = ಮೋಹ ರಹಿತಳು
163. ಮೋಹನಾಶಿನೀ = ಮೋಹವನ್ನು ನಾಶ ಮಾಡುವವಳು
164. ನಿರ್ಮಮಾ = ಮಮಕಾರ ರಹಿತಳು
165. ಮಮತಾಹಂತ್ರೀ = ಮಮತೆಯನ್ನು ನಾಶಮಾಡುವವಳು
166. ನಿಷ್ಪಾಪಾ = ಪಾಪರಹಿತಳು
167. ಪಾಪನಾಶಿನೀ = ಪಾಪವನ್ನು ನಾಶ ಮಾಡುವವಳು
168. ನಿಷ್ಕ್ರೋಧಾ = ಕ್ರೋಧರಹಿತಳು
169. ಕ್ರೋಧಶಮನೀ = ಕ್ರೋಧವನ್ನು ಶಮನ ಮಾಡುವವಳು
170. ನಿರ್ಲೋಭಾ = ಲೋಭ ರಹಿತಳು (ಉದಾರ-ಸಹಿತಳು)
171. ಲೋಭನಾಶಿನೀ = ಲೋಭವನ್ನು ನಾಶ ಮಾಡುವವಳು
172. ನಿಃಸಂಶಯಾ = ಸಂಶಯ ರಹಿತಳು
173. ಸಂಶಯಘ್ನೀ = ಸಂಶಯವನ್ನು ನಾಶ ಮಾಡುವವಳು
174. ನಿರ್ಭವಾ = ಜನ್ಮ-ರಹಿತಳು
175. ಭವನಾಶಿನೀ = ಸಂಸಾರವನ್ನು ನಾಶ ಮಾಡುವವಳು (ಮೋಕ್ಷದಾಯಕಳು)
176. ನಿರ್ವಿಕಲ್ಪಾ = ವಿಕಲ್ಪ ರಹಿತಳು
177. ನಿರಾಬಾಧಾ = ಬಾಧಿತಳಾಗದವಳು
178. ನಿರ್ಭೇದಾ = ಭೇದ ರಹಿತಳು
179. ಭೇದನಾಶಿನೀ = ಭೇದವನ್ನು ನಾಶ ಮಾಡುವವಳು
180. ನಿರ್ನಾಶಾ = ನಾಶ ರಹಿತಳು
181. ಮೃತ್ಯುಮಥನೀ = ಮೃತ್ಯುವನ್ನು ನಾಶ ಮಾಡುವವಳು (ಭಕ್ತರಿಗೆ ಅಮರತ್ವ ಕೊಡುವವಳು)
182. ನಿಷ್ಕ್ರಿಯಾ = ಕ್ರಿಯಾರಹಿತಳು
183. ನಿಷ್ಪರಿಗ್ರಹಾ = ಸಹಾಯ, ಕುಟುಂಬ ಎಂಬಿತ್ಯಾದಿ ಪರಿಗ್ರಹ ಇಲ್ಲದವಳು
184. ನಿಸ್ತುಲಾ = ಹೋಲಿಕೆ ಇಲ್ಲದವಳು
185. ನೀಲಚಿಕುರಾ = ದಟ್ಟ ಕಪ್ಪುಕೂದಲುಳ್ಳವಳು
186. ನಿರಪಾಯಾ = ನಾಶ ರಹಿತಳು
187. ನಿರತ್ಯಯಾ = ಅತಿಕ್ರಮ/ದೋಷ/ವಿನಾಶ ರಹಿತಳು
188. ದುರ್ಲಭಾ = ಯೋಗಿಗಳಿಗೂ ಕೂಡ ಸುಲಭವಾಗಿ ಸಿಗದವಳು
189. ದುರ್ಗಮಾ = ಪಡೆಯಲು ಸುಲಭವಲ್ಲದವಳು
190. ದುರ್ಗಾ = ದುರ್ಗನೆಂಬ ಅಸುರನನ್ನು ಕೊಂದವಳು, ಸುರರನ್ನು ದುರ್ಗಸಂಕಟದಿಂದ ಪಾರು ಮಾಡಿದವಳು
191. ದುಃಖಹಂತ್ರೀ = ದುಃಖವನ್ನು ನಾಶಮಾಡುವವಳು
192. ಸುಖಪ್ರದಾ = ಸುಖವನ್ನು ಕೊಡುವವಳು
193. ದುಷ್ಟದೂರಾ = ದುಷ್ಟರಿಗೆ ಅಪ್ರಾಪ್ಯಳಾಗಿರುವವಳು
194. ದುರಾಚಾರಶಮನೀ = ಶಾಸ್ತ್ರವಿರೋಧಿಯಾದ ಆಚಾರವನ್ನು ನಾಶಮಾಡುವವಳು
195. ದೋಷವರ್ಜಿತ = ರಾಗ, ದ್ವೇಷ ಮೊದಲಾದ ದೋಷಗಳಿಲ್ಲದವಳು
196. ಸರ್ವಜ್ಞಾ = ಸರ್ವವನ್ನೂ ತಿಳಿದಿರುವವಳು
197. ಸಾಂದ್ರಕರುಣಾ = ಅತ್ಯಂತ ಕರುಣೆಯುಳ್ಳವಳು
198. ಸಮಾನಾಧಿಕವರ್ಜಿತಾ = ಸಮನಾರಾದವರಾಗಲಿ ಅಧಿಕರಾದವರಾಗಲಿ ಇಲ್ಲದವಳು
199. ಸರ್ವಶಕ್ತಿಮಯೀ = ಸರ್ವಶಕ್ತಿ ಸಂಪನ್ನಳು
200. ಸರ್ವಮಂಗಲಾ = ಸಮಸ್ತ ಮಂಗಳವನ್ನೂ ಅನುಗ್ರಹಿಸುವವಳು
201. ಸದ್ಗತಿಪ್ರದಾ = ಸ್ವರ್ಗವನ್ನು, ಬ್ರಹ್ಮಜ್ಞಾನ/ಮೋಕ್ಷವನ್ನು/ಸಾಧುಗತಿಯನ್ನು ಅನುಗ್ರಹಿಸುವವಳು
202. ಸರ್ವೇಶ್ವರೀ = ಸರ್ವಕ್ಕೂ ಒಡೆಯಳು
203. ಸರ್ವಮಯೀ = ಸರ್ವವೂ ಆಗಿರುವವಳು
204. ಸರ್ವಮಂತ್ರ ಸ್ವರೂಪಿಣೀ = ಸರ್ವಮಂತ್ರಗಳ ಸ್ವರೂಪಳಾಗಿರುವವಳು
205. ಸರ್ವಯಂತ್ರಾತ್ಮಿಕಾ = ಸರ್ವ ಯಂತ್ರಗಳ ಸ್ವರೂಪಳು
206. ಸರ್ವತಂತ್ರರೂಪಾ = ಸರ್ವತಂತ್ರಗಳ ಸ್ವರೂಪಳು
207. ಮನೋನ್ಮನೀ = ಬ್ರಹ್ಮರಂಧ್ರದ ಕೆಳಗಿರುವ ಸ್ಥಾನವಾದ ಮನೋನ್ಮನೀ ರೂಪದಲ್ಲಿ ಇರುವವಳು
208. ಮಾಹೇಶ್ವರೀ = ಮಹೇಶ್ವರನ ಪತ್ನಿ
209. ಮಹಾದೇವೀ = ಮಹತ್ತಾದ ರೂಪವುಳ್ಳವಳು, ಮಹದೇವನ ಪತ್ನಿ
210. ಮಹಾಲಕ್ಷ್ಮಿ = ಸಂಪತ್ತಿನ ಅಧಿಪತಿಯಾದ ಮಹಾಲಕ್ಷ್ಮಿಯ ರೂಪದವಳು
211. ಮೃಡಪ್ರಿಯಾ = ಮೃಡನ (ಸತ್ತ್ವಗುಣ, ಸುಖ) ಪ್ರಿಯೆ
212. ಮಹಾರೂಪಾ = ಮಹಾರೂಪವುಳ್ಳವಳು
213. ಮಹಾಪೂಜ್ಯಾ = ಪೂಜ್ಯರಿಂದಲೂ ಕೂಡ ಪೂಜ್ಯಳು, ಮಹಾಪೂಜ್ಯಳು
214. ಮಹಾಪಾತಕನಾಶಿನೀ = ಮಹಾಪಾತಕಗಳನ್ನು ನಾಶ ಮಾಡುವವಳು
215. ಮಹಾಮಾಯಾ = ಬ್ರಹ್ಮಾದಿಗಳಿಗೂ ಮೋಹವನ್ನುಂಟು ಮಾಡುವ ಮಹಾ ಮಾಯೆ
216. ಮಹಾಸತ್ತ್ವಾ = ಮಹಾ ಸತ್ತ್ವವುಳ್ಳವಳು (ಬಲ, ಗುಣ)
217. ಮಹಾಶಕ್ತಿಃ = ಸರ್ವಜಗತ್ತನ್ನೂ ನಿರ್ವಹಿಸಲು ವಿಸ್ತೃತ, ವಿವಿಧ ಶಕ್ತಿಯುಳ್ಳವಳು
218. ಮಹಾರತಿಃ = ಮಹಾ ಪ್ರೀತಿಯುಳ್ಳವಳು
219. ಮಹಾಭೋಗಾ = ಮಹಾ ಸುಖ, ಧನವಿರುವವಳು
220. ಮಹೈಶ್ವರ್ಯಾ = ಮಹತ್ತಾದ ಈಶ್ವರಭಾವವನ್ನೂ ವಿಭೂತಿಯನ್ನೂ ಹೊಂದಿರುವ ದೇವಿ
221. ಮಹಾವೀರ್ಯಾ = ಮಹತ್ತಾದ ತೇಜಃಸಾಮಥ್ರ್ಯಗಳಿಂದ ಕೂಡಿರುವವಳು
222. ಮಹಾಬಲಾ = ಮಹಾ ಬಲವುಳ್ಳವಳು
223. ಮಹಾಬುದ್ಧಿಃ = ಮಹಾಬುದ್ಧಿಯಾಗಿರುವವಳು, ದೇವಿಯು ಬುದ್ಧಿಯಲ್ಲಿ ಪ್ರಕಾಶಿಸಿದರೆ ಶ್ರೇಷ್ಠವು
224. ಮಹಾಸಿದ್ಧಿಃ = ಅಣಿಮಾದಿ ಮಹಾಸಿದ್ಧಿಯುಳ್ಳವಳು
225. ಮಹಾಯೋಗೇಶ್ವರೇಶ್ವರೀ = ಮಹಾಯೋಗೇಶ್ವರರಿಗೆ ಈಶ್ವರಿಯಾಗಿರುವವಳು
226. ಮಹಾತಂತ್ರಾ = ಕುಲಾರ್ಣವ, ಜ್ಞಾನಾರ್ಣವ ಮುಂತಾದ ಮಹಾತಂತ್ರಗಳುಳ್ಳವಳು
227. ಮಹಾಮಂತ್ರಾ = ಬಾಲಾ, ಬಗಲಾ ಮುಂತಾದ ಮಹಾಮಂತ್ರವುಳ್ಳವಳು
228. ಮಹಾಯಂತ್ರಾ = ಪೂಜಾಚಕ್ರ, ಪದ್ಮಚಕ್ರ, ಸಿದ್ಧಿವಜ್ರ ಮುಂತಾದ ಮಹಾ ಯಂತ್ರವುಳ್ಳವಳು
229. ಮಹಾಸನಾ = ಪೃಥ್ವಿ ಮೊದಲಾದ 36 ತತ್ತ್ವಗಳನ್ನೇ ಆಸನವಾಗಿರುವ ದೇವಿಯು
230. ಮಹಾಯಾಗಕ್ರಮಾರಾಧ್ಯಾ = ಮಹಾಯಾಗ ಕ್ರಮದಿಂದ ಆರಾಧಿಸಲ್ಪಡುವವಳು
231. ಮಹಾಭೈರವಪೂಜಿತಾ = ಮಹಾಭೈರವನಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
232. ಮಹೇಶ್ವರಮಹಾಕಲ್ಪ ಮಹಾತಾಂಡವಸಾಕ್ಷಿಣೀ = ಮಹಾ ಪ್ರಳಯದಲ್ಲಿ ಮಹೇಶ್ವರನ ಮಹಾತಾಂಡವ ನೃತ್ಯಕ್ಕೆ ಸಾಕ್ಷಿಯಾಗಿರುವವಳು
233. ಮಹಾಕಾಮೇಶಮಹಿಷೀ = ಮಹಾಕಾಮೇಶನ ಪತ್ನಿಯಾಗಿರುವವಳು
234. ಮಹಾತ್ರಿಪುರ ಸುಂದರೀ = ಪ್ರಮಾತೃ, ಪ್ರಮಾಣ, ಪ್ರಮೇಯ ಎಂಬ ತ್ರಿಪುರದ ಆತ್ಮಸ್ವರೂಪಳಾಗಿರುವವಳು
235. ಚತುಃಷಷ್ಟ್ಯುಪಚಾರಾಢ್ಯಾ = 64 ಉಪಚಾರಗಳಿಂದ ಆರಾಧಿಸಲ್ಪಡುವವಳು
236. ಚತುಃಷಷ್ಟಿಕಲಾಮಯೀ = 64 ಕಲೆಗಳ ಸ್ವರೂಪಳಾಗಿರುವವಳು
237. ಮಹಾಚತುಃಷಷ್ಟಿಕೋಟಿ ಯೋಗಿನೀಗಣಸೇವಿತಾ = 64 ಕೋಟಿ ಮಹಾಯೋಗಿನಿಯರ ಗಣಗಳಿಂದ ಸೇವಿತಳಾಗಿರುವವಳು
238. ಮನುವಿದ್ಯಾ = ಶ್ರೀವಿದ್ಯೆಯ 12 ಬಗೆಯಲ್ಲಿ ಒಂದಾದ ಮನುವಿದ್ಯೆಯ ಸ್ವರೂಪಳು
239. ಚಂದ್ರವಿದ್ಯಾ = ಚಂದ್ರವಿದ್ಯೆಯ ಸ್ವರೂಪಳು
240. ಚಂದ್ರಮಂಡಲ ಮಧ್ಯಗಾ = ಚಂದ್ರಮಂಡಲದ ಮಧ್ಯದಲ್ಲಿರುವವಳು
241. ಚಾರುರೂಪಾ = ಸುಂದರವಾದ ಲಾವಣ್ಯವುಳ್ಳವಳು
242. ಚಾರುಹಾಸಾ = ಮನೋಹರವಾದ ಮುಗುಳ್ನಗೆಯುಳ್ಳವಳು
243. ಚಾರುಚಂದ್ರಕಲಾಧರಾ = ಸುಂದರವಾದ ಚಂದ್ರಕಲೆಯನ್ನು ಧರಿಸಿರುವವಳು
244. ಚರಾಚರಜಗನ್ನಾಥಾ = ಸ್ಥಾವರ-ಜಂಗಮ ರೂಪವಾದ ಜಗತ್ತಿಗೆ ಅಧೀಶ್ವರಿಯಾಗಿರುವವಳು
245. ಚಕ್ರರಾಜನಿಕೇತನಾ = ಶ್ರೀಚಕ್ರವೇ ವಾಸಸ್ಥಾನವಿ ಇರುವವಳು
246. ಪಾರ್ವತೀ = ಪರ್ವತ ರಾಜನ ಮಗಳಾದ ಪಾರ್ವತಿಯು
247. ಪದ್ಮನಯನಾ = ಪದ್ಮದಂತೆ ನಯನವುಳ್ಳವಳು
248. ಪದ್ಮರಾಗಸಮಪ್ರಭಾ = ಪದ್ಮರಾಗಕ್ಕೆ ಸಮನಾದ ಕಾಂತಿಯುಳ್ಳವಳು
249. ಪಂಚಪ್ರೇತಾಸನಾಸೀನಾ = ಪಂಚಬ್ರಹ್ಮರೇ ಪಂಚಪ್ರೇತಗಳಾದ ಆಸನದಲ್ಲಿ ಕುಳಿತಿರುವವಳು

250. ಪಂಚಬ್ರಹ್ಮಸ್ವರೂಪಿಣೀ = ಪಂಚಬ್ರಹ್ಮದ ರೂಪವಾಗಿರುವವಳು
251. ಚಿನ್ಮಯೀ = ಜ್ಞಾನಸ್ವರೂಪಳು
252. ಪರಮಾನಂದಾ = ಪರಮಸುಖ, ಪರಮ ಆನಂದ ಸ್ವರೂಪಳು
253. ವಿಜ್ಞಾನಘನರೂಪಿಣೀ = ಚೈತನ್ಯ ಘನ ಸ್ವರೂಪಳು
254. ಧ್ಯಾನಾಧ್ಯಾತೃಧ್ಯೇಯರೂಪಾ = ಧ್ಯಾನ, ಧ್ಯಾನ ಮಾಡುವವನು, ಧ್ಯಾನ ಮಾಡಿಸಿಕೊಳ್ಳುವವನು - ಈ ಮೂರು ರೂಪವೂ ಆಗಿರುವವಳು
255. ಧರ್ಮಾಧರ್ಮವಿವರ್ಜಿತಾ = ಧರ್ಮ-ಅಧರ್ಮಗಳ ನಿರ್ಬಂಧವಿಲ್ಲದವಳು
256. ವಿಶ್ವರೂಪಾ = ಜಗತ್ ಸ್ವರೂಪಳು
257. ಜಾಗರಿಣೀ = ಜಾಗ್ರತ್ ಸ್ವರೂಪಳು
258. ಸ್ವಪಂತೀ = ಸ್ವಪ್ನರೂಪಳು
259. ತೈಜಸಾತ್ಮಿಕಾ = ತೇಜೋಮಯಳು
260. ಸುಪ್ತಾ = ನಿದ್ರಾ ಸ್ವರೂಪಳು
261. ಪ್ರಾಜ್ಞಾತ್ಮಿಕಾ = ವಿಶೇಷ ಜ್ಞಾನಸ್ವರೂಪಳು
262. ತುರ್ಯಾ = ನಾಲ್ಕನೆಯ ಅವಸ್ಥೆಯುಳ್ಳವಳು (ತುರೀಯಾ ಸ್ವರೂಪಳು)
263. ಸರ್ವಾವಸ್ಥಾವಿವರ್ಜಿತಾ = ಎಲ್ಲಾ ಅವಸ್ಥೆಗಳಿಂದಲೂ ಬಿಡುಗಡೆಯಾದವಳು
264. ಸೃಷ್ಟಿಕತ್ರ್ರೀ = ಜಗತ್ತನ್ನು ಸೃಷ್ಟಿಸುವವಳು
265. ಬ್ರಹ್ಮರೂಪಾ = ಸೃಷ್ಟಿಕರ್ತ-ಬ್ರಹ್ಮನ ಸ್ವರೂಪವುಳ್ಳವಳು
266. ಗೋಪ್ತ್ರೀ = ರಕ್ಷಣೆ ಮಾಡುವವಳು
267. ಗೋವಿಂದರೂಪಿಣೀ = ವಿಷ್ಣು ಸ್ವರೂಪಳು
268. ಸಂಹಾರಿಣೀ = ನಾಶ ಮಾಡುವವಳು
269. ರುದ್ರರೂಪಾ = ರುದ್ರ ರೂಪಳು
270. ತಿರೋಧಾನಕರೀ = ಅಂತರ್ಧಾನ ಮಾಡಿಕೊಳ್ಳುವವಳು
271. ಈಶ್ವರೀ = ಈಶ್ವರ ಸ್ವರೂಪಳು
272. ಸದಾಶಿವಾ = ಸದಾಶಿವ ಸ್ವರೂಪಳು
273. ಅನುಗ್ರಹದಾ = ಅನುಗ್ರಹಿಸುವವಳು
274. ಪಂಚಕೃತ್ಯಪರಾಯಣಾ = ಸೃಷ್ಟಿ-ಸ್ಥಿತಿ-ಲಯ-ತಿರೋಧಾನ-ಅನುಗ್ರಹಗಳೆಂಬ ಐದು ಕೆಲಸಗಳಲ್ಲಿ ನಿರತಳಾದವಳು
275. ಭಾನುಮಂಡಲಮಧ್ಯಸ್ಥಾ = ಸೂರ್ಯ ಮಂಡಲದಲ್ಲಿರುವವಳು
276. ಭೈರವೀ = ಭೈರವನ ಪತ್ನಿಯು
277. ಭಗಮಾಲಿನೀ = ಭಗಾಂಕಿತಳು (ಆರು ವಿಶಿಷ್ಟತೆಗಳ ಮಾಲೆಯನ್ನು ಧರಿಸಿದವಳು)
278. ಪದ್ಮಾಸನಾ = ಕಮಲದಲ್ಲಿ ಕುಳಿತವಳು
279, ಭಗವತೀ = ಷಡ್ಗುಣೈಶ್ವರ್ಯ ಸಂಪನ್ನಳು
280. ಪದ್ಮನಾಭಸಹೋದರೀ = ವಿಷ್ಣುವಿನ ಸೋದರಿ
281. ಉನ್ಮೇಷನಿಮಿಷೋತ್ಪನ್ನ ವಿಪನ್ನ ಭುವನಾವಲಿಃ - ಕಣ್ಣು ಮುಚ್ಚು ತೆರೆಯುವುದರಲ್ಲಿ ಪ್ರಪಂಚದ ಉತ್ಪತ್ತಿ ಮತ್ತು ನಾಶಕ್ಕೆ ಕಾರಣಳಾದವಳು
282. ಸಹಸ್ರಶೀರ್ಷವದನಾ = ಅಸಂಹ್ಯ ಶಿರಗಳು ಮತ್ತು ಮುಖಗಳುಳ್ಳವಳು
283. ಸಹಸ್ರಾಕ್ಷೀ = ಅಸಂಖ್ಯ ಕಣ್ಣುಳ್ಳವಳು
284. ಸಹಸ್ರಪಾತ್ = ಅಸಂಖ್ಯ ಕಾಲುಗಳುಳ್ಳವಳು
285. ಆಬ್ರಹ್ಮಕೀತಜನನೀ = ಮಹತ್ತಾದ ಬ್ರಹ್ಮದಿಂದ ಕ್ಷುಲ್ಲಕ ಕ್ರಿಮಿಯವರೆಗೆ ಎಲ್ಲಾ ಜೀವವರ್ಗಕ್ಕೂ ತಾಯಿಯಾದವಳು
286. ವರ್ಣಾಶ್ರಮವಿಧಾಯಿನೀ = ವರ್ಣ-ಆಶ್ರಮಗಳಿಗೆ ನಿಯಮವನ್ನು ವಿಧಿಸುವವಳು
287. ನಿಜಾಜ್ಞಾರೂಪನಿಗಮಾ = ತನ್ನ ಆಜ್ಞೆಯೇ ವೇದಗಳಾಗಿ ಉಳ್ಳವಳು
288. ಪುಣ್ಯಾಪುಣ್ಯಫಲಪ್ರದಾ = ಪುಣ್ಯ ಮತ್ತು ಪಾಪಗಳಿಗೆ ಫಲವನ್ನು ಕೊಡುವವಳು
289. ಶ್ರುತಿಸೀಮಂತ ಸಿಂದೂರೀ ಕೃತಪಾದಾಬ್ಜ ಧೂಲಿಕಾ = ವೇದದೇವತೆಗಳು, ತಮ್ಮ ಮುಂಗುರುಳಿನಲ್ಲಿ ಸಿಂದೂರವಾಗಿ ಧರಿಸಿರುವ, ಚರಣ ಕಮಲಧೂಳಿಯುಳ್ಳವಳು
290. ಸಕಲಾಗಮಸಂದೋಹ ಶುಕ್ತಿಸಂಪುಟ ಮೌಕ್ತಿಕಾ = ಸಕಲ ವೇದದ ಸಮುದಾಯವೆಂಬ ಚಿಪ್ಪಿನ ಕರಡಿಗೆಯಲ್ಲಿರುವ ಮುತ್ತಿನ ಮಣಿಯಾಗಿರುವವಳು
291. ಪುರುಷಾರ್ಥಪ್ರದಾ = ಧರ್ಮ-ಅರ್ಥ-ಕಾಮ-ಮೋಕ್ಷಗಳನ್ನು ಕೊಡುವವಳು
292. ಪೂರ್ಣಾ = ಪರಿಪೂರ್ಣಳು
293. ಭೋಗಿನೀ = ನಾಗಕನ್ಯಾ ಸ್ವರೂಪಳು
294. ಭುವನೇಶ್ವರೀ = ಹದಿನಾಲ್ಕು ಲೋಕಕ್ಕೂ ಒಡೆಯಳು
295. ಅಂಬಿಕಾ = ಭುವನಕ್ಕೆ ತಾಯಿಸ್ವರೂಪಳು
296. ಅನಾದಿನಿಧನಾ = ಜನನ-ಮರಣ ರಹಿತಳು
297. ಹರಿಬ್ರಹ್ಮೇಂದ್ರಸೇವಿತಾ = ವಿಷ್ಣು-ಬ್ರಹ್ಮ-ಇಂದ್ರರಿಂದ ಸೇವಿಸಲ್ಪಡುವವಳು
298. ನಾರಾಯಣೀ = ನಾರಾಯಣನಂತೆ ಸ್ವರೂಪಳು
299. ನಾದರೂಪಾ = ಧ್ವನಿರೂಪಳು
300. ನಾಮರೂಪವಿವರ್ಜಿತಾ = ನಾಮ-ರೂಪಗಳು ಇಲ್ಲದವಳು
301. ಹ್ರೀಂಕಾರೀ = ಹ್ರೀಂಕಾರ ಸ್ವರೂಪಳು (ಹ್ರೀಂಕಾರ ಧ್ವನಿಯನ್ನು ನುಡಿಸುವವಳು)
302. ಹ್ರೀಮತೀ = ಲಜ್ಜಾಶೀಲಳು
303. ಹೃದ್ಯಾ = ಮನೋಹರಳಾದವಳು
304. ಹೇಯೋಪಾದೇಯವರ್ಜಿತಾ = ಬಿಡುವ ಮತ್ತು ತೆಗೆದುಕೊಳ್ಳುವ ವಿಷಯಗಳಿಂದ ಬಿಡಲ್ಪಟ್ಟವಳು
305. ರಾಜರಾಜಾರ್ಚಿತಾ = ಚಕ್ರವರ್ತಿಗಳಿಂದ ಪೂಜಿಸಲ್ಪಟ್ಟವಳು
306. ರಾಜ್ಞೀ = ಮಹಾರಾಣಿಯು
307. ರಮ್ಯಾ = ಸುಂದರಿಯು
308. ರಾಜೀವಲೋಚನಾ = ಕಮಲದಂತೆ ಕಣ್ಣುಳ್ಳವಳು
309. ರಂಜಿನೀ = ಆನಂದಪಡಿಸುವವಳು
310. ರಮಣೀ = ಕ್ರೀಡಿಸುವವಳು
311. ರಸ್ಯಾ = ಆಸ್ವಾದಿಸಲು ಯೋಗ್ಯಳು
312. ರಣತ್ಕಿಂಕಿಣಿಮೇಖಲಾ = ಶಬ್ದಮಾಡುತ್ತಲಿರುವ ಗೆಜ್ಜೆಯುಳ್ಳ ಸೊಂಟದಪಟ್ಟಿಯುಳ್ಳವಳು
313. ರಮಾ = ಲಕ್ಷ್ಮೀ ಸ್ವರೂಪಳು
314. ರಾಕೇಂದುವದನಾ = ಪೂರ್ಣಚಂದ್ರನಂತೆ ಮುಖವುಳ್ಳವಳು
315. ರತಿರೂಪಾ = ರತಿ ಸ್ವರೂಪಳು
316. ರತಿಪ್ರಿಯಾ = ರತಿಯಲ್ಲಿ ಪ್ರೀತಿಯುಳ್ಳವಳು
317. ರಕ್ಷಾಕರೀ = ರಕ್ಷಿಸುವವಳು
318. ರಾಕ್ಷಸಘ್ನೀ = ರಾಕ್ಷಸರನ್ನು ನಾಶಮಾಡುವವಳು
319. ರಾಮಾ = ಸ್ತ್ರೀರೂಪಳು
320. ರಮಣಲಂಪಟಾ = ಪತಿಯಲ್ಲಿ ಅತೀವ ಒಲುಮೆಯುಳ್ಳವಳು
321. ಕಾಮ್ಯಾ = ಎಲ್ಲರಿಂದಲೂ ಇಚ್ಛಿಸಲ್ಪಡುವವಳು
322. ಕಾಮಕಲಾರೂಪಾ = ಮದನಕಲಾ ಸ್ವರೂಪಳು
323. ಕದಂಬಕುಸುಮಪ್ರಿಯಾ = ಕದಂಬಪುಷ್ಪದಲ್ಲಿ ಆಸಕ್ತಿಯುಳ್ಳವಳು
324. ಕಲ್ಯಾಣೀ = ಮಂಗಳಸ್ವರೂಪಳು
325. ಜಗತೀಕಂದಾ = ಜಗತ್ತಿಗೆ ಮೂಲಭೂತಳು
326. ಕರುಣಾರಸಸಾಗರಾ = ದಯಾ ಸಮುದ್ರಳು
327. ಕಲಾವತೀ = 64 ಕಲೆಗಳನ್ನು ತಿಳಿದವಳು
328. ಕಲಾಲಾಪಾ = ಮಧುರವಾದ ಮಾತುಳ್ಳವಳು
329. ಕಾಂತಾ = ಮನೋಹರಿಣಿಯು
330. ಕಾದಂಬರಿಪ್ರಿಯಾ = ಕಾದಂಬರಿ ಎಂಬ ಮದ್ಯವಿಶೇಷವನ್ನು ಪ್ರೀತಿಸುವವಳು
331. ವರದಾ = ಇಷ್ಟಾರ್ಥವನ್ನು ಕೊಡುವವಳು
332. ವಾಮನಯನಾ = ಸುಂದರ ಕಣ್ಣುಳ್ಳವಳು
333. ವಾರುಣೀಮದವಿಹ್ವಲಾ = ವಾರುಣೀ ಎಂಬ ಮದ್ಯವಿಶೇಷ ಪಾನದಿಂದ ವ್ಯಾಕುಲಳಾದವಳು
334. ವಿಶ್ವಾಧಿಕಾ = ಪ್ರಪಂಚಕ್ಕೆ ಶ್ರೇಷ್ಠಳಾದವಳು
335. ವೇದವೇದ್ಯಾ = ವೇದಗಳಿಂದ ತಿಳಿಯಲ್ಪಡುವವಳು
336. ವಿಂಧ್ಯಾಚಲನಿವಾಸಿನೀ = ವಿಂಧ್ಯಪರ್ವತದಲ್ಲಿ ಇರುವವಳು
337. ವಿಧಾತ್ರೀ = ಪ್ರಪಂಚವನ್ನು ಪೋಷಿಸುವವಳು
338. ವೇದಜನನೀ = ವೇದಗಳಿಗೆ ಮಾತೃಸ್ವರೂಪಳು
339. ವಿಷ್ಣುಮಾಯಾ = ವಿಷ್ಣುವಿಗೆ ಸಂಬಂಧಿಸಿದ ಮಹಾಶಕ್ತಿಯು
340. ವಿಲಾಸಿನೀ = ವೈಯಾರವುಳ್ಳವಳು
341. ಕ್ಷೇತ್ರಸ್ವರೂಪಾ = ಪುಣ್ಯಕ್ಷೇತ್ರ ಸ್ವರೂಪಳು
342. ಕ್ಷೇತ್ರೇಶೀ = ಕ್ಷೇತ್ರಗಳಿಗೆ ಅಧಿಪತಿಯು
343. ಕ್ಷೇತ್ರಕ್ಷೇತ್ರಜ್ಞಪಾಲಿನೀ = ಶರೀರ-ಜೀವರನ್ನು ಕಾಪಾಡುವವಳು
344. ಕ್ಷಯವೃದ್ಧಿವಿನಿರ್ಮುಕ್ತಾ = ನಾಶ ಮತ್ತು ವೃದ್ಧಿ ಇಲ್ಲದವಳು
345. ಕ್ಷೇತ್ರಪಾಲ ಸಮರ್ಚಿತಾ = ಕ್ಷೇತ್ರಪಾಲನಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
346. ವಿಜಯಾ = ಜಯ ಸ್ವರೂಪಳು
347. ವಿಮಲಾ = ಪರಿಶುದ್ಧಳು
348. ವಂದ್ಯಾ = ನಮಸ್ಕಾರಾರ್ಹಳು
349. ವಂದಾರುಜನವತ್ಸಲಾ = ನಮಸ್ಕರಿಸುವವರನ್ನು ಪ್ರೀತಿಸುವವಳು
350. ವಾಗ್ವಾದಿನೀ = ವಾಕ್ಕನ್ನು ನುಡಿಸುವವಳು
351. ವಾಮಕೇಶೀ = ಸುಂದರವಾದ ಕೂಡಲುಳ್ಳವಳು
352. ವಹ್ನಿಮಂಡಲವಾಸಿನೀ = ಅಗ್ನಿಮಂಡಲದಲ್ಲಿ ವಾಸಿಸುವವಳು
353. ಭಕ್ತಿಮತ್ಕಲ್ಪಲತಿಕಾ = ಭಕ್ತಿಯುಳ್ಳವರಿಗೆ ಇಷ್ಟಾರ್ಥಗಳನ್ನು ಕೊಡುವ ಕಲ್ಪವೃಕ್ಷಳು
354. ಪಶುಪಾಶವಿಮೋಚಿನೀ = ಜೀವಿಗಳನ್ನು ಕರ್ಮಪಾಶದಿಂದ ಮುಕ್ತಗೊಳಿಸುವವಳು
355. ಸಂಹೃತಾಶೇಷಪಾಷಂಡಾ = ಸಕಲ ವೇದಶಾಸ್ತ್ರ ನಿಂದಕರನ್ನೂ ನಾಶಮಾಡುವವಳು
356. ಸದಾಚಾರಪ್ರವರ್ತಿಕಾ = ಸದಾಚಾರದಲ್ಲಿ ಪ್ರವೃತ್ತಿಯನ್ನು ಉಂಟುಮಾಡುವವಳು
357. ತಾಪತ್ರಯಾಗ್ನಿ ಸಂತಪ್ತಸಮಾಹ್ಲಾದನ ಚಂದ್ರಿಕಾ = ತಾಪತ್ರಯಗಳ ಅಗ್ನಿಯಿಂದ ಸುಡಲ್ಪಟ್ಟವರನ್ನು ಬೆಳದಿಂಗಳಂತೆ ಸಂತೋಷಗೊಳಿಸುವವಳು
358. ತರುಣೀ = ಯುವತಿ
359. ತಾಪಸಾರಾಧ್ಯಾ = ತಪಸ್ವಿಗಳಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
360. ತನುಮಧ್ಯಾ = ಸಣ್ಣ ನಡುವುಳ್ಳವಳು
361. ತಮೋಽಪಹಾ = ಅಜ್ಞಾನವನ್ನು ಹೋಗಲಾಡಿಸುವವಳು
362. ಚಿತಿಃ = ಜ್ಞಾನಸ್ವರೂಪಳು
363. ತತ್ಪದಲಕ್ಷ್ಯಾರ್ಥಾ = ತತ್ ಎಂಬ ಶಬ್ದದಿಂದ ಸತ್ಯದ ಗುರಿಯಾಗಿರುವವಳು
364, ಚಿದೇಕರಸರೂಪಿಣೀ = ಜ್ಞಾನವೇ ರೂಪವಾಗಿ ಉಳ್ಳವಳು
365. ಸ್ವಾತ್ಮಾನಂದಲವೀಭೂತ ಬ್ರಹ್ಮಾದ್ಯಾನಂದಸಂತತಿಃ = ಬ್ರಹ್ಮಾನಂದವೇ ಮುಂತಾದ ಸಕಲ ಆನಂದಗಳನ್ನು ತನ್ನ ಸ್ವಾತ್ಮಾನಂದದ ಮುಂದೆ ಲೇಶ ಮಾತ್ರವಾಗಿರುವವಳು
366. ಪರಾ = ಎಲ್ಲರನ್ನೂ ಮೀರಿದ ಪರಮಶಕ್ತಿಯುತಳು, ಪರಾ ಎಂಬ ವಾಗ್ದೇವತೆ
367. ಪ್ರತ್ಯಕ್ಚಿತೀರೂಪಾ = ಸ್ವಾಭಿಮುಖವಾದ ಜ್ಞಾನವುಳ್ಳವಳು, ಅವ್ಯಕ್ತ ಜ್ಞಾನರೂಪಳು
368. ಪಶ್ಯಂತೀ = ಎಲ್ಲವನ್ನೂ ತನ್ನೊಳಗೆ ನೋಡುವವಳು, ಪರಾ ನಂತರ ಸ್ವಾಧಿಷ್ಠಾನದಲ್ಲಿ ಪಶ್ಯಂತೀ ರೂಪದ ವಾಗ್ದೇವತೆಯಾಗಿರುವವಳು
369. ಪರದೇವತಾ = ಶ್ರೇಷ್ಠಳಾದ ದೇವತೆಯು, ಪರಾಶಕ್ತಿ ಸ್ವರೂಪಳು, ದೇವತೆಗಳು ಶಕ್ತಿದಾತಳು
370. ಮಧ್ಯಮಾ = ಎಲ್ಲದಕ್ಕೂ ಕೇಂದ್ರದಲ್ಲಿರುವವಳು, ಮಧ್ಯಮಾ ಎಂಬ ವಾಗ್ದೇವತೆ ಸ್ವರೂಪಳು
371. ವೈಖರೀರೂಪಾ = ಧ್ವನಿ ರೂಪದಲ್ಲಿರುವ ವಾಗ್ದೇವತೆ
372. ಭಕ್ತಮಾನಸಹಂಸಿಕಾ = ಭಕ್ತರ ಮನಸ್ಸಿನಲ್ಲಿ ಹಂಸ ರೂಪದಲ್ಲಿರುವವಳು
373. ಕಾಮೇಶ್ವರಪ್ರಾಣನಾಡೀ = ಕಾಮೇಶ್ವರನ ಜೀವನಾಡಿಯು
374. ಕೃತಜ್ಞಾ = (ಎಲ್ಲರ) ಕರ್ಮವನ್ನು ತಿಳಿದವಳು
375. ಕಾಮಪೂಜಿತಾ = ಮನ್ಮಥನಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
376. ಶೃಂಗಾರರಸಪೂರ್ಣಾ = ಶೃಂಗಾರ ರಸದಿಂದ ಪರಿಪೂರ್ಣಳು
377. ಜಯಾ = ವಿಜಯ ಸ್ವರೂಪಳು
378. ಜಾಲಂಧರಸ್ಥಿತಾ = ಜಾಲಂಧರ ಪೀಠದಲ್ಲಿ ವಾಸಿಸುವವಳು,
379. ಓಡ್ಯಾಣಪೀಠನಿಲಯಾ = (ಆಜ್ಞಾ ಚಕ್ರದ) ಓಡ್ಯಾಣ ಪೀಠದಲ್ಲಿ ನೆಲೆಸಿರುವವಳು
380. ಬಿಂದುಮಂಡಲವಾಸಿನೀ = (ಶ್ರೀ ಚಕ್ರದ) ಬಿಂದು ಮಂಡಲದಲ್ಲಿ ವಾಸಿಸುವವಳು
381. ರಹೋಯಾಗಕ್ರಮಾರಾಧ್ಯಾ = ಅಂತರ್ಯಾಗ (ರಹಸ್ಯವಾದ) ಕ್ರಮದಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
382. ರಹಸ್ತರ್ಪಣತರ್ಪಿತಾ = ಅಂತರ್ಯಾಗ (ರಹಸ್ಯವಾದ) ತರ್ಪಣದಿಂದ ತೃಪ್ತಿಪಡುವವಳು
383. ಸದ್ಯಃಪ್ರಸಾದಿನೀ = ಶಿಘ್ರವಾಗಿ ಅನುಗ್ರಹಿಸುವವಳು
384. ವಿಶ್ವಸಾಕ್ಷಿಣೀ = ಪ್ರಪಂಚಕ್ಕೆ ಸಾಕ್ಷಿರೂಪಳು
385. ಸಾಕ್ಷಿವರ್ಜಿತಾ = ಸ್ವತಃ ಸಾಕ್ಷಿ ಇಲ್ಲದವಳು
386. ಷಡಂಗದೇವತಾಯುಕ್ತಾ = ಹೃದಯ ದೇವತೆ ಮೊದಲಾದ (ಶಿರ, ಕೇಶ, ನೇತ್ರ, ಕವಚ, ಅಸ್ತ್ರ-ಶಸ್ತ್ರ) ಷಡಂಗ ದೇವತೆಗಳಿಂದ ಕೂಡಿದವಳು
387. ಷಾಡ್ಗುಣ್ಯಪರಿಪೂರಿತಾ = ಆರು (ಸಮೃದ್ಧಿ, ಶೌರ್ಯ, ವೈರಾಗ್ಯ, ಕೀರ್ತಿ, ಸಂಪತ್ತು, ವಿವೇಕ) ಸದ್ಗುಣಗಳಿಂದ ಕೂಡಿದವಳು
388. ನಿತ್ಯಕ್ಲಿನ್ನಾ = ಸದಾ ದಯಾಮಯಳು
389. ನಿರುಪಮಾ = ಹೋಲಿಕೆ ಇಲ್ಲದವಳು
390. ನಿರ್ವಾಣ ಸುಖದಾಯಿನೀ = ಮೋಕ್ಷಸುಖವನ್ನು ಕೊಡುವವಳು
391. ನಿತ್ಯಾ ಷೋಡಶಿಕಾರೂಪಾ = ಹದಿನಾರು ನಿತ್ಯಾ ದೇವಿಯರ ಸ್ವರೂಪಳು (ಕಾಮೇಶ್ವರೀ, ಭಗಮಾಲಿನೀ, ನಿತ್ಯಕ್ಲಿನ್ನಾ, ಭೇರುಂಡಾ, ವಹ್ನಿವಾಸಿನೀ, ಮಹಾವಜ್ರೇಶ್ವರೀ, ಶಿವದೂತೀ, ತ್ವರಿತಾ, ಕುಲಸುಂದರೀ, ನಿತ್ಯಾ, ನೀಲಪತಾಕಿನೀ, ವಿಜಯಾ, ಸರ್ವಮಂಗಲಾ, ಜ್ವಾಲಾಮಾಲಿನೀ, ಚಿತ್ರಾ, ತ್ರಿಪುರಸುಂದರೀ)
392. ಶ್ರೀಕಂಠಾರ್ಧಶರೀರಿಣೀ = ಶ್ರೀಕಂಠನ ಅರ್ಧಶರೀರವನ್ನು ಧರಿಸಿದವಳು, ಅರ್ಧನಾರೀಶ್ವರ ಸ್ವರೂಪಳು
393. ಪ್ರಭಾವತೀ = (ಅತೀಂದ್ರಿಯ ಶಕ್ತಿಗಳಿಂದ) ಕಾಂತಿಯುಕ್ತಳು
394. ಪ್ರಭಾರೂಪಾ = ಪ್ರಕಾಶ ಸ್ವರೂಪಳು
395. ಪ್ರಸಿದ್ಧಾ = ಎಲ್ಲರಿಗೂ ತಿಳಿದವಳು
396. ಪರಮೇಶ್ವರೀ = ಎಲ್ಲರಿಗೂ ಒಡೆಯಳು
397. ಮೂಲಪ್ರಕೃತಿಃ = ಎಲ್ಲದಕ್ಕೂ ಕಾರಣೀಭೂತಳು
398. ಅವ್ಯಕ್ತಾ = ಅಗೋಚರಳು
399. ವ್ಯಕ್ತಾವ್ಯಕ್ತಸ್ವರೂಪಿಣೀ = ಸಗುಣ-ನಿರ್ಗುಣ ಸ್ವರೂಪಳು
400. ವ್ಯಾಪಿನೀ = ಸರ್ವಜಗದ್ವ್ಯಾಪಕಳು
401. ವಿವಿಧಾಕಾರಾ = ಅನೇಕ ವಿಧವಾದ ರೂಪವುಳ್ಳವಳು
402. ವಿದ್ಯಾಽವಿದ್ಯಾಸ್ವರೂಪಿಣೀ = ಮೋಕ್ಷಹೇತುವಾದ ವಿದ್ಯಾ ಮತ್ತು ಬಂಧಹೇತುವಾದ ಅವಿದ್ಯಾ ಸ್ವರೂಪಳು
403. ಮಹಾಕಾಮೇಶನಯನ ಕುಮುದಾಹ್ಲಾದ ಕೌಮುದೀ = ಕಾಮೇಶನ ಕಣ್ಣುಗಳೆಂಬ ಕನ್ನೈದಿಲೆಗೆ ಆನಂದವನ್ನುಂಟು ಮಾಡುವ ಬೆಳದಿಂಗಳ ಸ್ವರೂಪಳು
404. ಭಕ್ತಹಾರ್ದ ತಮೋಭೇದ ಭಾನುಮದ್ಭಾನು ಸಂತತಿಃ = ಭಕ್ತರ ಹೃದಯದಲ್ಲಿರುವ ಅಜ್ಞಾನವನ್ನು ಹೋಗಲಾಡಿಸುವ ಪ್ರಕಾಶಮಾನ ಸೂರ್ಯಕಿರಣ ಸ್ವರೂಪಳು
405. ಶಿವದೂತೀ = ಶಿವನ ದೌತ್ಯವನ್ನು ಮಾಡಿದವಳು
406. ಶಿವಾರಾಧ್ಯಾ = ಶಿವನಿಂದ ಪೂಜಿಸಲ್ಪಟ್ಟವಳು
407. ಶಿವಮೂರ್ತಿಃ = ಮಂಗಳರೂಪಳು
408. ಶಿವಂಕರೀ = ಮಂಗಳವನ್ನುಂಟು ಮಾಡುವವಳು
409. ಶಿವಪ್ರಿಯಾ = ಈಶ್ವರನಿಗೆ ಪ್ರಿಯಳು
410. ಶಿವಪರಾ = ಶಿವನ ಪರವಾದವಳು, ಶಿವನನ್ನೇ ಗುರಿಯಾಗಿಟ್ಟುಕೊಂಡಿರುವವಳು
411. ಶಿಷ್ಟೇಷ್ಟಾ = ಶಿಷ್ಟರಿಗೆ ಇಷ್ಟಳು, ಶಿಷ್ಟರನ್ನು ಇಷ್ಟಪಡುವವಳು
412. ಶಿಷ್ಟಪೂಜಿತಾ = ಶಿಷ್ಟರಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
413. ಅಪ್ರಮೇಯಾ = ಅಳತೆಗೆ/ಇಂದ್ರಿಯಗಳಿಗೆ ನಿಲುಕದವಳು
414. ಸ್ವಪ್ರಕಾಶಾ = ಸ್ವಯಂ ಪ್ರಕಾಶವುಳ್ಳವಳು
415. ಮನೋವಾಚಾಮಗೋಚರಾ = ಮನಸ್ಸು-ಮಾತುಗಳಿಗೆ ನಿಲುಕದವಳು
416. ಚಿಚ್ಛಕ್ತಿಃ = ಚೈತನ್ಯ ಶಕ್ತಿ ಸ್ವರೂಪಳು
417. ಚೇತನಾರೂಪಾ = ಚೇತನಾ ಸ್ವರೂಪಳು, ಶುದ್ಧ ಚೈತನ್ಯಳು
418. ಜಡಶಕ್ತಿಃ = ಜಗತ್ ಸೃಷ್ಟಿ ಮಾಡುವ ಮಾಯೆಯ ಜಡಶಕ್ತಿ ಸ್ವರೂಪಳು
419. ಜಡಾತ್ಮಿಕಾ = ಜಡವಸ್ತು ಸ್ವರೂಪಳು
420. ಗಾಯತ್ರೀ = ಗಾಯತ್ರೀ ಮಂತ್ರಸ್ವರೂಪಳು
421. ವ್ಯಾಹೃತಿಃ = (ಮಾತಿನ) ಉಚ್ಚಾರಣೆಯ ಶಕ್ತಿಸ್ವರೂಪಳು, ಭೂರ್ಭುವಃಸ್ವಃ ಸ್ವರೂಪಳು
422. ಸಂಧ್ಯಾ = ಸಂಧ್ಯಾಕಾಲದಲ್ಲಿ ಉಪಾಸಿಸಲ್ಪಡುವವಳು, ಸಂಧ್ಯಾ (ಜೀವಾತ್ಮ-ಪರಮಾತ್ಮ ಯೋಗದ) ಸ್ವರೂಪಳು
423. ದ್ವಿಜವೃಂದ ನಿಷೇವಿತಾ = ದ್ವಿಜರಿಂದ ಉಪಾಸಿಸಲ್ಪಡುವವಳು
424. ತತ್ವಾಸನಾ = ತತ್ತ್ವವನ್ನೇ ಆಸನವನ್ನಾಗಿರುವವಳು, ತತ್ತ್ವಾಧಾರಳು
425. ತತ್ = ಆತ್ಮಸ್ವರೂಪಳು
426. ತ್ವಂ = ಜೀವಸ್ವರೂಪಳು
427. ಅಯಿ = (ತತ್-ತ್ವಂ) ಇವೆರಡರ ಐಕ್ಯ ಸ್ವರೂಪಳು
428. ಪಂಚಕೋಶಾಂತರಸ್ಥಿತಾ = ಅನ್ನಮಯ-ಪ್ರಾಣಮಯ-ಮನೋಮಯ-ವಿಜ್ಞನಮಯ-ಆನಂದಮಯ ಕೋಶಗಳ ಒಳಗಿರುವವಳು
429. ನಿಃಸೀಮ ಮಹಿಮಾ = ಎಣೆ ಇಲ್ಲದ ಮಹಿಮೆಯುಳ್ಳವಳು
430. ನಿತ್ಯಯೌವನಾ = ಸದಾ ಯುವತಿಯು
431. ಮದಶಾಲಿನೀ = ಸದಾ ಉತ್ಸಾಹ-ಆನಂದಗಳಿಂದ ಕೂಡಿರುವವಳು
432. ಮದಘೂರ್ಣಿತರಕ್ತಾಕ್ಷೀ = ಆನಂದದಿಂದ/ಉತ್ಸಾಹದಿಂದ ಚಲಿಸುತ್ತಿರುವ ಕೆಂಪಾದ ಕಣ್ಣುಳ್ಳವಳು
433. ಮದಪಾಟಲಗಂಡಭೂಃ = ಆನಂದ-ಲವಲವಿಕೆಯಿಂದ ಕೆಂಪಾದ ಕೆನ್ನೆಯುಳ್ಳವಳು
434. ಚಂದನದ್ರವದಿಗ್ಧಾಂಗೀ = ಶ್ರೀಗಂಧದ ರಸಲೇಪಿಸಿದ ಅಂಗಗಳುಳ್ಳವಳು
435. ಚಾಂಪೇಯ ಕುಸುಮಪ್ರಿಯಾ = ಸಂಪಿಗೆಯ ಹೂವನ್ನು ಇಷ್ಟಪಡುವವಳು
436. ಕುಶಲಾ = ಚತುರಳು
437. ಕೋಮಲಾಕಾರಾ = ಮೃದುವಾದ ಅವಯವಗಳುಳ್ಳವಳು
438. ಕುರುಕುಲ್ಲಾ = ವಿಮರ್ಶ ವಾಪಿಕಾದೇವಿ ಸ್ವರೂಪಳು, ಕುರುವಿಂದಾ ಮಾಣಿಕ್ಯದಲ್ಲಿ ಶಕ್ತಿರೂಪಳಾಗಿರುವವಳು
439. ಕುಲೇಶ್ವರೀ = ವಂಶಾಧೀಶ್ವರೀ, ಜ್ಞಾನ-ಜ್ಞೇಯ-ಜ್ಞಾತೃ ತ್ರಿಪುಟಿಯ ಅಧಿಪತಿ
440. ಕುಲಕುಂಡಾಲಯಾ = ಮೂಲಾಧಾರದಲ್ಲಿ (ಕುಲಕುಂಡಾ-ದಲ್ಲಿ) ವಾಸಿಸುವವಳು, ಕುಂಡಲಿನಿಯ ಶಕ್ತಿರೂಪಳು
441. ಕೌಲಮಾರ್ಗತತ್ಪರಸೇವಿತಾ = ಬಹಿರ್ಯಾಗ ನಿರತರಿಂದ / ಕೌಲ ಮತದವರಿಂದ ಸೇವಿಸಲ್ಪಡುವವಳು
442. ಕುಮಾರಗಣನಾಥಾಂಬಾ = ಸ್ಕಂದ-ಗಣಪತಿಯರ ತಾಯಿ
443. ತುಷ್ಟಿಃ = ಸಂತೋಷ ಸ್ವರೂಪಳು
444. ಪುಷ್ಟಿಃ = ಪುಷ್ಟಿ ಸ್ವರೂಪಳು
445. ಮತಿಃ = ಬುದ್ಧಿ ಸ್ವರೂಪಳು
446. ಧೃತಿಃ = ಧೈರ್ಯ ಸ್ವರೂಪಳು
447. ಶಾಂತಿಃ = ಶಾಂತಿ ಸ್ವರೂಪಳು
448. ಸ್ವಸ್ತಿಮತೀ = ಮಂಗಳವೇ ಆಗಿರುವವಳು
449. ಕಾಂತಿಃ = ತೇಜಸ್ವರೂಪಳು
450. ನಂದಿನೀ = ಸಂತೋಷವನ್ನುಂಟು ಮಾಡುವವಳು
451. ವಿಘ್ನನಾಶಿನೀ = ವಿಘ್ನವನ್ನು ನಿವಾರಿಸುವವಳು
452. ತೇಜೋವತೀ = ಪ್ರಕಾಶ ಉಳ್ಳವಳು
453. ತ್ರಿನಯನಾ = ಸೂರ್ಯ-ಚಂದ್ರ-ಅಗ್ನಿ ರೂಪದ ಮೂರು ಕಣ್ಣುಳ್ಳವಳು
454. ಲೋಲಾಕ್ಷೀ ಕಾಮರೂಪಿಣೀ = ಉಂಡಾದ ಕಣ್ಣುಗಳುಳ್ಳವಳು, ಸ್ತ್ರೀಯಲ್ಲಿರುವ ಕಾಮದ ಸ್ವರೂಪಳು, ಕಾಮದಿಂದ ಕೂಡಿದ ಉಂಡಾದ ಕಣ್ಣುಗಳುಳ್ಳವಳು
455. ಮಾಲಿನೀ = ಮಾಲೆಯನ್ನು ಧರಿಸಿದವಳು
456. ಹಂಸಿನೀ = ಹಂಸಮಂತ್ರ ಸ್ವರೂಪಳು, ಹಂಸಗಳಿಂದ ಕೂಡಿದವಳು
457. ಮಾತಾ = ಮಾತೃ ಸ್ವರೂಪಳು
458. ಮಲಯಾಚಲವಾಸಿನೀ = ಮಲಯಾಚಲ ಪರ್ವತದಲ್ಲಿ ವಾಸಿಸುವವಳು
459. ಸುಮುಖೀ = ಸುಂದರ ಮುಖವುಳ್ಳವಳು
460. ನಲಿನೀ = ಪದ್ಮದಂತೆ ಕೋಮಲವಾಗಿ, ಸುಂದರವಾಗಿ ಇರುವವಳು
461. ಸುಭ್ರೂಃ = ಸುಂದರವಾದ ಹುಬ್ಬುಗಳುಳ್ಳವಳು
462. ಶೋಭನಾ = ಸೌಂದರ್ಯವತಿ, ಕಾಂತಿಯುಕ್ತಳು
463. ಸುರನಾಯಿಕಾ = ದೇವತೆಗಳ ನಾಯಿಕೆ
464. ಕಾಲಕಂಠೀ = ನೀಲಕಂಠನ ಪತ್ನಿಯು
465. ಕಾಂತಿಮತೀ = ಶೋಭೆಯುಳ್ಳವಳು, ಕಾಂತಿಯುಕ್ತಳು
466. ಕ್ಷೋಭಿಣೀ = ಉದ್ರೇಕಗೊಳಿಸುವವಳು
467. ಸೂಕ್ಷ್ಮರೂಪಿಣೀ = ಸೂಕ್ಷ್ಮ ರೂಪವುಳ್ಳವಳು (ಇಂದ್ರಿಯಗಳಿಗೆ ಗೋಚರವಾಗದವಳು)
468. ವಜ್ರೇಶ್ವರೀ = ವಜ್ರಾಯುಧಕ್ಕೆ ಅಧಿದೇವತೆ, ಜಾಲಂಧರ ಪೀಠದಲ್ಲಿ ವಾಸಿಸುವ ವಜ್ರೇಶ್ವರೀ ದೇವಿಯು
469. ವಾಮದೇವೀ = ವಾಮದೇವನ ಪತ್ನಿಯು
470. ವಯೋಽವಸ್ಥಾವಿವರ್ಜಿತಾ = ಬಾಲ್ಯಾದಿ (ಯೌವ್ವನ, ಕೌಮಾರ್ಯ, ಮುಪ್ಪು) ಅವಸ್ಥೆಗಳಿಂದ ಹೊರತಾದವಳು
471. ಸಿದ್ಧೇಶ್ವರೀ = ಸಿದ್ಧರಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು, ಸಿದ್ಧರ ಒಡತಿ
472. ಸಿದ್ಧವಿದ್ಯಾ = ಹದಿನೈದು ಮಾತ್ರೆಗಳುಳ್ಳ ಸಿದ್ಧವಿದ್ಯೆಯ ಮಂತ್ರಸ್ವರೂಪಳು
473. ಸಿದ್ಧಮಾತಾ = ಸಿದ್ಧರಿಗೆ ತಾಯಿಸ್ವರೂಪಳು
474. ಯಶಸ್ವಿನೀ = ಕೀರ್ತಿವಂತಳು
475. ವಿಶುದ್ಧಿಚಕ್ರನಿಲಯಾ = ಹದಿನಾರು ದಳದ ವಿಶುದ್ಧಿ ಚಕ್ರದಲ್ಲಿ ನೆಲೆಸಿರುವವಳು
476. ಆರಕ್ತವರ್ಣಾ = ಸಂಪೂರ್ಣವಾಗಿ ಕೆಂಪು(ಚಂದನದ) ಬಣ್ಣದವಳು
477. ತ್ರಿಲೋಚನಾ = ಮೂರು ಕಣ್ಣುಗಳುಳ್ಳವಳು
478. ಖಟ್ವಾಂಗಾದಿಪ್ರಹರಣಾ = ಖಟ್ವಾಂಗ (ಗದೆ) ಮೊದಲಾದ ಆಯುಧವುಳ್ಳವಳು
479. ವದನೈಕಸಮನ್ವಿತಾ = ಒಂದು ಮುಖದಿಂದ ಕೂಡಿದವಳು
480. ಪಾಯಸಾನ್ನಪ್ರಿಯಾ = ಪಾಯಸಾನ್ನ ಪ್ರಿಯಳು
481. ತ್ವಕ್ಸ್ಥಾ = ಚರ್ಮದ ಅಧಿದೇವತೆ
482. ಪಶುಲೋಕಭಯಂಕರೀ = ಲೋಕದ ಜೀವಿಗಳಿಗೆ (ಅಜ್ಞಾನಿಗಳಿಗೆ) ಭಯವನ್ನುಂಟು ಮಾಡುವವಳು
483. ಅಮೃತಾದಿಮಹಾಶಕ್ತಿಸಂವೃತಾ = ಅಮೃತಾದಿ (ಕರ್ಷಿಣೀ, ಇಂದ್ರಾಣೀ, ಈಶಾನೀ, ಉಮಾ, ಊಧ್ರ್ವಕೇಶೀ) ಮಹಾಶಕ್ತಿಗಳಿಂದ ಕೂಡಿದವಳು
484. ಡಾಕಿನೀಶ್ವರೀ = ಡಾಕಿನ್ಯಂಬಾ ಎಂಬ ದೇವತೆಯು
485. ಅನಾಹತಾಬ್ಜನಿಲಯ = (ಹೃದಯದ) ಅನಾಹತ ಚಕ್ರದಲ್ಲಿ ಇರುವವಳು
486. ಶ್ಯಾಮಾಭಾ = ಶ್ಯಾಮಲ(ಕಪ್ಪು) ವರ್ಣದವಳು
487. ವದನದ್ವಯಾ = ಎರಡು ಮುಖ ಉಳ್ಳವಳು
488. ದಂಷ್ಟ್ರೋಜ್ಜ್ವಲಾ = ಹೊಳೆಯುವ ಕೋರೆದಾಡೆಯುಳ್ಳವಳು
489. ಅಕ್ಷಮಾಲಾದಿಧರಾ = ರುದ್ರಾಕ್ಷಿ/ಜಪಮಾಲೆ ಮೊದಲಾದವನ್ನು ಧರಿಸಿದವಳು
490. ರುಧಿರಸಂಸ್ಥಿತಾ = (ಜೀವಿಗಳ) ರಕ್ತದ ಅಧಿದೇವತೆ
491. ಕಾಲರಾತ್ರ್ಯಾದಿಶಕ್ತ್ಯೌಘವೃತಾ = ಕಾಲರಾತ್ರಿ ಮೊದಲಾದ ಶಕ್ತಿ ಸಮೂಹದಿಂದ ಕೂಡಿದವಳು
492. ಸ್ನಿಗ್ಧೌದನಪ್ರಿಯಾ = ತುಪ್ಪದನ್ನ ಮುಂತಾದ (ಜಿಡ್ಡಿನ) ಪದಾರ್ಥವನ್ನು ಇಷ್ಟಪಡುವವಳು
493. ಮಹಾವೀರೇಂದ್ರವರದಾ = ಮಹಾವೀರರಿಗೆ ವರವನ್ನು ಕೊಡುವವಳು
494. ರಾಕಿಣ್ಯಂಬಾಸ್ವರೂಪಿಣೀ = ರಾಕಿಣೀ ಎಂಬ ದೇವತೆ
495. ಮಣಿಪೂರಾಬ್ಜನಿಲಯಾ = ಹತ್ತು ದಳದ ಮಣಿಪೂರ ಚಕ್ರದಲ್ಲಿ ನೆಲೆಸಿದವಳು
496. ವದನತ್ರಯಸಂಯುತಾ = ಮೂರು ಮುಖಗಳುಳ್ಳವಳು
497. ವಜ್ರಾದಿಕಾಯುಧೋಪೇತಾ = ವಜ್ರಾಯುಧ ಮೊದಲಾದ ಆಯುಧಗಳಿಂದ ಕೂಡಿದವಳು
498. ಡಾಮರ್ಯಾದಿಭಿರಾವೃತಾ = ಡಾಮರೀ ಮೊದಲಾದ ದೇವತೆಗಳಿಂದ ಕೂಡಿಕೊಂಡವಳು
499. ರಕ್ತವರ್ಣಾ = ಕೆಂಪು ಬಣ್ಣದವಳು
500. ಮಾಂಸನಿಷ್ಠಾ = (ಜೀವಿಗಳಲ್ಲಿರುವ) ಮಾಂಸದ ಅಧಿದೇವತೆ
