ಭವಾನ್ ಕತಿ ದಿನಾನಿ ವಿರಾಮಂ ಸ್ವೀಕರೋತಿ? = ನೀವು ಎಷ್ಟು ದಿನ ರಜೆ ತಗೊಳ್ತೀರಾ?
ಏಷು ದಿನೇಷು ಮಹಾನ್ ಕಾರ್ಯಭಾರಃ = ಇತ್ತೀಚೆಗೆ ತುಂಬಾ ಕೆಲಸದ ಹೊರೆ
ಏತದ್ ಸೂಚನಾಫಲಕೇ ಸ್ಥಾಪಯತು = ಇದನ್ನು ನೋಟಿಸ್ ಬೋರ್ಡಿಗೆ ಹಾಕು
ಅತ್ರ ಹಸ್ತಾಂಕನಂ ಕರೋತು = ಇಲ್ಲಿ ಸಹಿ ಮಾಡಿ.
ಸಃ ವಿರಾಮಂ ಸ್ವೀಕೃತವಾನ್ = ಅವರು ರಜೆ ತಗೊಂಡಿದ್ದಾರೆ.
ಏತಸ್ಮಿನ್ ವಿಷಯೇ ಪುನಃ ಚಿಂತಯಾಮಿ = ಇದರ ಬಗ್ಗೆ ಮತ್ತೊಮ್ಮೆ ಯೋಚಿಸ್ತೀನಿ.
ಆಗಾಮಿ ಸಪ್ತಾಹೇ ಮಾಂ ಪಶ್ಯತು = ಮುಂದಿನ ವಾರ ನನ್ನನು ನೋಡಿ.
ಏತಸ್ಮಿನ್ ವಿಷಯೇ ಅನಂತರಂ ವದಾಮಿ = ಇದರ ಬಗ್ಗೆ ಆಮೇಲೆ ಹೇಳ್ತೀನಿ.
ಏತತ್ ಅವಶ್ಯಂ ಮನಸಿ ಸ್ಥಾಪಯಾಮಿ = ಇದನ್ನು ಖಂಡಿತ ಮನಸ್ಸಿನಲ್ಲಿ ಇಟ್ಟುಕೊಂಡಿರ್ತೀನಿ.
ಭವದುಕ್ತಂ ಸರ್ವಂ ಜ್ಞಾತಂ ಭೋಃ = ನೀವು ಹೇಳಿದ್ದೆಲ್ಲಾ ತಿಳೀತು ಕಣ್ರೀ.
ಅತ್ರ ತಸ್ಯ ಏವ ಸರ್ವಾಧಿಕಾರಃ = ಇಲ್ಲಿ ಅವನದೇ ಸರ್ವಾಧಿಕಾರ.
ದೂರವಾಣೀ ಕಾಽಪಿ ಆಗತಾ ವಾ? = ಫೋನು ಏನಾದರೂ ಬಂತಾ?
ದೂರವಾಣೀ ಆಗತಾ ಆಸೀತ್ = ಫೋನು ಬಂದಿತ್ತು.
ಭವಾನ್ ಕಸ್ಮಿನ್ ಸ್ಥಾನೇ ನಿಯುಕ್ತಃ ಅಸ್ತಿ? = ನೀವು ಯಾವ ಸ್ಥಾನದಲ್ಲಿ (ಫೊಸ್ತ್)ಇದ್ದೀರಾ?
ಏಷಃ ಸದಾ ಅಗತ್ಯ ಪೀಡಯತಿ = ಇವನು ಯಾವಾಗಲೂ ಬಂದು ಪೀಡಿಸ್ತಾನೆ.
ಇದಾನೀಂ ಸಮಯಃ ಅತೀತಃ = ಈಗ ವೇಳೆಯಾಯಿತು.
ಶ್ವಃ ಆಗಚ್ಛತು = ನಾಳೆ ಬನ್ನಿ.
ಸಃ ಅಪಿ ಆಗತವಾನ್ ಇತಿ ಸ್ಮರಾಮಿ = ಅವನು ಬಂದಿದ್ದ ಅಂತ ನೆನಪು.
ಪಂಚವಾದನಪರ್ಯಂತಂ ಅತ್ರೈವ ಆಸೀತ್ = ಐದು ಗಂಟೆಯವರೆಗೂ ಇಲ್ಲಿಯೇ ಇದ್ದ.
ಮಾಂ ಆಹೂತವಾನ್ ವಾ? = ನನ್ನನು ಕರೆದಿರಾ?/ಕರೆದರಾ?
ತದ್ ವ್ಯವಸ್ಥಾಂ ಅಹಂ ಕರೋಮಿ = ಆ ವ್ಯವಸ್ಥೆಯನ್ನು ನಾನು ಮಾಡ್ತೀನಿ.
ಕಾರ್ಯಾಲಯಸ್ಯ ಸಮಾಪ್ತಿಃ ಕದಾ? = ಕಾರ್ಯಲಯ ಮುಗಿಯೋದು ಎಷ್ಟೊತ್ತಿಗೆ?
ಏತದ್ವಿಷಯೇ ಶ್ವಃ ಪುನರಪಿ ಸ್ಮಾರಯತು = ಇದರ ಬಗ್ಗೆ ನಾಳೆ ಜ್ಞಾಪಿಸಿ
ತಂ ಅತ್ರ ಆಗಂತುಂ ಸೂಚಯತು = ಅವನಿಗೆ ಇಲ್ಲಿ ಬರಲು ಹೇಳಿ.
ಕಿಮರ್ಥಂ ಇದಾನೀಂ ಅಪಿ ಕಾರ್ಯಂ ನ ಆರಬ್ಧಮ್? = ಯಾಕಿನ್ನೂ ಕೆಲಸ ಶುರು ಮಾಡಿಲ್ಲ?
ಅನ್ಯೇಷಾಂ ಉಪಹಾಸೇನ ಏವ ಕಾಲಂ ಯಾಪಯತಿ = ಬೇರೆಯವರನ್ನು ಟೀಕಿಸುವುದರಲ್ಲಿಯೇ ಕಾಲ ಕಳೆಯುತ್ತಾನೆ.
ಮಯಾ ಕಿಂ ಕರಣೀಯಮ್ ವದತು? = ನಾನು ಏನು ಮಾಡಬೇಕು ಹೇಳಿ?
ಅಹಂ ಕಿಂ ಕರೋಮಿ ಭೋಃ? = ನಾನೇನು ಮಾಡಲಪ್ಪಾ?
ಅಸ್ತು, ಪರಿಶೀಲಯಾಮಃ = ಆಗಲಿ, ನೋಡೋಣ.
ಆಗಚ್ಛತು, ಕಾಫೀ ಪೀತ್ವಾ ಆಗಚ್ಛಾಮಃ = ಬನ್ನಿ, ಕಾಫೀ ಕುಡಿದು ಬರೋಣ.
ಭವಾನ್ ಶೀಘ್ರಂ ಪ್ರತ್ಯಾಗಚ್ಛತಿ ವಾ? = ನೀವು ಬೇಗ ವಾಪಾಸ್ ಬರ್ತೀರಾ?
ಕೃಪಯಾ ಉಪವಿಶತು = ದಯವಿಟ್ಟು ಕುಳಿತುಕೊಳ್ಳಿ.
ಪಂಚ ನಿಮಿಷೇಷು ಕೃತ್ವಾ ದದಾಮಿ = ಐದು ನಿಮಿಷಗಳಲ್ಲಿ ಮಾಡಿಕೊಡ್ತೀನಿ.
ಅದ್ಯ ಸಃ ನಾಸ್ತಿ ಕಿಲ! = ಇವತ್ತು ಇವರು ಇಲ್ವಲ್ಲ!
ಏಕಸಪ್ತಾಹಾನಂತರಂ ಆಗಚ್ಛೇತ್ = ಒಂದು ವಾರದ ನಂತರ ಬರಬಹುದು.
