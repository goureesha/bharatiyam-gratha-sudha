|| ಲಲಿತಾಸಹಸ್ರನಾಮ ಅರ್ಥ - 2 ||
501. ಗುಡಾನ್ನಪ್ರೀತಮಾನಸಾ = ಬೆಲ್ಲದನ್ನವನ್ನು ಪ್ರೀತಿಸುವವಳು
502. ಸಮಸ್ತಭಕ್ತಸುಖದಾ = ಎಲ್ಲ ಭಕ್ತರಿಗೂ ಸುಖವನ್ನು ಕೊಡುವವಳು
503. ಲಾಕಿನ್ಯಂಬಾ ಸ್ವರೂಪಿಣೀ = ಲಾಕಿನೀ (ಯೋಗಿನೀ) ಎಂಬ ದೇವತೆ
504. ಸ್ವಾಧಿಷ್ಠಾನಾಂಬುಜಗತಾ = ಆರು ದಳಗಳ ಸ್ವಾಧಿಷ್ಠಾನ ಚಕ್ರದಲ್ಲಿ (ಕಾಕಿನೀ/ಯೋಗಿನೀ ರೂಪದಲ್ಲಿ) ನೆಲೆಸಿದವಳು
505. ಚತುರ್ವಕ್ತ್ರಮನೋಹರಾ = ನಾಲ್ಕು ಮನೋಹರವಾದ ಮುಖಗಳುಳ್ಳವಳು
506. ಶೂಲಾದ್ಯಾಯುಧಸಂಪನ್ನಾ = ಶೂಲ ಮೊದಲಾದ (ಪಾಶ, ಅಭಯ) ಆಯುಧಗಳನ್ನು ಧರಿಸಿರುವವಳು
507. ಪೀತವರ್ಣಾ = ಹಳದಿ ಬಣ್ಣದವಳು
508. ಅತಿಗರ್ವಿತಾ = ಅತ್ಯಂತ ಆತ್ಮಗೌರವ ಉಳ್ಳವಳು
509. ಮೇದೋನಿಷ್ಠಾ = ಮೇದಸ್ಸಿಗೆ ಅಧಿದೇವತೆ
510. ಮಧುಪ್ರೀತಾ = ಜೇನಿನ ಸಿಹಿಯಲ್ಲಿ ಆಸಕ್ತಳು
511. ಬಂಧಿನ್ಯಾದಿ ಸಮನ್ವಿತಾ = ಬಂಧಿನಿ ಮುಂತಾದ ಶಕ್ತಿಯರಿಂದ ಕೂಡಿದವಳು
512. ದಧ್ಯಾನ್ನಾಸಕ್ತಹೃದಯಾ = ಮೊಸರನ್ನವನ್ನು ಪ್ರೀತಿಸುವವಳು
513. ಕಾಕಿನೀರೂಪಧಾರಿಣೀ = ಕಾಕಿನೀ (ಯೋಗಿನೀ) ಎಂಬ ದೇವತೆಯ ರೂಪವನ್ನು ಧರಿಸಿದವಳು
514. ಮೂಲಾಧಾರಾಂಬುಜಾರೂಢಾ = ಮೂಲಾಧಾರ ಚಕ್ರದ ಕಮಲದಲ್ಲಿ ಇರುವವಳು
515. ಪಂಚವಕ್ತ್ರಾ = ಐದು ಮುಖಗಳುಳ್ಳವಳು
516. ಅಸ್ಥಿಸಂಸ್ಥಿತಾ = (ಜೀವಿಗಳ) ಮೂಳೆಗಳಿಗೆ ಅಧಿದೇವತೆ
517. ಅಂಕುಶಾದಿಪ್ರಹರಣಾ = ಅಂಕುಶ ಮೊದಲಾದ ಆಯುಧಗಳನ್ನು ಹಿಡಿದವಳು
518. ವರದಾದಿನಿಷೇವಿತಾ = ವರದಾ ಮೊದಲಾದ ಶಕ್ತಿಯರಿಂದ ಸೇವಿಸಲ್ಪಡುವವಳು
519. ಮುದ್ಗೌದನಾಸಕ್ತಚಿತ್ತಾ = ಹೆಸರು ಬೇಳೆ ಮಿಶ್ರಿತ ಅನ್ನವನ್ನು ಪ್ರೀತಿಸುವವಳು
520. ಸಾಕಿನ್ಯಂಬಾಸ್ವರೂಪಿಣೀ = ಸಾಕಿನೀ (ಯೋಗಿನೀ) ಎಂಬ ದೇವತೆಯು
521. ಆಜ್ಞಾಚಕ್ರಾಬ್ಜನಿಲಯಾ = ಆಜ್ಞಾಚಕ್ರದಲ್ಲಿ ನೆಲೆಸಿದವಳು
522. ಶುಕ್ಲವರ್ಣಾ = ಬಿಳಿಯ ಬಣ್ಣದವಳು
523. ಷಡಾನನಾ = ಆರು ಮುಖದವಳು
524. ಮಜ್ಜಾಸಂಸ್ಥಾ = ಮಜ್ಜಕ್ಕೆ (ಮೂಳೆಯ ತಿರುಳು) ಅಧಿದೇವತೆ
525. ಹಂಸವತೀಮುಖ್ಯಶಕ್ತಿಸಮನ್ವಿತಾ = ಹಂಸವತೀ ಮೊದಲಾದ ಶಕ್ತಿಯರಿಂದ ಕೂಡಿದವಳು
526. ಹರಿದ್ರಾನ್ನೈಕರಸಿಕಾ = ಅರಿಶಿನ ಮಿಶ್ರಿತ ಅನ್ನವನ್ನು ಪ್ರೀತಿಸುವವಳು
527. ಹಾಕಿನೀರೂಪಧಾರಿಣೀ = ಹಾಕಿನೀ ದೇವಿಯ ರೂಪವನ್ನು ಧರಿಸಿದವಳು
528. ಸಹಸ್ರಪದ್ಮದಲಸ್ಥಾ = ಸಾವಿರ ದಳದ ಸಹಸ್ರಾರ ಪದ್ಮದಲ್ಲಿ ಇರುವವಳು
529. ಸರ್ವವರ್ಣೋಪಶೋಭಿತಾ = ಎಲ್ಲಾ ವರ್ಣಗಳಿಂದಲೂ ಕೂಡಿದವಳು
530. ಸರ್ವಾಯುಧಧರಾ = ಎಲ್ಲಾ ಆಯುಧಗಳಿಂದಲೂ ಕೂಡಿದವಳು
531. ಶುಕ್ಲಸಂಸ್ಥಿತಾ = ವೀರ್ಯಕ್ಕೆ (ರೇತಸ್ಸಿಗೆ) ಅಧಿದೇವತೆ
532. ಸರ್ವತೋಮುಖೀ = ಎಲ್ಲಾ ದಿಕ್ಕಿಗೂ ಮುಖಗಳುಳ್ಳವಳು
533. ಸರ್ವೌದನಪ್ರೀತಚಿತ್ತಾ = ಎಲ್ಲಾ ವಿಧದ ಅನ್ನವನ್ನು (ಆಹಾರವನ್ನು) ಪ್ರೀತಿಸುವವಳು
534. ಯಾಕಿನ್ಯಂಬಾಸ್ವರೂಪಿಣೀ = ಯಾಕಿನೀ ಎಂಬ ದೇವತೆಯ ಸ್ವರೂಪಳು
535. ಸ್ವಾಹಾ = ಯಾಗಗಳಲ್ಲಿ ದೇವತೆಗಳಿಗೆ ಹವಿಸ್ಸನ್ನು ಒಯ್ಯುವವಳು, ಸ್ವಾಹಾ ಮಂತ್ರದ ಮೂರ್ತಿರೂಪಳು
536. ಸ್ವಧಾ = ಪಿತೃಗಳಿಗೆ ಹವಿಸ್ಸನ್ನು ಒಯ್ಯುವವಳು
537. ಅಮತಿಃ = ಅವಿದ್ಯಾ ಸ್ವರೂಪಳು
538. ಮೇಧಾ = ಬುದ್ಧಿ (ವಿದ್ಯಾ) ಸ್ವರೂಪಳು
539. ಶ್ರುತಿಃ = ವೇದ ಸ್ವರೂಪಳು
540. ಸ್ಮೃತಿಃ = (ವೇದಾರ್ಥಗಳ) ಸ್ಮರಣ ಸ್ವರೂಪಳು
541. ಅನುತ್ತಮಾ = ಶ್ರೇಷ್ಠಳು, ಅತ್ಯುತ್ತಮಳು
542. ಪುಣ್ಯಕೀರ್ತಿಃ = ಪುಣ್ಯಪ್ರದವಾದ ಕೀರ್ತಿಯುಳ್ಳವಳು
543. ಪುಣ್ಯಲಭ್ಯಾ = ಪುಣ್ಯವಂತರಿಂದ ಪಡೆಯಲು ಸಾಧ್ಯಳು
544. ಪುಣ್ಯಶ್ರವಣಕೀರ್ತನಾ = ಪುಣ್ಯಕರವಾದ ನಾಮಸಂಕೀರ್ತನೆ ಉಳ್ಳವಳು
545. ಪುಲೋಮಜಾರ್ಚಿತಾ = (ಇಂದ್ರ ಪತ್ನಿ) ಶಚೀದೇವಿಯಿಂದ ಪೂಜಿತಳು
546. ಬಂಧಮೋಚನೀ = (ಸಂಸಾರ) ಬಂಧನವನ್ನು ಬಿಡಿಸುವವಳು
547. ಬಂಧುರಾಲಕಾ/ಬರ್ಬರಾಲಕಾ = ಗುಂಗುರಾದ ಮುಂಗುರುಳುಳ್ಳವಳು
548. ವಿಮರ್ಶರೂಪಿಣೀ = ವಿಚಾರಸ್ವರೂಪಳು
549. ವಿದ್ಯಾ = ಜ್ಞಾನಸ್ವರೂಪಳು
550. ವಿಯದಾದಿಜಗತ್ಪ್ರಸೂಃ = ಆಕಾಶವೇ ಮೊದಲಾದ ಜಗತ್ತನ್ನು ಸೃಷ್ಟಿಸುವವಳು
551. ಸರ್ವವ್ಯಾಧಿಪ್ರಶಮನೀ = ಎಲ್ಲಾ ರೋಗಗಳನ್ನೂ ಗುಣಪಡಿಸುವವಳು
552. ಸರ್ವಮೃತ್ಯುನಿವಾರಿಣೀ = ಎಲ್ಲ ವಿಧವಾದ ಮರಣವನ್ನು ತಡೆಯುವವಳು
553. ಅಗ್ರಗಣ್ಯಾ = ಅಗ್ರಗಣ್ಯಳು
554. ಅಚಿಂತ್ಯರೂಪಾ = ಯೋಚಿಸಲು ಅಸಾಧ್ಯವಾದ ಆಕೃತಿಯುಳ್ಳವಳು
555. ಕಲಿಕಲ್ಮಷನಾಶಿನೀ = ಕಲಿಯುಗದ ದೋಷವನ್ನು ಹೋಗಲಾಡಿಸುವವಳು
556. ಕಾತ್ಯಾಯನೀ = ಋಷಿ ಕಾತ್ಯಾಯನನ ಮಗಳು
557. ಕಾಲಹಂತ್ರೀ = ಕಾಲವನ್ನು(ಮೃತ್ಯುವನ್ನು) ನಾಶ ಮಾಡುವವಳು
558. ಕಮಲಾಕ್ಷನಿಷೇವಿತಾ = ವಿಷ್ಣುವಿನಿಂದ ಸ್ತುತಿಸಲ್ಪಡುವವಳು
559. ತಾಂಬೂಲಪೂರಿತಮುಖೀ = ತಾಂಬೂಲದಿಂದ ಕೂಡಿದ ಮುಖವುಳ್ಳವಳು
560. ದಾಡಿಮೀಕುಸುಮಪ್ರಭಾ = ದಾಳಿಂಬದ ಹೂವಿನ ಕಾಂತಿಯುಳ್ಳವಳು
561. ಮೃಗಾಕ್ಷೀ = ಜಿಂಕೆಯಂತೆ (ಸುಂದರವಾದ ಉದ್ದ) ಕಣ್ಣುಗಳುಳ್ಳವಳು
562. ಮೋಹಿನೀ = ಮೋಹಗೊಳಿಸುವವಳು
563. ಮುಖ್ಯಾ = ಪ್ರಮುಖಳು
564. ಮೃಡಾನೀ = ಮೃದನ(ಶಿವನ) ಪತ್ನಿಯು
565. ಮಿತ್ರರೂಪಿಣೀ = (ಜಗತ್ತಿಗೆ) ಸ್ನೇಹಿತೆಯ ರೂಪದಲ್ಲಿರುವವಳು
566. ನಿತ್ಯತೃಪ್ತಾ = ಸದಾ ತೃಪ್ತಳು
567. ಭಕ್ತನಿಧಿಃ = ಭಕ್ತರಿಗೆ ನಿಧಿಯು(ಗಣಿಯು)
568. ನಿಯಂತ್ರೀ = (ಜೀವಿಗಳನ್ನು) ಸನ್ಮಾರ್ಗದಲ್ಲಿ ನಿಯಮಿಸುವವಳು
569. ನಿಖಿಲೇಶ್ವರೀ = ಎಲ್ಲಕ್ಕೂ ಒಡೆಯಳು
570. ಮೈತ್ರಾದಿವಾಸನಾಲಭ್ಯಾ = ಮೈತ್ರಿ (ಸ್ನೇಹ) ಮೊದಲಾದ ವಾಸನೆಗಳಿಂದ ಪಡೆಯಲು ಯೋಗ್ಯಳಾದವಳು
571. ಮಹಾಪ್ರಲಯಸಾಕ್ಷಿಣೀ = ಮಹಾಪ್ರಳಯಕ್ಕೆ ಸಾಕ್ಷಿರೂಪಳು
572. ಪರಾಶಕ್ತಿಃ = ಉತ್ಕೃಷ್ಟವಾದ ಶಕ್ತಿಯುಳ್ಳವಳು
573. ಪರಾನಿಷ್ಠಾ = ಉತ್ಕೃಷ್ಟವಾದ ಏಕಾಗ್ರತೆಯುಳ್ಳವಳು
574. ಪ್ರಜ್ಞಾನಘನರೂಪಿಣೀ = ಶುದ್ಧ ಜ್ಞಾನದ ಘನರೂಪಳು
575. ಮಾಧ್ವೀಪಾನಾಲಸಾ = (ಬೇರೆ ಆಸಕ್ತಿ ಇಲ್ಲದ) ಮದ್ಯಪಾನದಲ್ಲಿ ಆಸಕ್ತಳು
576. ಮತ್ತಾ = (ಅಮಲಿನಿಂದ) ಮದವೇರಿದವಳು
577. ಮಾತೃಕಾವರ್ಣರೂಪಿಣೀ = (ಅಕಾರಾದಿ, ಕ್ಷಕಾರಾಂತ) ಅಕ್ಷರ/ವರ್ಣ ಸ್ವರೂಪಳು
578. ಮಹಾಕೈಲಾಸನಿಲಯಾ = ಮಹಾಕೈಲಾಸದಲ್ಲಿ ನೆಲೆಸಿದವಳು
579. ಮೃಣಾಲಮೃದುದೋರ್ಲತಾ = ತಾವರೆಯ ದಂಟಿನಂತೆ ಮೃದುವಾದ ತಣ್ಣನೆಯ ಬಾಹುಗಳುಳ್ಳವಳು
580. ಮಹನೀಯಾ = ಗೌರವಾನ್ವಿತಳು
581. ದಯಾಮೂರ್ತಿಃ = ದಯಾಸ್ವರೂಪಳು
582. ಮಹಾಸಾಮ್ರಾಜ್ಯಶಾಲಿನೀ = ಮೂರು ಲೋಕಗಳ ಮಹಾಸಾಮ್ರಾಜ್ಯವನ್ನು ನಿಯಂತ್ರಿಸುವವಳು, ಚಕ್ರವರ್ತಿನಿ
583. ಆತ್ಮವಿದ್ಯಾ = ಆತ್ಮಜ್ಞಾನಸ್ವರೂಪಳು
584. ಮಹಾವಿದ್ಯಾ = ಮಹಾವಿದ್ಯಾ ಮಂತ್ರಸ್ವರೂಪಳು
585. ಶ್ರೀವಿದ್ಯಾ = ಪಂಚದಶೀ ಮಂತ್ರಸ್ವರೂಪಳು
586. ಕಾಮಸೇವಿತಾ = ಕಾಮದೇವನಿಂದ ಸೇವಿಸಲ್ಪಡುವವಳು/ಪೂಜಿತಳು
587. ಶ್ರೀಷೋಡಶಾಕ್ಷರೀ ವಿದ್ಯಾ = ಹದಿನಾರು ಅಕ್ಷರಗಳ ಮಂತ್ರ ಸ್ವರೂಪಳು
588. ತ್ರಿಕೂಟಾ = ವಾಗ್ಭವ, ಕಾಮರಾಜ, ಶಕ್ತಿಕೂಟ ಸ್ವರೂಪಳು
589. ಕಾಮಕೋಟಿಕಾ = ಕಾಮೇಶ್ವರನ ಅರ್ಧಾಂಗಿ, ಕಾಮಕೋಟಿ ಪೀಠನಿವಾಸಿನಿ
590. ಕಟಾಕ್ಷಕಿಂಕರೀಭೂತ ಕಮಲಾಕೋಟಿಸೇವಿತಾ = ಕಡೆಗಣ್ಣಿನ ನೋಟದಿಂದಲೇ ಸೇವೆ ಮಾಡಲು ಸಿದ್ಧರಾದ ಲಕ್ಷ್ಮಿಯರ ಸಮೂಹವುಳ್ಳವಳು
591. ಶಿರಃಸ್ಥಿತಾ = ಶಿರಸ್ಸಿನಲ್ಲಿ (ಸಹಸ್ರಾರದಲ್ಲಿ) ಇರುವವಳು
592. ಚಂದ್ರನಿಭಾ = ಚಂದ್ರನಂತ ಕಾಂತಿಯುಕ್ತಳು
593. ಭಾಲಸ್ಥಾ = ಹಣೆಯಲ್ಲಿ ಇರುವವಳು
594. ಇಂದ್ರಧನುಃಪ್ರಭಾ = ಇಂದ್ರಧನಸ್ಸಿನ (ಕಾಮನಬಿಲ್ಲಿನ) ಪ್ರಭೆಯುಳ್ಳವಳು
595. ಹೃದಯಸ್ಥಾ = ಹೃದಯದಲ್ಲಿ ಇರುವವಳು
596. ರವಿಪ್ರಖ್ಯಾ = ಸೂರ್ಯನಂತೆ ಪ್ರಕಾಶಮಾನಳು
597. ತ್ರಿಕೋಣಾಂತರದೀಪಿಕಾ = ತ್ರಿಕೋಣಮಧ್ಯದಲ್ಲಿ ಪ್ರಕಾಶಿಸುವವಳು
598. ದಾಕ್ಷಾಯಣೀ = ದಕ್ಷಪ್ರಜಾಪತಿಯ ಮಗಳು, ಸತೀದೇವಿ
599. ದೈತ್ಯಹಂತ್ರೀ = ದೈತ್ಯರನ್ನು ಸಂಹರಿಸುವವಳು
600. ದಕ್ಷಯಜ್ಞವಿನಾಶಿನೀ = ದಕ್ಷಬ್ರಹ್ಮನ ಯಜ್ಞವನ್ನು ನಾಶಮಾಡಿದವಳು
601. ದರಾಂದೋಲಿತದೀರ್ಘಾಕ್ಷೀ = ವಿಶಾಲವದ ಚಂಚಲ ಕಣ್ಣುಗಳುಳ್ಳವಳು
602. ದರಹಾಸೋಜ್ಜ್ವಲನ್ಮುಖೀ = ಮುಗುಳ್ನಗೆಯಿಂದ ಶೋಭಿಸುವ ಮುಖವುಳ್ಳವಳು
603. ಗುರುಮೂರ್ತಿಃ = ಬೃಹತ್/ಗುರು ಸ್ವರೂಪಳು
604. ಗುಣನಿಧಿಃ = ಗುಣಗಳ ಗಣಿಯು
605. ಗೋಮಾತಾ = ಸುರಭಿ/ಕಾಮಧೇನು ಸ್ವರೂಪಳು
606. ಗುಹಜನ್ಮಭೂಃ = ಷಣ್ಮುಖನ ತಾಯಿ
607. ದೇವೇಶೀ = ದೇವತೆಗಳ ಒಡತಿ/ರಕ್ಷಕಿ
608. ದಂಡನೀತಿಸ್ಥಾ = ನ್ಯಾಯಕ್ಕೆ/ಶಿಕ್ಷೆಗೆ ಅಧಿಕಾರಿಣಿ
609. ದಹರಾಕಾಶರೂಪಿಣೀ = ಹೃದಯದ ಸೂಕ್ಷ್ಮಾಕಾಶದಲ್ಲಿ ಇರುವವಳು
610. ಪ್ರತಿಪನ್ಮುಖ್ಯರಾಕಾಂತ ತಿಥಿಮಂಡಲಪೂಜಿತಾ = ಪಾಡ್ಯದಿಂದ ಪೂರ್ಣಿಮೆಯವರೆಗೆ ತಿಥಿದೇವತೆಗಳಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
611. ಕಲಾತ್ಮಿಕಾ = ಕಲಾ ಸ್ವರೂಪಳು, ಸಕಲ ಕಲೆಗಳ ಆತ್ಮರೂಪಳು
612. ಕಲಾನಾಥಾ = ಕಲೆಗಳಿಗೆ ಒಡತಿಯು
613. ಕಾವ್ಯಾಲಾಪವಿನೋದಿನೀ = ಕಾವ್ಯವಾಚನದಿಂದ ಸಂತೋಷಪಡುವವಳು
614. ಸಚಾಮರರಮಾವಾಣೀ ಸವ್ಯದಕ್ಷಿಣಸೇವಿತಾ = ಚಾಮರವನ್ನು ಹಿಡಿದ ಲಕ್ಷ್ಮೀ-ಸರಸ್ವತಿಯರಿಂದ ಎರಡೂ ಕಡೆಗಳಲ್ಲಿ ಸೇವಿಸಲ್ಪಡುವವಳು
615. ಆದಿಶಕ್ತಿಃ = ಆದಿಶಕ್ತಿಯು, ಜಗತ್ಸೃಷ್ಟಿಗೆ ಮೂಲ ಕಾರಣಳು
616. ಅಮೇಯಾ = ಅಳತೆಗೆ ಮೀರಿದವಳು, ಪರಿಮಾಣ ರಹಿತಳು
617. ಆತ್ಮಾ = ಆತ್ಮಸ್ವರೂಪಳು
618. ಪರಮಾ = ಸರ್ವೋತ್ಕೃಷ್ಟಳು
619. ಪಾವನಾಕೃತಿಃ = ಪವಿತ್ರವಾದ ಆಕಾರವುಳ್ಳವಳು
620. ಅನೇಕಕೋಟಿಬ್ರಹ್ಮಾಂಡಜನನೀ = ಕೋಟಿಗಟ್ಟಲೆ ಬ್ರಹ್ಮಾಂಡಗಳ ತಾಯಿ
621. ದಿವ್ಯವಿಗ್ರಹಾ = ಸುಂದರವಾದ ದೇಹವುಳ್ಳವಳು
622. ಕ್ಲೀಂಕಾರೀ = ಕ್ಲೀಂ (ಕಾಮಬೀಜ) ಸ್ವರೂಪಳು
623. ಕೇವಲಾ = ಸರ್ವಧರ್ಮ ರಹಿತಳು
624. ಗುಹ್ಯಾ = ಗುಟ್ಟಾಗಿರುವವಳು/ರಹಸ್ಯಮಯಳು
625. ಕೈವಲ್ಯಪದದಾಯಿನೀ = ಮೋಕ್ಷವನ್ನು ಕೊಡುವವಳು
626. ತ್ರಿಪುರಾ = ತ್ರಿಪುರಾ ಚಕ್ರದ ಒಡತಿ, ಬ್ರಹ್ಮ-ವಿಷ್ಣು-ಈಶ್ವರರಿಗಿಂತಲೂ ಪುರಾತನಳು, ಮೂರು ರೂಪದಲ್ಲಿ ವಾಸಿಸುವವಳು
627. ತ್ರಿಜಗದ್ವಂದ್ಯಾ = ಮೂರು ಲೋಕಗಳ ವಾಸಿಗಳಿಂದಲೂ ನಮಸ್ಕರಿಸಲ್ಪಡುವವಳು
628. ತ್ರಿಮೂರ್ತಿಃ = ಬ್ರಹ್ಮ-ವಿಷ್ಣು-ರುದ್ರ ರೂಪಳು
629. ತ್ರಿದಶೇಶ್ವರೀ = ದೇವತೆಗಳಿಗೆ ಒಡತಿ
630. ತ್ರ್ಯಕ್ಷರೀ = (ಅ ಉ ಮ್) ಮೂರು ಅಕ್ಷರದ ಮಂತ್ರ ಸ್ವರೂಪಳು
631. ದಿವ್ಯಗಂಧಾಢ್ಯಾ = ಸುಗಂಧದಿಂದ ಕೂಡಿದವಳು
632. ಸಿಂದೂರತಿಲಕಾಂಚಿತಾ = ಸಿಂದೂರದ ತಿಲಕದಿಂದ ಕೂಡಿದವಳು
633. ಉಮಾ = ಪಾರ್ವತಿ, ಶಿವನ ಪತ್ನಿ, ಓಂಕಾರ ರೂಪಳು
634. ಶೈಲೇಂದ್ರತನಯಾ = ಪರ್ವತರಾಜನ (ಹಿಮವಂತನ) ಮಗಳು
635. ಗೌರೀ = ಶುಭ್ರವರ್ಣದವಳು
636. ಗಂಧರ್ವಸೇವಿತಾ = ಗಂಧರ್ವರಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
637. ವಿಶ್ವಗರ್ಭಾ = ವಿಶ್ವವನ್ನು ಗರ್ಭದಲ್ಲಿ ಧರಿಸಿರುವವಳು
638. ಸ್ವರ್ಣಗರ್ಭಾ = ಸುವರ್ಣ ಸ್ವರೂಪಳು, ಜಗತ್ತಿನ ಸೃಷ್ಟಿಕಾರಣಳು
639. ಅವರದಾ = ದುಷ್ಟರನ್ನು ನಾಶಮಾಡುವವಳು
640. ವಾಗಧೀಶ್ವರೀ = ಮಾತಿಗೆ ಅಧಿದೇವತೆ
641. ಧ್ಯಾನಗಮ್ಯಾ = ಧ್ಯಾನದಿಂದ ಹೊಂದಲ್ಪಡುವವಳು
642. ಅಪರಿಚ್ಛೇದ್ಯಾ = ದೇಶ, ಕಾಲ, ವಸ್ತುಗಳಿಂದ ಬಿಡಿಸಲಾಗದವಳು, ಸೀಮಾತೀತಳು
643. ಜ್ಞಾನದಾ = (ಆತ್ಮ)ಜ್ಞಾನವನ್ನು ಕೊಡುವವಳು
644. ಜ್ಞಾನವಿಗ್ರಹಾ = ಜ್ಞಾನದ ಮೂರ್ತಿರೂಪಳು
645. ಸರ್ವವೇದಾಂತಸಂವೇದ್ಯಾ = ಎಲ್ಲಾ ಉಪನಿಷತ್ತುಗಳಿಂದಲೂ ತಿಳಿಯಲ್ಪಡುವವಳು
646. ಸತ್ಯಾನಂದಸ್ವರೂಪಿಣೀ = ಶಾಶ್ವತವಾದ ಸತ್ಯ-ಆನಂದದ ರೂಪಳು
647. ಲೋಪಾಮುದ್ರಾರ್ಚಿತಾ = (ಅಗಸ್ತ್ಯರ ಪತ್ನೀ) ಲೋಪಮುದ್ರಾಳಿಂದ ಪೂಜಿಸಲ್ಪಟ್ಟವಳು
648. ಲೀಲಾಕ್ಲೃಪ್ತ ಬ್ರಹ್ಮಾಂಡಮಂಡಲಾ = ಲೀಲೆಯಿಂದಲೇ ರಚಿಸಿದ ಬ್ರಹ್ಮಾಂಡ ಸಮೂಹವುಳ್ಳವಳು
649. ಅದೃಶ್ಯಾ = ಅಗೋಚರಳು
650. ದೃಶ್ಯರಹಿತಾ = ಬಾಹ್ಯವಸ್ತುಗಳನ್ನು ನೋಡದವಳು, ವಸ್ತುಗಳ ನಿಜಸ್ವರೂಪವನ್ನಲ್ಲದೆ ಬೇರೆಯನ್ನು ನೋಡದವಳು
651. ವಿಜ್ಞಾತ್ರೀ = ಸರ್ವಜ್ಞಳು, ಜಗತ್ಸತ್ಯವನ್ನು ಬಲ್ಲವಳು
652. ವೇದ್ಯವರ್ಜಿತಾ = ತಿಳಿಯಬೇಕಾದ ವಿಷಯವಿಲ್ಲದವಳು
653. ಯೋಗಿನೀ = ಯೋಗಶಕ್ತಿ ಉಳ್ಳವಳು, ಸದಾ ಶಿವನೊಂದಿಗೆ ಒಂದಾಗಿರುವವಳು (ಯೋಗದಲ್ಲಿ ನಿರತಳು)
654. ಯೋಗದಾ = ಯೋಗವನ್ನು ಉಪದೇಶಿಸುವವಳು, ಯೋಗಶಕ್ತಿಯನ್ನು ನೀಡುವವಳು
655. ಯೋಗ್ಯಾ = ಯೋಗದಿಂದ ಪಡೆಯಬಲ್ಲಳು, ಯೋಗದ ವಿಷಯವಾಗಿರುವವಳು (ಗುರಿ)
656. ಯೋಗಾನಂದಾ = ಯೋಗದಿಂದ ಉಂಟಾಗುವ ಆನಂದಸ್ವರೂಪಳು, ಯೋಗದಿಂದ ಆನಂದವನ್ನು ಪಡೆಯುವವಳು
657. ಯುಗಂಧರಾ = ಕೃತಾದಿ ಯುಗಗಳನ್ನು ಧರಿಸಿದವಳು
658. ಇಚ್ಛಾಶಕ್ತಿ ಜ್ಞಾನಶಕ್ತಿ ಕ್ರಿಯಾಶಕ್ತಿ ಸ್ವರೂಪಿಣೀ = ಇಚ್ಛಾ, ಜ್ಞಾನ, ಕ್ರಿಯಾಶಕ್ತಿ ಸ್ವರೂಪಳು
659. ಸರ್ವಾಧಾರಾ = ಎಲ್ಲಕ್ಕೂ ಆಧಾರವಾದವಳು
660. ಸುಪ್ರತಿಷ್ಠಾ = ಜಗತ್ತಿಗೆ ಅಧಿಷ್ಠಾನವಾದವಳು
661. ಸದಸದ್ರೂಪಧಾರಿಣೀ = ನಿತ್ಯ/ಅನಿತ್ಯ ವಸ್ತುಗಳ ರೂಪವನ್ನು ತಾಳಿದವಳು
662. ಅಷ್ಟಮೂರ್ತಿಃ = ಪೃಥಿವಿ ಮುಂತಾದ ಅಷ್ಟಮೂರ್ತಿ ಸ್ವರೂಪಳು
663. ಅಜಾಜೇತ್ರೀ = ಅವಿದ್ಯೆಯನ್ನು ನಾಶಪಡಿಸುವವಳು/ ಗೆಲ್ಲುವವಳು
664. ಲೋಕಯಾತ್ರವಿಧಾಯಿನೀ = ಲೋಕಗಳ ಸೃಷ್ಟಿ/ನಾಶಗಳನ್ನು ನಡೆಸುವವಳು
665. ಏಕಾಕಿನೀ = ಅದ್ವಿತೀಯಳು
666. ಭೂಮರೂಪಾ = ಬ್ರಹ್ಮಸ್ವರೂಪಳು
667. ನಿದ್ರ್ವೈತಾ = ಭಿನ್ನವಸ್ತು ಇಲ್ಲದವಳು
668. ದ್ವೈತವರ್ಜಿತಾ = ಭೇದ(ಕಲ್ಪನೆ) ಇಲ್ಲದವಳು
669. ಅನ್ನದಾ = ಅನ್ನವನ್ನು (ಆಹಾರವನ್ನು) ಕೊಡುವವಳು
670. ವಸುದಾ = ದ್ರವ್ಯವನ್ನು(ಐಶ್ವರ್ಯ) ಕೊಡುವವಳು
671. ವೃದ್ಧಾ = ಪುರಾತನಳು
672. ಬ್ರಹ್ಮಾತ್ಮೈಕ್ಯಸ್ವರೂಪಿಣೀ = ಜೀವ-ಆತ್ಮಗಳ ಐಕ್ಯಸ್ವರೂಪಳು
673. ಬೃಹತೀ = ಎಲ್ಲಕ್ಕೂ ದೊಡ್ಡವಳು
674. ಬ್ರಾಹ್ಮಣೀ = ಈಶ್ವರನ(ಬ್ರಾಹ್ಮಣನ) ಪತ್ನಿ
675. ಬ್ರಾಹ್ಮೀ = ವಾಗ್ದೇವಿ, ಬ್ರಹ್ಮನ ಭಾಗ
676. ಬ್ರಹ್ಮಾನಂದಾ = ಬ್ರಹ್ಮಾನಂದ ಸ್ವರೂಪಳು, ಬ್ರಹ್ಮಾನಂದದಲ್ಲೇ ಇರುವವಳು
677. ಬಲಿಪ್ರಿಯಾ = ಬಲಿಯನ್ನು/ಶಕ್ತಿವಂತರನ್ನು ಪ್ರೀತಿಸುವವಳು
678. ಭಾಷಾರೂಪಾ = ಭಾಷೆಗಳ ಸ್ವರೂಪಳು
679. ಬೃಹತ್ಸೇನಾ = ಅಪಾರ ಸೇನೆಯುಳ್ಳವಳು
680. ಭಾವಾಭಾವವಿವರ್ಜಿತಾ = ಇರುವಿಕೆ-ಇಲ್ಲದಿರುವಿಕೆಯನ್ನು ವರ್ಜಿಸಿದವಳು
681. ಸುಖಾರಾಧ್ಯಾ = ಸುಖವಾಗಿ/ಸುಲಭವಾಗಿ ಪೂಜಿಸಲ್ಪಡುವವಳು
682. ಶುಭಕರೀ = ಮಂಗಳವನ್ನು ಉಂಟುಮಾಡುವವಳು
683. ಶೋಭನಾಸುಲಭಾಗತಿಃ = ಸುಖಕರವೂ, ಸುಲಭವೂ ಆದ ಮೋಕ್ಷಸ್ವರೂಪಳು
684. ರಾಜರಾಜೇಶ್ವರೀ = ಮಹಾರಾಜರಿಗೂ ರಾಜ್ಞಿಯು
685. ರಾಜ್ಯದಾಯಿನೀ = ರಾಜ್ಯವನ್ನು ಕೊಡುವವಳು
686. ರಾಜ್ಯವಲ್ಲಭಾ = ರಾಜ್ಯದ ಒಡತಿಯು/ರಕ್ಷಕಿಯು
687. ರಾಜತ್ಕೃಪಾ = ಎಲ್ಲೆಡೆಯೂ/ಎಲ್ಲರಿಗೂ ಕೃಪೆತೋರಿ ರಾರಾಜಿಸುತ್ತಿರುವವಳು
688. ರಾಜಪೀಠನಿವೇಶಿತ ನಿಜಾಶ್ರಿತಾ = ರಾಜಪದವಿಯಲ್ಲಿ ತನ್ನನ್ನು ಆಶ್ರಯಿಸಿದವರನ್ನು ಕೂರಿಸುವವಳು
689. ರಾಜ್ಯಲಕ್ಷ್ಮೀ = ರಾಜ್ಯಾಭಿಮಾನಿ ಮಹಾಲಕ್ಷ್ಮಿಯು
690. ಕೋಶನಾಥಾ = ಕೋಶದ/ದ್ರವ್ಯಗಳ ಅಧಿಕಾರಿಣಿ
691. ಚತುರಂಗಬಲೇಶ್ವರೀ = ಚತುರಂಗ ಬಲದ ಒಡತಿ
692, ಸಾಮ್ರಾಜ್ಯದಾಯಿನೀ = ಚಕ್ರವರ್ತಿ ಪದವಿಯನ್ನು ಕೊಡುವವಳು
693. ಸತ್ಯಸಂಧಾ = ಸತ್ಯನಿಷ್ಠಳು
694. ಸಾಗರಮೇಖಲಾ = ಸಮುದ್ರವನ್ನೇ ಡಾಬುಆಗಿ(ಒಡ್ಯಾಣವಾಗಿ) ಉಳ್ಳವಳು
695. ದೀಕ್ಷಿತಾ = ಕರ್ಮನಿರತಳು
696. ದೈತ್ಯಶಮನೀ = ದೈತ್ಯರನ್ನು ಕೊಲ್ಲುವವಳು
697. ಸರ್ವಲೋಕವಶಂಕರೀ = ಎಲ್ಲಾ ಲೋಕಗಳನ್ನೂ ಅಧೀನದಲ್ಲಿ ಇಟ್ಟುಕೊಂಡಿರುವವಳು
698. ಸರ್ವಾರ್ಥದಾತ್ರೀ = ಇಷ್ಟಾರ್ಥವನ್ನು ನೆರವೇರಿಸುವವಳು (ಕೊಡುವವಳು)
699. ಸಾವಿತ್ರೀ = ಸೂರ್ಯಾಭಿಮಾನಿ ದೇವತೆ
700. ಸಚ್ಚಿದಾನಂದರೂಪಿಣೀ = ಸತ್ಯಜ್ಞಾನಾನಂದ ಸ್ವರೂಪಳು
701. ದೇಶಕಾಲಾಪರಿಚ್ಛಿನ್ನಾ = ದೇಶ-ಕಾಲಗಳಿಂದ ಅಳೆಯಲಾಗದವಳು
702. ಸರ್ವಗಾ = ಎಲ್ಲೆಡೆಯೂ ಇರುವವಳು
703. ಸರ್ವಮೋಹಿನೀ = ಎಲ್ಲರನ್ನೂ ಮೋಹಗೊಳಿಸುವವಳು
704. ಸರಸ್ವತೀ = ವಿದ್ಯಾಭಿಮಾನಿ ದೇವತೆ
705. ಶಾಸ್ತ್ರಮಯೀ = ಶಾಸ್ತ್ರಸ್ವರೂಪಳು
706. ಗುಹಾಂಬಾ = ಷಣ್ಮುಖನ ತಾಯಿ, ಹೃದಯದ ಗುಹೆಯಲ್ಲಿ ಇರುವವಳು
707. ಗುಹ್ಯರೂಪಿಣೀ = ರಹಸ್ಯರೂಪಳು
708. ಸರ್ವೋಪಾಧಿವಿನಿರ್ಮುಕ್ತಾ = ಎಲ್ಲಾ ಉಪಾಧಿಗಳಿಂದಲೂ ಮುಕ್ತಳು
709. ಸದಾಶಿವಪತಿವ್ರತಾ = ಸದಾಶಿವನಲ್ಲಿ ಪತಿನಿಷ್ಠೆಯುಳ್ಳವಳು
710. ಸಂಪ್ರದಾಯೇಶ್ವರೀ = ಸಂಪ್ರದಾಯಗಳಿಗೆ ಒಡತಿಯು
711. ಸಾಧು/ ಸಾಧ್ವೀ = ಸ್ವಸ್ಥಳು, ಸಮಚಿತ್ತತೆಯುಳ್ಳವಳು, ಯೋಗ್ಯಳು
712. ಈ = ಕಾಮಕಲಾ ಸ್ವರೂಪಳು
713. ಗುರುಮಂಡಲರೂಪಿಣೀ = ಪರಶಿವಾದಿ ಸ್ವಗುರು ಪರ್ಯಂತ ಗುರುಮಂಡಲ ಸ್ವರೂಪಳು, ಗುರು ಪರಂಪರೆಯನ್ನು ತನ್ನಲ್ಲೇ ಧರಿಸಿದವಳು
714. ಕುಲೋತ್ತೀರ್ಣಾ = ಇಂದ್ರಿಯಾತೀತಳು
715. ಭಗಾರಾಧ್ಯಾ = ಸೂರ್ಯಮಂಡಲದಲ್ಲಿ ಉಪಾಸಿಸಲ್ಪಡುವವಳು
716. ಮಾಯಾ = ಮಾಯಾಶಕ್ತಿಯು
717. ಮಧುಮತೀ = ಮಧುಮತೀ ದೇವತೆಯು, ಜೇನಿನಂತೆ ಮಧುರಪ್ರಕೃತಿಯುಳ್ಳವಳು,
718. ಮಹೀ = ಪೃಥ್ವೀಸ್ವರೂಪಳು
719. ಗಣಾಂಬಾ = ಗಣಪತಿಯ ತಾಯಿಯು, ಭೂತಗಣಗಳಿಗೆ ತಾಯಿಸ್ವರೂಪಳು
720. ಗುಹ್ಯಕಾರಾಧ್ಯಾ = ಗುಹ್ಯಕರಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
721. ಕೋಮಲಾಂಗೀ = ಮೃದುವಾದ ಅವಯವಗಳುಳ್ಳವಳು
722. ಗುರುಪ್ರಿಯಾ = ಗುರುಗಳಿಗೆ ಪ್ರಿಯಳು
723. ಸ್ವತಂತ್ರಾ = ಪರಾಧೀನತೆ ಇಲ್ಲದವಳು
724. ಸರ್ವತಂತ್ರೇಶೀ = ಎಲ್ಲಾ ತಂತ್ರಶಾಸ್ತ್ರಗಳಿಗೂ ಅಧ್ಕಾರಿಣಿಯು
725. ದಕ್ಷಿಣಾಮೂರ್ತಿರೂಪಿಣೀ = ದಕ್ಷಿಣಾಮೂರ್ತಿ ಸ್ವರೂಪಳು
726. ಸನಕಾದಿಸಮಾರಾಧ್ಯಾ = ಸನಕಾದಿಗಳಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
727. ಶಿವಜ್ಞಾನಪ್ರದಾಯಿನೀ = ಶಿವ-ಸಂಬಂಧಿತ ಜ್ಞಾನವನ್ನು ಕೊಡುವವಳು
728. ಚಿತ್ಕಲಾ = ಜ್ಞಾನದ/ಪರಬ್ರಹ್ಮನ ಅಂಶಳು
729. ಆನಂದಕಲಿಕಾ = (ಜೀವಿಗಳಲ್ಲಿ) ಆನಂದದ ಅಂಕುರಳು
730. ಪ್ರೇಮರೂಪಾ = ಪ್ರೀತಿಸ್ವರೂಪಳು
731. ಪ್ರಿಯಂಕರೀ = (ಭಕ್ತರಿಗೆ) ಒಳ್ಳೆಯದನ್ನು ಮಾಡುವವಳು
732. ನಾಮಪಾರಾಯಣಪ್ರೀತಾ = ನಾಮಕೀರ್ತನೆಯಿಂದ ಸಂತೋಷಪಡುವವಳು
733. ನಂದಿವಿದ್ಯಾ = ನಂದಿಯಿಂದ ಉಪಾಸಿಸಲ್ಪಟ್ಟ ಮಂತ್ರ ಸ್ವರೂಪಳು
734. ನಟೇಶ್ವರೀ = ನಟೇಶನ ಪತ್ನಿಯು
735. ಮಿಥ್ಯಾಜಗದಧಿಷ್ಠಾನಾ = ಮಿಥ್ಯವಾದ ಜಗತ್ತಿಗೆ ಅಧಿಷ್ಠಾನಳು
736. ಮುಕ್ತಿದಾ = ಮೋಕ್ಷವನ್ನು ಕೊಡುವವಳು
737. ಮುಕ್ತಿರೂಪಿಣೀ = ಮೋಕ್ಷಸ್ವರೂಪಳು
738. ಲಾಸ್ಯಪ್ರಿಯಾ = ನರ್ತನಪ್ರಿಯಳು
739. ಲಯಕರೀ = ಪ್ರಳಯಕಾರಕಳು
740. ಲಜ್ಜಾ = (ಜೀವಿಗಳಲ್ಲಿ) ನಾಚಿಕೆಯ ಸ್ವರೂಪಳು
741. ರಂಭಾದಿವಂದಿತಾ = ರಂಭಾ ಮುಂತಾದ ಅಪ್ಸರಸ್ತ್ರೀಯರಿಂದ ನಮಸ್ಕರಿಸಲ್ಪಡುವವಳು
742. ಭವದಾವಸುಧಾವೃಷ್ಟಿಃ = ಸಂಸಾರವೆಂಬ ಕಾಡ್ಗಿಚ್ಚಿಗೆ ಅಮೃತದ ಮಳೆಗರೆವವಳು
743. ಪಾಪಾರಣ್ಯದವಾನಲಾ = ಪಾಪವೆಂಬ ಕಾಡಿಗೆ ಕಾಡ್ಗಿಚ್ಚಿನಂತಿರುವವಳು
744. ದೌರ್ಭಾಗ್ಯತೂಲವಾತೂಲಾ = ದಾರಿದ್ರ್ಯವೆಂಬ ಹತ್ತಿಗೆ ಬಿರುಗಾಳಿಯಂತೆ ಇರುವವಳು
745. ಜರಾಧ್ವಾಂತರವಿಪ್ರಭಾ = ಮುಪ್ಪು ಎಂಬ ಕತ್ತಲೆಗೆ ಸೂರ್ಯಪ್ರಭೆಯತೆ ಇರುವವಳು
746. ಭಾಗ್ಯಾಬ್ಧಿಚಂದ್ರಿಕಾ = ಭಾಗ್ಯವೆಂಬ ಸಮುದ್ರಕ್ಕೆ ಬೆಳದಿಂಗಳಿನಂತಿರುವವಳು
747. ಭಕ್ತಚಿತ್ತಕೇಕಿಘನಾಘನಾ = ಭಕ್ತರ ಮನಸ್ಸೆಂಬ ನವಿಲಿಗೆ ಮೇಘದಂತಿರುವವಳು
748. ರೋಗಪರ್ವತದಂಭೋಲಿಃ = ರೋಗವೆಂಬ ಪರ್ವತಕ್ಕೆ ವಜ್ರಾಯುಧದಂತೆ ಇರುವವಳು
749. ಮೃತ್ಯುದಾರುಕುಠಾರಿಕಾ = ಮೃತ್ಯುವೆಂಬ ಮರಕ್ಕೆ ಕೊಡಲಿಯಂತಿರುವವಳು
750. ಮಹೇಶ್ವರೀ = ಮಹೇಶ್ವರನ ಪತ್ನಿ
751. ಮಹಾಕಾಲೀ = ಮಹಾಕಾಲನ ಪತ್ನಿ
752. ಮಹಾಗ್ರಾಸಾ = ಸಮೃದ್ಧ ಆಹಾರ(ಪೇಯ) ಉಳ್ಳವಳು
753. ಮಹಾಶನಾ = ಚರಾಚರರನ್ನೇ ನುಂಗುವವಳು, ಅತ್ಯಂತ ಹಸಿವಿರುವವಳು
754. ಅಪರ್ಣಾ = ಎಲೆಯನ್ನೂ ತಿನ್ನದೆ ತಪಸ್ಸು ಮಾಡಿದ ಪಾರ್ವತಿ
755. ಚಂಡಿಕಾ = ಮಹಾಕ್ರೋಧವುಳ್ಳವಳು
756. ಚಂಡಮುಂಡಾಸುರನುಷೂದಿನೀ = ಚಂಡ, ಮುಂಡ ಮುಂತಾದ ಅಸುರರನ್ನು ಕೊಂದವಳು
757. ಕ್ಷರಾಕ್ಷರಾತ್ಮಿಕಾ = ನಾಶವಾಗುವ ಮತ್ತು ನಾಶವಾಗದ ಆತ್ಮಸ್ವರೂಪಳು
758. ಸರ್ವಲೋಕೇಶೀ = ಸಮಸ್ತ ಲೋಕಗಳಿಗೂ ಒಡತಿಯು
759. ವಿಶ್ವಧಾರಿಣೀ = ವಿಶ್ವವನ್ನು ಧರಿಸಿದವಳು
760. ತ್ರಿವರ್ಗದಾತ್ರೀ = (ಧರ್ಮ-ಅರ್ಥ-ಕಾಮ ಈ) ಮೂರನ್ನೂ ಕೊಡುವವಳು
761. ಸುಭಗಾ = ಸೌಭಾಗ್ಯವತಿಯು
762. ತ್ರ್ಯಂಬಕಾ = ಮೂರು ಕಣ್ಣುಳ್ಳವಳು
763. ತ್ರಿಗುಣಾತ್ಮಿಕಾ = ಸತ್ತ್ವ-ರಜಸ್-ತಮೋ ಗುಣಗಳಿಂದ ಕೂಡಿದವಳು
764. ಸ್ವರ್ಗಾಪವರ್ಗದಾ = ಸ್ವರ್ಗ-ಮೋಕ್ಷಗಳನ್ನು ಕೊಡುವವಳು
765. ಶುದ್ಧಾ = ಪವಿತ್ರಳು
766. ಜಪಾಪುಷ್ಪನಿಭಾಕೃತಿಃ = ಕೆಂಪುದಾಸವಾಳದ ಬಣ್ಣವುಳ್ಳವಳು
767. ಓಜೋವತೀ = ತೇಜಸ್ಸು-ಬಲ ಉಳ್ಳವಳು
768. ದ್ಯುತಿಧರಾ = ಉಜ್ವಲ ಕಾಂತಿಯುಳ್ಳವಳು
769. ಯಜ್ಞರೂಪಾ = ಯಜ್ಞಸ್ವರೂಪಳು
770. ಪ್ರಿಯವ್ರತಾ = ವ್ರತಗಳಲ್ಲಿ ಪ್ರೀತಿಯುಳ್ಳವಳು
771. ದುರಾರಾಧ್ಯಾ = ಕಷ್ಟದಿಂದ ಆರಾಧಿಸಲ್ಪಡುವವಳು
772. ದುರಾಧರ್ಷಾ = ಸ್ವಾಧೀನಪಡಿಸಿಕೊಳ್ಳಲು ಕಷ್ಟಸಾಧ್ಯಳು
773. ಪಾಟಲೀಕುಸುಮಪ್ರಿಯಾ = ಪಾಟಲೀ ಪುಷ್ಪವನ್ನು ಪ್ರೀತಿಸುವವಳು
774. ಮಹತೀ = ಮಹತ್ತಾದವಳು, (ನಾರದ ವೀಣೆ) ಮಹತೀ ಸ್ವರೂಪಳು
775. ಮೇರುನಿಲಯಾ = ಮೇರು ಪರ್ವತಗಳಲ್ಲಿ ವಾಸಿಸುವವಳು
776. ಮಂದಾರಕುಸುಮಪ್ರಿಯಾ = ಮಂದಾರಪುಷ್ಪ ಪ್ರಿಯಳು
777. ವೀರಾರಾಧ್ಯಾ = (ಭಾವನಾ) ದಾಢ್ರ್ಯವಿರುವವರಿಂದ/ ವೀರರಿಂದ ಪೂಜಿಸಿದವಳು
778. ವಿರಾಡ್ರೂಪಾ = ಸ್ಥಾವರ-ಜಂಗಮಾತ್ಮಕ ವಿರಾಟ್ ರೂಪಳು
779. ವಿರಜಾ = ಕಾಮ-ಕ್ರೋಧವಿಲ್ಲದವಳು, ಪಾಪವಿಲ್ಲದವಳು
780. ವಿಶ್ವತೋಮುಖೀ = ಎಲ್ಲ ಕಡೆಗಳಲ್ಲಿಯೂ ಮುಖವುಳ್ಳವಳು
781. ಪ್ರತ್ಯಗ್ರೂಪಾ = ಅಂತರ್ಮುಖದಿಂದ ನೋಡಲು ಸಾಧ್ಯಳು
782. ಪರಾಕಾಶಾ = (ಬ್ರಹ್ಮನ ಅಭಿವ್ಯಕ್ತಿ ಸ್ಥಾನವಾದ) ಪರಾಕಾಶ ಸ್ವರೂಪಳು
783. ಪ್ರಾಣದಾ = ಪ್ರಾಣವನ್ನು ಕೊಡುವವಳು
784. ಪ್ರಾಣರೂಪಿಣೀ = ಪ್ರಾಣ ಸ್ವರೂಪಳು
785. ಮಾರ್ತಂಡಭೈರವಾರಾಧ್ಯಾ = ಮಾರ್ತಾಂಡ ಭೈರವನಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
786. ಮಂತ್ರಿಣೀನ್ಯಸ್ತರಾಜ್ಯಧೂಃ = ಶ್ಯಾಮಲೆಗೆ (ಮಂತ್ರಿಣಿಗೆ) ವಹಿಸಿರುವ ರಾಜ್ಯಭಾರವುಳ್ಳವಳು
787. ತ್ರಿಪುರೇಶೀ = ತ್ರಿಪುರದ ದೇವತೆ, ತ್ರಿಪುರಾ ಚಕ್ರದ ಒಡತಿ
788. ಜಯತ್ಸೇನಾ = ಜಯಿಸುವ ಸ್ವಭಾವವುಳ್ಳ ಸೇನೆಯುಳ್ಳವಳು
789. ನಿಸ್ತ್ರೈಗುಣ್ಯಾ = ಸತ್ವಾದಿ ಗುಣತ್ರಯ ರಹಿತಳು
790. ಪರಾಪರಾ = ಪರ-ಅಪರ, ಬಾಹ್ಯ-ಅಂತರ ಪೂಜಾ ಸ್ವರೂಪಳು
791. ಸತ್ಯಜ್ಞಾನಾನಂದರೂಪಾ = ಸತ್ಯ-ಜ್ಞಾನ-ಆನಂದ ರೂಪಳು
792. ಸಾಮರಸ್ಯ ಪರಾಯಣಾ = ಸಮಚಿತ್ತತೆಯಲ್ಲಿ, ಹೊಂದಾಣಿಕೆ ಮಾಡುವುದರಲ್ಲಿ ನಿರತಳು
793. ಕಪರ್ದಿನೀ = ಗಂಟುಗಳಿರುವ ಕೂದಲುಳ್ಳ ಕಪರ್ದಿಯ(ಶಿವನ) ಪತ್ನಿ
794. ಕಲಾಮಾಲಾ = ಅರವತ್ತನಾಲ್ಕು ಕಲೆಗಳನ್ನೇ ಮಾಲೆಯಾಗಿ ಉಳ್ಳವಳು
795. ಕಾಮಧುಕ್ = ಇಷ್ಟಾರ್ಥವನ್ನು ಕೊಡುವವಳು
796. ಕಾಮರೂಪಿಣೀ = ಇಷ್ಟಬಂದ ರೂಪವನ್ನು ಧರಿಸುವವಳು
797. ಕಲಾನಿಧಿಃ = ಕಲೆಗೆ ಆವಾಸಸ್ಥಾನಳು
798. ಕಾವ್ಯಕಲಾ = ಕಾವ್ಯ ಕಲಾಸ್ವರೂಪಳು
799. ರಸಜ್ಞಾ = (ನವ)ರಸಗಳನ್ನು ತಿಳಿದವಳು
800. ರಸಶೇವಧಿಃ = ರಸಗಳಿಗೆ ಗಣಿಯು
801. ಪುಷ್ಟಾ = ಪುಷ್ಟಿಸ್ವರೂಪಳು
802. ಪುರಾತನಾ = ಪ್ರಾಚೀನಳು
803. ಪೂಜ್ಯಾ = ಪೂಜಿಸಲ್ಪಡುವವಳು
804. ಪುಷ್ಕರಾ = ಪುಷ್ಕರವೆಂಬ ತೀರ್ಥಸ್ವರೂಪಳು, ಪೂರ್ಣಳು, ಎಲ್ಲರಿಗೂ ಪುಷ್ಟಿಯನ್ನು ನೀಡುವವಳು
805. ಪುಷ್ಕರೇಕ್ಷಣಾ = ಕಮಲದಂತೆ ಕಣ್ಣುಳ್ಳವಳು
806. ಪರಂಜ್ಯೋತಿಃ = ಉತ್ಕೃಷ್ಟ ಪ್ರಕಾಶಸ್ವರೂಪಳು
807. ಪರಂಧಾಮ = ಸರ್ವೋಚ್ಚ ನಿವಾಸಸ್ಥಾನ ಸ್ವರೂಪಳು
808. ಪರಮಾಣುಃ = ಅತ್ಯಂತ ಸೂಕ್ಷ್ಮಸ್ವರೂಪಳು
809. ಪರಾತ್ಪರಾ = ಉತ್ತಮಕ್ಕಿಂತಲೂ ಉತ್ತಮಳು
810. ಪಾಶಹಸ್ತಾ = ಕೈಯಲ್ಲಿ ಪಾಶವನ್ನು ಹಿಡಿದವಳು
811. ಪಾಶಹಂತ್ರೀ = (ಭವ) ಪಾಶವನ್ನು ನಾಶಮಾಡುವವಳು
812. ಪರಮಂತ್ರವಿಭೇದಿನೀ = ಅಭಿಚಾರಕವನ್ನು ನಾಶ ಮಾಡುವವಳು
813. ಮೂರ್ತಾ = ರೂಪಸಹಿತಳಾದ ಜಗತ್ ಸ್ವರೂಪಳು
814. ಅಮೂರ್ತಾ = ರೂಪರಹಿತವಾದ ಜಗತ್ ಸ್ವರೂಪಳು
815. ಅನಿತ್ಯತೃಪ್ತಾ = ಅನಿತ್ಯ ವಸ್ತುಗಳಿಂದ ತೃಪ್ತಿ ಹೊಂದುವವಳು
816. ಮುನಿಮಾನಸಹಂಸಿಕಾ = ಮುನಿಗಳ ಮಾನಸವೆಂಬ ಸರೋವರದಲ್ಲಿ ಹಂಸದಂತೆ ವಿಹರಿಸುವವಳು
817. ಸತ್ಯವ್ರತಾ = ಸತ್ಯವನ್ನೇ ವ್ರತವನ್ನಾಗಿ ಇಟ್ಟುಕೊಂಡವಳು
818. ಸತ್ಯರೂಪಾ = ಸತ್ಯ ಸ್ವರೂಪಳು
819. ಸರ್ವಾಂತರ್ಯಾಮಿನೀ = ಎಲ್ಲಾ ಕಡೆಯಲ್ಲಿಯೂ ಇರುವವಳು
820. ಸತೀ = ಪತಿವ್ರತೆಯು
821. ಬ್ರಹ್ಮಾಣೀ = ಸೃಷ್ಟಿಕರ್ತ ಬ್ರಹ್ಮನ ಹಿಂದಿರುವ ಶಕ್ತಿಯು, ಸರ್ವಕ್ಕೂ ಆಧಾರಳು.
822. ಬ್ರಹ್ಮ = ಪರಬ್ರಹ್ಮವಸ್ತು
823. ಜನನೀ = (ಜಗತ್ತಿಗೆ) ತಾಯಿ
824. ಬಹುರೂಪಾ = ಅನೇಕ ರೂಪವುಳ್ಳವಳು
825. ಬುಧಾರ್ಚಿತಾ = ಜ್ಞಾನಿಗಳಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
826. ಪ್ರಸವಿತ್ರೀ = ಚರಾಚರ ಜಗತ್ತನ್ನು ಪಡೆದವಳು, ಜಗತ್ತಿಗೆ ತಾಯಿ
827. ಪ್ರಚಂಡಾ = ಪ್ರಬಲಳು
828. ಆಜ್ಞಾ = ಅಪ್ಪಣೆಯ/ಆಜ್ಞೆಯ ಸ್ವರೂಪಳು
829. ಪ್ರತಿಷ್ಠಾ = ಸಮಸ್ತ ಜಗತ್ತಿಗೆ ಆಧಾರವಾದವಳು
830. ಪ್ರಕಟಾಕೃತಿಃ = ಎಲ್ಲರೂ ತಿಳಿಯಬಹುದಾದ ಆಕೃತಿಯುಳ್ಳವಳು
831. ಪ್ರಾಣೇಶ್ವರೀ = (ಪಂಚ) ಪ್ರಾಣಗಳಿಗೆ ಅಧಿಪತಿಯು,
832. ಪ್ರಾಣದಾತ್ರೀ = ಪ್ರಾಣವನ್ನು ಕೊಡುವವಳು
833. ಪಂಚಾಶತ್ಪೀಠರೂಪಿಣೀ = ಕಾಮರೂಪಾದಿ ಐವತ್ತು ಪೀಠಸ್ವರೂಪಳು
834. ವಿಶೃಂಖಲಾ = ಬಂಧನರಹಿತಳು
835. ವಿವಕ್ತಸ್ಥಾ = ನಿರ್ಜನ ಸ್ಥಳದಲ್ಲಿರುವವಳು
836. ವೀರಮಾತಾ = ಪರಾಕ್ರಮಿಗಳ ತಾಯಿಯು
837. ವಿಯತ್ಪ್ರಸೂಃ = ಆಕಾಶವನ್ನು ಹಡೆದವಳು
838. ಮುಕುಂದಾ = ವಿಷ್ಣುಸ್ವರೂಪಳು, ಮೋಕ್ಷಸ್ವರೂಪಳು
839. ಮುಕ್ತಿನಿಲಯಾ = ಮೋಕ್ಷವೇ ಸ್ಥಾನವಾಗಿ ಉಳ್ಳವಳು
840. ಮೂಲವಿಗ್ರಹರೂಪಿಣೀ = ಪ್ರಧಾನ ರೂಪಿಣಿಯು
841. ಭಾವಜ್ಞಾ = ಯೋಚನೆ/ಭಾವನೆಗಳನ್ನು ತಿಳಿದವಳು
842. ಭವರೋಗಘ್ನೀ = ಸಂಸಾರ ವ್ಯಾಧಿಯನ್ನು ಹೋಗಲಾಡಿಸುವವಳು
843. ಭವಚಕ್ರಪ್ರವರ್ತಿನೀ = ಸಂಸಾರ ಚಕ್ರವನ್ನು ನಡೆಸುವವಳು
844. ಛಂದಃಸಾರಾ = ವೇದದ ಸಾರಸ್ವರೂಪಳು
845. ಶಾಸ್ತ್ರಸಾರಾ = ಶಾಸ್ತ್ರದ ಸಾರಸ್ವರೂಪಳು
846. ಮಂತ್ರಸಾರಾ = ಮಂತ್ರಶಾಸ್ತ್ರದ ಸಾರಸ್ವರೂಪಳು
847. ತಲೋದರೀ = ಸಮತಟ್ಟಾದ/ನೀಳವಾದ ಉದರವುಳ್ಳವಳು
848. ಉದಾರಕೀರ್ತಿಃ = ವ್ಯಾಪಿಸಿದ ಕೀರ್ತಿಯುಳ್ಳವಳು
849. ಉದ್ದಾಮವೈಭವಾ = ಅಳತೆ ಮೀರಿದ ಸಂಪತ್ತುಳ್ಳವಳು
850. ವರ್ಣರೂಪಿಣೀ = ಅಕ್ಷರ ಸ್ವರೂಪಳು
851. ಜನ್ಮ-ಮೃತ್ಯು-ಜರಾ ತಪ್ತಜನ ವಿಶ್ರಾಂತಿದಾಯಿನೀ = ಹುಟ್ಟು-ಸಾವು-ಮುಪ್ಪುಗಳಿಂದ ಕಂಗೆಟ್ಟಿರುವವರಿಗೆ ವಿಶ್ರಾಂತಿಯನ್ನು ಕೊಡುವವಳು
852. ಸರ್ವೋಪನಿಷದುದ್ಘುಷ್ಟಾ = ಎಲ್ಲಾ ಉಪನಿಷತ್ತಿಗಳಲ್ಲೂ ಹೊಗಳಲ್ಪಟ್ಟವಳು
853. ಶಾಂತ್ಯತೀತಕಲಾತ್ಮಿಕಾ = ಶಾಂತ್ಯತೀತ ಎಂಬ ಕಲಾಸ್ವರೂಪಳು
854. ಗಂಭೀರಾ = ಗಾಂಭೀರ್ಯವುಳ್ಳವಳು
855. ಗಗನಾಂತಸ್ಥಾ = ಆಕಾಶದ ಅಂತರ್ಭಾಗದಲ್ಲಿ ಇರುವವಳು
856. ಗರ್ವಿತಾ = ಗರ್ವಿತಳು
857, ಗಾನಲೋಲುಪಾ = ಸಂಗೀತದಲ್ಲಿ ಅತ್ಯಾಸಕ್ತಳು
858. ಕಲ್ಪನಾರಹಿತಾ = ಕಲ್ಪಿಸಿಕೊಳ್ಳಲು ಅಸಾಧ್ಯಳು
859. ಕಾಷ್ಠಾ = ದಿಕ್ ಸ್ವರೂಪಳು, ಅತ್ಯುನ್ನತ ಸ್ಥಾನದಲ್ಲಿ ಇರುವವಳು
860. ಅಕಾಂತಾ = ಪಾಪ-ದುಃಖಗಳನ್ನು ನಾಶ ಮಾಡುವವಳು
861. ಕಾಂತಾರ್ಧವಿಗ್ರಹಾ = ಕಾಂತನ(ಶಿವನ) ಅರ್ಧದೇಹವನ್ನು ಪಡೆದವಳು
862. ಕಾರ್ಯಕಾರಣನಿರ್ಮುಕ್ತಾ = ಕಾರ್ಯ-ಕಾರಣಗಳಿಗೆ ಅತೀತಳು
863. ಕಾಮಕೇಲಿತರಂಗಿತಾ = ಕಾಮೇಶ್ವರನ ಲೀಲಾಪರಂಪರಎಯ ತರಂಗಗಳನ್ನು ಉಂಟುಮಾಡುವವಳು
864. ಕನತ್ಕನಕತಾಟಂಕಾ = ಹೊಳೆಯುತ್ತಿರುವ ಚಿನ್ನದ ಓಲೆಯುಳ್ಳವಳು
865. ಲೀಲಾವಿಗ್ರಹಧಾರಿಣೀ = ಅನಾಯಾಸವಾಗಿ ಅವತಾರಗಳನ್ನು ಧರಿಸುವವಳು
866. ಅಜಾ = ಹುಟ್ಟಿಲ್ಲದವಳು
867. ಕ್ಷಯವಿನಿರ್ಮುಕ್ತಾ = ಸಾವಿಲ್ಲದವಳು
868. ಮುಗ್ಧಾ = ಸುಂದರಿ
869. ಕ್ಷಿಪ್ರಪ್ರಸಾದಿನೀ = ಶೀಘ್ರವಾಗಿ ಅನುಗ್ರಹಿಸುವವಳು
870. ಅಂತರ್ಮುಖಸಮಾರಾಧ್ಯಾ = ಅಂತರ್ಮುಖಿಗಳಿಂದ ಸುಲಭವಾಗಿ ಆರಾಧಿಸಲ್ಪಡುವವಳು
871. ಬಹಿರ್ಮುಖಸುದುರ್ಲಭಾ = ಬಹಿರ್ಮುಖಿಗಳಿಂದ ಕಷ್ಟದಿಂದ ಆರಾಧಿಸಲ್ಪಡುವವಳು
872. ತ್ರಯೀ = ಋಗ್-ಯಜುಸ್-ಸಾಮ ವೇದರೂಪಳು
873. ತ್ರಿವರ್ಗನಿಲಯಾ = ಧರ್ಮ-ಅರ್ಥ-ಕಾಮಗಳಲ್ಲಿ ನೆಲೆಸುವವಳು
874. ತ್ರಿಸ್ಥಾ = ಅಗ್ನಿ-ವೇದ-ಲೋಕ-ಗುಣ ಇತ್ಯಾದಿ ಮೂರರ ಸಮೂಹಗಳಲ್ಲಿ ಇರುವವಳು
875. ತ್ರಿಪುರಮಾಲಿನೀ = ಶ್ರೀಚಕ್ರದ ಅಂತರ್ದಶಾರಚಕ್ರದ ಈಶ್ವರಿಯು
876. ನಿರಾಮಯಾ = ರೋಗಾದಿ ಉಪದ್ರವ ರಹಿತಳು
877. ನಿರಾಲಂಬಾ = ಯಾವ ಅವಲಂಬನೆಯೂ ಇಲ್ಲದವಳು
878. ಸ್ವಾತ್ಮಾರಾಮಾ = ತನ್ನಲ್ಲಿಯೇ ತಾನೇ ಪೀಡಿಸುವವಳು
879. ಸುಧಾಸೃತಿಃ/ಸುಧಾಸ್ರುತಿಃ = ಅಮೃತಧಾರಸ್ವರೂಪಳು
880. ಸಂಸಾರಪಂಕನಿರ್ಮಗ್ನ ಸಮುದ್ಧರಣಪಂಡಿತಾ = ಸಂಸಾರವೆಂಬ ಕೆಸರಿನಲ್ಲಿ ಹೂತುಕೊಂದವರನ್ನು ಎತ್ತುವುದರಲ್ಲಿ ಸಮರ್ಥಳು
881. ಯಜ್ಞಪ್ರಿಯಾ = ಯಜ್ಞದಲ್ಲಿ ಪ್ರೀತಿವುಳ್ಳವಳು
882. ಯಜ್ಞಕತ್ರ್ರೀ = ಯಜ್ಞವನ್ನು ಮಾಡುವವಳು
883. ಯಜಮಾನಸ್ವರೂಪಿಣೀ = ಯಜಮಾನ ಸ್ವರೂಪಳು
884. ಧರ್ಮಾಧಾರಾ = ಧರ್ಮಕ್ಕೆ ಆಧಾರಸ್ವರೂಪಳು
885. ಧನಾಧ್ಯಕ್ಷಾ = ಧನಕ್ಕೆ ಅಧಿಕಾರಿಣಿಯು
886. ಧನಧಾನ್ಯವಿವರ್ಧಿನೀ = ಧನ-ಧಾನ್ಯಗಳನ್ನು ಅಭಿವೃದ್ಧಿಪಡಿಸುವವಳು
887. ವಿಪ್ರಪ್ರಿಯಾ = ವಿಪ್ರರಿಗೆ ಪ್ರಿಯಳು, ವಿಪ್ರರನ್ನು ಪ್ರೀತಿಸುವವಳು
888. ವಿಪ್ರರೂಪಾ = ವಿಪ್ರಸ್ವರೂಪಳು
889. ವಿಶ್ವಭ್ರಮಣಕಾರಿಣೀ = ಪ್ರಪಂಚ ನಡೆಯಲು ಕಾರಣಳು
890. ವಿಶ್ವಗ್ರಾಸಾ = ಪ್ರಪಂಚವನ್ನು ಕಬಳಿಸುವವಳು
891. ವಿದ್ರುಮಾಭಾ = ಹವಳದ ಬಣ್ಣವುಳ್ಳವಳು
892. ವೈಷ್ಣವೀ = ವಿಷ್ಣುಮಾಯಾರೂಪಳು
893. ವಿಷ್ಣುರೂಪಿಣೀ = ವಿಶ್ವವನ್ನು ವ್ಯಾಪಿಸುವ ವಿಷ್ಣುರೂಪಳು
894. ಅಯೋನಿಃ = ಉತ್ಪನ್ನ ರಹಿತಳು
895. ಯೋನಿನಿಲಯಾ = ಎಲ್ಲದರ ಕಾರಣ/ಮೂಲಳು
896. ಕೂಟಸ್ಥಾ = ವಾಗ್ಭವ-ಕಾಮರಾಜ-ಶಕ್ತಿ ಕೂಟದಲ್ಲಿ ಅಡಿಗಲ್ಲಿನಂತೆ ಸ್ಥಿರವಾಗಿರುವವಳು
897. ಕುಲರೂಪಿಣೀ = ಆಚಾರಸ್ವರೂಪಳು, ಕೌಲ ಮತದ ದೇವತೆ
898. ವೀರಗೋಷ್ಠೀಪ್ರಿಯಾ = ವೀರರ ಸಮೂಹವನ್ನು ಪ್ರೀತಿಸುವವಳು
899. ವೀರಾ = ಸ್ವತಃ ಸಮರ್ಥಳು/ವೀರಳು
900. ನೈಷ್ಕಮ್ರ್ಯಾ = ಕರ್ಮಲೇಪವಿಲ್ಲದವಳು
901. ನಾದರೂಪಿಣೀ = (ಓಂಕಾರದ ತುದಿಯಲ್ಲಿರುವ) ನಾದ ಸ್ವರೂಪಳು
902. ವಿಜ್ಞಾನಕಲನಾ = ಜ್ಞಾನದಿಂದಲೇ ತಿಳಿದುಕೊಳ್ಳಬೇಕಾದವಳು
903. ಕಲ್ಯಾ = ಉಷಃಕಾಲ ಸ್ವರೂಪಳು
904. ವಿದಗ್ಧಾ = ಚತುರಳು, ಎಲ್ಲದರಲ್ಲೂ ನಿಪುಣಳು
905. ಬೈಂದವಾಸನಾ = ಭ್ರೂಮಧ್ಯದಲ್ಲಿರುವ ಬೈಂದವ ಚಕ್ರವನ್ನೇ ಆಸನವನ್ನಾಗಿ ಉಳ್ಳವಳು
906. ತತ್ತ್ವಾಧಿಕಾ = ತತ್ತ್ವಗಳನ್ನು ಮೀರಿದವಳು
907. ತತ್ತ್ವಮಯೀ = ತತ್ತ್ವ ಸ್ವರೂಪಿಣಿ
908. ತತ್ತ್ವಮರ್ಥಸ್ವರೂಪಿಣೀ = ತತ್, ತ್ವಂ ಎಂಬುದರ ಅರ್ಥ ಸ್ವರೂಪಳು
909. ಸಾಮಗಾನಪ್ರಿಯಾ = ಸಾಮಗಾನವನ್ನು ಪ್ರೀತಿಸುವವಳು
910. ಸೌಮ್ಯಾ = (ಚಂದ್ರನಂತೆ) ಶೀತಲವನ್ನು/ಆನಂದವನ್ನು ಉಂಟುಮಾಡುವವಳು
911. ಸದಾಶಿವಕುಟುಂಬಿನೀ = ಸದಾಶಿವನ ಪತ್ನಿಯು
912. ಸವ್ಯಾಪಸವ್ಯಮಾರ್ಗಸ್ಥಾ = ಸವ್ಯ-ಅಪಸವ್ಯ ಮಾರ್ಗಗಳೆರಡರಲ್ಲೂ ಇರುವವಳು/ಪಡೆಯಬಲ್ಲಳು
913. ಸರ್ವಾಪದ್ವಿನಿವಾರಿಣೀ = ಎಲ್ಲಾ ಆಪತ್ತುಗಳನ್ನೂ ನಿವಾರಿಸುವವಳು
914. ಸ್ವಸ್ಥಾ = ಚಾಂಚಲ್ಯವಿಲ್ಲದವಳು
915. ಸ್ವಭಾವಮಧುರಾ = ಸ್ವಾಭಾವಿಕವಾಗಿ ಮಧುರ ಸ್ವಭಾವವುಳ್ಳವಳು
916. ಧೀರಾ = ಪಂಡಿತಳು, ಪಾಂಡಿತ್ಯವನ್ನು ನೀಡುವವಳು
917. ಧೀರಸಮರ್ಚಿತಾ = ಪಂಡಿತರಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
918. ಚೈತನ್ಯಾಘ್ರ್ಯ ಸಮಾರಾಧ್ಯಾ = ಜ್ಞಾನವೆಂಬ ಅಘ್ರ್ಯದಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
919. ಚೈತನ್ಯಕುಸುಮಪ್ರಿಯಾ = ಜ್ಞಾನವೆಂಬ ಪುಷ್ಪವನ್ನು ಪ್ರೀತಿಸುವವಳು
920. ಸದೋದಿತಾ = ಯಾವಾಗಲೂ ಉದಿಸಿಯೇ ಇರುವವಳು
921. ಸದಾತುಷ್ಟಾ = ಸದಾ ಸಂತುಷ್ಟಳಾಗಿರುವವಳು
922. ತರುಣಾದಿತ್ಯಪಾಟಲಾ = ಮಧ್ಯಾಹ್ನದ ಸೂರ್ಯನಂತೆ ಕೆಂಪು-ಬಿಳುಪು ಮಿಶ್ರ ಬಣ್ಣವುಳ್ಳವಳು
923. ದಕ್ಷಿಣಾದಕ್ಷಿಣಾರಾಧ್ಯಾ = ವಾಮಮಾರ್ಗ-ದಕ್ಷಿಣ (ಎಡ-ಬಲ) ಮಾರ್ಗಗಳಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು
924. ದರಸ್ಮೇರಮುಖಾಂಬುಜಾ = ಮುಗುಳ್ನಗೆಯಿಂದ ಕೂಡಿದ ಮುಖವುಳ್ಳವಳು
925. ಕೌಲಿನೀ ಕೇವಲಾ = (ಕೌಲ ಮತದವರಿಂದ) ಪರಂಪರೆಯ ಕ್ರಮದಿಂದಲೇ, ಜ್ಞಾನರೂಪದಿಂದಲೇ ಪೂಜಿಸಲ್ಪಡುವವಳು
926. ಅನಘ್ರ್ಯಕೈವಲ್ಯ ಪದದಾಯಿನೀ = ಅಮೂಲ್ಯವಾದ ಮುಕ್ತಿ ಪದವಿಯನ್ನು ಕೊಡುವವಳು
927. ಸ್ತೋತ್ರಪ್ರಿಯಾ = ಸ್ತೋತ್ರಗಳಿಂದ ಪ್ರೀತಳಾಗುವವಳು
928. ಸ್ತುತಿಮತೀ = ಸ್ತುತಿಸಲು ಯೋಗ್ಯಳು
929. ಶ್ರುತಿಸಂಸ್ತುತ ವೈಭವಾ = ವೇದಗಳಿಂದ ಸ್ತುತಿಸಲ್ಪಟ್ಟ ಚರಿತ್ರೆಯುಳ್ಳವಳು
930. ಮನಸ್ವಿನೀ = ಮನಸ್ಸನ್ನು ಅಧೀನದಲ್ಲಿ ಇಟ್ಟುಕೊಂಡಿರುವವಳು, ಸ್ಥಿರ ಮನಸ್ಸಿನವಳು
931. ಮಾನವತೀ = ಗೌರವವುಳ್ಳವಳು
932. ಮಹೇಶೀ = ಈಶ್ವರನ ಪತ್ನಿಯು
933. ಮಂಗಲಾಕೃತಿಃ = ಪವಿತ್ರವಾದ ಆಕೃತಿಯುಳ್ಳವಳು
934. ವಿಶ್ವಮಾತಾ = ಜಗತ್ತಿಗೆ ಜನನಿಯು
935. ಜಗದ್ಧಾತ್ರೀ = ಜಗತ್ತನ್ನು ಧರಿಸಿದವಳು/ರಕ್ಷಿಸುವವಳು
936. ವಿಶಾಲಾಕ್ಷೀ = ವಿಶಾಲವಾದ ಕಣ್ಣುಗಳುಳ್ಳವಳು
937. ವಿರಾಗಿಣೀ = ವೈರಾಗ್ಯಸಂಪನ್ನಳು
938. ಪ್ರಗಲ್ಭಾ = ಪ್ರೌಢಿಮೆಯುಳ್ಳವಳು, ನೈಪುಣ್ಯ-ದಿಟ್ಟತನವುಳ್ಳವಳು
939. ಪರಮೋದಾರಾ = ಪರಮ ಉದಾರಳು
940. ಪರಾಮೋದಾ = ಪರಮ ಸಂತುಷ್ಟಳು
941. ಮನೋಮಯೀ = ಮನಸ್ಸಿನ ಸ್ವರೂಪಳು
942. ವ್ಯೋಮಕೇಶೀ = ವ್ಯೋಮಕೇಶನ ಪತ್ನಿ, ಆಕಾಶವನ್ನೇ ಕೂದಲಾಗಿ ಉಳ್ಳವಳು
943. ವಿಮಾನಸ್ಥಾ = ವಿಮಾನದಲ್ಲಿರುವವಳು, ವಿಮಾನದಲ್ಲಿ ಇದ್ದು ದೇವತೆಗಳೊಂದಿಗೆ ಸಂಚರಿಸುವವಳು
944. ವಜ್ರಿಣೀ = ವಜ್ರಾಯುಧವನ್ನು ಧರಿಸಿರುವವಳು
945. ವಾಮಕೇಶ್ವರೀ = ವಾಮಕೇಶ್ವರ ತಂತ್ರಸ್ವರೂಪಳು
946. ಪಂಚಯಜ್ಞಪ್ರಿಯಾ = ಅಗ್ನಿಹೋತ್ರ, ದರ್ಶಪೂರ್ಣಮಾಸ, ಚಾತುರ್ಮಾಸ್ಯ, ಗೋಯಜ್ಞ, ಸೋಮಯಜ್ಞ ಈ ಪಂಚಯಜ್ಞಗಳನ್ನು ಪ್ರೀತಿಸುವವಳು, ದೇವಯಜ್ಞಾದಿ ಪಂಚಯಜ್ಞ ಪ್ರಿಯಳು
947. ಪಂಚಪ್ರೇತಮಂಚಾಧಿಶಾಯಿನೀ = ಬ್ರಹ್ಮಾದಿ ಪಂಚ ಪ್ರೇತಗಳ ಮಂಚದಲ್ಲಿ ಮಲಗಿರುವವಳು
948. ಪಂಚಮೀ = (ಬ್ರಹ್ಮ, ವಿಷ್ಣು, ರುದ್ರ, ಈಶ್ವರನ ನಂತರದ) ಐದನೆಯವಳು, ವಾರಾಹೀ ಸ್ವರೂಪಿಣಿಯು
949. ಪಂಚಭೂತೇಶೀ = ಪೃಥಿವ್ಯಾದಿ ಪಂಚಭೂತಗಳಿಗೆ ಒಡತಿ
950. ಪಂಚಸಂಖ್ಯೋಪಚಾರಿಣೀ = ಗಂಧಾದಿ ಪಂಚೋಪಚಾರ ಉಳ್ಳವಳು
951. ಶಾಶ್ವತೀ = ನಾಶರಹಿತಳು
952. ಶಾಶ್ವತೈಶ್ವರ್ಯಾ = ನಾಶರಹಿತ ಐಶ್ವರ್ಯವುಳ್ಳವಳು
953. ಶರ್ಮದಾ = ಸುಖವನ್ನು ಕೊಡುವವಳು
954. ಶಂಭುಮೋಹಿನೀ = ಶಂಭು/ಶಿವನನ್ನು ಮೋಹಪರವಶನನ್ನಾಗಿ ಮಾಡಿದವಳು
955. ಧರಾ = (ಲೋಕವನ್ನು) ಧರಿಸಿದವಳು
956. ಧರಸುತಾ = ಹಿಮವಂತನ ಮಗಳು
957. ಧನ್ಯಾ = ಕೃತಾರ್ಥಳು, ಮಹತ್ತಾದ ಐಶ್ವರ್ಯವನ್ನು ಹೊಂದಿದವಳು
958. ಧರ್ಮಿಣೀ = ಧರ್ಮಸ್ವಭಾವ ಉಳ್ಳವಳು
959. ಧರ್ಮವರ್ಧಿನೀ = ಧರ್ಮವನ್ನು ಅಭಿವೃದ್ಧಿಗೊಳಿಸುವವಳು
960. ಲೋಕಾತೀತಾ = ಲೋಕಗಳಿಗೆ ಅತೀತಳು
961. ಗುಣಾತೀತಾ = (ತ್ರಿ)ಗುಣಗಳಿಗೆ ಅತೀತಳು
962. ಸರ್ವಾತೀತಾ = ಎಲ್ಲಕ್ಕೂ ಅತೀತಳು
963. ಶಮಾತ್ಮಿಕಾ = ಸುಖಸ್ವರೂಪಳು
964. ಬಂಧೂಕಕುಸುಮಪ್ರಖ್ಯಾ = ಕೆಂಪು ದಾಸವಾಳದಂತೆ ಸೌಂದರ್ಯ/ಬಣ್ಣವುಳ್ಳವಳು
965. ಬಾಲಾ = ಬಾಲಾಂಬಿಕೆ, ಮಗುವಿನಂತೆ ಪ್ರಕೃತಿಯುಳ್ಳವಳು
966. ಲೀಲಾವಿನೋದಿನೀ = (ತನ್ನ) ಲೀಲೆಯಿಂದಲೇ ಆನಂದಪಡುವವಳು
967. ಸುಮಂಗಲೀ = ಸದಾ ಸೌಮಂಗಲ್ಯ ಉಳ್ಳವಳು (ವಿಧವೆ ಆಗದವಳು)
968. ಸುಖಕರೀ = ಸುಖವನ್ನು ಉಂಟುಮಾಡುವವಳು
969. ಸುವೇಷಾಢ್ಯಾ = ಒಳ್ಳೆಯ ವೇಷವನ್ನು ಧರಿಸಿರುವವಳು
970. ಸುವಾಸಿನೀ = (ಸದಾ) ಮುತ್ತೈದೆಯು
971. ಸುವಾಸಿನ್ಯರ್ಚನಪ್ರೀತಾ = ಮುತ್ತೈದೆಯರ ಪೂಜೆಯಿಂದ ತೃಪ್ತಿಪಡುವವಳು
972. ಆಶೋಭನಾ = ಪರಿಪೂರ್ಣ ಸುಂದರಿಯು
973. ಶುದ್ಧಮಾನಸಾ = ಪರಿಶುದ್ಧ ಮನಸ್ಸಿನವಳು, (ಭಕ್ತರ) ಮನಸ್ಸನ್ನು ಶುದ್ಧೀಕರಿಸುವವಳು
974. ಬಿಂದುತರ್ಪಣ ಸಂತುಷ್ಟಾ = (ಶ್ರೀಚಕ್ರದ) ಬಿಂದುವಿನಲ್ಲಿ ತರ್ಪಣವನ್ನು ಕೊಡುವುದರಿಂದ ಸಂತುಷ್ಟಳು
975. ಪೂರ್ವಜಾ = ಎಲ್ಲಕ್ಕಿಂತಲೂ ಮೊದಲು ಹುಟ್ಟಿದವಳು
976. ತ್ರಿಪುರಾಂಬಿಕಾ = ಶ್ರೀಚಕ್ರದ ಎಂಟನೆಯ ಚಕ್ರದ ಈಶ್ವರಿ, ತ್ರಿಪುರದ ತಾಯಿ
977. ದಶಮುದ್ರಾ ಸಮಾರಾಧ್ಯಾ = ಹತ್ತು ವಿಧ ಮುದ್ರೆಗಳಿಂದ ಪೂಜಿಸಲ್ಪಡುವವಳು (ಸರ್ವಸಂಕ್ಷೋಭಿಣೀ, ಸರ್ವವಿದ್ರಾವಿಣೀ, ಸರ್ವಾಕರ್ಷಿಣೀ, ಸರ್ವವಶಂಕರೀ, ಸರ್ವೋನ್ಮಾದಿನೀ, ಸರ್ವಮಹಾಂಕುಶ, ಸರ್ವಖೇಚರೀ, ಸರ್ವಬೀಜ, ಸರ್ವಯೋನಿ, ಸರ್ವತ್ರಿಖಂಡಾ)
978. ತ್ರಿಪುರಾಶ್ರೀವಶಂಕರೀ = ಶ್ರೀಚಕ್ರದ ಐದನೇ ಚಕ್ರದ ಚಕ್ರೇಶ್ವರಿಯನ್ನು (ತ್ರಿಪುರಾಶ್ರೀಯನ್ನು) ಅಧೀನದಲ್ಲಿಟ್ಟುಕೊಂಡವಳು
979. ಜ್ಞಾನಮುದ್ರಾ = ಜ್ಞಾನಮುದ್ರಾ ಸ್ವರೂಪಳು
980. ಜ್ಞಾನಗಮ್ಯಾ = ಜ್ಞಾನದಿಂದ ತಿಳಿಯಲ್ಪಡುವವಳು
981. ಜ್ಞಾನಜ್ಞೇಯಸ್ವರೂಪಿಣೀ = ತಿಳುವಳಿಕೆ (ಜ್ಞಾನ) ಮತ್ತು ತಿಳಿಯಬೇಕಾದ್ದರ (ಜ್ಞೇಯದ) ಸ್ವರೂಪಳು
982. ಯೋನಿಮುದ್ರಾ = (ಒಂಬತ್ತನೆಯ ಮುದ್ರೆ) ಯೋನಿಮುದ್ರಾ ಸ್ವರೂಪಳು
983. ತ್ರಿಖಂಡೇಶೀ = (ಹತ್ತನೆಯ ಮುದ್ರೆ) ತ್ರಿಖಂಡಾ ಸ್ವರೂಪಳು
984. ತ್ರಿಗುಣಾ = ಸತ್ವ-ರಜಸ್-ತಮಸ್ ಗುಣಸ್ವರೂಪಳು
985. ಅಂಬಾ = (ಎಲ್ಲಾ ಜೀವಿಗಳ, ಜಗತ್ತಿನ) ಜನನಿ
986. ತ್ರಿಕೋಣಗಾ = ತ್ರಿಕೋಣ ಚಕ್ರದಲ್ಲಿ ಇರುವವಳು
987. ಅನಘಾ = ದೋಷ ರಹಿತಳು
988. ಅದ್ಭುತಚಾರಿತ್ರಾ = ಆಶ್ಚರ್ಯಕರವಾದ ಚರಿತ್ರೆ ಉಳ್ಳವಳು
989. ವಾಂಛಿತಾರ್ಥಪ್ರದಾಯಿನೀ = ಇಚ್ಛಿಸಿದ್ದನ್ನು ಕೊಡುವವಳು
990. ಅಭ್ಯಾಸಾತಿಶಯಜ್ಞಾತಾ = ಹೆಚ್ಚಿನ ಅಭ್ಯಾಸದಿಂದ ತಿಳಿಯಲ್ಪಡುವವಳು
991. ಷಡಧ್ವಾತೀತರೂಪಿಣೀ = ಪದ-ಮಂತ್ರ-ವರ್ಣ-ಭುವನ-ತತ್ತ್ವ-ಕಾಲ ಆರು ದಾರಿಗಳನ್ನು ಮೀರಿದವಳು
992. ಅವ್ಯಾಜಕರುಣಾಮೂರ್ತಿಃ = ನಿಷ್ಕಾರಣವಾದ ದಯೆಯುಳ್ಳವಳು
993. ಅಜ್ಞಾನಧ್ವಾಂತದೀಪಿಕಾ = ಅಜ್ಞಾನವೆಂಬ ಕತ್ತಲೆಗೆ ದೀಪಸ್ವರೂಪಳು
994. ಆಬಾಲಗೋಪವಿದಿತಾ = ಬಾಲಕ-ಗೋಪಾಲಕರಿಗೂ ತಿಳಿದವಳು
995. ಸರ್ವಾನುಲ್ಲಂಘ್ಯಶಾಸನಾ = ಬ್ರಹ್ಮಾದಿಗಳಿಂದಲೂ ಮೀರಲಸಾಧ್ಯವಾದ ಅಪ್ಪಣೆಯುಳ್ಳವಳು/ಶಾಸನವುಳ್ಳವಳು
996. ಶ್ರೀಚಕ್ರರಾಜನಿಲಯಾ = ಶ್ರೀಚಕ್ರ ಯಂತ್ರದಲ್ಲೇ ನೆಲೆಸಿದವಳು
997. ಶ್ರೀಮತ್‍ತ್ರಿಪುರಸುಂದರೀ = ತ್ರಿಪುರ ಸುಂದರೀ ದೇವಿಯು
998. ಶ್ರೀಶಿವಾ = ಮಂಗಳಾತ್ಮಕಳು
999. ಶಿವಶಕ್ತ್ಯೈಕ್ಯರೂಪಿಣೀ = ಶಿವ-ಶಕ್ತಿಯರ ಸಂಯೋಗದ ರೂಪಳು
1000. ಲಲಿತಾಂಬಿಕಾ = ಶ್ರೀ ಪಂಚದಶೀ-ಷೋಡಶೀ ಮಂತ್ರಿಗಳಿಂದ ಪ್ರತಿಪಾದ್ಯಳಾದ, ಕೋಮಲ ಮನಸ್ಸಿನ, ಸುಲಭವಾಗಿ ಹೊಂದುವ ತಾಯಿ ಲಲಿತಾಂಬಿಕೆ!
