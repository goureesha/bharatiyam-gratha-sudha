ಪ್ರಯತ್ನಂ ಕರೋಮಿ = ಪ್ರಯತ್ನ ಮಾಡುತ್ತೇನೆ
ತಥಾ ನ ವದತು = ಹಾಗೆ ಹೇಳಬೇಡ
ತತ್ರ ಕೋಽಪಿ ಸಂದೇಹಃ ನಾಸ್ತಿ = ಅದರಲ್ಲಿ ಸಂದೇಹವಿಲ್ಲ
ಅಹಂ ನ ಜ್ಞಾತವಾನ್ = ನನಗೆ ಗೊತ್ತಾಗಿಲ್ಲ
ಕದಾ ದದಾತಿ? = ಯಾವಾಗ ಕೊಡುವೆ?
ಅಹಂ ಕಥಂ ವದಾಮಿ? = ನಾನು ಹೇಗೆ ಹೇಳಲಿ?
ತಥಾ ಭವತಿ ವಾ? = ಹಾಗೆ ಆಗುತ್ತಾ?
ಭವತಃ ಸಮಯಾವಕಾಶಃ ಅಸ್ತಿ ವಾ? = ನಿಮಗೆ ಪುರುಸೊತ್ತು ಇದೆಯಾ?
ಅದ್ಯ ಭವತಃ ಕಾರ್ಯಕ್ರಮಃ ಕಃ? = ಇವತ್ತು ನಿಮ್ಮ ಕಾರ್ಯಕ್ರಮವೇನು?
ಹಸ್ತಸ್ಯ/ಪಾದಸ್ಯ ಕಿಂ ಅಭವತ್? = ಕೈಗೆ/ಕಾಲಿಗೆ ಏನಾಯಿತು?
ಬಹುದಿನೈಃ ತೇ ಪರಿಚಿತಾಃ = ಬಹಳ ದಿನಗಳಿಂದ ಅವರು ಪರಿಚಿತರು
ತಸ್ಯ ಕಿಯದ್ ಧೈರ್ಯಂ? = ಅವನಿಗೆ ಎಷ್ಟು ಧೈರ್ಯ?
ಭವಾನ್ ನ ಉಕ್ತವಾನ್ ಏವ = ನೀನು ಹೇಳಲೇ ಇಲ್ಲ
ಅಹಂ ಕಿಂ ಕರೋಮಿ? = ನಾನು ಏನು ಮಾಡಲಿ?
ಅಹಂ ನ ಜಾನಾಮಿ = ನನಗೆ ಗೊತ್ತಿಲ್ಲ
ಯಥಾ ಭವಾನ್ ಇಚ್ಛತಿ ತಥಾ = ನೀವು ಹೇಗೆ ಇಚ್ಛಿಸುತ್ತೀರೋ/ಹೇಳುತ್ತೀರೋ ಹಾಗೆ
ಭವತು, ಕಿಂ ತೇನ = ಆಗಲಿ, ಅದರಿಂದೇನು?
ತೇನ ಕಿಮಪಿ ನ ಸಿದ್ಧ್ಯತಿ = ಅದರಿಂದ ಏನೂ ಆಗಲ್ಲ
ಸಃ ಸರ್ವಥಾ ಅಪ್ರಯೋಜಕಃ = ಅವನು ಯಾವುದಕ್ಕೂ ಪ್ರಯೋಜನವಿಲ್ಲ
ಏಕವಾರಂ ಪ್ರಯತ್ನಂ ಕುರ್ಮಃ = ಒಮ್ಮೆ ಪ್ರಯತ್ನ ಮಾಡೋಣ
ಮೌನಂ ಏವ ಉಚಿತಂ = ಸುಮ್ಮನಿರುವುದೇ ವಾಸಿ
ತಸ್ಮಿನ್ ವಿಷಯೇ ಅಹಂ ಕಿಮಪಿ ನ ವದಾಮಿ = ಆ ವಿಷಯದಲ್ಲಿ ನಾನು ಏನೂ ಹೇಳುವುದಿಲ್ಲ
ವಿಚಿಂತ್ಯ ವಕ್ತವ್ಯಂ = ಯೋಚಿಸಿ ಹೇಳಬೇಕು
ತರ್ಹಿ ಸಮೀಚಿನಂ = ಹಾಗಾದರೆ ಒಳ್ಳೆಯದು
ಏವಂ ಚೇತ್ ಕಥಂ? = ಹೀಗಾದರೆ ಹೇಗೆ?
ಮಾಂ ಕಿಂಚಿತ್ ಸ್ಮಾರಯತು = ನನಗೆ ಸ್ವಲ್ಪ ಜ್ಞಾಪಿಸು
ತಂ ಅಹಂ ಸಮ್ಯಕ್ ಜಾನಾಮಿ = ಅವನು ನನಗೆ ಚೆನ್ನಾಗಿ ಗೊತ್ತು
ತದಾನೀಂ ಏವ ಉಕ್ತಂ ಕಿಲ = ಆಗಲೇ ಹೇಳಲಿಲ್ಲವೇ?
ಕದಾ ಉಕ್ತಂ ಭೋಃ? = ಯಾವಾಗಯ್ಯ ಹೇಳಿದ್ದು?
ಯತ್ಕಿಮಪಿ ಭವತು = ಏನು ಬೇಕಾದರೂ ಆಗಲಿ
ಸಃ ಬಹು ಸಮೀಚೀನಃ = ಅವನು ಬಹಳ ಒಳ್ಳೆಯವನು
ಸಃ ಬಹು Iೂಕ್ಷಃ = ಅವನು ತುಂಬಾ ಒರಟ
ತದ್ವಿಷಯೇ ಚಿಂತಾ ಮಾಸ್ತು = ಆ ವಿಷಯದಲ್ಲಿ ಚಿಂತೆ ಬೇಡ
ತಥೈವ ಇತಿ ನ ನಿಯಮಃ = 'ಹಾಗೆ' ಅಂತೇನೂ ಇಲ್ಲ
ಕರ್ತುಂ ಶಕ್ಯಂ ಕಿಂಚಿತ್ ಸಮಯಃ ಅಪೇಕ್ಷ್ಯತೇ = ಮಾಡಬಹುದು, ಸ್ವಲ್ಪ ಸಮಯ ಬೇಕಾಗುತ್ತೆ
ಏತಾವತ್ ವಾ ಕೃತವಾನ್ = ಇಷ್ಟಾದರೂ ಮಾಡಿದ
ದೃಷ್ಟಂ ಏವ ನ ಶಕ್ಯತೇ = ನೋಡಲಾಗುವುದಿಲ್ಲ
ತತ್ರೈವ ಕುತ್ರಾಪಿ ಸ್ಯಾತ್ = ಅಲ್ಲೇ ಎಲ್ಲಾದರೂ ಇದ್ದೀತು
ಯಥಾರ್ಥಂ ವದಾಮಿ = ನಿಜವಾಗಿ ಹೇಳುತ್ತಿದ್ದೇನೆ
ಏವಂ ಭವಿತುಂ ಅರ್ಹತಿ = ಹೀಗೆ ಆಗಬಹುದು
ಕದಾಚಿತ್ ಏವಂ ಅಪಿ ಸ್ಯಾತ್ = ಕೆಲವೊಮ್ಮೆ ಹೀಗೂ ಇರಬಹುದು
ಅಹಂ ತಾವದಪಿ ನ ಜಾನಾಮಿ ವಾ = ನನಗೆ ಅಷ್ಟೂ ಗೊತ್ತಿಲ್ಲವೇ?
ಕಃ ವಾ ನಾ ಇಚ್ಛತಿ? = ಯಾರಿಗೆ ತಾನೆ ಬೇಡ ಹೇಳು!?
ತತ್ರ ಗತ್ವಾ ಕಿಂ ಕರೋತಿ? = ಅಲ್ಲಿ ಹೋಗಿ ಏನು ಮಾಡುತ್ತೀಯಾ?
ಪುನಃ ಆಗಚ್ಛಂತು = ಮತ್ತೆ ಬನ್ನಿ
ಅತ್ರ ಕಿಂ ಕಷ್ಟಂ = ಇದರಲ್ಲೇನು ಕಷ್ಟ?
ಭೋಃ ಆನೀತವಾನ್ ವಾ? = ತಂದಿದ್ದೀಯೇನಪ್ಪ?
ಭವಂತಂ ಕಃ ಉಕ್ತವಾನ್? = ನಿಮಗೆ ಯಾರು ಹೇಳಿದ್ದು?
ಯದಾ ಕದಾ ವಾ ಆಗಚ್ಛೇತ್ = ಯಾವಾಗಲಾದರೂ ಬಂದಾನು/ಬಂದೀತು
ಪ್ರಾಯಃ ತಥಾ ನ ಸ್ಯಾತ್ = ಬಹುಷಃ ಹಾಗಿರಲಾರದು
ಶ್ವಃ ದದಾತು = ನಾಳೆ ಕೊಡಿ
ಅಹಂ ಪುನಃ ಸೂಚಯಾಮಿ = ನಾನು ಮತ್ತೆ ತಿಳಿಸುತ್ತೇನೆ
ಅದ್ಯ ಆಸೀತ್ ವಾ? = ಇಂದು ಇತ್ತಾ?
ಅವಶ್ಯಂ ಆಗಚ್ಛಾಮಿ = ಖಂಡಿತ ಬರುತ್ತೇನೆ
ನಾಗರಾಜಃ ಅಸ್ತಿ ವಾ? = ನಾಗರಾಜನು ಇದ್ದಾನಾ?
ಇದಂ ಕಥಂ ಏವಂ ಜಾತಂ = ಇದೇಕೆ ಹೀಗಾಯ್ತು?
ತತ್ರ ಆಸೀತ್ ವಾ? = ಅಲ್ಲಿ ಇತ್ತಾ?
ಕಿಮಪಿ ಉಕ್ತಂ ವಾ? = ಏನಾದರೂ ಹೇಳಿದ್ದಾರೋ?
ಕುತಃ ಆನೀತಂ? = ಎಲ್ಲಿಂದ ತಂದದ್ದು?
ಅನ್ಯತ್ ಕಿಮಪಿ ಕಾರ್ಯಂ ನಾಸ್ತಿ = ಬೇರೆ ಏನೂ ಕೆಲಸವಿಲ್ಲ
ಮಮ ವಚನಂ ಕಿಂಚಿತ್ ಶೃಣೋತು = ನನ್ನ ಮಾತು ಸ್ವಲ್ಪ ಕೇಳು
ಏತತ್ ಸತ್ಯಂ ಕಿಲ? = ಇದು ನಿಜ ತಾನೇ?
ತತ್ ಅಹಂ ಅಪಿ ಜಾನಾಮಿ = ಅದು ನನಗೂ ಗೊತ್ತು
ತಾವತ್ ಅವಶ್ಯಕಂ ನ = ಅಷ್ಟೇನೂ ಬೇಕಾದದ್ದಲ್ಲ
ಭವತಃ ಕಾ ಹಾನಿಃ? = ನಿಮಗೇನು ನಷ್ಟ?
ಕಿಮರ್ಥಂ ಏತಾವಾನ್ ವಿಲಂಬಃ? = ಏಕೆ ಇಷ್ಟು ಹೊತ್ತು?
ಯಥೇಷ್ಟಂ ಅಸ್ತಿ = ಎಷ್ಟು ಬೇಕಾದರೂ ಇದೆ
ಭವತಃ ಅಭಿಪ್ರಾಯಃ ಕಃ? = ನಿಮ್ಮ ಅಭಿಪ್ರಾಯವೇನು?
ತಸ್ಯ ಕಿಂ ಕಾರಣಂ ? = ಅದಕ್ಕೇನು ಕಾರಣ?
ಸ್ವಯಂ ಏವ ಕರೋತಿ ? = ಸ್ವತಃ ಮಾಡುತ್ತೀಯ?
ತತ್ ಮಮ ಕೃತೇ ನ ರೋಚತೇ = ಅದು ನನಗೆ ಹಿಡಿಸುವುದಿಲ್ಲ
ಉಕ್ತಂ ಏವ ವದತಿ ಸಃ = ಹೇಳಿದ್ದನ್ನೇ ಹೇಳುತ್ತಾನೆ ಅವನು
ಅನ್ಯಥಾ ಬಹು ಕಷ್ಟಂ = ಇಲ್ಲ ಎಂದರೆ ಬಹಳ ಕಷ್ಟ
ಕಿಮರ್ಥಂ ಪೂರ್ವಂ ನ ಉಕ್ತಂ? = ಯಾಕೆ ಮೊದಲೇ ಹೇಳಲಿಲ್ಲ?
ಸಮ್ಯಕ್ ನ ಜಾನಾಮಿ = ಸರಿಯಾಗಿ ಗೊತ್ತಿಲ್ಲ
ನಿಶ್ಚಯಃ ನಾಸ್ತಿ = ನಿಶ್ಚಯವಿಲ್ಲ
ಕುತ್ರ ಆಸೀತ್ ಭವಾನ್ ? = ಎಲ್ಲಿದ್ದೆ ನೀನು?
ಭಯಸ್ಯ ಕಾರಣಂ ನಾಸ್ತಿ = ಹೆದರಬೇಕಾಗಿಲ್ಲ
ತದ್ವಿಷಯೇ ಮಹತೀ ಪ್ರೀತಿಃ = ಆ/ಅವನ ವಿಷಯದಲ್ಲಿ ತುಂಬಾ ಪ್ರೀತಿ
ಕಿಯತ್ ಲಜ್ಜಾಸ್ಪದಂ! = ಎಂಥಾ ನಾಚಿಕೆಗೇಡು!
ಏಷಃ ಮಮ ದೋಷಃ ನ = ಇದು ನನ್ನ ತಪ್ಪಲ್ಲ
ಮಮ ತು ಆಕ್ಷೇಪಃ ನಾಸ್ತಿ = ನನ್ನದೇನೂ ಅಭ್ಯಂತರವಿಲ್ಲ
ಸಃ ಶೀಘ್ರಕೋಪೀ = ಅವನು ಮುಂಗೋಪಿ
ತೀವ್ರಂ ಮಾ ಪರಿಗಣಯತು = ಗಂಭೀರವಾಗಿ ತಗೋಬೇಡ
ಆಗತಃ ಏಷಃ ವರಾಕಃ = ಬಂದು ವಕ್ಕರಿಸಿದ
ಯುಕ್ತೇ ಸಮಯೇ ಆಗತವಾನ್ = ಸರಿಯಾದ ಸಮಯಕ್ಕೆ ಬಂದ
ಬಹು ಜಲ್ಪತಿ ಭೋಃ = ಬಹಳ ಹರಟೆ ಹೊಡೆಯುತ್ತಾನೆ
ಏಷಾ ಕೇವಲಂ ಕಿಂವದಂತೀ = ಇದು ಕೇವಲ ಗಾಳಿ ಮಾತು(ಸುದ್ದಿ)
ಕಿಮಪಿ ನ ಭವತಿ = ಏನೂ ಆಗಲ್ಲ
ಏವಮೇವ ಆಗತವಾನ್ = ಹೀಗೆ ಬಂದ
ವಿನಾ ಕಾರಣಂ ಕಿಮರ್ಥಂ ಗಂತವ್ಯಂ? = ಸುಮ್ಮನೆ ಏಕೆ ಹೋಗಬೇಕು?
ಭವತಃ ವಚನಂ ಸತ್ಯಂ = ನಿಮ್ಮ ಮಾತು ನಿಜ
ಮಮ ವಚನಂ ಕಃ ಶೃಣೋತಿ? = ನನ್ನ ಮಾತನ್ನು ಯಾರು ಕೇಳುತ್ತಾರೆ?
ತದಾ ಕಿಮಪಿ ನ ಸ್ಫುರಿತಂ = ಆಗ ಏನೂ ತೋಚಲೇ ಇಲ್ಲ
ಕಿಮರ್ಥಂ ತಾವತೀ ಚಿಂತಾ? = ಯಾಕೆ ಅಷ್ಟೆಲ್ಲಾ ಚಿಂತೆ?
ಭವತಃ ಕಿಮಸ್ತಿ ಕಷ್ಟಂ? = ನಿಮಗೇನು ತೊಂದರೆ ಇದೆ?
ಏವಂ ನ ಭವಿತವ್ಯಂ ಆಸೀತ್ = ಹೀಗಾಗಬಾರದಿತ್ತು
ಅನ್ಯಥಾ ಮಾ ಚಿಂತಯತು = ಅನ್ಯಥಾ ತಿಳಿದುಕೊಳ್ಳಬೇಡಿ
