ಭವತಃ ನಾಮ ಕಿಂ? = ನಿಮ್ಮ ಹೆಸರೇನು? (ಪುರುಷರಿಗೆ)
ಭವತ್ಯಾಃ ನಾಮ ಕಿಂ? = ನಿಮ್ಮ ಹೆಸರೇನು? (ಸ್ತ್ರೀಯರಿಗೆ)
ಮಮ ನಾಮ .. = ನನ್ನ ಹೆಸರು ..
ಏಷಃ ಮಮ ಮಿತ್ರಂ = ಇವನು ನನ್ನ ಗೆಳೆಯ
ಏತಸ್ಯಂ ನಾಮ ಶ್ರುತವಾನ್ = ಇವರ ಹೆಸರನ್ನು ಕೇಳಿದ್ದೇನೆ
(ಪುರುಷರಿಗೆ) ಭವಾನ್ ಕಿಂ ಉದ್ಯೋಗಂ ಕರೋತಿ? = ನೀವು ಏನು ಕೆಲಸ ಮಾಡುತ್ತಿದ್ದೀರಿ?
(ಸ್ತ್ರೀಯರಿಗೆ) ಭವತೀ ಕಿಂ ಉದ್ಯೋಗಂ ಕರೋತಿ? = ನೀವು ಏನು ಕೆಲಸ ಮಾಡುತ್ತಿದ್ದೀರಿ?
ಅಹಂ ಅಧ್ಯಾಪಕಃ ಅಸ್ಮಿ = ನಾನು ಅಧ್ಯಾಪಕನಾಗಿದ್ದೇನೆ
ಅಹಂ ಅಧ್ಯಾಪಿಕಾ ಅಸ್ಮಿ = ನಾನು ಅಧ್ಯಾಪಕಿಯಾಗಿದ್ದೇನೆ
ನ್ಯಾಯವಾದೀ = ವಕೀಲ
ತಂತ್ರಜ್ಞಃ = ಎಂಜಿನಿಯರ್
ವಿಕ್ರಯಿಕಃ = ವ್ಯಾಪಾರಸ್ಥ
ಅಹಂ ಯಂತ್ರಾಗಾರೇ ಅಸ್ಮಿ = ನಾನು ಕಾರ್ಖಾನೆಯಲ್ಲಿ ಇದ್ದೇನೆ
ಕಾರ್ಯಾಲಯೇ = ಕಚೇರಿಯಲ್ಲಿ
ಚಿಕಿತ್ಸಾಲಯೇ = ಆಸ್ಪತ್ರೆಯಲ್ಲಿ
ಪ್ರೌಢಶಾಲಾಯಾಂ = ಪ್ರೌಢಶಾಲೆಯಲ್ಲಿ
ಭವಾನ್/ಭವತೀ ಕಿಂ ಪಠತಿ? = ನೀವು ಏನು ಓದುತ್ತಿದ್ದೀರಿ?
ಅಹಂ ನವಮಕಕ್ಷಾಯಾಂ ಪಠಾಮಿ = ನಾನು ಒಂಭತ್ತನೇ ತರಗತಿಯಲ್ಲಿ ಓದುತ್ತಿದ್ದೇನೆ
ಭವತಃ ಗ್ರಾಮಃ? = ನಿಮ್ಮ ಊರು?
ಕುಶಲಂ ವಾ? = ಕುಶಲವೇ?
ಗೃಹೇ ಸರ್ವೇ ಕುಶಲಿನಃ ವಾ? = ಮನೆಯಲ್ಲಿ ಎಲ್ಲರೂ ಕುಶಲವೇ?
ಏವಂ ಅಸ್ಮಿ = ಹೀಗಿದ್ದೇನೆ
ಕಃ ವಿಶೇಷಃ? = ಏನು ವಿಶೇಷ?
ಕುತಃ ಆಗಚ್ಛಂತಿ? = ಎಲ್ಲಿಂದ ಬರುತ್ತಿದ್ದೀರಿ?
ಗೃಹತಃ ಆಗಚ್ಛಾಮಿ = ಮನೆಯಿಂದ ಬರುತ್ತಿದ್ದೇನೆ
ಕುತ್ರ ಗಚ್ಛಂತಿ? = ಎಲ್ಲಿ ಹೋಗುತ್ತಿದ್ದೀರಿ?
ದೇವಾಲಯಂ ಗಚ್ಛಾಮಿ = ದೇವಸ್ಥಾನಕ್ಕೆ ಹೊರಟಿದ್ದೇನೆ
ವಿಪಣಿಂ = ಮಾರುಕಟ್ಟೆಗೆ
ಕಿಂ ಚಿರಾದ್ ದರ್ಶನಂ? = ಏನು, ಬಹಳ ಅಪರೂಪ?
ಭವಂತಂ ಕುತ್ರಾಪಿ ದೃಷ್ಟವಾನ್ = ನಿಮ್ಮನ್ನು ಎಲ್ಲೋ ನೋಡಿದ್ದೇನೆ
ಭವಾನ್ ಸಂಭಾಷಣಶಿಬಿರಂ ಆಗತವಾನ್ ವಾ? = ನೀವು ಸಂಭಾಷಣ ಶಿಬಿರಕ್ಕೆ ಬಂದಿದ್ದಿರಾ?
ತರ್ಹಿ ಕುತ್ರ ದೃಷ್ಟವಾನ್? = ಹಾಗಾದರೆ ಎಲ್ಲಿ ನೋಡಿದ್ದೀನಿ?
ತರ್ಹಿ ತತ್ರೈವ ದೃಷ್ಟವಾನ್ = ಹಾಗಾದರೆ ಅಲ್ಲಿಯೇ ನೋಡಿದ್ದು
