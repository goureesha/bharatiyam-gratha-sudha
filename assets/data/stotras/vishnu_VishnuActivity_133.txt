601. ಶ್ರೀವತ್ಸವಕ್ಷಾಃ = ಪ್ರಕೃತಿ ತತ್ತ್ವರೂಪವಾದ ಶ್ರೀವತ್ಸವೆಂಬ ಮಚ್ಚೆಯಿಂದ ಎದೆಯಲ್ಲಿ ಅಲಂಕೃತನಾದವನು

602. ಶ್ರೀವಾಸಃ = ಲಕ್ಷ್ಮೀದೇವಿಗೆ ನಿತ್ಯನಿವಾಸ ಸ್ಥಾನನಾಗಿರುವವನು

603. ಶ್ರೀಪತಿಃ = ಶ್ರೀದೇವಿಗೆ ಸಹಜ ಪತಿಯಾಗಿರುವವನು

604. ಶ್ರೀಮತಾಂ ವರಃ = ಶ್ರೀಸಂಪನ್ನರಾದ ಎಲ್ಲರಿಗಿಂತಲೂ ಶ್ರೇಷ್ಠತಮನಾದವನು

605. ಶ್ರೀದಃ = ಭಕ್ತರಿಗೆ ಸಂಪತ್ತನ್ನು ಕೊಡುವವನು

606. ಶ್ರೀಶಃ = ಶ್ರೀಗೆ ಈಶ್ವರನಾದವನು

607. ಶ್ರೀನಿವಾಸಃ = ಸಂಪತ್ತುಗಳಿಗೆ ವಾಸಸ್ಥಾನನಾದವನು

608. ಶ್ರೀನಿಧಿಃ = ಸಮಸ್ತ ಸಂಪತ್ತುಗಳೂ ಯಾವನಲ್ಲಿ ಒಂದಾಗಿ ಸೇರಿವೆಯೋ ಅವನು

609. ಶ್ರೀವಿಭಾವನಃ = ಶ್ರೀದೆವಿಯನ್ನು ಚೆನ್ನಾಗಿ ಭಾವಿಸಿ ಗೌರವಿಸುವವನು, ಸಂಪತ್ತನ್ನು ಬಗೆ ಬಗೆಯಾಗಿ ವಿತರಣೆ ಮಾಡುವವನು

610. ಶ್ರೀಧರಃ = ಶ್ರೀದೇವಿಯನ್ನು ಧರಿಸಿರುವವನು

611. ಶ್ರೀಕರಃ = ಭಕ್ತರಿಗೆ ಸಂಪತ್ತು-ಕಾಂತಿಗಳನ್ನು ಉಂಟು ಮಾಡುವವನು

612. ಶ್ರೇಯಃ = ಅತ್ಯಂತ ಪ್ರಶಸ್ತ್ಯನಾದವನು, ಯಾವ ದೋಷವೂ ಇಲ್ಲದ ಪರಿಪೂರ್ಣನಾದ ಸುಖ ಸ್ವರೂಪಿಯು

613. ಶ್ರೀಮಾನ್ = ಶ್ರೀಗಳಿಂದ ಸಂಪನ್ನನಾದವನು

614. ಲೋಕತ್ರಯಾಶ್ರಯಃ = ಮೂರು ಲೋಕಗಳಿಗೂ ಆಶ್ರಯನಾದವನು

615. ಸ್ವಕ್ಷಃ = ಶ್ರೇಷ್ಠವಾದ ಇಂದ್ರಿಯಗಳು/ಕಣ್ಣುಗಳು ಉಳ್ಳವನು

616. ಸ್ವಂಗಃ = ಸುಂದರವಾದ ಅಂಗಗಳುಳ್ಳವನು

617. ಶತಾನಂದಃ = ಅನಂತವಾದ/ಅಪರಿಮಿತವಾದ ಆನಂದವುಳ್ಳವನು, ಬ್ರಹ್ಮದೇವ ಸ್ವರೂಪಿಯು

618. ನಂದಿಃ = ಆನಂದವುಳ್ಳವನು, ಆನಂದವನ್ನು ಉಂಟುಮಾಡುವವನು

619. ಜ್ಯೋತಿರ್ಗಣೇಶ್ವರಃ = ಜ್ಯೋತಿಗಳ/ನಕ್ಷತ್ರಗಳ ಸಮೂಹಕ್ಕೆ ಅಧಿಪತಿಯಾದವನು

620. ವಿಜಿತಾತ್ಮಾ = ಮನಸ್ಸನ್ನು ಗೆದ್ದಿರುವವನು

621. ವಿಧೇಯಾತ್ಮಾ/ಅವಿಧೇಯಾತ್ಮಾ = ಜಿತೇಂದ್ರಿಯನು, ವಿಧಾನ ಮಾಡಲ್ಪಡುವ ವಿಷಯಗಳಲ್ಲಿ ಮನಸ್ಸುಳ್ಳವನು, ಯಾರಿಗೂ ವಿಧೇಯವಲ್ಲದ ಸ್ವರೂಪವುಳ್ಳವನು

622. ಸತ್ಕೀರ್ತಿಃ = ಸತ್ಯವಾದ ಕೀರ್ತಿಯುಳ್ಳವನು, ಒಳ್ಳೆಯ ಕೀರ್ತಿಯುಳ್ಳವನು

623. ಛಿನ್ನಸಂಶಯಃ = ಯಾವ ಸಂಶಯವೂ ಇಲ್ಲದವನು, ಜ್ಞಾನಿಗಳ ಸಂಶಯವನ್ನು ಧ್ವಂಸಮಾಡುವವನು

624. ಉದೀರ್ಣಃ = ಎಲ್ಲ ಪ್ರಾಣಿಗಳಿಗಿಂತಲೂ ಉತ್ಕೃಷ್ಟನಾದವನು

625. ಸರ್ವತಶ್ಚಕ್ಷುಃ = ಎಲ್ಲೆಲ್ಲಿಯೂ ಕಣ್ಣುಗಳುಳ್ಳವನು

626. ಅನೀಶಃ = (ತಾನೇ ಸರ್ವೇಶ್ವರನಾದ್ದರಿಂದ) ಬೇರೆ ಪ್ರಭು ಇಲ್ಲದವನು

627. ಶಾಶ್ವತಸ್ಥಿರಃ = ನಿತ್ಯನೂ ಸ್ಥಿರನೂ ಆಗಿರುವವನು

628. ಭೂಶಯಃ = ಭಕ್ತರ ದರ್ಶನಕ್ಕಾಗಿ ಅರ್ಚಾಮೂರ್ತಿಯಾಗಿ ಭೂಮಿಯಲ್ಲಿ ಅನಂತ ಪದ್ಮನಾಭ, ಗೋವಿಂದರಾಜ, ಶ್ರೀರಂಗನಾಥ ಮುಂತಾದ ರೂಪದಲ್ಲಿ ಪವಡಿಸಿರುವವನು, ಭೂಮಿಯಲ್ಲಿ ದರ್ಭಶಯನ ಮಾಡಿದ ಶ್ರೀರಾಮ ರೂಪಿಯು

629. ಭೂಷಣಃ = ಅಲಂಕಾರವಾಗಿರುವವನು, ಅಲಂಕೃತನಾಗಿರುವವನು, ತನ್ನ ಅವತಾರಗಳಿಂದ ಸ್ವರ್ಗ-ಭೂಮಿ-ಪಾತಾಳಗಳೇ ಮುಂತಾದ ಸ್ಥಾನಗಳನ್ನು ಅಲಂಕರಿಸುವವನು, ತತ್ತ್ವಮಯವಾದ ದಿವ್ಯಭೂಷಣವುಳ್ಳವನು

630. ಭೂತಿಃ = ಸತ್-ಸ್ವರೂಪನು, ವಿಭೂತಿ ರೂಪನು

631. ವಿಶೋಕಃ = ಶೋಕರಹಿತನು, ಪರಮಾನಂದ ರೂಪಿಯು

632. ಶೋಕನಾಶನಃ = ಆಧ್ಯಾತ್ಮಿಕ, ಆಧಿಭೌತಿಕ, ಆಧಿದೈವಿಕ ಎಂಬ ಮೂರು ಬಗೆಯ ಶೋಕಗಳನ್ನು ನಾಶಪಡಿಸುವವನು

633. ಅರ್ಚಿಷ್ಮಾನ್ = ಎಲ್ಲ ತೇಜಸ್ಸುಗಳಿಗೂ ತೇಜಸ್ಸನ್ನು ಕೊಡುವ ಮುಖ್ಯವಾದ ತೇಜಸ್ವಿಯು

634. ಅರ್ಚಿತಃ = ಪೂಜ್ಯರಿಂದಲೂ ಪೂಜಿತನಾದವನು

635. ಕುಂಭಃ = ಪ್ರಾಣಾಪಾನ ನಿರೋಧರೂಪವಾದ ಕುಂಭಕ ಯೋಗ ಸ್ವರೂಪಿಯು

636. ವಿಶುದ್ಧಾತ್ಮಾ = ಅತ್ಯಂತ ಪರಿಶುದ್ಧವಾದ (ತ್ರಿಗುಣಾತೀತನಾದ) ಆತ್ಮ ಸ್ವರೂಪವುಳ್ಳವನು

637. ವಿಶೋಧನಃ = ಭಕ್ತರನ್ನು ಚೆನ್ನಾಗಿ ಶುದ್ಧಿ ಮಾಡುವವನು

638. ಅನಿರುದ್ಧಃ = ತಡೆಯಿಲ್ಲದವನು, ವಾಸುದೇವಾದಿ ವ್ಯೂಹಗಳಲ್ಲಿ ನಾಲ್ಕನೆಯವನು, ಅಹಂಕಾರ ತತ್ತ್ವಕ್ಕೆ ಅಭಿಮಾನಿಯಾದ ದೇವತೆಯು, ತನ್ನ ಸೃಷ್ಟಿ ವ್ಯಾಪಾರದಲ್ಲಿ ಯಾವ ತಡೆಯೂ ಇಲ್ಲದವನು, ಯಾರಿಂದಲೂ ತಡೆಯಲು ಸಾಧ್ಯವಾಗದವನು

639. ಅಪ್ರತಿರಥಃ = ತಡೆಯುವ ಎದುರಾಳಿಯಿಲ್ಲದವನು, ತಡೆಯಿಲ್ಲದ ಧಾರಾಳವಾದ ವೇದಮಯನಾದ ಗರುತ್ಮಂತನೆಂಬ ರಥವುಳ್ಳವನು

640. ಪ್ರದ್ಯುಮ್ನಃ = ಕಾಮದೇವ ಸ್ವರೂಪನು, ಶ್ರೇಷ್ಠನಾದ ರೂಪಾದಿಗಳೇ ಧನವುಳ್ಳವನು, ವಾಸುದೇವ ವ್ಯೂಹದಲ್ಲಿ ಮೂರನೆಯವನು, ಮನಸ್ತತ್ತ್ವಕ್ಕೆ ಅಭಿಮಾನಿಯಾದ ಭಗವದ್ರೂಪನು

641. ಅಮಿತವಿಕ್ರಮಃ = ಅಳೆಯಲಾಗದ ಪರಾಕ್ರಮವುಳ್ಳವನು, ದೀರ್ಘವಾದ ಪಾದವಿನ್ಯಾಸವುಳ್ಳವನು, ತ್ರಿವಿಕ್ರಮ ಸ್ವರೂಪಿಯು

642. ಕಾಲನೇಮಿನಿಹಾ = ಅವಿದ್ಯಾಸ್ವರೂಪಿಯಾದ ಅಸುರನನ್ನು ಸಂಹರಿಸಿದವನು

643. ವೀರಃ = ಶೂರನು

644. ಶೌರಿಃ = ಶೂರನೆಂಬ ಯದುವಂಶೀಯನಾದ ರಾಜನ ಕುಲದಲ್ಲಿ ಹುಟ್ಟಿದವನು

645. ಶೂರಜನೇಶ್ವರಃ = ಶೂರರೆಂಬ ಯಾದವಗಣಗಳಿಗೆ ಅಧಿಪತಿಯಾದವನು, ಅತಿಶೂರರಾದ ಜನರಿಗೂ/ದೇವಾದಿಗಳಿಗೂ ಅಧಿಪತಿಯಾದವನು

646. ತ್ರಿಲೋಕಾತ್ಮಾ = ಮೂರು ಲೋಕಗಳಿಗೂ ಅಂತರ್ಯಾಮಿ ಚೈತನ್ಯವಾಗಿರುವವನು

647. ತ್ರಿಲೋಕೇಶಃ = ಮೂರು ಲೋಕಗಳಿಗೂ ಸ್ವಾಮಿಯಾದವನು

648. ಕೇಶವಃ = ತ್ರಿಮೂರ್ತಿ ಪ್ರಭುವು, ಸುಂದರವಾದ ನೀಲವಾದ ಕೇಶಪಾಶಗಳುಳ್ಳವನು, ಸೂರ್ಯಾದಿ ಕಿರಣಗಳುಳ್ಳವನು

649. ಕೇಶಿಹಾ = ಕೇಶಿಯೆಂಬ ದಾನವನನ್ನು ಸಂಹರಿಸಿದವನು

650. ಹರಿಃ = ಮನಸ್ಸನ್ನು ಅಪಹರಿಸುವವನು, ಪಾಪಗಳನ್ನು ಹೋಗಲಾಡಿಸುವವನು

651. ಕಾಮದೇವಃ = ಮನ್ಮಥ ಸ್ವರೂಪಿಯು, ಕಾಮಿಸಲ್ಪಡುವ ದೇವನು

652. ಕಾಮಪಾಲಃ = ಭಕ್ತರ ಕಾಮಗಳನ್ನು ಪಾಲಿಸುವವನು, ಮನ್ಮಥನನ್ನು ಪಾಲಿಸುವವನು, ಬಲರಾಮ ಸ್ವರೂಪನು

653. ಕಾಮೀ = ಪೂರ್ಣಕಾಮನಾದವನು, ಲೋಕಕಲ್ಯಾಣಾರ್ಥವಾದ ಕಾಮಗಳುಳ್ಳವನು

654. ಕಾಂತಃ = ಆಕರ್ಷಕನಾದವನು, ಕಮನೀಯನು

655. ಕೃತಾಗಮಃ = ಆಗಮಗಳನ್ನು (ವೇದ-ಸ್ಮೃತಿ-ವೈಷ್ಣವ/ಶೈವ) ರಚಿಸಿದವನು

656. ಅನಿರ್ದೇಶ್ಯವಪುಃ = ತ್ರಿಗುಣಾತೀತನೂ, ಅತೀಂದ್ರಿಯನೂ ಆದ್ದರಿಂದ ಯಾವನ ಶರೀರ 'ಹೀಗೆಯೇ ಇದೆ' ಎಂದು ನಿರ್ದಿಷ್ಟವಾಗಿ ಹೇಳಲಾಗುವುದಿಲ್ಲವೋ ಅಂಥವನು

657. ವಿಷ್ಣುಃ = ಸರ್ವವ್ಯಾಪಕನು, ತನ್ನ ಪಾದವಿನ್ಯಾಸದಿಂದ ಎಲ್ಲವನ್ನೂ ಆಕ್ರಮಿಸಿದವನು

658. ವೀರಃ = ಮಹಾವೀರನು

659. ಅನಂತಃ = ದೇಶ-ಕಾಲ ವಸ್ತುಗಳ ಪರಿಮಿತಿ ಇಲ್ಲದವನು, ನಾಶವಿಲ್ಲದವನು

660. ಧನಂಜಯಃ = ಎಲ್ಲ ಧನಗಳನ್ನೂ ಗೆದ್ದ ಪರಮಧನ ಸ್ವರೂಪನು, ಅರ್ಜುನ ಸ್ವರೂಪನು, ಅಗ್ನಿ ಸ್ವರೂಪನು

661. ಬ್ರಹ್ಮಣ್ಯಃ = ತಪಸ್ಸು, ವೇದ, ಬ್ರಾಹ್ಮಣರು ಮತ್ತು ಜ್ಞಾನಗಳಿಗೆ ಹಿತವನ್ನು ಉಂಟುಮಾಡುವವನು

662. ಬ್ರಹ್ಮಕೃತ್ = ವೇದ, ತಪಸ್ಸು, ಬ್ರಾಹ್ಮಣರನ್ನು ಉಂಟು ಮಾಡುವವನು

663. ಬ್ರಹ್ಮಾ = ಸೃಷ್ಟಿಕರ್ತನು

664. ಬ್ರಹ್ಮ = ಪರಬ್ರಹ್ಮನು, ವೇದ-ತಪಸ್ಸು-ಬ್ರಾಹ್ಮಣ ಸ್ವರೂಪಿಯು

665. ಬ್ರಹ್ಮವಿವರ್ಧನಃ = ವೇದ,ತಪಸ್ಸು ಮತ್ತು ಬ್ರಾಹ್ಮಣರನ್ನು ವೃದ್ಧಿ ಹೊಂದಿಸುವವನು

666. ಬ್ರಹ್ಮವಿತ್ = ವೇದವನ್ನೂ ತಪಸ್ಸನ್ನೂ ಚೆನ್ನಾಗಿ ಯಥಾವತ್ತಾಗಿ ಅರಿತವನು

667. ಬ್ರಾಹ್ಮಣಃ = ಬ್ರಹ್ಮಜ್ಞಾನಿಯಾದ ಬ್ರಾಹ್ಮಣ ಸ್ವರೂಪಿಯು

668. ಬ್ರಹ್ಮೀ = ವೇದ, ತಪಸ್ಸು ಮತ್ತು ಬ್ರಹ್ಮರನ್ನುಳ್ಳವನು

669. ಬ್ರಹ್ಮಜ್ಞಃ = ವೇದ, ತಪಸ್ಸು ಮತ್ತು (ಸ್ವರೂಪ) ಪರತತ್ತ್ವವನ್ನು ತಿಳಿದವನು

670. ಬ್ರಾಹ್ಮಣಪ್ರಿಯಃ = ಬ್ರಹ್ಮಜ್ಞಾನಿಗಳಾದ ಬ್ರಾಹ್ಮಣರಿಗೆ ಪ್ರಿಯನಾದವನು, ಬ್ರಹ್ಮಜ್ಞಾನಿಗಳಾದ ಬ್ರಾಹ್ಮಣರನ್ನು ಪ್ರೀತಿಸುವವನು

671. ಮಹಾಕ್ರಮಃ = ಅತ್ಯಂತ ದೊಡ್ಡ ಪಾದವಿನ್ಯಾಸಗಳುಳ್ಳವನು, ಹೆಜ್ಜೆ-ಹೆಜ್ಜೆಯಾಗಿ ಭಕ್ತರನ್ನು ಮೇಲಕ್ಕೆ ಒಯ್ಯುವವನು, ದೊಡ್ಡ ಪರ್ವಗಳುಳ್ಳವನು

672. ಮಹಾಕರ್ಮಾ = ಜಗತ್ತಿನ ಸೃಷ್ಟಿ ಮುಂತಾದ ದೊಡ್ಡ ವ್ಯಾಪಾರಗಳುಳ್ಳವನು, ದೊಡ್ಡ ಆಶಯವುಳ್ಳ ಕರ್ಮಗಳುಳ್ಳವನು

673. ಮಹಾತೇಜಾಃ = ಮಹಾ ತೇಜಸ್ವಿಯು

674. ಮಹೋರಗಃ = ಮಹಾಶೇಷನಾಗ ವಾಸುಕಿ-ಸ್ವರೂಪನು, ಮಹಾಕುಂಡಲಿನೀ ಪ್ರಾಣಶಕ್ತಿ ಸ್ವರೂಪನು

675. ಮಹಾಕ್ರತುಃ = ಮಹಾಯಜ್ಞ ಅಥವಾ ಜ್ಞಾನಸ್ವರೂಪಿಯು

676. ಮಹಾಯಜ್ವಾ = ಸೃಷ್ಟಿ-ಸ್ಥಿತಿ-ಲಯ ರೂಪಗಳಾದ ಮಹಾಯಜ್ಞಗಳನ್ನು ಮಾಡುವವನು, ಶ್ರೀರಾಮಾದಿ ಅವತಾರಗಳಲ್ಲಿ ದೊಡ್ಡ ಯಜ್ಞಗಳನ್ನು ಮಾಡಿದವನು, ಮಹಾಯಜ್ಞಗಳನ್ನು ಮಾಡುವವರಲ್ಲಿ ಅಂತರ್ಯಾಮಿ ಆಗಿರುವವನು

677. ಮಹಾಯಜ್ಞಃ = ಪಂಚಮಹಾಯಜ್ಞ ಸ್ವರೂಪನು, ಪಂಚ ಮಹಾಯಜ್ಞಗಳಿಂದ ಆರಾಧಿಸಲ್ಪಡುವವನು, ಸೃಷ್ಟಿ-ಸ್ಥಿತಿ-ಲಯ ರೂಪವಾದ ದೊಡ್ಡ ಯಜ್ಞವನ್ನು ಮಾಡುವವನು

678. ಮಹಾಹವಿಃ = ಮಹಾಗ್ನಿಯಾದ ತನ್ನಲ್ಲಿ ಹೋಮ ಮಾಡಿಕೊಳ್ಳುವ ಮಹಾಹವಿಸ್ಸೂ ತಾನೇ ಆಗಿರುವವನು, ಮಹಾಹವಿಸ್ಸಾಗಿ ಜಗತ್ತೆಲ್ಲವೂ ಯಾವನಲ್ಲಿ ಹೋಮ ಮಾಡಲ್ಪಡುತ್ತದೆಯೋ ಅವನು

679. ಸ್ತವ್ಯಃ = ಎಲ್ಲರಿಂದಲೂ ಸ್ತುತಿಸಲ್ಪಡಲು ಯೋಗ್ಯನಾದವನು

680. ಸ್ತವಪ್ರಿಯಃ = ಸ್ತೋತ್ರ ಪ್ರಿಯನಾದವನು

681. ಸ್ತೋತ್ರಂ = ಸ್ತುತಿ ರೂಪನಾದವನು

682. ಸ್ತುತಿಃ = ಸ್ತೋತ್ರಕ್ರಿಯೆಯೂ ಆಗಿರುವವನು, ಬ್ರಹ್ಮಾದಿಗಳಿಂದಲೂ ಸ್ತೋತ್ರ ಮಾಡಲ್ಪಡುವವನು

683. ಸ್ತೋತಾ = ಸ್ತೋತ್ರ ಮಾಡುವವನೂ ತಾನೇ ಆಗಿರುವವನು

684. ರಣಪ್ರಿಯಃ = ಧರ್ಮರಕ್ಷಣಾರ್ಥವಾದ ಯುದ್ಧದಲ್ಲಿ ಪ್ರೀತಿಯುಳ್ಳವನು

685. ಪೂರ್ಣಃ = ಜ್ಞಾನಾನಂದಾದಿ ಗುಣಗಳಿಂದ ಪೂರ್ಣನಾದವನು

686. ಪೂರಯಿತಾ = (ಎಲ್ಲವನ್ನೂ ಸಂಪತ್ತಿನಿಂದ) ತುಂಬುವವನು

687. ಪುಣ್ಯಃ = ಪಾವನಗೊಳಿಸುವವನು, ಸುಖಕ್ಕೆ ಕಾರಣನಾದವನು

688. ಪುಣ್ಯಕೀರ್ತಿಃ = ಪಾವನವಾದ ಕೀರ್ತಿಯುಳ್ಳವನು

689. ಅನಾಮಯಃ = ವ್ಯಾಧಿಗಳಿಂದ ರಹಿತನಾದವನು

690. ಮನೋಜವಃ = ಮನಸ್ಸಿನ ವೇಗಕ್ಕೆ ಸಮಾನವಾದ ವೇಗವುಳ್ಳವನು, ಮನಸ್ಸಿಗೆ ವೇಗವನ್ನು ಉಂಟುಮಾಡುವವನು

691. ತೀರ್ಥಕರಃ = ಶಾಸ್ತ್ರಕರ್ತೃವು, ತೀರ್ಥಸದೃಶವಾದ ಪವಿತ್ರವಾದ ಹಸ್ತವುಳ್ಳವನು

692. ವಸುರೇತಾಃ = ವಸುದೇವತೆಗಳಿಗೆ ಕಾರಣನಾದವನು, ಸುವರ್ಣರೇತಸ್ಸುಳ್ಳವನು, ಸಂಪನ್ಮಯವಾದ ವೀರ್ಯವುಳ್ಳವನು

693. ವಸುಪ್ರದಃ = ಭಕ್ತರಿಗೆ ಹೇರಳವಾಗಿ ಧನವನ್ನು ಕೊಡುವವನು, ಕುಬೇರ ಸ್ವರೂಪಿಯು

694. ವಸುಪ್ರದಃ = ಮೋಕ್ಷರೂಪವಾದ ಫಲವನ್ನು ಕೊಡುವವನು, ದೈವೀ ಸಂಪತ್ತನ್ನು ಕೊಡುವವನು, ವಸುದೇವತಾ ಪದವಿಯನ್ನು ಕೊಡುವವನು

695. ವಾಸುದೇವಃ = ವಸುದೇವನ ಮಗನು, ಎಲ್ಲೆಲ್ಲಿಯೂ ವಾಸಮಾಡುವ ದೇವನು, ಜೀವಗಳಿಗೆ ಸ್ವಾಮಿಯಾದವನು, ವ್ಯೂಹದಲ್ಲಿ ಮೊದಲನೆಯವನು, ಜ್ಞಾನ-ಬಲ-ಐಶ್ವರ್ಯ-ವೀರ್ಯ-ಶಕ್ತಿ-ತೇಜಸ್ಸುಗಳೆಂಬ ಆರು ಗುಣಗಳಿಂದ ಸಂಪನ್ನನಾದವನು

696. ವಸುಃ = ವಸುದೇವತಾ ರೂಪಿಯು, ಸಂಪತ್ತುಗಳುಳ್ಳವನು, ಭೂತಗಳಿಗೆ ನಿವಾಸ ಸ್ಥಾನವಾದವನು

697. ವಸುಮನಾಃ = ವಸುದೇವತೆಗಳಲ್ಲಿ ಮನಸ್ಸುಳ್ಳವನು, ಸಂಪತ್ತಿನ ವಿಷಯದಲ್ಲಿ ಮನಸ್ಸುಳ್ಳವನು, ವಾಸ ಮಾಡುವ ಮನಸ್ಸುಳ್ಳವನು, ವಸುದೇವತೆಗಳ ಮನಸ್ಸು ಯಾವನಲ್ಲಿ ನೆಟ್ಟಿದೆಯೋ ಅವನು

698. ಹವಿಃ = ಅಗ್ನಿಯಲ್ಲಿ ದೇವತೆಗಳಿಗೆ ಹೋಮ ಮಾಡಲ್ಪಡಬೇಕಾದ ಹವಿಸ್ಸಿನ ರೂಪವಾಗಿರುವವನು

699. ಸದ್ಗತಿಃ = ಸತ್ಪುರುಷರಿಗೆ ಗತಿಯಾದವನು, ಒಳ್ಳೆಯ ಗತಿಯಾಗಿರುವವನು

700. ಸತ್ಕೃತಿಃ = (ಜಗತ್ತಿನ ರಕ್ಷಣೆಯೆಂಬ) ಶ್ರೇಷ್ಠವಾದ ಕೃತಿಗಳುಳ್ಳವನು

701. ಸತ್ತಾ = ಸದ್ಭಾವ ರೂಪನು

702. ಸದ್ಭೂತಿಃ = ಸತ್ಪುರುಷರಿಗೆ ಐಶ್ವರ್ಯವಾಗಿರುವವನು

703. ಸತ್ಪರಾಯಣಃ/ಸತ್ಪರಾಯಣಂ = ಸತ್ಪುರುಷರಿಗೆ ಶ್ರೇಷ್ಠತಮವಾದ ಸ್ಥಾನವಾಗಿರುವವನು

704. ಶೂರಸೇನಃ = ಶೂರಸೇನ ವಂಶದವನು, ಶೂರ-ವೀರರಾದ ಸೈನ್ಯಗಳುಳ್ಳವನು, ವಿಷ್ವಕ್ಸೇನ ರೂಪಿಯು

705. ಯದುಶ್ರೇಷ್ಠಃ = ಯಾದವರಲ್ಲಿ ಶ್ರೇಷ್ಠನಾದವನು

706. ಸನ್ನಿವಾಸಃ = ಸತ್ಪುರುಷರಿಗೆ ನಿವಾಸಭೂಮಿ/ಆಶ್ರಯ ಆಗಿರುವವನು

707. ಸುಯಾಮುನಃ = ಒಳ್ಳೆಯ ಯಮುನಾ ನಡಿಯ ಸಂಬಂಧವುಳ್ಳವನು, ಯಮುನಾತೀರ ವಾಸಿಗಳ ಪರಿವಾರವುಳ್ಳವನು

708. ಭೂತಾವಾಸಃ = ಭೂತಗಳಿಗೆ ನಿವಾಸಸ್ಥಾನ ಆಗಿರುವವನು

709. ವಾಸುದೇವಃ = ಸರ್ವಭೂತಗಳಲ್ಲಿಯೂ ನಿವಾಸ ಮಾಡುವ ಸ್ವಾಮಿಯು

710. ಸರ್ವಾಸುನಿಲಯಃ = ಎಲ್ಲ ಪ್ರಾಣಿಗಳಿಗೂ ನೆಲೆಮನೆ ಅಥವಾ ಲಯಸ್ಥಾನ ಆಗಿರುವವನು

711. ಅನಲಃ = ಅಗ್ನಿ ಸ್ವರೂಪಿಯು, ಶಕ್ತಿ-ಸಂಪತ್ತುಗಳ ಪರ್ಯಾಪ್ತಿ ಇಲ್ಲ್ದವನು

712. ದರ್ಪಹಾ = ದುಷ್ಟರ ದರ್ಪವನ್ನು ಧ್ವಂಸ ಮಾಡುವವನು

713. ದರ್ಪದಃ = ದುಷ್ಟರಿಗೆ ದರ್ಪವನ್ನುಂಟು ಮಾಡುವವನು, ಮಿತ್ರರಿಗೆ ವಿಜಯದರ್ಪವನ್ನು ಉಂಟು ಮಾಡುವವನು, ಮನ್ಮಥ ಮುಂತಾದವರಿಗೆ ರೂಪಾದಿ ದರ್ಪಗಳನ್ನು ಕೊಟ್ಟವನು

714. ದೃಪ್ತಃ/ಅದೃಪ್ತಃ = ನಿತ್ಯಾನಂದದಿಂದ ಉಬ್ಬಿದವನು, ಎಣೆಯಿಲ್ಲದ ಶೌರ್ಯಾದಿಗಳಿದ್ದರೂ ಆ ವಿಷಯದಲ್ಲಿ ದರ್ಪವಿಲ್ಲದವನು

715. ದುರ್ಧರಃ = ಧರಿಸಲು ಕಷ್ಟಸಾಧ್ಯನಾದವನು, ಇತರರಿಗೆ ಧರಿಸಲು ಕಷ್ಟಸಾಧ್ಯವಾದ ವೀರ್ಯ-ಪರಾಕ್ರಮ ಉಳ್ಳವನು

716. ಅಪರಾಜಿತಃ = ಸರ್ವ ವಿಜಯಿಯಾಗಿ ಯಾರಿಗೂ ಸೋಲದವನು

717. ವಿಶ್ವಮೂರ್ತಿಃ = ವಿಶ್ವರೂಪಿಯಾದವನು, ವಿಶ್ವವು ಯಾವನ ಆಕಾರವಾಗಿದೆಯೋ ಅವನು

718. ಮಹಾಮೂರ್ತಿಃ = ವಿಶ್ವಾಕಾರವಾದ ಹೆಚ್ಚು ದೊಡ್ಡ ಮೂರ್ತಿಯುಳ್ಳವನು

719. ದೀಪ್ತಮೂರ್ತಿಃ = ಜ್ಞಾನಪ್ರಕಾಶದಿಂದ ಉಜ್ಜ್ವಲವಾದ ಮೂರ್ತಿಯುಳ್ಳವನು, ತೇಜೋಮಯ ಮೂರ್ತಿಯು

720. ಅಮೂರ್ತಿಮಾನ್ = ಆಕಾರರಹಿತನು, ಪ್ರಾಕೃತವಾದ ಆಕಾರವಿಲ್ಲದವನು, ಕರ್ಮಜನ್ಯವಾದ ದೇಹವಿಲ್ಲದವನು

721. ಅನೇಕಮೂರ್ತಿಃ = ಅನಂತ ರೂಪಗಳುಳ್ಳವನು, ಸ್ವೇಚ್ಛೆಯಿಂದ ಸ್ವೀಕರಿಸಲ್ಪಟ್ಟ ನಾನಾ ರೂಪಗಳುಳ್ಳವನು

722. ಅವ್ಯಕ್ತಃ = ಅಜ್ಞಾನಿಗಳಿಗೆ ಪ್ರಕಾಶಗೊಳ್ಳದವನು, ಅಕ್ಷರ ರೂಪಿಯು, ವ್ಯಕ್ತನಂತೆ ಕಂಡರೂ ಅವ್ಯಕ್ತನಾಗಿರುವವನು

723. ಶತಮೂರ್ತಿಃ = ನೂರಾರು ಮೂರ್ತಿಗಳುಳ್ಳವನು

724. ಶತಾನನಃ = ನೂರಾರು ಬಾಯಿಗಳುಳ್ಳವನು

725. ಏಕಃ = ಅದ್ವಿತೀಯನು

726. ನೈಕಃ = ಅನೇಕರೂಪಗಳುಳ್ಳವನು

727. ಸವಃ = ಯಜ್ಞ/ಸೋಮಯಜ್ಞ ಸ್ವರೂಪಿಯು, ಅವ್ಯಕ್ತನಾಗಿರುವುದರಿಂದ ಅವತಾರರೂಪದಲ್ಲಿ ವ್ಯಕ್ತನಾದಾಗ 'ನೀನು-ನಿಮ್ಮವನು' ಎಂದು ಕರೆಯಲ್ಪಡುವವನು

728. ಕಃ = ಇವನು ಯಾರು? ಎಂಬ ಪ್ರಶ್ನೆಗೆ ವಿಷಯವಾಗಿರುವವನು, ಬ್ರಹ್ಮ/ಸುಖ ಸ್ವರೂಪಿಯು

729. ಕಿಂ = ಇದು ಯಾವುದು? ಎಂಬ ವಿಚಾರ ಮಾಡಲ್ಪಡಬೇಕಾದ ಪರಬ್ರಹ್ಮವಸ್ತುವು

730. ಯತ್ = ಸ್ವತಃ ಸಿದ್ಧನಾದವನು, ಅತ್ಯಂತ ಪ್ರಸಿದ್ಧನಾದವನು

731. ತತ್ = ಇಂದ್ರಿಯಗಳಿಗೆ ಅತಿ ದೂರನಾಗಿ ಗ್ರಹಿಸಲು ಕಷ್ಟ ಸಾಧ್ಯನಾಗಿರುವುದರಿಂದ 'ಅದು' ಎಂದು ಹೇಳಲ್ಪಡುವವನು

732. ಪದಮನುತ್ತಮಂ = ಯಾವುದಕ್ಕಿಂತಲೂ ಉತ್ತಮವಾದುದು ಬೇರೊಂದಿಲ್ಲವೋ ಅಂತಹ ಪರಮೋತ್ತಮವಾದ ಸ್ಥಾನವು/ಗತಿಯು

733. ಲೋಕಬಂಧುಃ = ಸಮಸ್ತ ಲೋಕಗಳಿಗೂ ನೆಂಟನಾಗಿರುವವನು

734. ಲೋಕನಾಥಃ = ಸಮಸ್ತ ಲೋಕಗಳಿಗೂ ಒಡೆಯನಾದವನು

735. ಮಾಧವಃ = ಲಕ್ಷ್ಮೀ ಪತಿಯು, ಮಧು ವಂಶದಲ್ಲಿ ಹುಟ್ಟಿದವನು, ಮಧುವಿಗೆ/ಪರಮಾನಂದಕ್ಕೆ ಸೇರಿದವನು

736. ಭಕ್ತವತ್ಸಲಃ = ಭಕ್ತರಲ್ಲಿ ವಾತ್ಸಲ್ಯವನ್ನು/ಪರಮ-ಪ್ರೇಮವನ್ನು ಹೊಂದಿರುವವನು (ಹಸುವು ಆಗ ತಾನೇ ಈದ ತನ್ನ ಕರುವಿನಲ್ಲಿ ತೋರುವ ಹಾಗೆ)

737. ಸುವರ್ಣವರ್ಣಃ = (ಒರೆ ಹಚ್ಚಿದ) ಚಿನ್ನದ ಬಣ್ಣಕ್ಕೆ ಸಮನಾದ ಬಣ್ಣವುಳ್ಳವನು

738. ಹೇಮಾಂಗಃ = ಚಿನ್ನಕ್ಕೆ ಸಮಾನವಾದ ಅವಯವ/ದೇಹ ಉಳ್ಳವನು, ಬಣ್ಣ, ಕಾಂತಿ, ಅಮೂಲ್ಯತೆ, ರಮಣೀಯತೆ ಮುಂತಾದವುಗಳಿಂದ ಚಿನ್ನಕ್ಕೆ ಸದೃಶವಾದ ಅಂಗಗಳುಳ್ಳವನು

739. ವರಾಂಗಃ = ಶ್ರೇಷ್ಠವಾದ ಅಂಗಗಳುಳ್ಳವನು

740. ಚಂದನಾಂಗದೀ = ಭೂಮಿಯಲ್ಲಿ ಅವತರಿಸಿದಾಗ ಶ್ರೀಗಂಧದಿಂದ ರಚಿತವಾದ/ಪೂಜಿಸಿದ (ಆಹ್ಲಾದಕರವಾದ) ತೋಳ್ಬಳೆಗಳುಳ್ಳವನು, ಶ್ರೀಗಂಧದಿಂದ ಪೂಜಿಸಲ್ಪಟ್ಟವನು, ಕೇಯೂರಗಳಿಂದ ಅಲಂಕೃತನಾದವನು

741. ವೀರಹಾ = ವೀರರಾದ ಶತ್ರುಗಳನ್ನು ಸಂಹರಿಸಿದವನು

742. ವಿಷಮಃ/ಅವಿಷಮಃ = ಎಣೆಯಿಲ್ಲದವನು/ ವೈಷಮ್ಯವಿಲ್ಲದವನು, ಸರ್ವಸಮನಾದವನು, ಋಜು ಬುದ್ಧಿಯುಳ್ಳವರಿಗೆ ಕ್ಷೇಮವನ್ನೂ, ಕುಟಿಲ ಬುದ್ಧಿಯುಳ್ಳವರಿಗೆ ಭೀತಿಯನ್ನೂ ಉಂಟು ಮಾಡುವವನು

743. ಶೂನ್ಯಃ = ಆಕಾಶ ಸದೃಶನಾದವನು, ವಿಶೇಷ ರಹಿತನಾದವನು, ದೋಷ ಶೂನ್ಯನಾದವನು, ಸರ್ವಪದಾರ್ಥ ಶೂನ್ಯನಾದವನು

744. ಘೃತಾಶೀ = ಹವ್ಯ-ಕವ್ಯ ರೂಪವಾದ ತುಪ್ಪದಲ್ಲಿ ಆಸೆಯುಳ್ಳವನು, ಗೋಪಾಲಕರ ಮನೆಯ ತುಪ್ಪದಲ್ಲಿ ಆಶೆಯುಳ್ಳವನು, ಪೂರ್ಣ ಕಾಮನಾದುದರಿಂದ ಕರಗಿ ಹೋದ ಆಶೆಗಳುಳ್ಳವನು

745. ಅಚಲಃ = ತನ್ನ ಸ್ವರೂಪ-ಸ್ಥಾನಗಳಿಂದ ಚಲಿಸದವನು, ಮಹಾಪರ್ವತ ವಿಭುತಿ ರೂಪನು, ಪರ್ವತ ಸದೃಶನು

746. ಚಲಃ = ಅಚಲನಾಗಿದ್ದರೂ ಅವತಾರ ರೂಪದಲ್ಲಿ ಚಲಿಸುವವನು, ಜಗತ್ತಿನ ಚಲನೆಗೆ ಕಾರಣನಾದವನು, ಚರವಿಭೂತಿ ಉಳ್ಳವನು, ವಾಯುರೂಪದಿಂದ ಚಲಿಸುವವನು

747. ಅಮಾನೀ = ಅಭಿಮಾನವಿಲ್ಲದವನು, ಅಳತೆಯಿಲ್ಲದವನು

748. ಮಾನದಃ = ಭಕ್ತರಿಗೆ ಬಹುಮಾನಗಳನ್ನು ಕೊಡುವವನು, ಸಂಸಾರದಲ್ಲಿ ಅಭಿಮಾನವನ್ನು ಉಂಟು ಮಾಡುವವನು, ಸತ್ಕರಿಸುವವನು, ತತ್ತ್ವಜ್ಞರಿಗೆ ಅನಾತ್ಮಗಳಲ್ಲಿ ಆತ್ಮಾಭಿಮಾನವನ್ನು ಖಂಡಿಸುವವನು

749. ಮಾನ್ಯಃ = ಪೂಜ್ಯರಿಗೂ ಪೂಜ್ಯನಾದವನು

750. ಲೋಕಸ್ವಾಮೀ = ಸಮಸ್ತ ಲೋಕಗಳಿಗೂ ಸ್ವಾಮಿಯು

751. ತ್ರಿಲೋಕಧೃಕ್ = ಮೂರು ಲೋಕಗಳನ್ನೂ ಧರಿಸಿರುವವನು

752. ಸುಮೇಧಾಃ = ಒಳ್ಳೆಯ ಮೇಧಾಶಕ್ತಿ ಉಳ್ಳವನು, ಮೇಧಾ ಹಯಗ್ರೀವ ಮತ್ತು ಮೇಧಾ ದಕ್ಷಿಣಾಮೂರ್ತಿ ರೂಪನಾದವನು

753. ಮೇಧಜಃ = ಯಜ್ಞದಲ್ಲಿ ಹುಟ್ಟುವವನು/ಪ್ರಕಟವಾಗುವವನು, ಮೇಧಾಶಕ್ತಿಯಿಂದ ಆವಿರ್ಭವಿಸುವವನು

754. ಧನ್ಯಃ = ಸುಕೃತಿಯು, ಕೃತಾರ್ಥನಾದವನು

755. ಸತ್ಯಮೇಧಾಃ = ಅಮೋಘವಾದ ಮೇಧೆಯುಳ್ಳವನು

756. ಧರಾಧರಃ = ಭೂಮಿಯನ್ನು ಧರಿಸಿರುವವನು

757. ತೇಜೋವೃಷಃ = ತೇಜಸ್ಸನ್ನು ಮಳೆಯಂತೆ ಚೆಲ್ಲುವವನು, ಆದಿತ್ಯ ರೂಪದಿಂದ ಜಲವನ್ನು ವರ್ಷಿಸುವವನು

758. ದ್ಯುತಿಧರಃ = ಎಲ್ಲರ ಕಾಂತಿಯನ್ನೂ ಮೀರಿಸಿದ ಕಾಂತಿಯನ್ನು ಧರಿಸಿರುವವನು

759. ಸರ್ವಶಸ್ತ್ರಭೃತಾಂ ವರಃ = ಶಸ್ತ್ರಧಾರಿಗಳಲ್ಲಿ ಶ್ರೇಷ್ಠವಾದ ಶ್ರೀರಾಮ ರೂಪಿಯು

760. ಪ್ರಗ್ರಹಃ = ಸಂಸಾರಪಾಶ ಸ್ವರೂಪನು, ಇಂದ್ರಿಯಗಳೆಂಬ ಕುದುರೆಗಳನ್ನು ಕಟ್ಟಿಹಾಕುವವನು, ರಜ್ಜು ಸ್ವರೂಪನು, ನೆಚ್ಚಿದ ಭಕ್ತರನ್ನು ಚೆನ್ನಾಗಿ ಹಿಡಿದುಕೊಳ್ಳುವವನು, ಕಡಿವಾಣವನ್ನು ಧರಿಸಿದವನು

761. ನಿಗ್ರಹಃ = ಶತ್ರುಗಳನ್ನು ನಿಗ್ರಹಿಸುವವನು

762. ವ್ಯಗ್ರಃ/ಅವ್ಯಗ್ರಃ = (ಭಕ್ತರ ರಕ್ಷಣೆಯಲ್ಲಿ) ಅತ್ಯಂತಾಸಕ್ತನಾದವನು, ಸ್ವಾನಂದದಲ್ಲಿ ಸಂತೃಪ್ತನಾದ್ದರಿಂದ ಅನಾಸಕ್ತನಾದವನು, ಚಿಂತೆಯಿಲ್ಲದವನು

763. ನೈಕಶೃಂಗಃ = ಅನೇಕ ಶೃಂಗಗಳುಳ್ಳವನು, ಪ್ರಣವ ಸ್ವರೂಪಿಯು, ಅಗ್ನಿ ರೂಪಿಯು

764. ಗದಾಗ್ರಜಃ = ಗದನಿಗೆ ಅಣ್ಣನಾದವನು

765. ಚತುರ್ಮೂರ್ತಿಃ = ಕೃತಯುಗಾದಿಗಳ ಭೇದಕ್ಕೆ ಅನುಗುಣವಾಗಿ ಕ್ರಮವಾಗಿ ಬಿಳುಪು, ಕೆಂಪು, ಹಳದಿ ಮತ್ತು ಕಪ್ಪು ಬಣ್ಣಗಳ ಮೂರ್ತಿಗಳುಳ್ಳವನು, ವಿಶ್ವ-ತೈಜಸ-ಪ್ರಾಜ್ಞ-ತುರೀಯಗಳೆಂಬ ನಾಲ್ಕು ಮೂರ್ತಿಗಳುಳ್ಳವನು

766. ಚತುರ್ಬಾಹುಃ = ನಾಲ್ಕು ತೋಳುಗಳುಳ್ಳವನು

767. ಚತುವ್ರ್ಯೂಹಃ = ವಾಸುದೇವ, ಸಂಕರ್ಷಣ, ಅನಿರುದ್ಧ, ಪ್ರದ್ಯುಮ್ನ - ಎಂಬ ನಾಲ್ಕು ವ್ಯೂಹಗಳುಳ್ಳವನು

768. ಚತುರ್ಗತಿಃ = ನಾಲ್ಕು ಪುರುಷಾರ್ಥಗಳನ್ನೂ ಪಡೆಯಲು ಗತಿಯಾದವನು, ಆರ್ತ-ಜಿಜ್ಞಾಸು-ಅರ್ಥಾರ್ಥೀ-ಜ್ಞಾನೀ ಎಂಬ ನಾಲ್ಕು ಬಗೆಯ ಭಕ್ತರಿಗೂ ಗತಿಯಾದವನು, ನಾಲ್ಕು ವರ್ಣಗಳಿಗೂ, ನಾಲ್ಕು ಆಶ್ರಮಗಳಿಗೂ ಏಕಮಾತ್ರ ಲಕ್ಷ್ಯನಾದವನು

769. ಚತುರಾತ್ಮಾ = ಪರಮಾತ್ಮಾ, ಜೀವ, ಮನಸ್ಸು ಮತ್ತು ಅಹಂಕಾರ ಎಂಬ ನಾಲ್ಕು ರೂಪವುಳ್ಳವನು

770. ಚತುರ್ಭಾವಃ = ವ್ಯೂಹ, ವಿಭವ, ಅಂತರ್ಯಾಮೀ, ಅರ್ಚಾಮೂರ್ತಿ ಎಂಬ ನಾಲ್ಕು ಭಾವಗಳಲ್ಲಿ ಇರುವವನು, ಸಾಲೋಕ್ಯ-ಸಾಮೀಪ್ಯ-ಸಾರೂಪ್ಯ-ಸಾಯುಜ್ಯ ಎಂಬ ನಾಲ್ಕು ಮೋಕ್ಷಭಾವಗಳಲ್ಲಿ ಇರುವವನು

771. ಚತುರ್ವೇದವಿತ್ = ನಾಲ್ಕು ವೇದಗಳನ್ನೂ ವೇದಾರ್ಥಗಳನ್ನೂ ಪ್ರಾಮಾಣಿಕವಾಗಿ ಸಂಪೂರ್ಣವಾಗಿ ತಿಳಿದವನು

772. ಏಕಪಾತ್ = ವಿಶ್ವರೂಪವಾದ ಏಕಪಾದವುಳ್ಳವನು, ಏಕಾಂಶದಿಂದ ಜಗತ್ತನ್ನು ವ್ಯಾಪಿಸಿರುವವನು

773. ಸಮಾವರ್ತಃ = ಸೃಷ್ಟಿ-ಸ್ಥಿತಿ-ಲಯ ವ್ಯಾಪಾರಗಳ ವರ್ತನೆಯನ್ನು ಉಂಟುಮಾಡುವವನು, ಮತ್ತೆ-ಮತ್ತೆ ಅವತಾರ ರೂಪಗಳಲ್ಲಿ ಕಾಣಿಸಿಕೊಳ್ಳುವವನು, ಏಕರೂಪವಾಗಿ ವರ್ತಿಸುವವನು

774. ಅನಿವೃತ್ತಾತ್ಮಾ/ನಿವೃತ್ತಾತ್ಮಾ = ಎಲ್ಲ ಕಡೆಗಳಲ್ಲಿಯೂ ಇರುವುದರಿಂದ ಎಲ್ಲಿಂದಲೂ ನಿವೃತ್ತಿ ಹೊಂದದ ಆತ್ಮವುಳ್ಳವನು, ವಿಷಯಗಳಿಂದ ಹಿಂದಿರುಗಿದ ಮನಸ್ಸುಳ್ಳವನು

775. ದುರ್ಜಯಃ = ಜಯಿಸಲು ಅಸಾಧ್ಯವಾದವನು, ಬಹಳ ಕಷ್ಟದಿಂದ ಉಪಾಸಕ ಭಕ್ತರಿಂದ ವಶಪಡಿಸಿಕೊಳ್ಳಲು ಸಾಧ್ಯನಾದವನು

776. ದುರತಿಕ್ರಮಃ = ಅತಿಕ್ರಮಿಸಲು/ಉಲ್ಲಂಘಿಸಲು/ಮೀರಿ-ಹೋಗಲು ಅಸಾಧ್ಯನಾಗಿರುವವನು, ಸರ್ವೋತ್ತಮನೂ, ಸರ್ವವ್ಯಾಪಕನೂ ಆದ್ದರಿಂದ ಮೀರಲು ಸಾಧ್ಯವಿಲ್ಲ, ಸರ್ವೇಶ್ವರನಾದ್ದರಿಂದ ಆಜ್ಞೆಯನ್ನು ಯಾರೂ ಉಲ್ಲಂಘಿಸಲಾರರು

777. ದುರ್ಲಭಃ = ಪಡೆಯಲು ಅತಿಕಷ್ಟ ಸಾಧ್ಯನಾದವನು, ಯೋಗಿಗಳಿಗೆ ಮಾತ್ರವೇ ಪಡೆಯಲು ಸಾಧ್ಯನಾದವನು

778. ದುರ್ಗಮಃ = ಅತಿಕಷ್ಟದಿಂದ ಹೊಂದಲ್ಪಡುವವನು

779. ದುರ್ಗಃ = ಗಹನವಾದ ತತ್ತ್ವವು, ದುಃಖ/ಅತಿಕಷ್ಟ-ದಿಂದ ಹೊಂದಲ್ಪಡುವವನು, ದುರ್ಗ/ಗಹನ-ವಾದ ಮಾರ್ಗವುಳ್ಳವನು

780. ದುರಾವಾಸಃ = ಯೋಗಿಗಳಿಗೆ ಅತಿಕಷ್ಟದಿಂದ ಚಿತ್ತದಲ್ಲಿ ನೆಲೆಸಲ್ಪಡಲು ಸಾಧ್ಯನಾದವನು, ಇತರರಿಂದ ಅತಿಕಷ್ಟದಿಂದ ಹೊಂದಲ್ಪಡುವ ಪರಮಪದ ನಿವಾಸವುಳ್ಳವನು

781. ದುರಾರಿಹಾ = ದುರ್ಮಾರ್ಗಗಾಮಿಗಳನ್ನು (ದಾನವರನ್ನು) ಸಂಹರಿಸುವವನು

782. ಶುಭಾಂಗಃ = ಯೋಗಿಗಳಿಗೆ ಶುಭಾಲಂಬನಕ್ಕೆ ಯೋಗ್ಯವಾದ ಶುಭ ಅವಯವಗಳುಳ್ಳವನು

783. ಲೋಕಸಾರಂಗಃ = ಚಿತ್ರ-ವಿಚಿತ್ರವಾದ ಆಕರ್ಷಕವಾದ ಮತ್ತು ಆಕಾರದಿಂದಲೂ ಗತಿವೇಗದಿಂದಲೂ ಸಾರಂಗದಂತಿರುವ ಜಗತ್ತುಳ್ಳವನು, ದುಂಬಿಗಳು ಪುಷ್ಪಗಳ ಸಾರವನ್ನು ಗ್ರಹಿಸುವಂತೆ ಲೋಕಗಳ ಸಾರವನ್ನು ಗ್ರಹಿಸುವವನು

784. ಸುತಂತುಃ = ಶೋಭನವಾದ/ ವಿಜ್ಞಾನರೂಪವಾದ/ಧರ್ಮ-ರೂಪವಾದ ವಿಸ್ತಾರವುಳ್ಳವನು

785. ತಂತುವರ್ಧನಃ = ವಿಸ್ತಾರದಿಂದ (ವಿಜ್ಞಾನಧರ್ಮ-ರೂಪವಾದ) ತಂತುವನ್ನು ವೃದ್ಧಿಪಡಿಸುವವನು

786. ಇಂದ್ರಕರ್ಮಾ = ಉಪೇಂದ್ರನಾಗಿ ಇಂದ್ರನಿಗೋಸ್ಕರ ದೈತ್ಯ ಸಂಹಾರ ಮುಂತಾದ ಕೆಲಸ ಮಾಡುವವನು, ಪರಮೈಶ್ವರ್ಯಕ್ಕೆ ಅನುಗುಣವಾದ ಕರ್ಮಗಳುಳ್ಳವನು

787. ಮಹಾಕರ್ಮಾ = ಜಗದ್ರೂಪವಾದ ದೊಡ್ಡ ಕರ್ಮಗಳುಳ್ಳವನು

788. ಕೃತಕರ್ಮಾ = ಕೃತಕೃತ್ಯನಾದವನು, ಆಪ್ತಕಾಮನು

789. ಕೃತಾಗಮಃ = ಆಗಮ ಶಾಸ್ತ್ರವನ್ನು ಮಾಡಿದವನು, ಶಾಸ್ತ್ರ ಕರ್ತೃವು

790. ಉದ್ಭವಃ = ಅವತಾರ ರೂಪದಲ್ಲಿ ಕಾಣಿಸಿಕೊಂಡವನು, ಸೃಷ್ಟಿಗೆ ಕಾರಣನಾದವನು, ಸಂಸಾರದಿಂದ (ಭವದಿಂದ) ಬಿಡುಗಡೆ ಹೊಂದಿಸುವವನು

791. ಸುಂದರಃ = ಎಣೆಯಿಲ್ಲದ ಸೌಂದರ್ಯವುಳ್ಳವನು

792. ಸುಂದಃ = ಶುಭವಾದ ಕರುಣೆಯಿಂದ ಆದ್ರ್ರನಾದವನು

793. ರತ್ನನಾಭಃ = ರತ್ನದಂತೆ ಹೊಳೆಯುವ ನಾಭಿಯುಳ್ಳವನು

794. ಸುಲೋಚನಃ = ಸುಂದರವಾದ ಕಣ್ಣುಗಳುಳ್ಳವನು, ಶ್ರೇಷ್ಠವಾದ ಜ್ಞಾನದೃಷ್ಟಿಯುಳ್ಳವನು

795. ಅರ್ಕಃ = ಸೂರ್ಯ ರೂಪಿಯು, ಪೂಜ್ಯನಾದವನು

796. ವಾಜಸನಃ = ಅಶ್ವ ರೂಪನಾದ ಸೂರ್ಯನು, ವಾಜಸನ ಸಂಹಿತೆಯನ್ನು ಯಾಜ್ಞವಲ್ಕ್ಯರಿಗೆ ಉಪದೇಶ ಮಾಡಿದವನು, ಅನ್ನವನ್ನು ಅನುಗ್ರಹಿಸಿದವನು

797. ಶೃಂಗೀ = ಏಕಶೃಂಗವುಳ್ಳ ವರಾಹಸ್ವಾಮಿಯು, ನಾಲ್ಕು ಶೃಂಗಗಳುಳ್ಳ ಪ್ರಣವ/ಅಗ್ನಿ, ಮತ್ಸ್ಯಾವತಾರಿಯು, ಶಿಖರ ಸ್ಥಾನದಲ್ಲಿರುವವನು

798. ಜಯಂತಃ = ಶತ್ರುಗಳನ್ನು ಜಯಿಸುವವನು, ಇಂದ್ರಪುತ್ರ ರೂಪಿಯು

799. ಸರ್ವವಿಜ್ಜಯೀ = ಸರ್ವಜ್ಞನಾಗಿ ಎಲ್ಲ ಒಳ-ಹೊರ ಶತ್ರುಗಳನ್ನೂ ಜಯಿಸುವ ಸ್ವಭಾವವುಳ್ಳವನು

800. ಸುವರ್ಣಬಿಂದುಃ = ನಾದಕ್ಕೆ ಮೂಲವಾದ ಸುವರ್ಣಬಿಂದು ಸ್ವರೂಪನು

801. ಅಕ್ಷೋಭ್ಯಃ = ಯಾರಿಂದಲೂ ಯಾವುದರಿಂದಲೂ ಕ್ಷೋಭೆಗೆ ಒಳಗಾಗದವನು, ಅವಿಕಾರಿಯು

802. ಸರ್ವವಾಗೀಶ್ವರೇಶ್ವರಃ = ವಾಣಿಗೆ ಈಶ್ವರರಾದ ಬೃಹಸ್ಪತಿ-ಬ್ರಹ್ಮ ಮೊದಲಾದಎ ಎಲ್ಲ ದೇವತೆಗಳಿಗೂ ಈಶ್ವರನಾದವನು

803. ಮಹಾಹ್ರದಃ = ಜ್ಞಾನಿಗಳು ಅತ್ಯಂತ ಹಿತವಾಗಿ ಅವಗಾಹನ ಮಾಡುವ ಮಹಾನಂದ ಸರೋವರದ ಮಡುವಿನಂತೆ ಇರುವವನು

804. ಮಹಾಗರ್ತಃ = ಹೃದಯ ಗುಹೆಯ ದೊಡ್ಡ ಆಳದಲ್ಲಿ ಇರುವವನು, ಆಳವಾದ ಸಂಸಾರ/ಮಾಯೆ ಎಂಬ ಹಳ್ಳವುಳ್ಳವನು

805. ಮಹಾಭೂತಃ = ಪರಮ ಸ್ವರೂಪಿಯು, ಅಖಂಡ ಸತ್ತಾ ಸ್ವರೂಪಿಯು, ಪಂಚಮಹಾಭೂತ ಸ್ವರೂಪಿಯು

806. ಮಹಾನಿಧಿಃ = ಅಕ್ಷಯವಾದ ಪರಮಾನಂದ ಶಾಂತಿನಿಧಿ ಆಗಿರುವವನು, ಯೋಗಿಗಳಿಗೆ ಅಮೂಲ್ಯವಾದ ನಿಧಿಯಾಗಿರುವವನು

807. ಕುಮುದಃ = ಭೂಮಿಗ ಆನಂದವನ್ನು ಉಂಟುಮಾಡುವವನು

808. ಕುಂದರಃ = ಭೂಮಿಯನ್ನು ಸೀಳಿದ ವರಾಹರೂಪಿಯು

809. ಕುಂದಃ = ಮಾಘಮಾಸದ ಮಲ್ಲೆ ಹೂವಂತೆ ಶುದ್ಧವಾಗಿರುವವನು, ಭೂಮಿಯನ್ನು ಅನುಗ್ರಹಿಸಿದವನು

810. ಪರ್ಜನ್ಯಃ = ತಾಪತ್ರಯಗಳನ್ನು ಶಮನಗೊಳಿಸುವವನು, ಎಲ್ಲ ಇಷ್ಟಾರ್ಥಗಳನ್ನು ಮಳೆಗರೆಯುವ ಪರ್ಜನ್ಯ ದೇವತಾ ರೂಪಿಯು

811. ಪಾವನಃ = ಪವಿತ್ರಗೊಳಿಸುವವನು

812. ಅನಿಲಃ = ವಾಯುದೇವತಾ ಸ್ವರೂಪಿಯು, ಪ್ರೇರಕರಿಲ್ಲದವನು

813. ಅಮೃತಾಶಃ = ಪರಮಾನಂದಾಮೃತವನ್ನು ಭೋಗಿಸುವವನು

814. ಅಮೃತವಪುಃ = ಅಮೃತಮಯವಾದ ಶರೀರವುಳ್ಳವನು

815. ಸರ್ವಜ್ಞಃ = ಎಲ್ಲವನ್ನೂ ತಿಳಿದವನು

816. ಸರ್ವತೋಮುಖಃ = ಎಲ್ಲ ಕಡೆಗಳಲ್ಲಿಯೂ ಮುಖಗಳುಳ್ಳವನು

817. ಸುಲಭಃ = ಭಕ್ತರಿಗೆ ಸುಲಭೋಪಾಯಗಳಿಂದ ಸುಲಭವಾಗಿ ದೊರೆಯುವವನು

818. ಸುವ್ರತಃ = ರಕ್ಷಣೆಯೆಂಬ ಒಳ್ಳೆಯ ವ್ರತವುಳ್ಳವನು, ಒಳ್ಳೆಯ ವ್ರತಗಳ ಆಚರಣೆಯಿಂದ ದೊರಕುವವನು, ಒಳ್ಳೆಯ ವ್ರತಗಳನ್ನು ಆಚರಿಸುವವನು

819. ಸಿದ್ಧಃ = ಸ್ವತಃ ಸಿದ್ಧನಾದವನು, ಸ್ವಾಧೀನವಾದ ಸರ್ವಸಿದ್ಧಿಗಳುಳ್ಳವನು

820. ಶತ್ರುಜಿತ್ = ಒಳ-ಹೊರ ಶತ್ರುಗಳನ್ನು ಜಯಿಸಿದವನು

821. ಶತ್ರುತಾಪನಃ = ಒಳ-ಹೊರ ಶತ್ರುಗಳಿಗೆ ಸಂತಾಪವನ್ನು ಉಂಟುಮಾಡುವವನು

822. ನ್ಯಗ್ರೋಧಃ = (ಸಂಸಾರ) ವಟವೃಕ್ಷ ಸ್ವರೂಪಿಯಾದವನು, ಕೆಳ ಮುಖವಾಗಿ ಬೆಳೆಯುವ ವಿಶ್ವ/ದೇಹ ವಟವೃಕ್ಷ ರೂಪಿಯು

823. ಉದುಂಬರಃ = ಅತ್ತೀ ಮರವೆಂಬ ಯಜ್ಞ ವೃಕ್ಷ ರೂಪಿಯಾದವನು

824. ಅಶ್ವತ್ಥಃ = ಅಶ್ವತ್ಥ ವೃಕ್ಷ ಸ್ವರೂಪಿಯು, ಊಧ್ರ್ವ ಮೂಲ - ಅಧಃ ಶಾಖವೂ ಆದ ಸಂಸಾರ/ದೇಹ ಎಂಬ ಅಶ್ವತ್ಥ ವೃಕ್ಷ ರೂಪಿಯು

825. ಚಾಣೂರಾಂಧ್ರನಿಷೂದನಃ = ಆಂಧ್ರ ಜಾತಿಗೆ ಸೇರಿದ ಚಾಣುರನೆಂಬ ದ್ವೇಷರೂಪಿಯಾದ ಜಟ್ಟಿಯನ್ನು ಸಂಹರಿಸಿದವನು

826. ಸಹಸ್ರಾರ್ಚಿಃ = ಅನಂತವಾದ ಕಿರಣಗಳುಳ್ಳವನು, ಸೂರ್ಯ ರೂಪಿಯು

827. ಸಪ್ತಜಿಹ್ವಃ = ಕಾಳೀ-ಕರಾಳೀ ಮುಂತಾದ ಏಳು ನಾಲಿಗೆಗಳುಳ್ಳ ಅಗ್ನಿದೇವ ಸ್ವರೂಪನು

828. ಸಪ್ತೈಧಾಃ = ಏಳು ಬಗೆಯ ಸಮಿತ್ತುಗಳುಳ್ಳ ಅಗ್ನಿದೇವ ಸ್ವರೂಪಿಯು

829. ಸಪ್ತವಾಹನಃ = ಸಪ್ತಪ್ರಾಣ ರೂಪವಾದ ಏಳು ಕುದುರೆಗಳುಳ್ಳ ಸೂರ್ಯದೇವ ಸ್ವರೂಪಿಯು

830. ಅಮೂರ್ತಿಃ = ನಿರಾಕಾರನು, ಪ್ರಾಕೃತವಾದ ಅವಯವಗಳಿಲ್ಲದವನು

831. ಅನಘಃ = ಪಾಪರಹಿತನು

832. ಅಚಿಂತ್ಯಃ = ಅತೀಂದ್ರಿಯನಾದ್ದರಿಂದ ಸರ್ವವಿಲಕ್ಷಣನಾದ್ದರಿಂದ ಚಿಂತಿಸಲು ಅಸಾಧ್ಯನಾದವನು

833. ಭಯಕೃತ್ = ದುಷ್ಟರಿಗೆ ಭಯವನ್ನು ಉಂಟುಮಾಡುವವನು, ಶತ್ರುಗಳಿಗೆ ಭಯಂಕರನಾದವನು, ಅಧ್ಯಾತ್ಮ ಯಾತ್ರಿಗಳಿಗೆ ಮಾರ್ಗದಲ್ಲಿ (ಮೊದಲಿಗೆ) ಭಯವನ್ನು ಉಂಟು ಮಾಡುವವನು

834. ಭಯನಾಶನಃ = ಸಾಧುಗಳದ ಭಕ್ತರಿಗೆ ಭಯವನ್ನು ನಾಶ ಮಾಡುವವನು, ಅಧ್ಯಾತ್ಮ ಯಾತ್ರಿಗಳಿಗೆ ಮಾರ್ಗದಲ್ಲಿ (ನಂತರ) ಭಯವನ್ನು ಹೋಗಲಾಡಿಸುವವನು

835. ಅಣುಃ = ಅತ್ಯಂತ ಸೂಕ್ಷ್ಮನಾದವನು

836. ಬೃಹತ್ = ಅತ್ಯಂತ ಮಹತ್ತ್ವವುಳ್ಳವನು

837. ಕೃಶಃ = ಲಘುತಮನಾದವನು, ಸ್ಥೂಲ-ಸೂಕ್ಷ್ಮ ಪ್ರಕೃತಿ ಸಂಬಂಧವಿಲ್ಲದವನು, ವೃದ್ಧಿ-ಕ್ಷಯಗಳಿಗೆ ಒಳಪಟ್ಟ ಹದಿನೈದು ಕಲೆಗಳನ್ನು ಬಿಟ್ಟಿರುವ ಕಲಾಮಾತ್ರ ಸ್ವರೂಪನು

838. ಸ್ಥೂಲ = ಪ್ರಕೃತಿ ಕಲೆಗಳ ಸಂಬಂಧದಿಂದ ಸ್ಥೂಲನಾದವನು, ಸರ್ವಾತ್ಮಕನು

839. ಗುಣಭೃತ್ = ಕಲ್ಯಾಣ ಗುಣಸಂಪನ್ನನು, ಲೋಕ ವ್ಯಾಪಾರಕ್ಕೋಸ್ಕರ ತ್ರಿಗುಣ ರೂಪವಾದ ಕವಚವನ್ನು ಧರಿಸಿರುವವನು

840. ನಿರ್ಗುಣಃ = ಪ್ರಾಕೃತ ಗುಣರಹಿತನು, ಶುದ್ಧನು

841. ಮಹಾನ್ = ಮಹತ್ತಮನು, ಯಾವ ವಿಷಯದಲ್ಲಿಯೂ ಕೀಳ್ಮೆ ಇಲ್ಲದವನು

842. ಅಧೃತಃ = ಯಾರಿಂದಲೂ ಧರಿಸಲ್ಪಡದವನು, ಪೋಷಿಸಲ್ಪಡದವನು

843. ಸ್ವಧೃತಃ = ತನ್ನಿಂದಲೇ ಧರಿಸಲ್ಪಟ್ಟವನು

844. ಸ್ವಾಸ್ಯಃ = ಶ್ರೇಷ್ಠವಾದ ಮುಖವುಳ್ಳವನು, ವೇದ/ಬ್ರಾಹ್ಮಣ ಮುಖವುಳ್ಳವನು

845. ಪ್ರಾಗ್ವಂಶಃ = ಪೂರ್ವ(ಪ್ರವೃತ್ತಿ) ಮುಖವಾದ ವಂಶವುಳ್ಳವನು, ಯಜ್ಞಶಲೆಯಲ್ಲಿ ಪೂರ್ವಕ್ಕೆ ಇರುವ ಪೂರ್ವಾಗ್ರವಾದ ಪ್ರಾಗ್ವಂಶವೆಂಬ ಗ್ರಹ ರೂಪಿಯಾದವನು, ಪ್ರಾಗ್ವಂಶವೆಂಬ ಸ್ಥಳದಲ್ಲಿ ಸ್ತುತಿಸಲ್ಪಡುವವನು, ವಂಶ ವಿಸ್ತಾರಕ್ಕೆ ಹಿಂದಿರುವವನು

846. ವಂಶವರ್ಧನಃ = ಪ್ರಪಂಚ ರೂಪವಾದ ವಂಶವೃದ್ಧಿಗೆ ಕಾರಣನಾದವನು, ವಂಶವೃದ್ಧಿಪಡಿಸುವ ಶಕ್ತಿಯನ್ನು ಕೊಡುವವನು

847. ಭಾರಭೃತ್ = ಜಗತ್ತಿನ ಭಾರವನ್ನು ಧರಿಸಿರುವವನು, ಭಕ್ತರ ಯೋಗ-ಕ್ಷೇಮಗಳನ್ನು ವಹಿಸುವವನು

848. ಕಥಿತಃ = ಆಗಮಗಳಲ್ಲಿ ಕೊಂಡಾಡಲ್ಪಟ್ಟವನು

849. ಯೋಗೀ = ಸಮಾಧಿ ಸಂಪನ್ನನು, ಜೀವಯೋಗವುಳ್ಳವನು

850. ಯೋಗೀಶಃ = ನರ ನಾರಾಯಣ ಕಪಿಲ ದತ್ತಾತ್ರೇಯಾದಿ ರೂಪಿಯಾದ ಯೋಗೀಂದ್ರನು, ಯೋಗಿರಾಜನು, ಯೋಗದಿಂದ ಎಂದಿಗೂ ವಿಚಲಿತನಾಗದವನು, ಅದ್ಭುತ ಯೋಗೈಶ್ವರ್ಯ ಸಂಪನ್ನನು, ಯೋಗಿಗಳನ್ನು ಆಳುವವನು

851. ಸರ್ವಕಾಮದಃ = ಭಕ್ತರ ಎಲ್ಲ ಕಾಮನೆಗಳನ್ನೂ ಈಡೇರಿಸುವವನು, ಧರ್ಮಕ್ಕೆ ವಿರೋಧವಲ್ಲದ ಎಲ್ಲ ಕಾಮಗಳನ್ನೂ ಉಂಟುಮಾಡುವವನು

852. ಆಶ್ರಮಃ = ಪ್ರಾಣಗಳಿಗೆ ಆರಾಮವಾಗಿರುವವನು, ಸಂಸಾರದಲ್ಲಿ ಬಳಲಿದ ಜೀವಿಗಳಿಗೆ ವಿಶ್ರಾಂತಿ ಧಾಮನಾಗಿರುವವನು

853. ಶ್ರಮಣಃ = ಮೋಕ್ಷಧರ್ಮ ನಿಷ್ಠನಾದ ಸಂನ್ಯಾಸಿ ಸ್ವರೂಪನು, ತಪಸ್ವಿಯು

854. ಕ್ಷಾಮಃ = ತಪೋವ್ರತ ನಿಷ್ಠೆಯಿಂದ ಕ್ಷೀಣದೇಹನಾದ ತಪಸ್ವಿ ರೂಪನಾದವನು, ಕ್ಷಮಾಶೀಲನು

855. ಸುಪರ್ಣಃ = ಸುಂದರವಾದ ರೆಕ್ಕೆಗಳುಳ್ಳ ವೇದಮಯನಾದ ಗರುಡ ಸ್ವರೂಪಿಯು, ಪ್ರಾಣಾಪಾನಗಳೆಂಬ ಶ್ರೇಷ್ಠವಾದ ರೆಕ್ಕೆಗಳುಳ್ಳ ಮಹಾಪ್ರಾಣ ಸ್ವರೂಪನು

856. ವಾಯುವಾಹನಃ = ಪ್ರಾಣವಾಯುಗಳ ಮೂಲಕ ಸಂಚರಿಸುವವನು, ಪ್ರಾಣವಾಯುವನ್ನು ಒಯ್ಯುವವನು, ವಾಯುಗಳಿಗೆ ನಿಯಾಮಕನಾದವನು

857. ಧನುರ್ಧರಃ = ಅಹಂಕಾರ ತತ್ತ್ವರೂಪವಾದ ಶಾಙ್ರ್ಗ ಧನುಸ್ಸನ್ನು ಧರಿಸಿರುವವನು, ಪ್ರಣವ ಎಂಬ ಧನುಸ್ಸನ್ನು ಬಳಸಬಲ್ಲ ಮಹಾಯೋಗಿಯು, ದಸ್ಯ್ಗಳನ್ನು ದಂಡಿಸಲು ವೈಷ್ಣವ ಧನುಸ್ಸನ್ನು ಧರಿಸಿದ ಕೋದಂಡರಾಮನು, ತ್ರಿಪುರ ಸಂಹಾರಕ್ಕೆ ಪಿನಾಕ ಧನುಸ್ಸನ್ನು ಧರಿಸಿದ ಮಹಾದೇವನು

858. ಧನುರ್ವೇದಃ = ಯಜುರ್ವೇದಕ್ಕೆ ಉಪವೇದವಾದ ಧನುರ್ವೇದ ಸ್ವರೂಪಿಯು

859. ದಂಡಃ = ಒಳ-ಹೊರ ಜೀವನಗಳನ್ನು ಹತೋಟಿಯಲ್ಲಿಡುವ ಯೋಗದಂಡ-ರಾಜದಂಡ ರೂಪಿಯು, ಮನೋನಿಗ್ರಹವನ್ನು ಉಂಟುಮಾಡುವವನು, ಯೋಗಶಾಸ್ತ್ರಗಳು ಹೇಳುವ ವಜ್ರದಂಡ-ಮೇರುದಂಡ ಎನಿಸುವ ಸರ್ವದೇವತಾಮಯವಾದ ಬೆನ್ನೆಲುಬು ಆಗಿರುವವನು, ದಂಡನೀತಿ ಸ್ವರೂಪನು

860. ದಮಯಿತಾ = ಒಳ-ಹೊರ ಶತ್ರುಗಳನ್ನು ದಮನ ಮಾಡುವವನು

861. ದಮಃ = ಇಂದ್ರಿಯ ನಿಗ್ರಹ ಸ್ವರೂಪಿಯು

862. ಅಪರಾಜಿತಃ = ಸೋಲನ್ನು ಪಡೆಯದವನು, ಸದಾ ವಿಜಯಶಾಲಿಯಾದವನು

863. ಸರ್ವಸಹಃ = ಭಕ್ತರ ಎಲ್ಲ ಅಪರಾಧಗಳನ್ನೂ ಸಹಿಸುವವನು, ಶತ್ರುಗಳ ಆಕ್ರಮಣವನ್ನು ಸಹಿಸುವವನು, ಯಾವಾಗಲೂ ಎಲ್ಲರಲ್ಲೂ ತಾಳ್ಮೆಯುಳ್ಳವನು

864. ನಿಯಂತಾ = ಎಲ್ಲರಿಗೂ ನಿಯಾಮಕನಾದವನು, ಎಲ್ಲರನ್ನೂ ತಮ್ಮ-ತಮ್ಮ ಕಾರ್ಯಗಳಲ್ಲಿ ವ್ಯವಸ್ಥೆ ಮಾಡುವವನು

865. ನಿಯಮಃ/ಅನಿಯಮಃ = ವ್ಯವಸ್ಥಾ ರೂಪನಾದವನು, ಯಾವ ನಿಯಂತ್ರಣವೂ ಇಲ್ಲದವನು, ಸರ್ವ ಸ್ವತಂತ್ರನು

866. ಯಮಃ/ಅನಿಯಮಃ = ಯೋಗಕ್ಕೆ ಅಂಗವಾದ ಯಮರೂಪಿಯು, ಯಮಧರ್ಮ ರಾಜರೂಪಿಯು, ಸಂಹಾರಕನು, ಮೃತ್ಯುರಹಿತನು, ಎಲ್ಲ ನಿಯಮಗಳನ್ನೂ ಮೀರಿದವನು

867. ಸತ್ತ್ವವಾನ್ = ಶುದ್ಧ ಸತ್ತ್ವನಾಗಿರುವವನು, ಬಲಶಾಲಿಯು, ಸರ್ವಸಾರನಾದವನು, ಸ್ಥಿರ ಸ್ವರೂಪನಾದವನು

868. ಸಾತ್ತ್ವಿಕಃ = ಸತ್ತ್ವಗುಣಕ್ಕೆ ಅಧಿಷ್ಠಾನನಾಗಿರುವವನು

869. ಸತ್ಯಃ = ವಿಕಾರರಹಿತನು, ಸಾಧುವು, ಸತ್ಯ ಸ್ವರೂಪನು

870. ಸತ್ಯಧರ್ಮಪರಾಯಣಃ = ಸತ್ಯ ಧರ್ಮಗಳಲ್ಲಿ ಪರಮನಿಷ್ಠೆಯುಳ್ಳವನು, ಸತ್ಯ-ಧರ್ಮಗಳಿಗೆ ಪರಮಗತಿಯಾದವನು

871. ಅಭಿಪ್ರಾಯಃ = ಸರ್ವ ಶಾಸ್ತ್ರಗಳಿಗೂ ತಾತ್ಪರ್ಯವಾಗಿರುವವನು, ಪರಮಾರ್ಥವಾಗಿರುವವನು, ಸಾಧುಗಳಿಂದ ಯಾವಾಗಲೂ ಎಲ್ಲ ಸನ್ಮಾರ್ಗಗಳಿಂದಲೂ ಹೊಂದಲ್ಪಡುವವನು

872. ಪ್ರಿಯಾರ್ಹಃ = ಇಷ್ಟವಾದ ವಸ್ತುಗಳ ಸಮರ್ಪಣೆಗೆ ಯೋಗ್ಯನಾದವನು

873. ಅರ್ಹಃ = ಅಂತರಂಗ-ಬಹಿರಂಗ ಪೂಜೆಗಳಿಗೆ ಯೋಗ್ಯನಾದವನು

874. ಪ್ರಿಯಕೃತ್ = ಭಕ್ತರಿಗೆ ಪ್ರಿಯವನ್ನು ಉಂಟುಮಾಡುವವನು

875. ಪ್ರೀತಿವರ್ಧನಃ = ಭಕ್ತರ ಪ್ರೀತಿಯನ್ನು ವೃದ್ಧಿಪಡಿಸುವವನು

876. ವಿಹಾಯಸಗತಿಃ = ಜ್ಞಾನಾಕಾಶದಲ್ಲಿ ಸಂಚರಿಸುವವನು, ಆಕಾಸಸಂಚಾರಿಯಾದ ಆದಿತ್ಯ ರೂಪಿಯು

877. ಜ್ಯೋತಿಃ = ಪರಂಜ್ಯೋತಿಯು, ಸ್ವಯಂ ಜ್ಯೋತಿಸ್ವರೂಪನು

878. ಸುರುಚಿಃ = ಒಳ್ಳೆಯ ಕಾಂತಿಯುಕ್ತನು, ಅಭಿರುಚಿಯುಳ್ಳವನು

879. ಹುತಭುಕ್ = ಅಗ್ನಿ ರೂಪಿಯು, ದೇವತೆಗಳಿಗೆ ಅರ್ಪಿಸಿದ ಹವಿಸ್ಸನ್ನು ತಾನೇ ಭುಂಜಿಸುವವನು

880. ವಿಭುಃ = ಸರ್ವ ವ್ಯಾಪಕನು, ಸರ್ವಪ್ರಭುವು

881. ರವಿಃ = ಸೂರ್ಯ ರೂಪಿಯು, ರಸಗಳನ್ನು ಗ್ರಹಿಸುವವನು

882. ವಿರೋಚನಃ = ವಿಶೇಷವಾಗಿ/ವಿವಿಧವಾಗಿ ಬೆಳಗುವವನು

883. ಸೂರ್ಯಃ = ಜ್ಞಾನಮಾರ್ಗ ಮತ್ತು ಧರ್ಮಮಾರ್ಗಗಳಲ್ಲಿ ಚೆನ್ನಾಗಿ ಪ್ರೇರೇಪಿಸುವವನು, ಸೂರಿಗಳಿಗೆ ಯೋಗ್ಯನಾದವನು, ಆದಿತ್ಯ ರೂಪಿಯು

884. ಸವಿತಾ = ಜಗತ್ತನ್ನು ಪ್ರಸವಿಸುವವನು, ಪ್ರೇರಕನು

885. ರವಿಲೋಚನಃ = ಸೂರ್ಯನನ್ನು ಕಣ್ಣಾಗಿ ಹೊಂದಿರುವವನು, ಬುದ್ಧಿ ಸೂರ್ಯನ ಮೂಲಕ ಪ್ರಕಾಶಿಸುವವನು, ಸೂರ್ಯನಿಗೂ ಬುದ್ಧಿಗೂ ಕಣ್ಣಾಗಿರುವವನು

886. ಅನಂತಃ = ನಿತ್ಯನೂ ಸರ್ವಗತನೂ ಆದ್ದರಿಂದ ದೇಶ-ಕಾಲ-ವಸ್ತುಗಳ ಪರಿಚ್ಛೇದ ಇಲ್ಲದವನು, ಆದಿಶೇಷ ರೂಪನು

887. ಹುತಭುಕ್ = ಹೋಮ ಮಾಡಿದ್ದನ್ನು (ಹವ್ಯ-ಕವ್ಯಗಳಲ್ಲಿ) ಭುಂಜಿಸುವವನು

888. ಭೋಕ್ತಾ = ಪರಮಾನಂದವನ್ನು ಅನುಭವಿಸುವವನು

889. ಸುಖದಃ = ಪರಮಾನಂದ ಸುಖವನ್ನು ಕೊಡುವವನು

890. ನೈಕಜಃ = ಹಲವು ಬಾರಿ ಅವತರಿಸುವವನು

891. ಅಗ್ರಜಃ = ಎಲ್ಲಕ್ಕೂ ಮೊದಲು ಆವಿರ್ಭವಿಸಿದವನು

892. ಅನಿರ್ವಿಣ್ಣಃ = ನಿರ್ವೇದ/ಖೇದ ಇಲ್ಲದವನು, ಅವಾಪ್ತ ಸಮಸ್ತ ಕಾಮನಾದುದರಿಂದ ಯಾವ ಖೇದವೂ ಇಲ್ಲದವನು

893. ಸದಾಮರ್ಷೀ = ಸದಾ ಸಹನಶೀಲನು, ಸದಾ ಕ್ಷಮಾಶೀಲನಾಗಿರುವವನು, ದುಷ್ಟರ ವಿಷಯದಲ್ಲಿ ಸದಾ ಕ್ರೋಧವುಳ್ಳವನು

894. ಲೋಕಾಧಿಷ್ಠಾನಂ = ಎಲ್ಲ ಲೋಕಗಳಿಗೂ ಆಧಾರವಾದವನು

895. ಅದ್ಭುತಃ = ಯಾವನ ದರ್ಶನ-ಜ್ಞಾನ-ಪ್ರವಚನಆದಿಗಳು ಸ್ವರೂಪ, ರೂಪ, ಗುಣ, ವೈಭವ, ಐಶ್ವರ್ಯ, ಕೃತಿ ಮುಂತಾದವುಗಳೆಲ್ಲವೂ ಪರಮಾಶ್ಚರ್ಯಕರ ಆಗಿವೆಯೋ ಅವನು

896. ಸನಾತ್ = ಸರ್ವಕಾಲದಲ್ಲಿಯೂ ಇರುವವನು

897. ಸನಾತನತಮಃ = ಅತ್ಯಂತ ಪುರಾತನನೂ ನಿತ್ಯನೂ ಆಗಿರುವವನು

898. ಕಪಿಲಃ = ಸಿದ್ಧೇಶ್ವರ ಕಪಿಲ ಸ್ವರೂಪನು

899. ಕಪಿಃ = ಸೂರ್ಯ ರೂಪಿಯು, ಬಡಬಾನಲ ಸ್ವರೂಪನು, ಆಂಜನೇಯ ರೂಪಿಯು, ಶ್ರೇಷ್ಠನು, ಪರಮಸುಖವನ್ನು ಸೇವಿಸುವವನು

900. ಅಪ್ಯಯಃ = ಎಲ್ಲವೂ ಯಾವನಲ್ಲಿ ಲೀನವಾಗುವುದೋ ಅವನು, (ಅವ್ಯಯಃ=ವಿಕಾರವಿಲ್ಲದವನು)

901. ಸ್ವಸ್ತಿದಃ = ಮಂಗಳದಾಯಕವಾದ ಮನಸ್ಸೇ ಮುಂತಾದವುಗಳನ್ನು ಕೊಡುವವನು, ಭಕ್ತರಿಗೆ ಒಳ್ಳೆಯ ಜೀವನವನ್ನು ಕೊಡುವವನು

902. ಸ್ವಸ್ತಿಕೃತ್ = ಮಂಗಳವನ್ನು ಉಂಟುಮಾಡುವವನು

903. ಸ್ವಸ್ತಿ = ಪರಮಾನಂದ ಮಂಗಳ ಸ್ವರೂಪನು

904. ಸ್ವಸ್ತಿಭುಕ್ = ಮಂಗಳಾನಂದವನ್ನು ಅನುಭವಿಸುವವನು

905. ಸ್ವಸ್ತಿದಕ್ಷಿಣಃ = ಮಂಗಳವನ್ನುಂಟು ಮಾಡುವುದರಲ್ಲಿ ಕುಶಲನಾದವನು, ಮಂಗಳಮಯವಾದ ಔದಾರ್ಯವುಳ್ಳವನು, ಮಂಗಳಮಯವಾದ ಯಜ್ಞ ದಕ್ಷಿಣೆಯುಳ್ಳವನು

906. ಅರೌದ್ರಃ = ರೌದ್ರಗುಣ ರೂಪಾದಿಗಳಿಲ್ಲದವನು, ಅತ್ಯಂತ ಸೌಮ್ಯನು

907. ಕುಂಡಲೀ = ದಿವ್ಯ ಕುಂಡಲವನ್ನು ಧರಿಸಿರುವವನು, ಕುಂಡಲಿನೀ ಸರ್ಪರೂಪನಾದವನು

908. ಚಕ್ರೀ = ಮನಸ್ತತ್ತ್ವ ರೂಪವಾದ ಸುದರ್ಶನ ಚಕ್ರವುಳ್ಳವನು, ಧರ್ಮಚಕ್ರಕ್ಕೆ ಪ್ರವರ್ತಕನಾದವನು, ಸಂಸಾರ ಚಕ್ರವನ್ನು ತಿರುಗಿಸುವವನು, ಮೂಲಾಧಾರವೇ ಮುಂತಾದ ಚಕ್ರಗಳಲ್ಲಿರುವವನು

909. ವಿಕ್ರಮೀ = ಅತಿಶಯವಾದ ಪರಾಕ್ರಮವುಳ್ಳವನು/ ಪಾದವಿನ್ಯಾಸವುಳ್ಳವನು

910. ಊರ್ಜಿತಶಾಸನಃ = ಅತ್ಯಧಿಕಬಲ, ಶಕ್ತಿ, ಉತ್ಸಾಹ ಇವುಗಳಿಂದ ಕೂಡಿದ ಶಾಸನವುಳ್ಳವನು

911. ಶಬ್ದಾತಿಗಃ = ಶಬ್ದಗಳಿಗೆ ದೊರಕದವನು, ಶಬ್ದಬ್ರಹ್ಮವನ್ನು ಮೀರಿದವನು

912. ಶಬ್ದಸಹಃ = ಶಬ್ದಗಳನ್ನು ಮೀರಿದ್ದರೂ ಎಲ್ಲ ಶಬ್ದಗಳಿಗೂ ವಾಚ್ಯನಾಗಿರುವವನು, ಶಬ್ದಗಳಿಂದಲೇ ಸ್ತುತಿಸಲ್ಪಡಬೇಕಾದವನು, ಶತ್ರುಗಳ ನಿಂದಾ ಶಬ್ದಗಳನ್ನು ತಾಳಿಕೊಳ್ಳುವವನು

913. ಶಿಶಿರಃ = ಚಂದ್ರ ಶೀತಲನಾಗಿರುವವನು

914. ಶರ್ವರೀಕರಃ = ಚಂದ್ರರೂಪಿಯು, ಅಜ್ಞಾನಿಗಳಿಗೆ ಸಂಸಾರವೆಂಬ ಕಾಳರಾತ್ರಿಯನ್ನು ಉಂಟುಮಾಡುವವನು, ಜ್ಞಾನಿಗಳಿಗೆ ಸಂಸಾರವೂ ಮತ್ತು ಅಜ್ಞಾನಿಗಳಿಗೆ ಪರಮಾತ್ಮನೂ ಕತ್ತಲೆಯಾಗುವಂತೆ ಮಾಡುವವನು

915. ಅಕ್ರೂರಃ = ಕ್ರೌರ್ಯವಿಲ್ಲದವನು

916. ಪೇಶಲಃ = ಅತ್ಯಂತ ಸುಂದರನಾದವನು, ಸರ್ವಾತ್ಮನಾ ಸುಂದರನಾದವನು

917. ದಕ್ಷಃ = ಸಮರ್ಥನು, ಶೀಘ್ರವಾಗಿ ಕೆಲಸ ಮಾಡುವವನು

918. ದಕ್ಷಿಣಃ = ಉದಾರಿಯು, ಸರಳ ಸ್ವಭಾವನು, ಸಮರ್ಥನು, ಶೀಘ್ರಕಾರಿಯು, ಕುಶಲನು

919. ಕ್ಷಮಿಣಾಂ ವರಃ = ಕ್ಷಮಾಶೀಲರಲ್ಲಿ ಶ್ರೇಷ್ಠನಾದವನು, ಸಹನಶಕ್ತಿ/ತಾಳ್ಮೆ ಉಳ್ಳವರಲ್ಲಿ ಶ್ರೇಷ್ಠನು

920. ವಿದ್ವತ್ತಮಃ = ಎಣೆಯಿಲ್ಲದ ಪರಿಪೂರ್ಣ ಜ್ಞಾನವನ್ನು ಸದಾ ಹೊಂದಿರುವ ಜ್ಞಾನಿಶ್ರೇಷ್ಠನು

921. ವೀತಭಯಃ = ಸರ್ವಭಯ ರಹಿತನು, ಸರ್ವಭಯ ಪರಿಹಾರಕನು

922. ಪುಣ್ಯಶ್ರವಣ ಕೀರ್ತನಃ = ಯಾವನ ಶ್ರವಣ ಮತ್ತು ಸಂಕೀರ್ತನೆಗಳಿಂದ ಅತ್ಯಂತ ಅತಿಶಯವಾದ ಪುಣ್ಯಲಾಭ ಇದೆಯೋ ಅವನು

923. ಉತ್ತಾರಣಃ = ಸಂಸಾರ ಸಾಗರದಿಂದ ದಾಟಿಸುವವನು, ಸರ್ವ ಕಷ್ಟಗಳಿಂದಲೂ ಪಾರು ಮಾಡುವವನು

924. ದುಷ್ಕೃತಿಹಾ = ದುಷ್ಟಕಾರ್ಯಗಳನ್ನು/ ಪಾಪಗಳನ್ನು/ ದುಷ್ಟಕಾರ್ಯ-ಮಾಡಿದವರನ್ನು ಸಂಹರಿಸುವವನು

925. ಪುಣ್ಯಃ = ಪಾವನ ಮಾಡುವವನು, ಪುಣ್ಯವನ್ನು ಉಪದೇಶಿಸುವವನು

926. ದುಃಸ್ವಪ್ನನಾಶನಃ = ಧ್ಯಾನಿಸುವ ಭಕ್ತರ ಕೆಟ್ಟ ಕನಸುಗಳನ್ನು ನಾಶಪಡಿಸುವವನು, ದುಷ್ಟವಾದ ನಿದ್ರೆಯನ್ನು/ಅಜ್ಞಾನವನ್ನು ಹೋಗಲಾಡಿಸುವವನು

927. ವೀರಹಾ = ವೀರರಾದ ಒಳ-ಹೊರ ಶತ್ರುಗಳನ್ನು ಸಂಹರಿಸುವವನು

928. ರಕ್ಷಣಃ = ಜಗತ್ತನ್ನು ರಕ್ಷಿಸುವ ಸ್ಥಿತಿಕಾರಕವಾದ ವಿಷ್ಣುವು, ಭಕ್ತ ಸಂರಕ್ಷಕನು

929. ಸಂತಃ = ಸತ್ಪುರುಷನು, ಸತ್ತಾ ಸ್ವರೂಪನು

930. ಜೀವನಃ = ಆತ್ಮೋಜ್ಜೀವನವನ್ನು/ ನಿತ್ಯ-ಜೀವನವನ್ನು ಉಂಟುಮಾಡುವವನು

931. ಪರ್ಯವಸ್ಥಿತಃ = ವಿಶ್ವವನ್ನು ಎಲ್ಲೆಡೆಗಳಲ್ಲಿಯೂ ವ್ಯಾಪಿಸಿರುವವನು, ಎಲ್ಲೆಲ್ಲಿಯೂ ನೆಲೆಸಿರುವವನು

932. ಅನಂತರೂಪಃ = ಅನಂತವಾದ ರೂಪವುಳ್ಳವನು, ನಾಶ ರಹಿತವಾದ ರೂಪಗಳುಳ್ಳವನು

933. ಅನಂತಶ್ರೀಃ = ಅಸಂಖ್ಯಾತವಾದ/ ನಾಶ-ರಹಿತವಾದ ಸಂಪತ್ತುಗಳುಳ್ಳವನು

934. ಜಿತಮನ್ಯುಃ = ಕೋಪವನ್ನೂ/ ದೈನ್ಯವನ್ನೂ ಜಯಿಸಿದವನು/ ಜಯಿಸಲು-ಕಾರಣನಾದವನು

935. ಭಯಾಪಹಃ = ಎಲ್ಲ ಭಯಗಳನ್ನೂ ಹೋಗಲಾಡಿಸುವವನು

936. ಚತುರಶ್ರಃ = ನ್ಯಾಯಶಾಲಿಯು, ವೈಷಮ್ಯವಿಲ್ಲದೇ ಸರ್ವ ಸಮಾನನಾಗಿರುವವನು

937. ಗಭೀರಾತ್ಮಾ = ಹೆಚ್ಚು ಆಳವಾದ ಆತ್ಮಸ್ವರೂಪನು, ದೇವತೆಗಳಿಗೂ ಅಳೆಯಲು ಅಸಾಧ್ಯವಾದ ಹೆಚ್ಚು ಆಳವಾದ ಮನಸ್ಸುಳ್ಳವನು

938. ವಿದಿಶಃ = ಈಶಾನ್ಯ-ವಾಯವ್ಯ ಮುಂತಾದ ಉಪದಿಕ್ಕುಗಳ ರೂಪದಲ್ಲಿರುವವನು, ವಿವಿಧ ಫಲಗಳನ್ನು ವಿಶೇಷವಾಗಿ ಕೊಡುವವನು

939. ವ್ಯಾದಿಶಃ = ಸುತ್ತಲೂ ಇರುವ ದಿಕ್ಕುಗಳ ರೂಪನಾದವನು, ಬಗೆ-ಬಗೆಯ ಆಜ್ಞೆಗಳನ್ನು ಕೊಡುವವನು

940. ದಿಶಃ = ಎಲ್ಲ ದಿಕ್ಕುಗಳೂ ಆಗಿರುವವನು, ಕರ್ಮ-ಫಲಗಳನ್ನು /ಇಷ್ಟಾರ್ಥಗಳನ್ನು ಕೊಡುವವನು

941. ಅನಾದಿಃ = ಆದಿಯಿಲ್ಲದವನು, ಸರ್ವಕಾರಣನು

942. ಭೂರ್ಭುವಃ = ಭೂಮಿಯ ಉತ್ಪತ್ತಿಗೆ ಕಾರಣನಾದವನು, ಭೂಮಿಗೆ ಆಧಾರನಾದವನು, ಸತ್ತಾ ಸ್ವರೂಪನು, ಜಗತ್ತಿನ ಉತ್ಪತ್ತಿಗೆ ಕಾರಣನು, ಭೂಃ-ಭುವಃ ಎಂಬ ವ್ಯಾಹೃತಿಗಳ ಸ್ವರೂಪನು, ಭೂಮಿ-ಅಂತರಿಕ್ಷಗಳ ಸ್ವರೂಪನು

943. ಲಕ್ಷ್ಮೀಃ = ಅಷ್ಟಲಕ್ಷ್ಮೀಯೂ ಮತ್ತು ಆತ್ಮಲಕ್ಷ್ಮೀಯೂ ಆಗಿರುವವನು, ನಾರಾಯಣೀ/ಕಾಂತಿ ಸ್ವರೂಪಿಯು, ಸರ್ವಸಂಪತ್ತೂ ಆಗಿರುವವನು

944. ಸುವೀರಃ = ಮಹಾವೀರನು

945. ರುಚಿರಾಂಗದಃ = ಮಂಗಳವಾದ ಮನೋಹರವಾದ ತೋಳ್ಬಳೆಗಳುಳ್ಳವನು

946. ಜನನಃ = ಪ್ರಾಣಿಗಳ ಜನನಕ್ಕೆ ಕಾರಣನಾದವನು, ಕರ್ಮ ವಶನಾಗದೇ ಜನನವನ್ನು/ಅವತಾರವನ್ನು ಸ್ವೀಕರಿಸುವವನು

947. ಜನಜನ್ಮಾದಿಃ = ಪ್ರಾಣಿಗಳ ಜನ್ಮಕ್ಕೆ ಮೂಲಕಾರಣನಾದವನು

948. ಭೀಮಃ = (ದುಷ್ಟರಿಗೆ) ಭಯಂಕರನಾದವನು

949. ಭೀಮಪರಾಕ್ರಮಃ = ದುಷ್ಟರಿಗೆ ಭಯವನ್ನು ಉಂಟು ಮಾಡುವ ಪರಾಕ್ರಮವುಳ್ಳವನು

950. ಆಧಾರನಿಲಯಃ = ಎಲ್ಲಕ್ಕೂ ಆಧಾರವೂ ನೆಲೆಮನೆಯೂ ಆಗಿರುವವನು, ಆಧಾರವಾದ ಶೇಷನಾಗ ಮುಂತಾದವರಿಗೂ ನೆಲೆಮನೆಯಾಗಿರುವವನು

951. ಆಧಾತಾ/ಧಾತಾ = ತನಗೆ ಬೇರೆ ಧಾರಕರಿಲ್ಲದವನು, ಎಲ್ಲವನ್ನೂ ಧರಿಸಿ ಪೋಷಿಸುವವನು

952. ಪುಷ್ಪಹಾಸಃ = ಪುಷ್ಪದಂತೆ ಮೃದು, ಮಧುರ, ಮನೋಹರ, ಶುದ್ಧ, ಸುಗಂಧಯುಕ್ತ ಮಂದಹಾಸವುಳ್ಳವನು

953. ಪ್ರಜಾಗರಃ = ಸದಾ ಚೆನ್ನಾಗಿ ಎಚ್ಚರವಾಗಿಯೇ ಇರುವವನು

954. ಊಧ್ರ್ವಗಃ = ಮೇಲಕ್ಕೆ ಹೋಗುವ ವೃತ್ತಿಯುಳ್ಳವನು. ಊಧ್ರ್ವಗಾಮಿಯಾದ ವೀರ್ಯವುಳ್ಳವನು

955. ಸತ್ಪಥಾಚಾರಃ = ಸನ್ಮಾರ್ಗದಲ್ಲಿ ನಡೆಯುವವನು

956. ಪ್ರಾಣದಃ = ಪ್ರಾಣಸಧಾನ ಮಾಡುವವನು, ಆತ್ಮೋಜ್ಜೀವನವನ್ನು ಉಂಟು ಮಾಡುವವನು

957. ಪ್ರಣವಃ = ಓಂಕಾರ ವಾಚ್ಯನು, ಪ್ರಾಣಗಳನ್ನು ತನ್ನಲ್ಲಿ ಪ್ರಣಾಮ ಮಾಡಿಸುವವನು

958. ಪಣಃ = ವ್ಯವಹರಿಸುವವನು, ಸ್ತುತಿಸಲ್ಪಡುವವನು, ಪುಣ್ಯಪಣವಾಗಿರುವವನು

959. ಪ್ರಮಾಣಂ = ಪ್ರಮೆಗೆ (ಯಥಾರ್ಥ ಜ್ಞಾನದ ಅನುಭವಕ್ಕೆ) ಸಾಧನ ಮತ್ತು ಶ್ರೇಷ್ಠವಾದ ಅಳತೆ ಆಗಿರುವವನು

960. ಪ್ರಾಣನಿಲಯಃ = ಪ್ರಾಣಗಳಿಗೆ ಇಂದ್ರಿಯಗಳಿಗೆ ಲಯಸ್ಥಾನವೂ ಆರಾಮಗೃಹವೂ ಆಗಿರುವವನು

961. ಪ್ರಾಣಭೃತ್ = ಪ್ರಾಣಕ್ಕೂ ಪ್ರಾಣವಾಗಿದ್ದು ಅದನ್ನು ಪೋಷಿಸುವವನು

962. ಪ್ರಾಣಜೀವನಃ = ಪ್ರಾಣಗಳಿಗೆ ಚೈತನ್ಯನಾಗಿರುವವನು

963. ತತ್ತ್ವಂ = ಸತ್ ಸ್ವರೂಪನು, ಎಲ್ಲದರ ಸಾರಾಂಶವಾಗಿರುವವನು

964. ತತ್ತ್ವವಿತ್ = ಸ್ವರೂಪವನ್ನು ಯಥಾವತ್ತಾಗಿ ಅರಿತವನು

965. ಏಕಾತ್ಮಾ = ಕೇವಲಾತ್ಮನು, ಪರಮಾತ್ಮನು, ಅದ್ವಿತೀಯನಾದ ಆತ್ಮನು

966. ಜನ್ಮಮೃತ್ಯುಜರಾತಿಗಃ = ಹುಟ್ಟು, ಸಾವು, ಮುಪ್ಪು ಮುಂತಾದ ಸರ್ವ ವಿಕಾರಗಳನ್ನೂ ಮೀರಿದವನು

967. ಭೂರ್ಭುವಃಸ್ವಸ್ತರುಃ = ಸರ್ವಾಶ್ರಯವಾದ ವ್ಯಾಹೃತಿತ್ರಯ ಎಂಬ ಮತ್ತು ಜಗತ್ರಯವೆಂಬ ವೃಕ್ಷ ಸ್ವರೂಪನು

968. ತಾರಃ = ಪ್ರಣವ ಸ್ವರೂಪಿಯು, (ವಾಚಕವಾದ ಪ್ರಣವಕ್ಕೂ ವಾಚ್ಯವಾದ ಪರಮಾತ್ಮನಿಗೂ ಅಭೇದವೆಂಬಂತೆ ವಾಚಕವನ್ನು ಅರಿತರೆ ವಾಚ್ಯವು ಪ್ರಸನ್ನವಾಗುತ್ತದೆ)

969. ಸವಿತಾ = ಜಗತ್ತನ್ನು ಪ್ರಸವಿಸುವವನು, ಬುದ್ಧಿ ಮುಂತಾದದ್ದನ್ನು ಪ್ರೇರೇಪಿಸುವವನು, ಅನುಮತಿಸುವವನು

970. ಪ್ರಪಿತಾಮಹಃ = ಪಿತಾಮಹನಾದ ಬ್ರಹ್ಮದೇವನಿಗೂ ತಂದೆಯಾದವನು

971. ಯಜ್ಞಃ = ಯಜ್ಞ ಸ್ವರೂಪಿಯು, ಯಜ್ಞದಿಂದ ಪೂಜಿಸಲ್ಪಡುವವನು

972. ಯಜ್ಞಪತಿಃ = ಯಜ್ಞಗಳ ಸ್ವಾಮಿಯು, ಯಜ್ಞ ಸಂರಕ್ಷಕನು

973. ಯಜ್ವಾ = ಯಜಮಾನ ಸ್ವರೂಪಿಯು, ಸೃಷ್ಟಿ-ಸ್ಥಿತಿ-ಲಯಗಳೆಂಬ ವಿಶ್ವಯಜ್ಞವನ್ನು ಮಾಡುತ್ತಿರುವವನು, (ವಿಶ್ವಕರ್ಮಸೂಕ್ತ) ಶ್ರೀರಾಮಾದಿ ರೂಪಗಳಲ್ಲಿ ಯಜ್ಞಗಳನ್ನು ಮಾಡಿದವನು

974. ಯಜ್ಞಾಂಗಃ = ಯಜ್ಞದ ಅಂಗಗಳೂ ತಾನೇ ಆಗಿರುವವನು, ಯಜ್ಞಗಳನ್ನು ತನ್ನ ಅಂಗಗಳನ್ನಾಗಿ ಹೊಂದಿರುವ ಯಜ್ಞವರಾಹ ಮೂರ್ತಿಯು

975. ಯಜ್ಞವಾಹನಃ = ಯಜ್ಞವನ್ನು ವಾಹನವನ್ನಾಗಿ ಹೊಂದಿರುವವನು, ಯಜ್ಞದ ಮೂಲಕವಾಗಿ ಭಕ್ತರ ಬಳಿಗೆ ಬಂದು ಅವರೊಡನೆ ಸ್ವಧಾಮಕ್ಕೆ ಪ್ರಯಾಣ ಮಾಡುವವನು, ಋತ-ಸತ್ಯ ಸ್ವರೂಪಿಯು, ಯಜ್ಞಗಳನ್ನು ನಡೆಸಿಕೊಡುವವನು

976. ಯಜ್ಞಭೃತ್ = ಯಜ್ಞವನ್ನು ಧರಿಸಿ ರಕ್ಷಿಸುವವನು

977. ಯಜ್ಞಕೃತ್ = ಪ್ರಪಂಚದ ಸೃಷ್ಟಿ-ಸ್ಥಿತಿ-ಲಯ ರೂಪವಾದ ಯಜ್ಞವನ್ನು ಮಾಡುವವನು, ರಾಮಾದಿ ಅವತಾರಗಳಲ್ಲಿ ಯಜ್ಞವನ್ನು ಮಾಡುವವನು

978. ಯಜ್ಞೀ = ಯಜ್ಞಗಳನ್ನುಳ್ಳವನು, ಯಜ್ಞಕ್ಕೆ ಶೇಷಿಯಾಗಿರುವವನು, ಯಜ್ಞಗಳು ಯಾವನ ಪ್ರಯೋಜನಕ್ಕಾಗಿ ಇವೆಯೋ ಅವನು

979. ಯಜ್ಞಭುಕ್ = ಯಜ್ಞವನ್ನು ಅನುಭವಿಸುವವನು ಅಥವಾ ರಕ್ಷಿಸುವವನು

980. ಯಜ್ಞಸಾಧನಃ = ಯಜ್ಞದ ಉಪಕರಣಗಳೂ ತಾನೇ ಆಗಿರುವವನು, ಯಜ್ಞಗಳು ಯಾವನನ್ನು ಹೊಂದಲು ಸಾಧನಗಳಾಗಿವೆಯೋ ಅವನು

981. ಯಜ್ಞಾಂತಕೃತ್ = ಯಜ್ಞದ ಫಲವನ್ನು ಕೊಡುವವನು, ಪೂರ್ಣಗೊಳಿಸುವವನು, ಅವಿಧಿಪೂರ್ವಕವಾದ ಯಜ್ಞವನ್ನು ಧ್ವಂಸ ಮಾಡುವವನು, ಸೃಷ್ಟಿ-ಸ್ಥಿತಿ ಯಜ್ಞಗಳನ್ನು ಪ್ರಳಯಕಾಲದಲ್ಲಿ ಧ್ವಂಸ ಮಾಡುವವನು

982. ಯಜ್ಞಗುಹ್ಯಂ = ಯಜ್ಞಗಳ ರಹಸ್ಯವಾಗಿರುವವನು

983. ಅನ್ನಂ = ಜ್ಞಾನಿಗಳಿಗೆ ಭೋಗ್ಯವಾದ ಪರಮಾನ್ನವಾಗಿರುವವನು, ಜ್ಞಾನಿಗಳಿಂದ ಅನುಭವಿಸಲ್ಪಡುವವನು, ಪ್ರಳಯಕಾಲದಲ್ಲಿ ಎಲ್ಲವನ್ನೂ ತಿಂದು ಹಾಕುವವನು

984. ಅನ್ನಾದಃ = ಅನ್ನವನ್ನು ತಿನ್ನುವವನು, ಬ್ರಹ್ಮ-ಕ್ಷತ್ರಗಳೆಂಬ ಅನ್ನವನ್ನು ಮೃತ್ಯುವಿನ ಮೇಲೋಗರದೊಡನೆ ತಿಂದು ಹಾಕುವವನು

985. ಆತ್ಮಯೋನಿಃ = ತನಗೆ ತಾನೇ ಕಾರಣನಾದವನು, ಸ್ವಯಂಭುವು

986. ವೈಖಾನಃ = ವೈಖಾನಸಾಶ್ರಮ/ ವಾನಪ್ರಸ್ಥಾಶ್ರಮ ಮತ್ತು ವೈಖಾನಸ-ಮುನಿ ಸ್ವರೂಪನು, ಗೆಣಸುಗಳನ್ನು ಅಗೆದು ತಿನ್ನುವ ತಾಪಸ ಸ್ವರೂಪಿಯು, ಬ್ರಹ್ಮದೇವ ಸ್ವರೂಪಿಯು

987. ಸಾಮಗಾಯನಃ = ಸಾಮಗಳನ್ನು ಗಾನ ಮಾಡುವವನು, ಮುಕ್ತಾತ್ಮರಿಂದ ಸ್ವಾನುಭವ ರೂಪವಾದ ಸಾಮಗಾನವನ್ನು ಮಾಡಿಸುವವನು

988. ದೇವಕೀನಂದನಃ = ದೇವಕೀ ದೇವಿಗೆ ಆನಂದವನ್ನು ಉಂಟುಮಾಡಿದ ಸುಪುತ್ರನು

990. ಸ್ರಷ್ಟಾ = ಎಲ್ಲವನ್ನೂ ಸೃಷ್ಟಿ ಮಾಡಿದವನು

991. ಕ್ಷಿತೀಶಃ = ಭೂಪತಿಯು, ದಶರಥ ಪುತ್ರ ಶ್ರೀರಾಮನಾಗಿ ಭೂಮಿಯನ್ನು ಆಳಿದವನು

992. ಪಾಪನಾಶನಃ = ಸ್ಮರಣ, ಜಪ, ಕೀರ್ತನಾದಿಗಳಿಂದ ಪಾಪಗಳನ್ನು ನಾಶ ಮಾಡುವವನು

993. ಶಂಖಭೃತ್ = (ತಾಮಸ-ಅಹಂಕಾರ ರೂಪವಾದ ಪಾಂಚಜನ್ಯ) ಶಂಖವನ್ನು ಧರಿಸಿರುವವನು

994. ನಂದಕೀ = ವಿದ್ಯಾಮಯವಾದ ನಂದಕವೆಂಬ ಖಡ್ಗವುಳ್ಳವನು

995. ಚಕ್ರೀ = ಮನಸ್ತತ್ತ್ವಾತ್ಮಕವಾದ ಸುದರ್ಶನ ಚಕ್ರವುಳ್ಳವನು

996. ಶಾಙ್ರ್ಗಧನ್ವಾ = ರಾಜಸ-ಅಹಂಕಾರ ರೂಪವಾದ ಶಾಙ್ರ್ಗ ಎಂಬ ಧನುಸ್ಸನ್ನು ಹೊಂದಿರುವವನು

997. ಗದಾಧರಃ = ಮಹತ್ತತ್ತ್ವ ರೂಪವಾದ ಕೌಮೋದಕೀ ಎಂಬ ಗದೆಯನ್ನು ಧರಿಸಿರುವವನು

998. ರಥಾಂಗಪಾಣಿಃ = ಕೈಯಲ್ಲಿ ಚಕ್ರವನ್ನು ಧರಿಸಿರುವವನು

999. ಅಕ್ಷೋಭ್ಯಃ = (ಸರ್ವ ತತ್ತ್ವಗಳಿಗೂ ಅಧಿಪತಿಯಾದ್ದರಿಂದ) ಕ್ಷೋಭೆಗೊಳಿಸಲು ಅಸಾಧ್ಯನಾದವನು

1000. ಸರ್ವಪ್ರಹರಣಾಯುಧಃ = ಪ್ರಹರಿಸಲು ಯೋಗ್ಯವಾದ ಎಲ್ಲ ಪದಾರ್ಥಗಳನ್ನೂ ತನ್ನ ಆಯುಧಗಳನ್ನಾಗಿ ಹೊಂದಿರುವವನು, ಯಾವನ ಸಂಕಲ್ಪ ಮಾತ್ರದಿಂದ ದರ್ಭ, ಇಷೀಕ, ನಖ ಇವೇ ಮುಂತಾದ ಯಾವುದೇ ಪದಾರ್ಥವೂ ಅವನಿಗೆ ಆಯುಧವಾಗಬಹುದೋ ಅವನು!
