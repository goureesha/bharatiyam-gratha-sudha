ವಾಯುಃ ಏವ ನ ವಾತಿ = ಗಾಳಿಯೇ ಬೀಸುತ್ತಿಲ್ಲ
ಆರಾತ್ರಿಃ ವೃಷ್ಟಿಃ ಆಸೀತ್ = ರಾತ್ರಿಯಿಡೀ ಮಳೆ ಇತ್ತು
ಘರ್ಮೋ ಘರ್ಮಃ = ಸೆಕೆಯೋ ಸೆಕೆ!
ಕಿಂ ಭೋಃ ಕ್ಲಿನ್ನಃ ಅಸ್ತಿ? = ತೋಯ್ದು ಹೋಗಿದ್ದೀಯಲ್ಲಾ?
ಪ್ರಾತಃ ಆರಭ್ಯ ಏವಮೇವ ವೃಷ್ಟಿಃ = ಬೆಳಗ್ಗೆಯಿಂದ ಇದೇ ರೀತಿ ಮಳೆ
ಅದ್ಯ ವೃಷ್ಟಿಃ ಭವತಿ ವಾ? ಇತಿ = ಇವತ್ತು ಮಳೆ ಬರುತ್ತಾ? ಅಂತ
ಕಿಂ ಇಯಂ ಉಷ್ಣತಾ ಭೋಃ = ಏನು ಸೆಕೆ ಅಂತೀರಿ
ಶೈತ್ಯಂ ಅಹೋ ಶೈತ್ಯಂ = ಛಳಿ ಅಂದರೆ ಛಳಿ!
ಮಹತೀ ವೃಷ್ಟಿಃ = ತುಂಬಾ ಮಳೆ
ವೃಷ್ಟಿತಃ ಏವ ಭಯಂ = ಮಳೆಯದೇ ಹೆದರಿಕೆ
ದಿನೇ ದಿನೇ ಶೈತ್ಯಂ ಅಧಿಕಂ ಭವತಿ = ದಿನ ದಿನಕ್ಕೂ ಛಳಿ ಹೆಚ್ಚಾಗುತ್ತಿದೆ
ಯದ್ವಾ ತದ್ವಾ ವೃಷ್ಟಿಃ = ಸಿಕ್ಕಾಪಟ್ಟೆ ಮಳೆ
ವಾಯುರಹೋ ವಾಯುಃ = ಗಾಳಿ ಅಂದರೆ ಗಾಳಿ
ಅತ್ರ ವಾಯುಃ ಸುಷ್ಠು ವಾತಿ = ಇಲ್ಲಿ ಗಾಳಿ ಚೆನ್ನಾಗಿ ಬೀಸುತ್ತೆ
ಬಹು ಔಷ್ಣ್ಯಂ ಕಿಲ? = ತುಂಬಾ ಸೆಕೆ ಅಲ್ವಾ?
ಅದ್ಯ ಕಿಂಚಿತ್ ಶೈತ್ಯಂ ಅಧಿಕಂ = ಇಂದು ಸ್ವಲ್ಪ ಛಳಿ ಜಾಸ್ತಿ
ಭವತಃ ಪ್ರದೇಶೇ ವೃಷ್ಟಿಃ ಕಥಂ? = ನಿಮ್ಮೂರಲ್ಲಿ ಮಳೆ ಹೇಗೆ?
ಕುತ್ರಾಪಿ ವೃಷ್ಟಿಃ ನಾಸ್ತಿ = ಎಲ್ಲಿಯೂ ಮಳೆ ಇಲ್ಲ
