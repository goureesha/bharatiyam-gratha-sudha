ಅರುಣಾಚಲೇಶ್ವರನ ಪ್ರದಕ್ಷಿಣಾ ಮಹಾತ್ಮೆ

ಗೌತಮ ಮುನಿಯು ಅರುಣಾಚಲೇಶ್ವರನಲ್ಲಿ ಹೀಗೆಂದು ಪ್ರಾರ್ಥಿಸಿದರು: "ಎಲೈ ಅರುಣಾಚಲನೇ! ದೇವ ಪೂಜಿತವಾದ ಈ ಕ್ಷೇತ್ರವನ್ನು ಪವಿತ್ರ ಮಾಡಿರುವ ನಿನ್ನ ನಾಮ ಭೇದಗಳನ್ನು ತಿಳಿಯಬೇಕೆಂಬ ಆಸೆ. ಕೃಪಾಸಾಗರನಾದ ನೀನು ನನಗೆ ಅದನ್ನು ಹೇಳುವೆಯಾ?"

ಭಕ್ತನ ಮಾತುಗಳನ್ನು ಕೇಳಿ ಮಹೇಶ್ವರನು ಸುಪ್ರೀತನಾಗಿ ಹೀಗೆ ಹೇಳಿದನು, "ಎಲೈ ಬ್ರಾಹ್ಮಣೋತ್ತಮನೇ! ಮುಖ್ಯವಾದ ಕೆಲವು ನಾಮಗಳನ್ನು ಹೇಳುವೆನು, ಕೇಳು. ಪುಣ್ಯಹೀನರಿಗೆ ಈ ನಾಮಗಳು ದುರ್ಲಭವಾಗಿವೆ. ಭಕ್ತರಿಗೆ ಈ ನಾಮಗಳು ಬಹಳ ಪ್ರಿಯವಾದುವು. ಇವುಗಳನ್ನು ಪೂಜಾಸಮಯದಲ್ಲಿ ಅರ್ಚನೆ ಮಾಡುವಾಗ ಹೇಳಬೇಕು:

1. ಶೋಣಾದ್ರೀಶಃ = ತಾಮ್ರವರ್ಣದ ಪರ್ವತ ರೂಪದಲ್ಲಿರುವ ಪರಮಾತ್ಮ
2. ಅರುಣಾದ್ರೀಶಃ = ಕೆಂಪುವರ್ಣದ ಪರ್ವತ ರೂಪದಲ್ಲಿರುವ ಪರಮಾತ್ಮ
3. ದೇವಾಧೀಶಃ = ದೇವತೆಗಳಿಗೆಲ್ಲಾ ಒಡೆಯ
4. ಜನಪ್ರಿಯಃ = ಜನತೆಗೆಲ್ಲಾ ಪ್ರಿಯನಾದವನು
5. ಪ್ರಪನ್ನ ರಕ್ಷಕಃ = ಶರಣಾಗತರಾದವರಿಗೆ ರಕ್ಷಕ
6. ಧೀರಃ = ಧೈರ್ಯವಂತ
7. ಶಿವ ಸೇವಕ ವರ್ಧಕಃ = ತನ್ನನ್ನು ಸೇವಿಸುವವರನ್ನು ಉದ್ಧರಿಸುವವ
8. ಅಕ್ಷಿಪೇಯಃ = ಅವನ ದರ್ಶನದಿಂದ ಕಣ್ಣುಗಳಿಗೆ ಅಮೃತ ಪಾನ ಮಾಡಿಸುವವ
9. ಸ್ತ್ರೀಪುಂಭಾವ ಪ್ರದಾಯಕ = ಹೆಣ್ಣು ಗಂಡೆಂಬ ಭಾವನೆಗಳಿಗೆ ದಯೆಗೈಯುವವನು
10. ಅಮೃತೇಶಾನಃ = ಅಮೃತಕ್ಕೆ ಅಧಿಪತಿ
11. ಭಕ್ತ ವಿಜ್ಞಪ್ತಿ ಸಂದಾಯತಃ =ಭಕ್ತರು ಕೋರಿದ್ದನ್ನು ದಯಪಾಲಿಸುವನು
12. ದೀನ ಬಂಧ ವಿಮೋಚಕಃ = ದೀನರ ಬಂಧವನ್ನು ಬಿಡಿಸುವವನು
13. ಮುಖರಾಷ್ಟ್ರಪತಿಃ = ಮುಖರಾತ್ರಿ ದೇವಿಯ ಪತಿ
14. ಶ್ರೀಮಾನ್ = ಕುಲೀನ, ಶ್ರೀಮಂತ
15. ಭೂತನಾಥಃ = ಭೂತಗಳಿಗೆಲ್ಲಾ ಒಡೆಯ, ಜೀವಗಳಿಗೆಲ್ಲಾ ಒಡೆಯ
16. ಮೃಗಮದೇಶ್ವರಃ = ಕಸ್ತೂರಿ ಮೃಗ ಪ್ರಿಯ
17. ಭಕ್ತ ಪ್ರೇಕ್ಷಣಾಕ್ರತಃ = ಭಕ್ತರನ್ನು ಕಾಪಾಡುವವನು
18. ಸಾಕ್ಷೀ = ಎಲ್ಲ ವಿಷಯಗಳಿಗೂ ಸಾಕ್ಷಿ; ಲೋಕ ಸಾಕ್ಷಿ
19. ಭಕ್ತ ದೋಷ ನಿವಾರಕಃ = ಭಕ್ತರ ದೋಷಗಳನ್ನು ಪರಿಹರಿಸುವವನು
20. ಜ್ಞಾನ ಸಂಬಂಧ ನಾಥಃ = ಜ್ಞಾನ ಸಂಬಂಧನಿಗೆ ಒಡೆಯ
21. ಶ್ರೀಹಾಲಾಹಲ ಸುಂದರಃ = ವಿಷವನ್ನು ಕುಡಿದ ಸುಂದರ
22. ಅಹಮೈಶ್ವರ್ಯದಾತಾ = ಸ್ವಯಂಪ್ರೇರಣೆಯಿಂದ ಐಶ್ವರ್ಯವನ್ನು ನೀಡುವವನು
23. ಸ್ಮರ್ತೃಸರ್ವಾಘನಾಶನಃ = ತನ್ನನ್ನು ಧ್ಯಾನಿಸುವವರ ಪಾಪಗಳನ್ನು ನಾಶ ಮಾಡುವವನು
24. ವ್ಯತ್ಯಸ್ತ ನೃತ್ಯಃ = ಎಡಗಾಲನ್ನೆತ್ತಿ ನರ್ತಿಸುವವನು
25. ಧ್ವಜಾಧ್ಯಕ್ = ಧ್ವಜವನ್ನು ಎತ್ತಿ ಹಿಡಿದವನು
26. ಸಕಾಂತಿಃ = ಉತ್ಕೃಷ್ಟ ಕಾಂತಿಯುಕ್ತನು
27. ನಟೇಶ್ವರಃ = ನೃತ್ಯಗಳ ಒಡೆಯ
28. ಸಾಮಪ್ರಿಯಃ = ಸಾಮವೇದ ಸ್ತೋತ್ರ ಪ್ರಿಯನು
29. ಕಲಿಪೀಡಾಧ್ವಂಸಿಃ = ಕಲಿಯ ಪೀಡೆಯಿಂದ ರಕ್ಷಿಸುವವನು
30. ವೇದಮೂರ್ತಿಃ = ವೇದಸ್ವರೂಪಿ/ವೇದಗಳ ಮೂರ್ತಿವೆತ್ತಂತಿರುವವನು |
31. ನಿರಂಜನಃ = ಶುದ್ದ ಅಮಲ ರೂಪಿ
32. ಜಗನ್ನಾತಃ = ಬ್ರಹ್ಮಾಂಡಕ್ಕೆ/ಜಗತ್ತಿಗೆ ಒಡೆಯನು
33. ಮಹಾದೇವಃ = ದೇವಾದಿದೇವ/ಪರಮಾತ್ಮ
34. ತ್ರಿನೇತ್ರಃ = ಮೂರು ಕಣ್ಣುಗಳುಳ್ಳವನು/ಮುಕ್ಕಣ್ಣ
35. ತ್ರಿಪುರಾಂತಕಃ = ಮೂರು ಪುರಗಳನ್ನೂ ನಾಶ ಮಾಡಿದವನು
36. ಭಕ್ತ ಅಪರಾಧ ಸೋಢಃ = ಭಕ್ತರ ಅಪರಾಧಗಳನ್ನು ಕ್ಷಮಿಸುವವ/ ಸಹಿಸುವವನು
37. ಭೋಗನಾಯಕಃ = ಭೋಗೇಂದ್ರ; ಭೋಗಗಳಿಗೆಲ್ಲಾ ಒಡೆಯ
38. ಗಿರಿಪ್ರದಕ್ಷಿಣಾ ಪ್ರಿಯ = ಗಿರಿಪ್ರದಕ್ಷಿಣೆಯನ್ನು ಪ್ರೀತಿಸುವವನು
39. ಬಾಲಮೂರ್ತಿ = ಬಾಲಕನಂತ ರೂಪ ಧರಿಸಿರುವವ
40. ಕ್ಷಮದ್ರೂಪಿಃ = ಕ್ಷಮೆಯೇ ಮೂರ್ತಿವೆತ್ತಂತಿರುವವನು
41. ಧರ್ಮ ರಕ್ಷಕಃ = ಧರ್ಮ ಪಾಲಕ
42. ಹರಃ = ಪಾಪನಾಶಕ/ಅಧರ್ಮ ನಾಶಕ
43. ವೃಷಭಧ್ವಜಃ = ವೃಷಭವನ್ನು (ನಂದಿ) ಲಾಂಛನವಾಗಿ ಹೊಂದಿರುವವನು
44. ಗಿರೀಶ್ವರಃ = ಪರ್ವತಗಳಿಗೆ ಒಡೆಯ
45. ತೇಜಃಪುಂಜ, = ತೇಜಸ್ಸುಳ್ಳವನು
46. ಚಂದ್ರರೇಖಾವತಂಸಃ = ಅರ್ಧಚಂದ್ರನನ್ನು ಧರಿಸಿ ಶೋಭಿಸುವವನು
47. ಸ್ಮರಾಂತಕಃ = ಕಾಮದೇವನ ವೈರಿ
48. ಅಂಧಕರಿಪು = ಅಂಧಕಾಸುರನನ್ನು ಸಂಹರಿಸಿದವ/ಅಜ್ಞಾನವನ್ನು ಹೋಗಲಾಡಿಸಿದವನು
49. ಸಿದ್ಧರಾಜಃ = ಸಿದ್ದರ ರಾಜ
50. ದಿಗಂಬರಃ = ಆಕಾಶವನ್ನೇ ವಸ್ತ್ರವನ್ನಾಗಿ ಹೊಂದಿದವನು
51. ಆಗಮಪ್ರಿಯಃ = ಆಗಮಗಳನ್ನು ಇಷ್ಟಪಡುವವನು
52. ಈಶಾನಃ = ಜಗತ್ಕಾರಣ; ಜಗತ್ತಿಗೆಲ್ಲಾ ಆಧಾರ/ಒಡೆಯ
53. ಭಸ್ಮ ರುದ್ರಾಕ್ಷ ಲಾಂಛನಃ = ಭಸ್ಮ, ರುದ್ರಾಕ್ಷಿಗಳನನು ಧರಿಸಿದವನು
54. ಶ್ರೀಪತಿ = ಸಕಲೈಶ್ವರ್ಯಕ್ಕೂ ಒಡೆಯ
55. ಶಂಕರಃ = ಸಂತೋಷವನ್ನು ಕೊಡುವವನು
56. ಸೃಷ್ಠ = ಜಗತ್ ಕರ್ತ
57. ಸರ್ವ ವಿದ್ಯೆಶ್ವರಃ = ಎಲ್ಲ ವಿದ್ಯೆಗಳನ್ನೂ ಅರಿತವನು
58. ಅನಘಃ = ದುರಿತಗಳಿಲ್ಲದವನು; ಪಾಪಗಳಿಲ್ಲದವನು
59. ಗಂಗಾಧರಃ = ಗಂಗೆಯನ್ನು (ಜಟೆಯಲ್ಲಿ) ಧರಿಸಿದವನು
60. ದಕ್ಷಯಾಗ ಧ್ವಂಸಕಃ = ದಕ್ಷಯಾಗವನ್ನು ಧ್ವಂಸ ಮಾಡಿದವನು
61. ವಿಮಲಃ = ಯಾವ ರೀತಿಯ ಮಲಗಳೂ ಇಲ್ಲದವನು (ನಿರ್ಮಲ)
62. ನಾಗಭೂಷಣಃ = ಸರ್ಪಗಳನ್ನು ಆಭರಣದಂತೆ ಧರಿಸಿರುವವನ್ನು ಕೆಂಪು ವರ್ಣದವನು
63. ಅರುಣಃ = ಕೆಂಪು ವರ್ಣದವನು
64. ಬಹುರೂಪಃ = ನಾನಾ ರೂಪಗಳಲ್ಲಿರುವವನು
65. ವಿರೂಪಾಕ್ಷ = ಮೂರನೇ ಜ್ಞಾನಾಗ್ನಿ ಕಣ್ಣುಳ್ಳವನು
66. ಅಕ್ಷರಾಕೃತಿಃ = ನಾಶವಿಲ್ಲದ ಆಕೃತಿಯುಳ್ಳವನು
67. ಅನಾದಿ = ಆದಿ ಇಲ್ಲದವನು; ಹುಟ್ಟಿಲ್ಲದವ ಪ್ರಾರಂಭರಹಿತನು
68. ಅಂತರಹಿತಃ = ಅಂತ್ಯವಿಲ್ಲದವನು; ಕೊನೆಯಿಲ್ಲದವನು
69. ಶಿವಕಾಮಃ = ಸರ್ವರಿಗೂ ಶುಭವನ್ನೇ ಕೋರುವವನು
70. ಸ್ವಯಂಪ್ರಭುಃ = ತನಗೆ ತಾನೇ ಒಡೆಯನು
71. ಸಚ್ಚಿದಾನಂದ ಸ್ವರೂಪಃ = ಸತ್‍ಚಿತ್-ಆನಂದ ಸ್ವರೂಪನು
72. ಸರ್ವಾತ್ಮಃ = ಸರ್ವ ಜೀವಿಗಳಲ್ಲೂ ಇರುವ ಆತ್ಮ ಸ್ವರೂಪ
73. ಜೀವ ಧಾರಕಃ = ಸರ್ವ ಜೀವಿಗಳಿಗೂ ಆಧಾರನು
74. ಸ್ತ್ರೀಸಂಗವಾಮ ಸುಭಗಃ = ಎಡಭಾಗದಲ್ಲಿ ಸುಂದರ ಶಕ್ತಿ ಸ್ತ್ರೀ ಸಂಗವುಳ್ಳವನು
75. ವಿಹಿತ ಸುಂದರಃ = ಸುಂದರ ರೂಪವನ್ನು ಸೃಷ್ಟಿಸಿದವನು
76. ಜ್ಞಾನಪ್ರದಃ = ಜ್ಞಾನವನ್ನು ದಯಪಾಲಿಸುವವನು
77. ಮುಕ್ತಿಪ್ರದಃ = ಮುಕ್ತಿಯನ್ನು ಕರುಣಿಸುವವನು
78. ಭಕ್ತವಾಂಛಿತದಾಯಕಃ = ಭಕ್ತರಿಗೆ ಕೇಳಿದ ವರಗಳನ್ನು ಕೊಡುವವನು
79. ಆಶ್ಚರ್ಯ ವೈಭವಃ = ಅದ್ಭುತ ವೈಭವದ ವಿಭೂತಿಯನ್ನು ಹೊಂದಿದವನು
80. ಕಾಮಿ = ತುಂಬು ಪ್ರೇಮದಿಂದ ಕೂಡಿರುವವನು
81. ಸಂಗೀತವೇತ್ತಃ = ಸಂಗೀತದಲ್ಲಿ ನಿಪುಣ
82. ನೃತ್ಯಜ್ಞಃ = ನೃತ್ಯದಲ್ಲಿ ನಿಪುಣನು
83. ತ್ರಿವೇದಿಃ = ಮೂರು ವೇದಗಳಲ್ಲಿ ಪಾರಂಗತ
84. ವೃದ್ಧ ವೈದಿಕಃ = ಪುರಾತನದಿಂದಲೂ ವೇದಸ್ವರೂಪನು
85. ತ್ಯಾಗರಾಜಃ = ಕರುಣಾಕರ/ಕೃಪಾಕರ
86. ಕೃಪಾಸಿಂಧುಃ = ಕರುಣಾಸಾಗರ
87. ಸುಗಂಧಿಃ = ಸುವಾಸನೆಯುಳ್ಳವನು
88. ಸೌರಭೇಶ್ವರಃ = ನಂದಿಗೆ ಯಜಮಾನ
89. ಕಾರ್ತವೀರ್ಯೇಶ್ವರಃ = ಕಾರ್ತವೀರನಿಗೆ ಯಜಮಾನ
90. ಶಾಂತಃ = ಶಾಂತವಾಗಿರುವವನು
91. ಕಪಾಲಿಃ = ಕಪಾಲಗಳನ್ನು ಧರಿಸಿದವನು/ಕಪಾಲೇಶ್ವರ
92. ಕಲಶಪ್ರಭುಃ = ಕಲಶಗಳಿಗೆಲ್ಲಾ ಒಡೆಯ(ಪೂಜೆಯ ಕಲಶಗಳಲ್ಲಿ ಸಮಸ್ತ ದೇವತೆಗಳ ಆವಾಹನೆಯಾಗುತ್ತದೆ)
93. ನಿರವದ್ಯಃ = ಕಳಂಕರಹಿತನು
94. ನಿಧಿಪ್ರದಃ = ಐಶ್ವರ್ಯದಾತ
95. ಶಿವನಾಮಿಃ = ಶಿವನೆಂಬ ಹೆಸರುಳ್ಳವನು
96. ಮನೋವಾಸಃ = ಎಲ್ಲರ ಮನಸ್ಸಿನಲ್ಲೂ ನೆಲೆಸಿರುವವ (ಚಿದಾತ್ಮ)
97. ಸರ್ವಃ = ಸರ್ವರಿಗೂ ಪ್ರಭು/ದೇವ
98. ಕ್ರತುಧ್ವಂಸಿಃ = (ದಕ್ಷನ) ಯಜ್ಞವನ್ನು ನಾಶಮಾಡಿದವನು
99. ಶೂಲಿಃ = ತ್ರಿಶೂಲವನ್ನು ಧರಿಸಿದವನು
100. ಪಶುಪತಿಃ = ನಂದಿಕೇಶ್ವರನ/ಸಕಲ ಜೀವಿಗಳಿಗೂ ಒಡೆಯ
101. ಶಂಭುಃ = (ಸರ್ವರ) ಯೋಗಕ್ಷೇಮ ನೀಡುವವನು
102. ಸ್ವಯಂಭೂ = ಸ್ವಯಂ ಜನಿತ
103. ಗಿರೀಶಃ = ಗಿರಿಗಳಲ್ಲಿ/ಪರ್ವತಗಳಲ್ಲಿ ವಾಸಿಸುವವನು
104. ಮೃಡಃ = ಕರುಣಾಕರ
105. ವಿಧಿಪ್ರದಃ = ವಿಧಾಯಕ/ಕರ್ಮದಾತಾ
106. ಸರ್ವನಾಮಿ = ಎಲ್ಲ ಹೆಸರೂ ಅವನದೇ
107. ವೋಮಕೇಶಃ = ಸರ್ವವ್ಯಾಪೀ ಶಿವ
108. ಅರುಣಗಿರೀಶ್ವರ = ಅರುಣಪರ್ವತವೆಂಬ ರೂಪದಲ್ಲಿರುವ ಪರಮಾತ್ಮ/ಸದಾಶಿವ

ಇವುಗಳೇ ಈ ಕ್ಷೇತ್ರದಲ್ಲಿ ನನ್ನನ್ನು ಅರ್ಚಿಸುವಾಗ ಹೇಳಿಕೊಳ್ಳುವ ಮುಖ್ಯ ನಾಮಾವಳಿ. ಪುರಾಣದಲ್ಲಿ ಹೇಳಿರುವಂತಹ ಇನ್ನೂ ಅನೇಕ ನಾಮಗಳನ್ನೂ ಸ್ಮರಿಸಿಕೊಂಡು ನಿತ್ಯವೂ ವಿಶೇಷ ರೀತಿಯ ಪೂಜೆಯಿಂದ ನನ್ನ ಪ್ರದಕ್ಷಿಣೆಯನ್ನು ಮಾಡಬೇಕು. ಪರ್ವತರೂಪದಲ್ಲಿರುವ ಅರುಣಾಚಲಾಕಾರನಾದ ನನಗೆ ಪ್ರದಕ್ಷಿಣೆಯೇ ವಿಶೇಷವಾದ ಪ್ರೀತಿಯ ಪೂಜೆಯು." ಹೀಗೆ ಮಹಾದೇವನಿಂದ ಉಪದೇಶ ಹೊಂದಿದ ನಾನು ಅರುಣಾಚಲೇಶ್ವರನನ್ನು ಪೂಜಿಸುತ್ತಾ ನಿರಂತರವಾಗಿ ಇಲ್ಲಿಯೇ ವಾಸವಾಗಿದ್ದೇನೆ.

ಗೌರಿಯು ಆಗ ಗೌತಮ ಮುನಿಗಳನ್ನು ಕೇಳಿದಳು, "ಎಲೈ ಗೌತಮನೇ! ಷಡ್ಗುಣ ಸಂಪನ್ನರೂ, ಸಕಲ ಧರ್ಮಗಳನ್ನು ಅರಿತವರೂ, ವೃದ್ಧರೂ, ಆಗಿರುವ ತಮ್ಮಿಂದ ಪರ್ವತದ ರೂಪದಲ್ಲಿರುವ ಅರುಣಾಚಲನ ಪ್ರದಕ್ಷಿಣೆಯ ಮಹಾತ್ಮೆಯನ್ನು ಕೇಳ ಬಯಸುತ್ತೇನೆ. ಯಾವ ಕಾಲದಲ್ಲಿ, ಯಾವ ಸಂದರ್ಭದಲ್ಲಿ ಹೇಗೆ ಪ್ರದಕ್ಷಿಣೆಯನ್ನು ಮಾಡಬೇಕು? ಈ ಮೊದಲು ಅರುಣಗಿರಿನಾಥನ ಪ್ರದಕ್ಷಿಣೆಯ ಯಾರು ಮಾಡಿ ತಮ್ಮ ಇಷ್ಟಾರ್ಥಗಳನ್ನು ಹಾಗು ಮೋಕ್ಷವನ್ನು ಪಡೆದರು?"

ಬ್ರಹ್ಮನು ತನ್ನ ಪುತ್ರ ಸನಕನಿಗೆ ಹೇಳಿದನು. "ಗೌತಮರು ಗಿರಿಜೆಗೆ ಹೇಳಿದ್ದು ನಾನು ನಿನಗೆ ಹೇಳುತ್ತೇನೆ, ಕೇಳು. 'ಎಲೈ ದೇವಿಯೇ! ಮಹೇಶ್ವರನು ನನಗೆ ಪ್ರದಕ್ಷಿಣೆಯ ಮಹಾತ್ಮೆಯ ವಿಚಾರವನ್ನು ಏನು ಹೇಳಿರುವನೋ, ಅದನ್ನು ನಾನು ನಿನಗೆ ಹೇಳುವೆ. ನಾನು ಭೂಲೋಕದಲ್ಲಿ ಅರುಣಾಚಲೇಶ್ವರನಾಗಿ ಪರ್ವತದ ರೂಪದಲ್ಲಿ ನೆಲೆಸಿರುವೆನು. ನನ್ನ ಸುತ್ತಲೂ ಮಹರ್ಷಿಗಳೂ ದೇವತೆಗಳೂ ನೆಲೆಸಿ, ನನ್ನನ್ನು ಸೇವಿಸುತ್ತಿರುವರು. ಮಾನವನು ಜನ್ಮಾಂತರಗಳಲ್ಲಿ ಯಾವ ಪಾಪವನ್ನು ಮಾಡಿದ್ದರೂ, ಗಿರಿ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುವುದರಿಂದ ಹೆಜ್ಜೆ ಹೆಜ್ಜೆಗೂ ಅವನ ಪಾಪಗಳು ನಾಶವಾಗುತ್ತವೆ. ಗಿರಿ ಪ್ರದಕ್ಷಿಣೆಯಿಂದ ಸಾವಿರ ಅಶ್ವಮೇಧ ಯಾಗಗಳು ಫಲವೂ, ಸರ್ವತೀರ್ಥ ಸ್ಥಾನ ಫಲವೂ ಒಮ್ಮೆಗೇ ಸಿದ್ಧಿಸುತ್ತವೆ. ಶುಭಲಕ್ಷಣ ಹೀನನೂ, ವೈದಿಕ ಕ್ರಿಯೆಗಳನ್ನು ಮಾಡದವನೂ, ನೀಚ ಜಾತಿಯಲ್ಲಿ ಹುಟ್ಟಿದವನಾದರೂ ಅರುಣಾಚಲ ಪ್ರದಕ್ಷಿಣೆಯಲ್ಲಿ ಮಾಡಿದರೆ ಅವನಿಗೆ ಸಕಲ ಸಿದ್ಧಿ ಗಳೂ ಸಿದ್ಧಿಸುವುವು ಎಂಬುದು ಖಚಿತ. ಈ ಪ್ರದಕ್ಷಿಣೆಯ ಸಮಯದಲ್ಲಿ ಮೊದಲನೆ ಹೆಜ್ಜೆಯಿಂದ ಭೂಲೋಕವನ್ನೂ, ಎರಡನೆ ಹೆಜ್ಜೆಯಿಂದ ಅಂತರಿಕ್ಷವನ್ನೂ, ಮೂರನೆ ಹೆಜ್ಜೆಯಿಂದ ಸ್ವರ್ಗಲೋಕವನ್ನೂ ಜಯಿಸಬಹುದು. ಒಂದು ಹೆಜ್ಜೆಯಿಂದ ಮಾನಸಿಕ ತಾಪಗಳನ್ನೂ, ಎರಡನೆಯದರಿಂದ ವಾಚಕ ಪಾಪಗಳನ್ನೂ ಮೂರನೆಯ ಹೆಜ್ಜೆಯಿಂದ ಕಾಯಿಕ ಪಾಪಗಳನ್ನೂ ಕಳೆದುಕೊಳ್ಳುವರು. ಈ ಕ್ಷೇತ್ರದಲ್ಲಿ ಸಾವಿರಾರು ಸಿದ್ಧರ, ಮಹರ್ಷಿಗಳ ಪರ್ಣಶಾಲೆಗಳೂ, ದೇವತೆಗಳ ನಿವಾಸ ಸ್ಥಾನಗಳು ಇರುವುವು. ಈ ಪರ್ವತ ಶಿಖರದಲ್ಲಿ ನಾನು ಸಿದ್ದನಾಗಿ ನಿರಂತರವೂ ದೇವತೆಗಳ ಪೂಜೆಗಳನ್ನು ಸ್ವೀಕರಿಸುತ್ತಾ ಇರುವೆನು. ಧ್ಯಾನ ಮಾಡುವವರು ನನ್ನೊಳಗಿರುವ ದಿವ್ಯಭೋಗಗಳಿಂದ ಕೂಡಿರುವ ಗುಹೆಯನ್ನು ಧ್ಯಾನಿಸಬೇಕು. ಅರುಣಾದ್ರಿ ಎಂದು ಪ್ರಖ್ಯಾತವಾಗಿರುವ ಪರ್ವತವನ್ನು ದೊಡ್ಡದಾದ ಅಗ್ನಿಸ್ತಂಭ ರೂಪದಲ್ಲಿ ಧ್ಯಾನಿಸುತ್ತಾ ನಿಧಾನವಾಗಿ ಗಿರಿ ಪ್ರದಕ್ಷಿಣೆಯನ್ನು ಮಾಡಬೇಕು. ಅಷ್ಟಮೂರ್ತಿಗಳನ್ನೊಳಗೊಂಡ ಈ ಅರುಣಾದ್ರಿಯನ್ನು ಈ ವಿಶೇಷ ರೀತಿಯಿಂದ ಯಾರು ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುತ್ತಾರೋ, ಅವರು ತಮ್ಮ ಸಕಲ ಪಾಪಗಳನ್ನೂ ನಾಶ ಮಾಡಿಕೊಳ್ಳಬಲ್ಲರು. ಪ್ರತಿದಿನವೂ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದ ಭಕ್ತರಿಗೆ ಪುನರ್ಜನ್ಮವಿಲ್ಲ. ಯಾರೇ ಈ ಗಿರಿಯ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಲು ಭಕ್ತಿಯಿಂದ ಒಂದು ಹೆಜ್ಜೆಯನ್ನಿಟ್ಟರೂ, ಭೂಮಿಯೆಲ್ಲಾ ಪವಿತ್ರವಾಗುವುದು. ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುವಾಗ ಭಕ್ತಿಯಿಂದ ಕರಗಳನ್ನು ಜೋಡಿಸಿ, ಪರ ಸಂಪರ್ಕವಿಲ್ಲದೆ ಧ್ಯಾನ ಸ್ತೋತ್ರಾದಿಗಳ ಸಹಿತ ಅಷ್ಟ ದಿಕ್ಕುಗಳನ್ನೂ ನಮಸ್ಕರಿಸುತ್ತಾ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಬೇಕು. ನಿಧಾನವಾಗಿ, ಮೆಲ್ಲ ಮೆಲ್ಲನೆ ಯಾವ ರೀತಿಯ ಕೋಪ, ಆತಂಕಗಳಿಲ್ಲದೆಯೆ ನಡೆಯುವ ತನ್ನ ಹೆಜ್ಜೆಯ ಶಬ್ದವೂ ತನಗೆ ಕೇಳಿಸದ ಹಾಗೆ ತುಂಬು ಗರ್ಭಿಣಿಯು ನಡೆಯುವ ರೀತಿಯಲ್ಲಿ ನಡೆಯಬೇಕು. ಭಸ್ಮವನ್ನು ಲೇಪಿಸಿ, ರುದ್ರಾಕ್ಷಿಯನ್ನು ಧರಿಸಿ, ಮನಸ್ಸಿನಲ್ಲೇ ಶಿವಧ್ಯಾನವನ್ನು ಮಾಡುತ್ತಾ ವಿವೇಕದಿಂದ ನಿಧಾನವಾಗಿ ಹೆಜ್ಜೆ ಹಾಕಬೇಕು. ಏಕೆಂದರೆ ಅದೃಶ್ಯರೂಪಗಳಲ್ಲಿ ಅನೇಕ ದೇವತೆಗಳೂ, ಮನುಗಳೂ, ಸಿದ್ದರೂ ವಾಯು ರೂಪರಾಗಿ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುತ್ತಲಿರುವರು. ಅವರಿಗೆ ಯಾವುದೇ ತೊಂದರೆಯಾಗದ ರೀತಿ ನಡೆಯಬೇಕು. ಗುಂಪಿನಲ್ಲಿ ಹೋದರೆ, ಶಿವಸಂಕೀರ್ತನೆ ಮಾಡುತ್ತಾ, ಶಿವನಾಟ್ಯಗಳನ್ನು ನಟಿಸುತ್ತಾ, ಭಕ್ತರೊಡನೆ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಬೇಕು. ನನ್ನ ಮಹಾತ್ಮೆಯ ಕಥೆಗಳನ್ನು ಕೇಳುತ್ತಾ, ಇತರೆ ಹವ್ಯಾಸಗಳಲ್ಲಿ ಮನವಿಡದೆ ಆನಂದವನ್ನು ಅನುಭವಿಸುತ್ತಾ ಪ್ರದಕ್ಷಿಣೆಯನ್ನು ನಾನಾ ವಿಧಗಳಾದ ದಾನಗಳನ್ನು ಮಾಡುತ್ತಾ, ಪುಣ್ಯ ಕಾರ್ಯಗಳಲ್ಲಿ ತೊಡಗಿ, ದಯಾಭರಿತರಾಗಿ, ಆಸ್ತಿಕ ಬುದ್ದಿಯಿಂದ ಪರ್ವತ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಬೇಕು.

ಈ ಪರ್ವತವು ಕೃತಯುಗದಲ್ಲಿ ಅಗ್ನಿಮಯವಾಗಿಯೂ, ತ್ರೇತಾಯುಗದಲ್ಲಿ ಮಣಿಮಯವಾಗಿಯೂ, ದ್ವಾಪರ ಯುಗದಲ್ಲಿ ಹೇಮಮಯವಾಗಿಯೂ, ಕಲಿಯುಗದಲ್ಲಿ ಪಚ್ಚೆ ವರ್ಣದ್ದಾಗಿಯೂ ಇರುವುದು. ಸ್ವಯಂಜ್ಯೋತಿಯಿಂದ ಕೂಡಿದ ಅರುಣ ವರ್ಣದ ಸ್ಪಟಿಕ ಲಿಂಗವನ್ನು ಧ್ಯಾನಿಸಿದರೆ, ಭಕ್ತರು ಸಕಲ ಪಾಪಗಳನ್ನು ಕಳೆದುಕೊಂಡು ಕೈಲಾಸದಲ್ಲಿ ನೆಲೆಸುವರು. ಈ ಸ್ಥಿರಲಿಂಗವು ಅಪ್ರಮೇಯ ಸ್ವರೂಪಿಯೂ, ಅಗೋಚರವೂ, ಅಗ್ನಿಮಯವೂ ಆಗಿರುವುದರಿಂದ ಸಾಮಾನ್ಯರಿಗೆ ಈ ಲಿಂಗದ ನಿಗೂಢಾರ್ಥದ ಅರಿವಾಗುವುದಿಲ್ಲ. ಆದರೆ ನನ್ನ ಧ್ಯಾನದಲ್ಲಿಯೇ ನಿರತರಾಗಿರುವವರಿಗೆ ಕೂಡಲೇ ನಾನು ಪ್ರತ್ಯಕ್ಷನಾಗುತ್ತೇನೆ. ಭಕ್ತಿಯಿಂದ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುವವರಿಗೆ ಪರ್ವತದ ಪಾದಸ್ಪರ್ಶದಿಂದಲೇ ಅಮರತ್ವ ಪ್ರಾಪ್ತಿಯಾಗುವುದು. ಗಿರಿ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುವ ಭಕ್ತರಿಗೆ ಸುಖ ನೀಡಲು ದೇವ ವೃಕ್ಷಗಳು ನೆರಳು ನೀಡುತ್ತವೆ. ದೇವತೆಗಳ, ಗಂಧರ್ವರ ವಿಮಾನಗಳು ನಾಮುಂದು ತಾಮುಂದು ಎಂದು ಧಾವಿಸಿ ಮುದ ನೀಡುತ್ತವೆ. ಗಿರಿ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದವರಿಗೆ ಸ್ವರ್ಗಲೋಕಯೋಗ್ಯವಾದ ವಜ್ರದಂತಹ ದೇಹ ದೊರಕಿ, ಅವರು ಯಾರಿಂದಲೂ ಪರಾಜಯ ಹೊಂದುವುದಿಲ್ಲ. ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದವರ ಪಾದಧೂಳಿಯಿಂದ ನಗರದ ಬೀದಿಗಳೆಲ್ಲಾ ಪಾವನವಾಗುವುವು. ವಿಮಾನಗಳಲ್ಲಿ ಸಂಚರಿಸುತ್ತಿರುವ ದೇವತೆಗಳು, ಸಿದ್ದರು, ಬ್ರಹ್ಮರ್ಷಿಗಳು, ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುತ್ತಿರುವ ಭಕ್ತರ ವಿನಯ ಭಕ್ತಿಗಳನ್ನು ಕಂಡು ಸಂತೋಷಭರಿತರಾಗಿ ಕೇಳಿದ ವರಗಳನ್ನು ನೀಡುವರು. ಪೂರ್ವದಲ್ಲಿ ಮೂವತ್ಮೂರು ದೇವತೆಗಳು ದಿನವೂ ನನ್ನ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದ್ದರಿಂದ ಒಬ್ಬೊಬ್ಬರೂ ಒಂದೊಂದು ಕೋಟಿ ರೂಪಗಳನ್ನು ಹೊಂದಿದರು. ಆದಿತ್ಯಾದಿ ನವಗ್ರಹಗಳೂ ಪೂರ್ವದಲ್ಲಿ ನನ್ನ ಈ ಗಿರಿರೂಪಕ್ಕೆ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದ ಪುಣ್ಯದ ಫಲವಾಗಿಯೇ ಸಕಲ ಲೋಕಗಳಲ್ಲೂ ಗ್ರಹಾಧಿಪತಿಗಳಾದರು.

ಭಾನುವಾರದ ದಿನ ಅರುಣಾಚಲದ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದರೆ, ಭಕ್ತರು ಸೂರ್ಯಮಂಡಲವನ್ನು ಛೇದಿಸಿ ಕೈಲಾಸವನ್ನು ಪಡೆಯುವರು. ಸೋಮವಾರದ ದಿನ ಪ್ರದಕ್ಷಿಣೆಯಿಂದ ಜರಾಮೃತ್ಯುಗಳ ಭಯದಿಂದ ಬಿಡುಗಡೆ ಹೊಂದಿ ಬ್ರಾಹ್ಮಣರಾಗಿ ಜನಿಸುವರು. ಮಂಗಳವಾರದ ದಿನ ಗಿರಿಯ ಪ್ರದಕ್ಷಿಣೆಯಿಂದ ಎಲ್ಲಾ ತರಹದ ಋಣಗಳಿಂದ ಮುಕ್ತರಾಗಿ ಶಾಶ್ವತವಾದ ಸಾರ್ವಭೌಮತ್ವವನ್ನು ಪಡೆಯುವರು. ಬುಧವಾರದ ಪ್ರದಕ್ಷಿಣೆಯ ಫಲವಾಗಿ ಮಾನವನು ಸರ್ವಜ್ಞನಾಗಿ ಬೃಹಸ್ಪತಿಯ ಸ್ಥಾನವನ್ನು ಪಡೆಯುವನು. ಗುರುವಾರದ ದಿನದ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದ ಮಾನವನನ್ನು ಲೋಕಗುರುವೆಂದು ನಮಸ್ಕರಿಸುವರು. ಶುಕ್ರವಾರದ ದಿನದಂದು ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದರೆ ಇಹಲೋಕದಲ್ಲಿ ಅಷ್ಟೈಶ್ವರ್ಯ ಹೊಂದಿ ಪರಲೋಕದಲ್ಲಿ ವಿಷ್ಣುಪದವನ್ನು ಹೊಂದುವರು. ಶನಿವಾರದ ದಿನ ಗಿರಿಪ್ರದಕ್ಷಿಣೆಯನ್ನು ಮಾಡುವುದರಿಂದ ಸಕಲಗ್ರಹ ಪೀಡೆಗಳಿಂದ ಮುಕ್ತರಾಗಿ ಮೂರು ಲೋಕಗಳಲ್ಲೂ ವಿಜಯ ಸಾಧಿಸುವರು.

ನನ್ನನ್ನು ಭಕ್ತಿಯಿಂದ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುವವನಿಗೆ, ಸಕಲ ನಕ್ಷತ್ರಗಳೂ ತಮ್ಮ ಅಧಿದೇವತೆಗಳೊಂದಿಗೆ ಸಕಲ ಪುಣ್ಯಗಳನ್ನೂ, ಸಕಲ ತಿಥಿಗಳು, ಕರಣಗಳು, ಯೋಗಗಳು ಸಕಲಾಭೀಷ್ಟಗಳನ್ನೂ ನೀಡಿ ನಿತ್ಯವೂ ಮಂಗಳಕರಗಳಾಗಿ ಶುಭವನ್ನುಂಟು ಮಾಡುತ್ತವೆ.

ಪ್ರ ಎಂಬಕ್ಷರವು ಪಾಪಗಳನ್ನು ಕಳೆಯುವುದು.
ದ ಕಾರವು ಇಷ್ಟಾರ್ಥ ಪ್ರದಾಯಕವಾಗಿರುವುದು.
ಕ್ಷಿ ಎಂಬ ಅಕ್ಷರದಿಂದ ಕರ್ಮಬಂಧವು ಕ್ಷಯವಾಗುವುದು.
ಣ ಕಾರದಿಂದ ಮುಕ್ತಿಯನ್ನು ಪಡೆಯಲು ಸಾಧ್ಯವಾಗುವುದು.

ದುರ್ಬಲರಾದವರು, ಕೃಶಾಂಗರು, ಮಾನಸಿಕ ಹಾಗು ದೈಹಿಕ ಬೇನೆಗಳಿಂದ ಬಳಲುತ್ತಿರುವವರು ನನ್ನನ್ನು ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದರೆ ಸರ್ವಪಾಪಗಳೂ ಪರಿಹಾರವಾಗಿ ಮುಕ್ತರಾಗುವರು. ಶ್ರದ್ಧಾಭಕ್ತಿಗಳಿಂದ ನಿತ್ಯವೂ ಗಿರಿಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದರೆ, ಮೂರು ಲೋಕಗಳ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದಂತಾಗುತ್ತದೆ. ಸಕಲ ಲೋಕಗಳ ನಾಯಕರೂ, ಅಷ್ಟ ದಿಕ್ಪಾಲಕರೂ, ಪ್ರಪಂಚದ ಸೃಷ್ಟ್ಯಾದಿಗಳಿಗೆ ಕಾರಣರಾದವರು ಕಾರ್ಯಗಳನ್ನು ಪ್ರಾರಂಭಿಸುವ ಮುನ್ನ ನನಗೆ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿಯೇ ಅವರವರ ಕಾರ್ಯಗಳನ್ನು ಮಾಡುವ ಅಧಿಕಾರವನ್ನು ಸ್ವೀಕರಿಸುವರು. ನಾನೂ ಸಹ ಉತ್ತರಾಯಣ ಕಾಲದಲ್ಲಿ ತ್ರಿಲೋಕ ಕಲ್ಯಾಣಕ್ಕಾಗಿ ನನ್ನ ಸಕಲ ಗಣಗಳೊಡನೆ ಆತ್ಮ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿಕೊಳ್ಳುತ್ತೇನೆ

ಗೌರಿಯೂ ಸಹ ತನ್ನ ದೇಹಾವಸಾನ ಕಾಲದಲ್ಲಿ ಇಲ್ಲಿಗೆ ಬಂದು ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿ ನನ್ನನ್ನು ಮರಳಿ ಪಡೆದಳು. ದೇವಿಯ ತಪಃಪ್ರಭಾವದಿಂದ ಅವಳಿಗೆ ತೋರುವ ಈ ಪ್ರತ್ಯಕ್ಷ ತೇಜೋಲಿಂಗದ ರೂವವು ಅಲ್ಪ ಪುಣ್ಯವಂತರಿಗೆ ಅತ್ಯಂತ ದುರ್ಲಭವಾಗಿರುವುದು. ಉತ್ತರಾಯಣದ ಸಂಗಮದ ಕಾಲದಲ್ಲಿ ಗಿರಿಜೆಯು ಅಪೀತಕುಚಾಂಬೆ ಎಂಬ ನಾಮದಿಂದ ನನ್ನೊಡನೆ ಇದ್ದು, ಸಕಲ ದೇವತೆಗಳಿಗೆ ಸಮಾಧಾನ ಹೇಳುವಳು. ಆ ಸಮಯದಲ್ಲಿ ದೇವತೆಗಳು, ಗಂಧರ್ವರು, ಯಕ್ಷರು, ಸಿದ್ದರು, ರಾಕ್ಷಸರು ಹಾಗೂ ಎಲ್ಲ ಪುಣ್ಯವಂತರೂ ಬಂದು ಇಲ್ಲಿ ಸೇರುವರು. ಆಗ ಯಾವ ತಪೋನಿಧಿಗಳು ನನ್ನನ್ನು ಪೂಜಿಸುವರೋ ಅವರ ಸಕಲ ಜನ್ಮಗಳ ಪಾಪಗಳಿಗೂ ಪ್ರಾಯಶ್ಚಿತ್ತ ಮಾಡಿಕೊಂಡಂತಾಗುವುದು. ಆ ಕಾಲದಲ್ಲಿ ನನ್ನನ್ನು ಮಾನವರು ಅರ್ಚಿಸಿದರೆ ಕೃತಾರ್ಥರಾಗುವರು. ಯಾವ ರಾಜನು ನನ್ನನ್ನು ಉತ್ತಮ ರೀತಿಯಲ್ಲಿ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುವನೋ ಅವನ ಸೈನ್ಯಕ್ಕೆ ನಾನೇ ಮುಂದಾಳಾಗಿ ಎಲ್ಲ ಶತ್ರುಗಳನ್ನು ಕೊಲ್ಲುವೆನು. ದೇಶಾಧಿಪತಿಯೂ, ಸಂಪದ್ಭರಿತನೂ ಆದ ರಾಜನು ತನ್ನ ರಾಜ್ಯದಲ್ಲಿರುವ ಶ್ರೋತ್ರಿಯ ಬ್ರಾಹ್ಮಣರಿಂದ ನನ್ನ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿಸಬೇಕು. ನನ್ನನ್ನು ಈ ರೀತಿಯಾಗಿ ಒಂದು ಮಂಡಲವಾಗಲೀ, ಮಂಡಲಾರ್ಧವಾಗಲಿ ವಿಧಿಪೂರ್ವಕವಾಗಿ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿದರೆ ಆ ರಾಜರ ರಾಜ್ಯವು ಸ್ಥಿರವಾಗಿ ನಿಲ್ಲುವುದು. ಭಕ್ತರು ಧರ್ಮಲೋಭದಿಂದಾಗಲೀ, ಶಿವಾಚಾರದ ವಿರೋಧವಾಗಲೀ, ವಾಹನಗಳ ಸಹಾಯದಿಂದಾಗಲೀ ಸರ್ವಥಾ ಪ್ರದಕ್ಷಿಣೆಯನ್ನು ಮಾಡಕೂಡದು.

ಪೂರ್ವದಲ್ಲಿ ಧರ್ಮಕೇತುವೆಂಬ ರಾಜನು ಯಮಲೋಕದಿಂದ ಬಂದು ತನ್ನ ಕುದುರೆಯನ್ನೇರಿದಾಗ, ಆ ಕುದುರೆ ಗಣನಾಯಕನ ರೂಪ ಧರಿಸಿ ದೇವತೆಗಳಿಂದ ಪೂಜಿಸಲ್ಪಟ್ಟಿತು. ರಾಜನನ್ನು ಭೂಲೋಕದಲ್ಲಿ ಬಿಟ್ಟು ತಾನು ಕೈಲಾಸವನ್ನು ಸೇರಿತು. ಆಗ ಹೀಗೆ ತನ್ನ ಕುದುರೆ ಗಣರೂಪವನ್ನು ಹೊಂದಿದ್ದನ್ನು ನೋಡಿ ತಾನೂ ಸಹ ಕಾಲ್ನಡಿಗೆಯಲ್ಲೇ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಿ, ಗಣಾಧೀಶನಾದನು. ಅಂದಿನಿಂದ ಇಂದ್ರ, ವಿಷ್ಣು ಮೊದಲಾದ ದೇವತೆಗಳು ನನ್ನ ಪ್ರದಕ್ಷಿಣೆಯನ್ನು ಪಾದ ಸಂಚಾರದಿಂದಲೇ ಮಾಡುವರು.

ಒಬ್ಬಾನೊಬ್ಬ ಸಿದ್ಧನು ತನ್ನ ತಪೋಬಲವು ಕ್ಷೀಣವಾದುದರಿಂದ ಸ್ವರ್ಗಚ್ಯುತನಾದನು. ಆವನು ಭೂಲೋಕಕ್ಕೆ ಬಂದು ನನ್ನ ಪ್ರದಕ್ಷಿಣೆ ಮಾಡಲು, ಆ ಪುಣ್ಯ ಫಲದಿಂದ ಸ್ವರ್ಗದಲ್ಲಿ ತನ್ನ ಸ್ಥಾನವನ್ನು ಪುನಃ ಪಡೆದನು. ನನ್ನ ಭಕ್ತರು ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುವಾಗ ಎಡವಿದರೆ ಅವರ ಕಾಲಿನಿಂದ ಬಿದ್ದ ರಕ್ತಬಿಂದುವು ದೇವೇಂದ್ರನ ಕಿರೀಟದಲ್ಲಿರುವ ಮಂದಾರ ಪುಷ್ಪಗಳ ಕೇಸರದಿಂದ ತೊಡೆಯಲ್ಪಡುತ್ತದೆ ಹಾಗು ಚೂಪಾದ ನೊರಜುಗಲ್ಲುಗಳಿಂದ ನೋವುಂಟಾದರೆ, ಲಕ್ಷ್ಮಿಯ ಕುಂಕುಮದಿಂದ ಸವರಲ್ಪಡುತ್ತದೆ. ನನ್ನನ್ನು ಪ್ರದಕ್ಷಿಣೆ ಮಾಡುವ ಮಾನವರು ಸ್ವರ್ಗಲೋಕದ ಮಣಿಪರ್ವತಗಳಲ್ಲಿಯೂ, ಕಲ್ಪವೃಕ್ಷದ ವನಗಳಲ್ಲಿಯೂ ವಿಹರಿಸುತ್ತಾರೆ."

ಅರುಣಾಚಲದ ಗಿರಿ ಪ್ರದಕ್ಷಿಣೆಯ ಮಹಾತ್ಮೆಯನ್ನು ಕೇಳಿದ ಗೌತಮ ಮುನಿಗಳನ್ನು ಕುರಿತು ಇಂತೆಂದಳು; "ಎಲೈ ತಪೋಧನನೇ! ಮಾನವನು ತನ್ನ ಶಕ್ತಿ ಅನುಸಾರವಾಗಿ ಅರುಣಾಚಲೇಶ್ವರನ ಪೂಜೆ ಹಾಗು ಸೇವೆಗಳನ್ನು ಮಾಡುವುದರಿಂದ ಸಿಗು ಫಲವೇನು?"

ಗೌತಮರು ಹೀಗೆ ಪರಮೇಶ್ವರನ ಸೇವೆಯಿಂದ, ಅವನಿಂದಲೇ ಹೇಳಲ್ಪಟ್ಟ ಸೇವೆಯ ಫಲವನ್ನು ಕೇಳು. ಒಂದು ಜೇಡರ ಹುಳುವು ಒಮ್ಮೆ ನನಗೆ (ಶಿವನಿಗೆ) ದಾರಗಳನ್ನು ಹೊದಿಸಿದ ಪೂಳೆ ಪ್ರಭಾವದಿಂದ ಜನ್ಮಾಂತರದಲ್ಲಿ ಮನುಷ್ಯನಾಗಿ ಜನಿಸಿ, ನನಗೆ ಉತ್ತಮ ವಸ್ತ್ರಗಳು ತೊಡಿಸಿತು. ಆನೆಯೊಂದು ಬಾಯಾರಿಕೆಯಿಂದ ಗಿಡದ ಚಿಗುರನ್ನು ತಿನ್ನುವಾಗ ಅಪ್ರಯತ್ನವಾಗಿ ಹಾರಿದ ಜೇನುತುಪ್ಪದ ಹನಿಯೊಂದು ನನ್ನ ಮೇಲೆ ಬಿದ್ದುದರಿಂದ ಆನೆಯು ಗಣಾಧಿಪತಿಯಾಯಿತು. ಕ್ರಿಮಿ ಕೀಟಗಳೂ ಸಹ ನನ್ನ ಮೇಲೆ ಹೊರಳಿದುದರ ಪರಿಣಾಮವಾಗಿ ಆ ಪುಣ್ಯದಿಂದ ಉತ್ತಮ ಜನ್ಮ ತಾಳಿ ನನ್ನ ಸಾಮೀಪ್ಯವನ್ನೆ; ಹೊಂದಿದವು. ಗಿಡಗಳಲ್ಲಿ ವಾಸಿಸುವ ಬೆಳಕಿನ ಪತಂಗವೊಂದು ಒಮ್ಮೆ ರಾತ್ರಿಯಲ್ಲಿ ದೀಪಕ್ಕೆರಗುವ ರೀತಿಯಲ್ಲಿ ನನ್ನ ಬಳಿಗೆ ಬಂದಾಗ, ಅದು ತನ್ನ ಬೆಳಕಿನಿಂದ ನನ್ನನ್ನ ಬೆಳಗಿ ಮೋಕ್ಷವನ್ನು ಪಡೆಯಿತು. ಭಕ್ತರು ಕ್ಷಣಕಾಲವಾದರೂ ನನಗೆ ಅಖಂಡ ಜ್ಯೋತಿಗಳ ಬೆಳಗುವ ಕೈಂಕರ್ಯ ಮಾಡಿದರೆ ಅವರೇ ಸ್ವಯಂ ಜ್ಯೋತಿಯಾಗಿ ನನ್ನ ಸಾರೂಪ್ಯವನ್ನ ಹೊಂದುವರು. ಕೆಲವು ಹಸುಗಳು ತಮ್ಮ ಕರುಗಳನ್ನು ನೆನೆದು ತಮ್ಮ ಕೆಚ್ಚಲಿನಲ್ಲಿ ಉಕ್ಕಿ ಬಂದ ಹಾಲನ್ನು ಅಪ್ರಯತ್ನವಾಗಿಯಾದರೂ ನನ್ನ ಪಾಶ್ರ್ವದಲ್ಲಿ ಸುರಿಸಿದ್ದರಿಂದ ಮೋಕ್ಷವನ್ನು ಹೊಂದಿದವು. ಒಂದು ಕಾಗೆಯು ತನ್ನ ಆಹಾರಕ್ಕಾಗಿ ನನ್ನ ಬಳಿ ಹಾರಿ ಬರುವಾಗ ಅದರ ರೆಕ್ಕೆಗಳಿಂದ ಬೀಸಿದ ತಂಗಾಳಿಯಿಂದ ನನ್ನ ಮುಖಕ್ಕೆ ತಂಪಡರಿ ಆ ಕಾಗೆಗೆ ಮುಕ್ತಿಯಾಯಿತು. ಒಂದಾನೊಂದು ಇಲಿಯು ಹೊಳೆಯುವ ಮಣಿಗಳನ್ನು ಎಳೆದು ತರುವಾಗ, ಆ ಮಣಿಗಳ ಪ್ರಕಾಶದಿಂದ ನನ್ನ ಗರ್ಭಗೃಹದಲ್ಲಿ ಕತ್ತಲು ಕರಗಿ ಬೆಳಗಲು ಆ ಪುಣ್ಯದ ಫಲದಿಂದ ಇಲಿಯು ಮೋಕ್ಷವನ್ನು ಪಡೆಯಿತು.

ಮಹರ್ಷಿಗಳು, ದೇವತೆಗಳು ಮೋಕ್ಷವನ್ನು ಪಡೆಯುವುದಕ್ಕಾಗಿ ನನ್ನ ಬಳಿಯ ವೃಕ್ಷಗಳಾಗಿ, ಭಕ್ತರಿಗೆ ನೆರಳು ನೀಡಿ ಪುನೀತರಾಗುತ್ತಾರೆ. ನನ್ನ ಸಮೀಪದಲ್ಲಿಯ ಗೋಪುರ, ಶಿಖರ, ಶಾಲೆ, ಮಂಟಪ, ಕೊಳ ಮುಂತಾದುವುಗಳನ್ನು ನಿರ್ಮಿಸಿದವರಿಗೆ ಸರ್ವ ಇಷ್ಟಾರ್ಥಗಳೂ ಸಿದ್ಧಿಸುವುವು. ಈ ಅಗ್ನಿಲಿಂಗದ ಸಮೀಪ ಬರುವುದು ಅಸಾಧ್ಯದ ವಿಷಯವಾದುದರಿಂದ ಮನುಷ್ಯರು ಅಚಲೇಶ್ವರ ಎಂಬ ರೂಪವನ್ನು ಪೂಜಿಸಲಿ. ಪರಾಶಕ್ತಿಯಾದ ಅಪೀತಕುಚಾಂಬೆಯನ್ನು ನೋಟದಿಂದ, ಸ್ಪರ್ಶದಿಂದ ಹಾಗೂ ಧ್ಯಾನದಿಂದ ಪೂಜಿಸಲಿ. ಏಕೈಕ ಜಗನ್ಮಾತೆಯಾದ ಚಿರ ಯೌವ್ವನೆಯಾದ ಅಪೀತಕುಚಾಂಬೆಯನ್ನು ಯೌವನ ಬಯಸುವ ಸರ್ವರೂ ಪೂಜಿಸಲಿ."

ಗೌತಮರು ಮುಂದುವರೆಸುತ್ತಾ ಹೇಳಿದರು: "ಎಲೈ ಪಾರ್ವತಿಯೇ! ಒಂದೇ ಒಂದು ಕ್ಷಣವಾದರೂ ಅವನ ಬಳಿ ನಿಂತವರಿಗೆ ಇಹಲೋಕದಲ್ಲಾಗಲೀ, ಪರಲೋಕದಲ್ಲಾಗಲೀ ದೊರಕದ ವಸ್ತುವೇ ಇಲ್ಲ. ಅನಂತ ಗುಣಗಳಿಂದ ಕೂಡಿದವನೂ, ವರದೇಶ್ವರನೂ, ಅತುಳ ಐಶ್ವರ್ಯವುಳ್ಳವನೂ ಆದ ಆ ಶೋಣಶೈಲನನ್ನು ಭಕ್ತಿಯಿಂದ ಪೂಜಿಸು. ನಿನ್ನ ವ್ರತಗಳ ಮಹಿಮೆಯಿಂದ ನಿನ್ನ ಇಷ್ಟಾರ್ಥಗಳು ನೆರವೇರಿ, ಪರಮೇಶ್ವರನನ್ನು ಸೇರುವೆ. ದೇವತೆಗಳೂ ಸಹ ಹಲವಾರು ಕಾರಣಗಳಿಂದಾಗಿ ಇಲ್ಲಿ ತಪವನ್ನು ಗೈಯ್ಯುವರು. ಅದರ ರಹಸ್ಯವು ಅವರ ತಪಸ್ಸುಗಳ ಫಲದಿಂದಲೇ ತಿಳಿಯುತ್ತದೆ. ನಾವೂ ಸಹ ನಿನ್ನ ಸಹವಾಸದಿಂದಲೂ, ನಿನ್ನ ವ್ರತ ನಿಯಮಗಳ ದರ್ಶನದಿಂದಲೂ ಕೃತಾರ್ಥರಾಗುತ್ತೇವೆ. ದೇವನಾಯಕಿಯಾದ ನಿನ್ನ ತಪಸ್ಸುಗಳು ಫಲದಿಂದ ನಮಗೆಲ್ಲಾ ಭಗವಂತನ ಆಶೀರ್ವಾದವೂ, ಜೀವನದಲ್ಲಿ ತೃಪ್ತಿಯೂ ಸಿಗುವುದು."
ಗೌತಮ ಮುನಿಗಳ ಅರ್ಥಗರ್ಭಿತ ವಾಕ್ಕುಗಳನ್ನು ಕೇಳಿದ ಗಿರಿಜೆಯು ಆಶ್ಚರ್ಯದಿಂದಲೂ, ಆನಂದದಿಂದಲೂ ಅವರನ್ನು ತುಂಬು ಹೃದಯದಿಂದ ಹೀಗೆ ಹೊಗಳಿದಳು: "ಎಲೈ ಮುನಿವರ್ಯರೇ! ನಾನು ಇನ್ನು ಯಾವ ತಪಸ್ಸನ್ನು ಮಾಡಬೇಕು? ತಮ್ಮ ಕೃಪೆಯಿಂದ ಅರುಣಾಚಲನ ಮಹಿಮೆಯನ್ನು ಕೇಳಿದೆನು. ಅರುಣಾಚಲದ ದರ್ಶನವಾಗಿ ನಿಮ್ಮ ದರ್ಶನದ ಲಾಭವೂ ಆಯಿತು. ಈ ಭೂಲೋಕವು ಸ್ವರ್ಗಕ್ಕಿಂತಲೂ ಮಿಗಿಲಾಗಿರುವುದು. ಇಲ್ಲಿಯೇ ದೇವತೆಗಳಿಗೂ ಪ್ರಿಯವಾದ ಸಕಲ ಅಭೀಷ್ಟಗಳನ್ನೂ ನೆರವೇರಿಸುವ ತೇಜೋಲಿಂಗವಿದೆ. ಸದಾಶಿವನು ನನಗೆ ಪ್ರಸನ್ನನಾಗಿರುವನು. ಅವನ ನಿವಾಸ ಸ್ಥಾನವೂ ನನಗೆ ತಿಳಿಯಿತು. ಇಲ್ಲಿಯೇ ನೆಲೆಸಿ ಆ ಪರಮೇಶ್ವರನನ್ನು ಗೆಲ್ಲುವೆನು. ನಾನು ಪರಮೇಶ್ವರನಿಂದ ಬೇರ್ಪಡದೆ ನಿರಂತರವಾಗಿ ಅವನೊಡನೆಯೇ ಇರಬೇಕು. ತಾವು ಮಾಡಿದ ಈ ಸಹಾಯದಿಂದ ನಾನು ಆ ಪರಶಿವನಿಗೆ ಪರಮಪ್ರಿಯಳಾಗುತ್ತೇನೆ."

ಹೀಗೆಂದು ಹೇಳಿ ಪಾರ್ವತೀ ದೇವಿಯು ಗೌತಮ ಋಷಿಗಳ ಸನ್ನಿಧಿಯಲ್ಲಿ ತಪಸ್ಸನ್ನು ಮಾಡಲು ನಿರ್ಧರಿಸಿದಳು. ಅವರ ಆಜ್ಞೆಯನ್ನು ಪಡೆದು, ಅವಳಿಗಾಗಿಯೇ ನಿರ್ಮಿಸಿದ ಸುಂದರವಾದ ಪರ್ಣಶಾಲೆಯಲ್ಲಿ ತಪಸ್ಸನ್ನು ಮಾಡಲು ನಿರ್ಧರಿಸಿದಳು. ಸುಕುಮಾರ ಶರೀರವನ್ನು ಹೊಂದಿದವಳೂ, ಕಮಲಾಕ್ಷಿಯೂ, ಸ್ತನಗಳಮೇಲೆ ಸೆರಗನ್ನು ಹೊದ್ದವಳೂ, ಜಟೆಗಳನ್ನು ಹೊಂದಿದವಳೂ, ಇಂದ್ರನೀಲ ರತ್ನದಂತಹ ಮೈಕಾಂತಿಯುಳ್ಳವಳೂ ಆದ ಗಿರಿಜೆಯು ತಪೋಲಕ್ಷ್ಮಿಯೇ ಮೂರ್ತಿವೆತ್ತಂತೆ ಕಾಣುತ್ತಿದ್ದಳು. ಅನೇಕ ನಿಯಮಗಳಿಂದಲೂ, ತಪಗಳಿಂದಲೂ, ಪ್ರಾಪ್ತವಾದ ಯೋಗಬಂಧಗಳಿಂದಲೂ, ವೇದ ಹಾಗೂ ಆಗಮಗಳಲ್ಲಿ ಹೇಳಿರುವ ಸಕಲ ಧರ್ಮ ಮಾರ್ಗಗಳನ್ನು ಸಾರ್ಥಕಪಡಿಸಿದಳು. ಬಹುವಿಧ ಕಠಿಣ ತಪಸ್ಸನ್ನು ಮಾಡಿದರೂ ಆಕೆಯು ಎಂದೂ ಬೇಸರ ಪಡದೆ, ಇಂದ್ರನೀಲ ಮಣಿಯಂತೆ ಕಾಂತಿಯುತವಾಗಿ ಸುಂದರವಾಗಿ ಕಂಗೊಳಿಸುತ್ತಿದ್ದಳು."
