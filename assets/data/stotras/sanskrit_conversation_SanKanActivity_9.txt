ತಥೈವ ಅಸ್ತು = ಹಾಗೇ ಆಗಲಿ
ಜಾನಾಮಿ ಭೋಃ = ಗೊತ್ತು ಕಣ್ರೀ
ಆಂ, ಏವಂ ಏವ = ಹೌದು ಹಾಗೆಯೇ!
ಸಮೀಚಿನಾ ಸೂಚನಾ = ಒಳ್ಳೆಯ ಸಲಹೆ
ಕೀಮ್ಚಿತ್ ಏವ = ಸ್ವಲ್ಪ ಮಾತ್ರ
ಕಿಮರ್ಥಂ ನ ಭವತಿ? = ಯಾಕೆ ಆಗಲ್ಲ?
ಏವಂ ವಾ? = ಹೀಗೋ?
ತಥಾ ವಾ? = ಹಾಗೋ?
ಏವಮಪಿ ಅಸ್ತಿ ವಾ? = ಹೀಗೂ ಉಂಟೇ?
ನೈವ ಕಿಲ = ಇಲ್ಲವಲ್ಲ!
ಭವತು = ಆಗಲಿ
ಆಗಚ್ಛಂತು = ಬನ್ನಿ
ಉಪವಿಶಂತು = ಕುಳಿತುಕೊಳ್ಳಿ
ಸರ್ವಥಾ ಮಾಸ್ತು = ಖಂಡಿತ ಬೇಡ
ಕಿಮರ್ಥಂ? = ಯಾಕೆ?
ಲಬ್ಧಂ ಕಿಲ? = ಸಿಕ್ಕಿತು ತಾನೇ?
ಶಕ್ಯತೇ ವಾ ಇತಿ ಪಶ್ಯಾಮಃ = ಆಗುತ್ತಾ ಅಂತ ನೋಡೋಣ
ಜ್ಞಾತಂ ವಾ? = ತಿಳಿಯಿತೇ?
ಕಥಂ ಆಸೀತ್? = ಹೇಗಿತ್ತು?
ಅಂಗೀಕೃತಂ ಕಿಲ? = ಒಪ್ಪಿಗೆ ತಾನೇ?
ಕತಿ ಅಪೇಕ್ಷಿತಾನಿ? = ಎಷ್ಟು ಬೇಕು?
ಅದ್ಯ ಏವ ವಾ? = ಇವತ್ತೇನಾ?
ಇದಾನೀಂ ಏವ ವಾ? = ಇವಾಗಲೇನಾ?
ತತ್ ಕಿಮಪಿ ಮಾಶ್ತು = ಅದೇನೂ ಬೇಡ
ನ ದೃಶ್ಯತೇ = ಕಾಣುತ್ತಿಲ್ಲ
ಸಮಾಪ್ತಂ ವಾ = ಮುಗಿಯಿತಾ?
ಕಸ್ಮಿನ್ ಸಮಯೇ? = ಎಷ್ಟು ಹೊತ್ತಿಗೆ?
ತಥಾ ಅಪಿ = ಹಾಗಿದ್ದರೂ ಕೂಡ
ಆವಶ್ಯಕಂ ನ ಆಸೀತ್ = ಬೇಡವಾಗಿತ್ತು
ತಿಷ್ಠತು ಭೋಃ = ಇರಯ್ಯ
ಸ್ಮರತಿ ಕಿಲ? = ನೆನಪಿದೆ ತಾನೆ?
ಕಥಂ ಅಸ್ತಿ ಭವಾನ್ = ಹೇಗಿದ್ದೀರಿ?
ಮಾ ವಿಸ್ಮರತು = ಮರೆಯಬೇಡ
ನಿಃಸಂಕೋಚಂ ವದತು = ಸಂಕೋಚವಿಲ್ಲದೆ ಹೇಳು
ಅನ್ಯಚ್ಚ = ಅಲ್ಲದೆ
ತದನಂತರಂ = ಅನಂತರ
ಏತಾವದೇವ ಕಿಲ? = ಇಷ್ಟೇ ತಾನೇ?
ತತ್ ತಥಾ ನ = ಅದು ಹಾಗಲ್ಲ
ತಸ್ಯ ಕಃ ಅರ್ಥಃ? = ಅದರ ಅರ್ಥವೇನು?
ಏವಮೇವ = (ಸುಮ್ಮನೆ) ಹೀಗೆಯೇ
ತನ್ನಾಮ = ಅಂದರೆ
