ಅದ್ಯ ಪ್ರಾತರಾಶಃ ಕಃ ? = ಇವತ್ತು ತಿಂಡಿ ಏನು ?
ಅದ್ಯ ಪಾಕೋ ನಾಮ ಪಾಕಃ = ಇವತ್ತು ಅಡಿಗೇ ಅಂದರೆ ಅಡಿಗೆ
ಕಿಂ ಅದ್ಯ ರುಚಿರೇವ ನಾಸ್ತಿ? = ಇವತ್ತೇಕೆ ರುಚಿಯೇ ಇಲ್ಲ?
ರುಚಿರೇವ ನಾಸ್ತಿ ವಾ? = ರುಚಿಯೇ ಇಲ್ಲವೇ?
ಲವಣಂ ಏವ ನಾಸ್ತಿ = ಉಪ್ಪೇ ಹಾಕಿಲ್ಲ.
ಲವಣಂ ಕಿಂಚಿತ್ ಅಧಿಕಮ್ = ಉಪ್ಪು ಸ್ವಲ್ಪ ಹೆಚ್ಚು
ವ್ಯಂಜನೇ ಲವಣಂ ನ್ಯೂನಮ್ = ಪಲ್ಯಕ್ಕೆ ಉಪ್ಪು ಕಡಿಮೆ.
ಅನ್ನಂ ಬಹು ಉಷ್ಣಮ್ = ಅನ್ನ ತುಂಬ ಬಿಸಿ.
ಪುನಃ ಕಿಂಚಿತ್/ಏಕಂ ಪರಿವೇಷಯತು = ಇನ್ನೊಂದು ಸ್ವಲ್ಪ/ಇನ್ನೊಂದನ್ನು ಬಡಿಸು.
ಜಲಂ ಪೂರಯತು = ನೀರು ಬಡಿಸು.
ಏಕಚಷಕಂ ಜಲಂ ಆನಯತು = ಒಂದು ಲೋಟ ನೀರು ತಗೊಂಡು ಬಾ.
ಕಿಂಚಿತ್ ವ್ಯಂಜನಂ ಪರಿವೇಷಯತು = ಸ್ವಲ್ಪ ಪಲ್ಯ ಬಡಿಸು.
ಅನ್ನಮ್ - ಅನ್ನವನ್ನು,
ಸಾರಮ್ - ಸಾರನ್ನು,
ಲವಣಮ್ - ಉಪ್ಪನ್ನು,
ಕ್ವಥಿತಮ್ - ಹುಳಿಯನ್ನು,
ತಕ್ರಮ್ - ಮಜ್ಜಿಗೆಯನ್ನು,
ಘೃತಮ್ - ತುಪ್ಪವನ್ನು,
ಪರ್ಪಟಮ್ - ಹಪ್ಪಳವನ್ನು,
ಭರ್ಜಮ್ - ಸಂಡಿಗೆಯನ್ನು,
ತೈಲಮ್ - ಎಣ್ಣೆಯನ್ನು,
ಉಪಸೇಚನಮ್ - ಚಟ್ನಿಯನ್ನು,
ಕಿಂ, ನ ರೋಚತೇ ವಾ? = ಏನು, ಸೇರುವುದಿಲ್ಲವೇ?
ಕಿಂ ಅಂಬ! ಪ್ರತಿದಿನಂ ಸಾರಃ ಏವ? = ಏನಮ್ಮ, ಪ್ರತಿದಿನವೂ ಬರೀ ಸಾರು?
ಅದ್ಯ ಅಪಿ ಸಾರಃ ಏಕಃ ಏವ ವಾ? = ಇವತ್ತೂ ಸಾರು ಒಂದೇನಾ?
ಕಿಂ ಅಂಬ! ಕಿಯತ್ ಪರಿವೇಷಿತವತೀ! = ಏನಮ್ಮ! ಎಷ್ಟು ಬಡಿಸಿ ಬಿಟ್ಟೆ!
ಕಿಯದ್ ಅಸ್ತಿ ತತ್? = ಎಷ್ಟಿದೆ ಅದು?
ಅಂಬ, ಕಿಂಚಿತ್ ಉಪದಂಶಂ ಪರಿವೇಷಯತು = ಅಮ್ಮ, ಸ್ವಲ್ಪ ಉಪ್ಪಿನಕಾಯಿಯನ್ನು ಬಡಿಸು.
ಅಂಬ, ಅದ್ಯ ಕದಾ ವಾ ಭೋಜನಮ್ ? = ಅಮ್ಮ, ಇವತ್ತು ಊಟ ಎಷ್ಟೊತ್ತಿಗೆ ?
ಸಾವಧಾನಂ ಪರಿವೇಷಯತು = ಮೆಲ್ಲಗೆ ಬಡಿಸು.
ಅದ್ಯ ಭೂರಿ ಭೋಜನಮ್! = ಇವತ್ತು ಗಡದ್ದು ಊಟ!
ಅಧಿಕಂ ಜಲಂ ಮಾ ಪಿಬತು, ಪೀನಸಃ ಭವತಿ = ಹೆಚ್ಚಿಗೆ ನೀರು ಕುಡಿಯಬೇಡ, ಶೀತವಾಗುತ್ತೆ.
ಏತೇನ ವ್ಯಂಜನಂ ಕರಣೀಯಂ ಆಸೀತ್ = ಇದರ ಪಲ್ಯ ಮಾಡಬೇಕಿತ್ತು.
ತೇಮನಂ ನ ಪರಿವೇಷಿತವತೀ ಏವ = ಮಜ್ಜಿಗೆಹುಳಿ ಬಡಿಸಿಯೇ ಇಲ್ಲ.
ಪುನಃ ಏಕವಾರಂ ಪಾಯಸಂ ಪರಿವೇಷಯತು = ಮತ್ತೊಮ್ಮೆ ಪಾಯಸ ಬಡಿಸು.
ಉತ್ತಿಷ್ಠತು , ಭೋಜನಂ ಕುರ್ಮಃ = ಏಳಿ, ಊಟ ಮಾಡೋಣ.
ಇದಮಿದಾನೀಂ ಭೋಜನಂ ಸಮಾಪ್ತಮ್ = ಈಗ ತಾನೇ ಊಟವಾಯಿತು.
ಮಮ ಚಪಾತಿ-ಅಭ್ಯಾಸಃ ನಾಸ್ತಿ = ನನಗೆ ಚಪಾತಿ ಅಭ್ಯಾಸವಿಲ್ಲ.
ಉಪದಂಶಃ ಅಸ್ತಿ ಚೇತ್ ಸಮೀಚಿನಂ (ಭವೇತ್) = ಉಪ್ಪಿನಕಾಯಿ ಇದ್ದಿದ್ದರೆ ಚೆನ್ನಾಗಿರುತ್ತಿತ್ತು.
ಕಿಂ ಭೋಃ? ಭೋಜನಮೇವ ನ ಕರೋತಿ! = ಯಾಕೋ? ಊಟವನ್ನೇ ಮಾಡೋಲ್ಲ!
ಅನ್ನೇ ಕೇವಲಂ ಪಾಷಾಣಾಃ = ಅನ್ನದಲ್ಲಿ ಬರೀ ಕಲ್ಲು .
ತಕ್ರಾರ್ಥಂ ಅನ್ನಂ ಪರಿವೇಷಯಾಮಿ ವಾ? = ಮಜ್ಜಿಗೆಗೆ ಅನ್ನ ಬಡಿಸಲಾ?
ತಕ್ರಂ ನ ಸ್ವೀಕರೋತಿ ವಾ? = ಮಜ್ಜಿಗೆ ಹಾಕಿಕೊಳ್ಳಲ್ವಾ?
ಭೋಜನಂ ಸಮ್ಯಕ್ ಕರೋತಿ ಚೇತ್ ಕ್ರೀಡನಕಂ ದದಾಮಿ = ಸರಿಯಾಗಿ/ಚೆನ್ನಾಗಿ ಊಟ ಮಾಡಿದರೆ ಆಟಿಕೆ (ಬೊಂಬೆ) ಕೊಡ್ತೀನಿ.
ತೇಷಾಂ ಗೃಹೇ ಕಿಂ ಭುಕ್ತವಾನ್? = ಅವರ ಮನೆಯಲ್ಲಿ ಏನು ತಿಂದೆ?
ಶೀಘ್ರಂ ಭೋಜನಂ ಕರೋತು, ವಿಲಂಬಃ ಜಾತಃ = ಹೊತ್ತಾಯಿತು, ಬೇಗ ಊಟ ಮಾಡು.
ಇದಾನೀಂ ಮಾಸ್ತು, ಅನಂತರಂ ದದಾಮಿ = ಈಗ ಬೇಡ ಆಮೇಲೆ ಕೊಡ್ತೀನಿ.
ಕಿಂಚಿತ್ ವಾ ತಕ್ರಾನ್ನಂ ಸ್ವೀಕರೋತು = ಸ್ವಲ್ಪವಾದರೂ ಮಜ್ಜಿಗೆ ಅನ್ನ ಊಟ ಮಾಡು.
ಆದ್ಯ ಕಿಂ ಭಕ್ಷ್ಯಂ? =  ಇವತ್ತು ಏನು ಭಕ್ಷ್ಯ?
ಬಹು ಮಧುರಂ = ತುಂಬಾ ಸಿಹಿಯಾಗಿದೆ.
ಅಂಬಾ, ಬುಭುಕ್ಷಾ  ಭವತಿ  = ಅಮ್ಮ, ಹಸಿವಾಗುತ್ತೆ.
ಮಮ ತು ಇದಾನೀಂ ಅತೀವ ಬುಭುಕ್ಷಾ = ನನಗೆ ಈಗ ತುಂಬಾ ಹಸಿವು.
ಭೋಜನಂ ಸಿದ್ಧಂ ವಾ? = ಊಟಕ್ಕೆ ಆಯ್ತಾ?
ಶಾಲಾಯಾಃ ವಿಲಂಬಃ ಭವತಿ = ಶಾಲೆಗೆ ಹೊತ್ತಾಗುತ್ತೆ.
ಭೋಜನಂ ಕೃತ್ವಾ ನಿದ್ರಾಂ ಕರೋತು = ಊಟ ಮಾಡಿ ನಿದ್ದೆ ಮಾಡು.
ಅಸ್ತು, ಪರಿವೇಷಯಾಮಿ = ಆಗಲಿ, ಬಡಿಸುತ್ತೇನೆ.
ಕಿಂಚಿತ್ ಸ್ವೀಕರೋತು = ಸ್ವಲ್ಪ ತಗೊಳ್ಳಿ.
ಮಾಸ್ತು, ಅಧಿಕಂ ಭವತಿ = ಬೇಡ, ಹೆಚ್ಚಾಗಿಬಿಡುತ್ತೆ .
ಪರಿವೇಷಣಾರ್ಥಂ ಕಿಯಾನ್ ವಿಲಂಬಃ ಅಯಮ್? = ಬಡಿಸೋಕೆ ಏಕಿಷ್ಟು ತಡ?
ಯಾವದ್ ರೋಚತೇ ತಾವದೇವ ಸ್ವೀಕರೋತು = ಎಷ್ಟು ಸೇರುತ್ತೋ ಅಷ್ಟೇ ಊಟ ಮಾಡು.
ಸರ್ವೇ ಏಕವಾರಂ ಏವ ಉಪವಿಶಂತು ನಾಮ = ಎಲ್ಲರೂ ಒಟ್ಟಿಗೆ (ಒಮ್ಮೆಗೇ) ಕುಳಿತುಕೊಳ್ಳಿ, ನೋಡೋಣ.
ಅಯ್ಯೋ, ಘೃತಂ ಏವ ನ ಪರಿವೇಷಿತವತೀ ಅಹಮ್ = ಅಯ್ಯೋ, ತುಪ್ಪವನ್ನೇ ಬಡಿಸಿಲ್ಲ ನಾನು.
ಭವತಿ ವಕ್ತವ್ಯಂ ಆಸೀತ್ ಕಿಲ? = ನೀನಾದರೂ ಹೇಳಬಾರದಿತ್ತಾ?
ಕಿಯಾನ್ ವಿಲಂಬಃ ಭೋಃ, ಶೀಘ್ರಂ ಆಗಚ್ಛತು = ಎಷ್ಟೊತ್ತಯ್ಯಾ? ಬೇಗ ಬಾ
ಸಃ ಇದಾನೀಂ ಅಪಿ ನ ಆಗತವಾನ್ ವಾ? = ಅವನು ಇನ್ನೂ ಬಂದಿಲ್ಲವಾ?
ಸ್ಥಾಲಿಕಾ ಸ್ಥಾಪಿತಾ ವಾ? = ತಟ್ಟೆ ಇಟ್ಟಾಯಿತಾ?
ಸ್ಥಾಲಿಕಾ ಏವ ನ ಸ್ಥಾಪಿತಾ = ತಟ್ಟೆಯನ್ನೇ ಹಾಕಿಲ್ಲ
ಲವಣಂ ಯೋಜಯತು ಸಮ್ಯಕ್ ಭವತಿ = ಉಪ್ಪು ಹಾಕ್ಕೋ, ಸರಿ ಹೋಗುತ್ತೆ
ಹಸ್ತಂ ಪ್ರಕ್ಷಾಲ್ಯ ಉಪವಿಶತು = ಕೈ ತೊಳೆದು ಕುಳಿತುಕೊ
ಕಿಮಪಿ ನ ಅವಶ್ಯಕಂ = ಏನೂ ಬೇಕಾಗಿಲ್ಲ
ಸಂಕೋಚಃ ಮಾಸ್ತು = ಸಂಕೋಚ ಪಟ್ಟುಕೊಳ್ಳಬೇಡಿ
ನ ಮಮ ಸಂಕೋಚಃ ಏವ ನಾಸ್ತಿ = ಇಲ್ಲ, ನನಗೆ ಸಂಕೋಚವೇ ಇಲ್ಲ
ಮಮ ಅಪಿ ಕಿಂಚಿತ್ = ನನಗೆ ಸ್ವಲ್ಪ
ಅಂಬಾ! ಅತ್ರ ಕಿಂಚಿತ್ ಪರಿವೇಷಯತು = ಅಮ್ಮಾ! ಇಲ್ಲಿ ಸ್ವಲ್ಪ ಬಡಿಸು
ಲವಣಂ ಅವಶ್ಯಕಂ ಇತಿ ಕಃ ಉಕ್ತವಾನ್? = ಉಪ್ಪು ಬೇಕು ಅಂತ ಹೇಳಿದ್ದು ಯಾರು?
ಕ್ವಥಿತಾಪೇಕ್ಷಯಾ ಸಾರಃ ಏವ ರುಚಿಕರಃ = ಹುಳಿಗಿಂತ ಸಾರು ರುಚಿಕರವಾಗಿದೆ
ಏಕೈಕಶಃ ವದತು ನಾಮ = ಒಬ್ಬೊಬ್ಬರಾಗಿ ಹೇಳಿ ನೋಡೋಣ
ವಿನಾ ಶಬ್ಧಂ ಭೋಜನಂ ಕುರ್ವಂತು ನಾಮ = ಗದ್ದಲವಿಲ್ಲದೆ ಊಟ ಮಾಡಿ ನೋಡೋಣ
ಪ್ರಥಮಃ ಸಃ ವಾ? ಭವಾನ್ ವಾ? = ಮೊದಲಿನವನು ಅವನೋ? ನೀನೋ?
ಪಾಕಃ ಶೀತಲಃ ಭವತಿ = ಅಡಿಗೆ ಆರಿ ಹೋಗುತ್ತೆ
ಪಾಕಃ ತದಾನೀಂ ಏವ ಸಿದ್ಧಃ = ಆಗಲೇ ಅಡಿಗೆಯಾಗಿದೆ
ವ್ಯಾಘರಣಂ ಕೃತಂ ಚೇತ್ ಪಾಕಃ ಸಿದ್ಧಃ = ಒಗ್ಗರಣೆ ಕೊಟ್ಟರಾಯಿತು ಅಡಿಗೆ ಸಿದ್ಧ!
ಶಾಕಃ ನಾಸ್ತಿ ಕಿಮಪಿ ನಾಸ್ತಿ, ಅಹಂ ಕಿಂ ಕರೋಮಿ? = ತರಕಾರಿಯೂ ಇಲ್ಲ, ಏನೂ ಇಲ್ಲ, ನಾನೇನು ಮಾಡಲಿ?
ಭವತಿ ಕ್ವಥಿತಂ ಕರ್ತುಂ ಏವ ನ ಜಾನಾತಿ, ಅಂಬಾ! = ನಿನಗೆ ಹುಳಿ ಮಾಡೋಕೆ ಬರಲ್ಲ ಅಮ್ಮಾ!
ಹ್ಯಸ್ತನ ವ್ಯಂಜನಂ ಕಿಯತ್ ರುಚಿಕರಂ ಆಸೀತ್ = ನಿನ್ನೆಯ ಪಲ್ಯ ಎಷ್ಟು ರುಚಿಯಾಗಿತ್ತು!
ಕತಿವಾರಂ ಉಕ್ತವಾನ್ ಏತತ್ ? = ಎಷ್ಟು ಸಾರಿ ಹೇಳಿದ್ದೀನಿ ಇದು?
ಮಹ್ಯಂ ನ ರೋಚತೇ ಇತಿ = ನನಗೆ ಹಿಡಿಸೊಲ್ಲ ಅಂತ
ಶ್ವಃ ಭವಾನ್ ಏವ ಪಾಕಂ ಕರೋತು = ನಾಳೆ ನೀನೇ ಅಡುಗೆ ಮಾಡು ನೋಡೋಣ
ಅಸ್ಯ ರುಚಿಂ ಪಶ್ಯತು = ಇದರ ರುಚಿ ನೋಡು
ಕ್ಷೀರಂ ದೂಷಿತಂ = ಹಾಲು ಕೆಟ್ಟು ಹೋಗಿದೆ
ತೂಷ್ಣೀಂ ಭೋಜನಂ ಕರೋತು = ಸುಮ್ಮನೆ ಊಟ ಮಾಡು
ಪುನಃ ಪುನಃ ಚರ್ವಿತ್ವಾ ಭೋಜನಂ ಕರೋತು = ಮತ್ತೆ ಮತ್ತೆ ಜಗಿದು ಊಟ ಮಾಡು
ಪರಿವೇಷಣಂ ಕೃತಂ, ಶೀಘ್ರಂ ಆಗಚ್ಛಂತು = ಬಡಿಸಿಯಾಯಿತು, ಬೇಗ ಬನ್ನಿ
ಕಿಮಪಿ ನ ಸ್ವೀಕೃತಂ = ಏನೂ ತಗೊಂಡೇ ಇಲ್ಲ
ಪುನಃ ಏಕವಾರಂ ಪೃಚ್ಛತು = ಇನ್ನೊಮ್ಮೆ ಕೇಳು
