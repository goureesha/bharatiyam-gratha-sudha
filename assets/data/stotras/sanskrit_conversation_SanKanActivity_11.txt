ಇದಾನೀಂ ಕುತ್ರ ವಾಸಃ? = ಈಗ ಎಲ್ಲಿ ವಾಸ?

ಪುನಃ ಅಸ್ಮಾಕಂ ಮಿಲನಂ ಕದಾ? = ಮತ್ತೆ ನಮ್ಮ ಭೇಟಿ ಎಂದು?
ಪುನಃ ಕದಾ ಮಿಲತಿ ಭವಾನ್? = ಮತ್ತೆ ನೀನು ಯಾವಾಗ ಸಿಗುವೆ?
ತದ್ದಿನೇ ಕಿಮರ್ಥಂ ಭವಾನ್ ನ ಆಗತವಾನ್? = ಆ ದಿನ ನೀನು ಯಾಕೆ ಬರಲಿಲ್ಲ?
ವಯಂ ತದಾನೀಂ ಏವ ಆಗತವಂತಃ = ನಾವು ಆಗಲೇ ಬಂದುಬಿಟ್ಟಿದ್ದೇವೆ

ಅಸ್ತು ಪಿಬಾಮಃ = ಆಗಲಿ ಕುಡಿಯೋಣ
ಸ್ಥಾತುಂ ಸಮಯಃ ನಾಸ್ತಿ = ಇರಲು ಸಮಯವಿಲ್ಲ
ಗಮನಸಂ ಅನಂತರಂ ಏವ ಪತ್ರಂ ಲಿಖತು = ಹೋದ ಕೂಡಲೇ ಪತ್ರ ಬರೆ

ಭವಾನ್ ಅಪಿ ಏವಂ ವದತಿ ವಾ? = ನೀನೂ ಹೀಗೆ ಹೇಳ್ತೀಯಾ?
ಭವಾನ್ ಏವಂ ಕರ್ತುಂ ಅರ್ಹತಿ ವಾ? = ನೀವು ಹೀಗೆ ಮಾಡಬಹುದಾ?
ಭವಾನ್ ಗಚ್ಛತು, ಮಮ ಕಿಂಚಿತ್ ಕಾರ್ಯಂ ಅಸ್ತಿ = ನೀನು ಹೋಗು, ನನಗೆ ಸ್ವಲ್ಪ ಕೆಲಸವಿದೆ
ಭವಾನ್ ವೃಥಾ ಚಿಂತಯತಿ = ನೀನು ಸುಮ್ಮನೆ ಚಿಂತೆ ಮಾಡುತ್ತೀಯ
ದೈವೇಚ್ಛಾ ತಥಾ ಆಸೀತ್, ಕಿಂ ಕುರ್ಮಃ? = ದೈವೇಚ್ಛೆ ಹಾಗಿತ್ತು, ಏನು ಮಾಡೋಣ?
ಅಹಂ ಅನ್ಯತ್ ಉಕ್ತವಾನ್, ಭವಾನ್ ಅನ್ಯತ್ ಗೃಹೀತವಾನ್ = ನಾನು ಒಂದು ಹೇಳಿದೆ, ನೀನು ಬೇರೆ ತಿಳಿದುಕೊಂಡೆ
ಏತಾವತ್ ಅನೃತಂ ವದತೀತಿ ನ ಜ್ಞಾತವಾನ್ = ಇಷ್ಟು ಸುಳ್ಳು ಹೇಳುತ್ತಾನೆಂದು ತಿಳಿದಿರಲಿಲ್ಲ
ಪ್ರಮಾದಾತ್ ಸಂವೃತ್ತಂ, ನ ತು ಬುದ್ಧ್ಯಾ = ಬೇಕಂತ ಮಾಡಿದ್ದಲ್ಲ, (ಅಕಸ್ಮಾತ್) ತಿಳಿಯದೆ ಆಗಿ ಹೋಯ್ತು
ಏಷಃ ಏಕಃ ಶನಿಃ = ಇವನೊಬ್ಬ ಶನಿ (ಪೀಡೆ)
ಭವದುಕ್ತಂ ಸರ್ವಂ ಅಪಿ ಅಂಗೀಕರ್ತುಂ ನ ಶಕ್ಯಂ = ನೀ ಹೇಳಿದ್ದೆಲ್ಲಾ ಒಪ್ಪಿಕೊಳ್ಳಲಾಗಲ್ಲ
ಅಹಂ ಗಂತುಂ ನ ಶಕ್ನೋಮಿ = ನನಗೆ ಹೋಗಲು ಆಗಲ್ಲ
ವಿಷಯಸ್ಯ ವರ್ಧನಂ ಮಾಸ್ತು = ವಿಷಯ ಬೆಳೆಸಬೇಡ
ಸರ್ವೇ ಅಪಿ ಪಲಾಯನಶೀಲಾಃ = ಎಲ್ಲರೂ ಪಲಾಯನವಾದಿಗಳು
ಅಸಂಬದ್ಧಂ ಮಾ ಪ್ರಲಪತು = ಹುಚ್ಚಾಗಿ ಹರಟಬೇಡ
ಸರ್ವಸ್ಯಾಪಿ ಭವಾನ್ ಏವ ಮೂಲಂ = ಎಲ್ಲದಕ್ಕೂ ನೀನೇ ಕಾರಣ (ಮೂಲ)
ಗತ್ವಾ ಗತ್ವಾ ತಸ್ಯ ಜಾಲೇ ಪತಿತವಾನ್ = ಹೋಗಿ ಹೋಗಿ ಅವನ ಬಲೆಗೆ ಬಿದ್ದ
ಅಸ್ಮಾಕಂ ಮಿಲನಾನಂತರಂ ಬಹುಕಾಲಃ ಅತೀತಃ = ನಾವು ಭೇಟಿಯಾಗಿ ಬಹಳ ಕಾಲವಾಗಿದೆ
ಇದಾನೀಂ ಆಗಂತುಂ ನ ಶಕ್ಯತೇ = ಈಗ ಬರಲು ಆಗುವುದಿಲ್ಲ
ಭವಾನ್ ಅಪಿ ವಿಶ್ವಸಿತಿ? = ನೀವೂ ಒಪ್ಪುವಿರಾ?
ಭವಾನ್ ಅಪಿ ವಿಶ್ವಸಿತವಾನ್? = ನೀವೂ ನಂಬಿಬಿಟ್ರಾ?
ಸಃ ವಿಶ್ವಾಸಯೋಗ್ಯಃ ವಾ? = ಅವನನ್ನು ನಂಬಬಹುದಾ?
ಕಿಂಚಿತ್ ಸಾಹಾಯ್ಯಂ ಕರೋತಿ ವಾ? = ಸ್ವಲ್ಪ ಸಹಾಯ ಮಾಡ್ತೀರಾ?
ಸಮಯಃ ಕಥಂ ಅತಿಶೀಘ್ರಂ ಗತಃ! = ಸಮಯ/ವೇಳೆ ಬೇಗ ಕಳೆದು ಹೋಯ್ತಲ್ಲ!
ಸಾಕ್ಷಾತ್ ಸಮಯೇ ಆಗತವಾನ್ = ಸರಿಯಾದ ಸಮಯಕ್ಕೆ ಬಂದೆ
ಏಕನಿಮೇಷಂ ವಿಲಂಬಃ ಸ್ಯಾತ್ ಅಹಂ ಗತಃ ಸ್ಯಾಮ್ = ಒಂದು ನಿಮಿಷ ತಡವಾಗಿದ್ದರೆ ನಾನು ಹೋಗಿರುತ್ತಿದ್ದೆ
ಅಹಂ ಅಪಿ ಭವತಾ ಸಹ ಆಗಚ್ಛಾಮಿ ವಾ? = ನಾನೂ ನಿನ್ನ ಜೊತೆ ಬರಲೇ?
ಕಿಂಚಿತ್ ಕಾಲಂ ದ್ವಿಚಕ್ರಿಕಾಂ ದದಾತಿ ವಾ? = ಸ್ವಲ್ಪ ಕಾಲ ದ್ವಿಚಕ್ರ (ಸೈಕಲ್/ಬೈಕ್) ವಾಹನ ಕೊಟ್ಟಿರುತ್ತೀಯಾ?
ಇದಾನೀಂ ಮಯಾ ಅಪಿ ಅನ್ಯತ್ರ ಗಂತವ್ಯಂ = ಈಗ ನನಗೂ ಬೇರೆ ಕಡೆ ಹೋಗಬೇಕು
ಭವಾನ್ ಸ್ವಕಾರ್ಯಂ ಪಶ್ಯತು = ನೀನು ನಿನ್ನ ಕೆಲಸ ನೋಡಿಕೋ
ಶೀಘ್ರಂ ಪ್ರತ್ಯಾಗಚ್ಛಾಮಿ = ಬೇಗ ಹಿಂದಿರುಗುತ್ತೇನೆ
ಆವಶ್ಯಕಂ ಚೇತ್ ಶ್ವಃ ಆನಯಾಮಿ = ಬೇಕಾಗಿದ್ದರೆ ನಾಳೆ ತರುತ್ತೇನೆ
ಮಾಸ್ತು ಇತ್ಯುಕ್ತೇ ಅಪಿ ನ ಶೃಣೋತಿ = ಬೇಡ ಎಂದರೂ ಕೇಳೋದಿಲ್ಲ
