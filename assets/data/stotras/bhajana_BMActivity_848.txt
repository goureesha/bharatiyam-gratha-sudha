|| ರಾಮ ಮಂಗಳಂ ||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ರಾಮ ಕೋದಂಡ ರಾಮ
ರಾಮ ಕಲ್ಯಾಣ ರಾಮ
ರಾಮ ಲೋಕಾಭಿರಾಮ
ರಾಮ ಪಟ್ಟಾಭಿರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||1||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ರಾಮ ದಿವಿಜಗೀತ ಮಹಿಮ
ರಾಮ ದಶರಥೇಷ್ಟಿಕಾಮ
ಕೋಸಲೇಂದ್ರಜಾಲಲಾಮ
ರಾಮರಘುಕುಲಾಬ್ಧಿ ಸೋಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||2||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಕುಶಿಕ ತನಯ ಶಿಷ್ಯ ರಾಮ
ರಾಮ ಮುನಿಮಖಾರಿ ಭೀಮ
ಮುನಿವದೂದ್ಧರಣ ರಾಮ
ಖಂಡಿತೇಷಚಾಪ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||3||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಜಾನಕಿ ಮನೋಭಿರಾಮ
ಜನಕ ಜಾಮಾತ್ರ ರಾಮ
ರಾಮ ಕಲ್ಯಾಣ ರಾಮ
ಭೃಗುಜದರ್ಪ ದಮನ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||4||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಈಪ್ಸಿತಾಭಿಷೇಕ ರಾಮ
ವಿಘ್ನಿತಾಭಿಷೇಕ ರಾಮ
ಆದೃತಾರಣ್ಯ ರಾಮ
ಚಿತ್ರಕೂಟ ವಾಸ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||5||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಪಿತೃವಿಯೋಗ ತಪ್ತ ರಾಮ
ಭರತಭಕ್ತಿ ಬದ್ಧ ರಾಮ
ದತ್ತಪಾದುಕಾಭಿರಾಮ
ಪಿತೃನಿಯೋಗನಿಷ್ಠರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||6||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ದಂಡಕಾಭಿಗಮನ ರಾಮ
ರಾಮಕೋದಂಡ ರಾಮ
ದುಷ್ಟದೈತ್ಯ ಕಾಲ ರಾಮ
ದನುಕಬಂಧ ಹರಣ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||7||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಖಲವಿರಾಧ ಶಮನ ರಾಮ
ಕುಂಭಜಾಪ್ತ ಚಾಪ ರಾಮ
ಪಂಚವಟೀವಾಸ ರಾಮ
ವಿಕೃತಶೂರ್ಪಣಖ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||8||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಹೇಮಮೃಗವಿಮುಗ್ಧ ರಾಮ
ಕಪಟಯತಿಹೃತಾತ್ಮ ರಾಮ
ಜಾನಕಿವಿಯುಕ್ತ ರಾಮ
ಪಕ್ಷಿಮುಕ್ತಿದಾತೃ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||9||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಲಬ್ಧರವಿಜಸಖ್ಯ ರಾಮ
ಸೂದಿತೇಂದ್ರಜಾತ ರಾಮ
ಜನಕಜಾನ್ವೇಷಣಾಯ
ಪ್ರಹಿತಸಕಲ ಸೈನ್ಯ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||10||

ಅಬ್ಧಿಲಂಘನಾಕ್ಷಹನನ
ರಾಕ್ಷಸೇಂದ್ರನಗರ ದಹನ
ಜಾನಕಿವಿಲೋಕನಾದಿ
ವೀರಕರ್ಮಸಾಹಿಸಿನ-
ದತ್ತಶೀರ್ಷರತ್ನ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||11||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಸೇತುಬಂಧಿತಾಬ್ಧಿ ರಾಮ
ಕುಂಭಕರ್ಣ ರಾವಣಾದಿ
ಮಾರಣಾರ್ತ ದೀಕ್ಷ ರಾಮ
ಅಗ್ನಿಪೂತದಾರ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||12||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಪುಷ್ಪಕಾಭಿಯಾನ ರಾಮ
ಭರತದತ್ತ ರಾಜ್ಯ ರಾಮ
ಮಂಗಳಾಭಿಷೇಕ ರಾಮ
ರಾಮ ಕಲ್ಯಾಣ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||13||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ರಾಮಾನಂದ ರಾಮ
ರಾಮ ಧರ್ಮಜ್ಞ ರಾಮ
ರಾಮ ಸತ್ಯಸಂಧ ರಾಮ
ರಾಮ ದಿವ್ಯನಾಮ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||14||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ಜಾನಕಿ ಸಕಾಮ ರಾಮ
ರಾಮ ಕಾರುಣ್ಯ ಧಾಮ
ರಾಮ ಪುಣ್ಯಮೂರ್ತಿ ರಾಮ
ರಾಮ ಶ್ರೀರಾಮ ರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||15||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ರಾಮ ಮಾಂಪಾಹಿ ರಾಮ
ರಾಮ ಲೋಕಾಭಿರಾಮ
ಚಿನ್ನದೀನಬಂಧು ರಾಮ
ಬೀಚಿದಂಬರಾತ್ಮ ಧಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||16||

ರಾಮ ರಾಮ ರಾಮ ರಾಮ
ರಾಮ ಕೋದಂಡ ರಾಮ
ರಾಮ ಕಲ್ಯಾಣ ರಾಮ
ರಾಮ ಲೋಕಾಭಿರಾಮ
ರಾಮ ಪಟ್ಟಾಭಿರಾಮ
ರಾಮ ಮಂಗಳಂ ರಾಮ ಮಂಗಳಂ ||17||

- ಚಿದಂಬರ ಭಟ್ಟರು
