|| ವಿಷ್ಣು ಸಹಸ್ರನಾಮ (ಅರ್ಥ ಸಹಿತ) ||

1. ವಿಶ್ವಂ = ವಿಶ್ವರೂಪಿಯು, ಜಾಗ್ರದವಸ್ಥೆಯಲ್ಲಿ ಮತ್ತು ಸ್ಥೂಲ ಶರೀರವ್ಯಷ್ಟಿಯಲ್ಲಿ ತೃಪ್ತಿಯನ್ನು ಹೊಂದುವ ಚೈತನ್ಯನು, ಪ್ರಣವದ ಪ್ರಥಮ ಮಾತ್ರೆಯಿಂದ ಸೂಚಿತವಾಗುವ ಚೈತನ್ಯನು

2. ವಿಷ್ಣುಃ = ವ್ಯಾಪಕನು, ಪ್ರವೇಶ ಮಾಡಿರುವವನು

3. ವಷಟ್ಕಾರಃ = ಯಜ್ಞಮಂತ್ರ ರೂಪಿಯಾಗಿ ಮತ್ತು ಉದ್ದೇಶವೂ ಆಗಿರುವವನು

4. ಭೂತಭವ್ಯಭವತ್ ಪ್ರಭುಃ = ತ್ರಿಕಾಲನಿಯಾಮಕನು, ಕಾಲತ್ರಯಾತೀತನು

5. ಭೂತಕೃತ್ = ಭೂತಗಳನ್ನು (ಪ್ರಾಣಿಗಳನ್ನು) ಸ್ವತಂತ್ರವಾಗಿ ಸೃಷ್ಟಿಸುವವನು

6. ಭೂತಭೃತ್ = ಭೂತಗಳನ್ನು ಭರಿಸುವವನು (ಪಾಲಿಸುವವನು, ಧರಿಸುವವನು, ಪೋಷಿಸುವವನು)

7. ಭಾವಃ = ಸತ್ತಾಸ್ವರೂಪಿಯು, ಸರ್ವ ವಿಭೂತಿ ಪುರುಷರಿಗಿಂತಲೂ ವಿಶಿಷ್ಟನಾದವನು

8. ಭೂತಾತ್ಮಾ = ಭೂತಗಳಿಗೆ ಅಂತರ್ಯಾಮಿಯಾದವನು

9. ಭೂತಭಾವನಃ = ಭೂತಗಳನ್ನು ಸಂಕಲ್ಪ ಮಾತ್ರದಿಂದ ಸೃಷ್ಟಿಸಿ ಪೋಷಿಸುವವನು

10. ಪೂತಾತ್ಮಾ = ಪವಿತ್ರ ಸ್ವರೂಪಿಯು, ಪವಿತ್ರನಾಗಿರುವ ಆತ್ಮನು

11. ಪರಮಾತ್ಮಾ = ಶ್ರೇಷ್ಠವಾದ ಆತ್ಮನು

12. ಮುಕ್ತಾನಾಂ ಪರಮಾ ಗತಿಃ = ಮುಕ್ತಪುರುಷರಿಗೆ ಶ್ರೇಷ್ಠವಾದ ಗತಿಯು

13. ಅವ್ಯಯಃ = ವಿಕಾರವಿಲ್ಲದವನು, ಅಳಿವಿಲ್ಲದವನು

14. ಪುರುಷಃ = ಶರೀರವೆಂಬ ಪುರದಲ್ಲಿ ಶಯಿನಿಸಿರುವವನು, ಪೂರ್ಣನಾಗಿರುವವನು

15. ಸಾಕ್ಷೀ = ಯಾರ ಅಡೆ-ತಡೆ ಇಲ್ಲದೆ ಎಲ್ಲವನ್ನು ಸದಾ ನೋಡುತ್ತಿರುವವನು

16. ಕ್ಷೇತ್ರಜ್ಞಃ = ಶರೀರಗಳನ್ನು ಅರಿತಿರುವ ಯೋಗಾತ್ಮನು

17. ಅಕ್ಷರಃ = ಅಳಿವಿಲ್ಲದವನು

18. ಯೋಗಃ = ಜೀವಾತ್ಮ-ಪರಮಾತ್ಮರ ಸಂಯೋಗವು, ಸಂಯೋಗದಲ್ಲಿ ಅನುಭವಕ್ಕೆ ಬರುವವನು

19. ಯೋಗವಿದಾಂ ನೇತಾ = ಯೋಗಜ್ಞರನ್ನು ಅವರ ಲಕ್ಷ್ಯಕ್ಕೆ ಒಯ್ಯುವವನು

20. ಪ್ರಧಾನ ಪುರುಷೇಶ್ವರಃ = ಪ್ರಕೃತಿ-ಪುರುಷರಿಗೆ ಈಶ್ವರನಾದವನು

21. ನಾರಸಿಂಹವಪುಃ = ನರ ಮತ್ತು ಸಿಂಹ ಎರಡರ ಅವಯವಗಳೂ ಕೂಡಿದ ದಿವ್ಯದೇಹ ಉಳ್ಳವನು

22. ಶ್ರೀಮಾನ್ = ಶ್ರೀಸಂಪನ್ನನು, ಶ್ರೀದೇವಿಯ ವಾಸ ಉಳ್ಳವನು

23. ಕೇಶವಃ = ತ್ರಿಮೂರ್ತಿಪ್ರಭು, ಸುಂದರಕೇಶರಾಶಿ ಉಳ್ಳವನು, ಕೇಶಿ-ದಾನವನನ್ನು ಸಂಹರಿಸಿದವನು

24. ಪುರುಷೋತ್ತಮಃ = ಕ್ಷರಪುರುಷ ಮತ್ತು ಅಕ್ಷರಪುರುಷರಿಗಿಂತಲೂ ಉತ್ತಮನಾದವನು, ಪರಮಪುರುಷನು, ಪರಮಾತ್ಮನು

25. ಸರ್ವಃ = ಸರ್ವಕಾರಣ, ಸರ್ವನಿಯಾಮಕ, ಸರ್ವಜ್ಞ, ಸರ್ವಶಕ್ತ, ಸರ್ವಾಧಾರ, ಸರ್ವವ್ಯಾಪಕ, ಸರ್ವಮಯ, ಸರ್ವರೂಪೀ

26. ಶರ್ವಃ = ಸರ್ವಸಂಹಾರಕನು

27. ಶಿವಃ = ಮಂಗಳಸ್ವರೂಪಿಯು

28. ಸ್ಥಾಣುಃ = ಸ್ಥಿರನು

29. ಭೂತಾದಿಃ = ಭೂತಗಳಿಗೆ ಆದಿ ಕಾರಣ, ಎಲ್ಲ ಭೂತಗಳಿಗೂ ಮೊದಲು ಇದ್ದವನು

30. ನಿಧಿರವ್ಯಯಃ = ಅಕ್ಷಯವಾದ ನಿಧಿಯು

31. ಸಂಭವಃ = ಉತ್ಪತ್ತಿ ಕಾರಣವು, ಸ್ವಸಂಕಲ್ಪದಿಂದ ಹುಟ್ಟುವವನು

32. ಭಾವನಃ = ಸತ್ಯಕಾಮ ಸಂಕಲ್ಪಗಳನ್ನು ಭಾವಿಸುವವನು

33. ಭರ್ತಾ = ಭರಿಸುವವನು, ಪೋಷಿಸುವವನು, ಒಡೆಯ

34. ಪ್ರಭವಃ = ಶ್ರೇಷ್ಠವಾದ ಜನ್ಮವುಳ್ಳವನು, ಪ್ರಾಣಿಗಳ ಜನ್ಮಕ್ಕೆ ಕಾರಣನಾದವನು

35. ಪ್ರಭುಃ = ಸರ್ವಸಮರ್ಥನು, ಸರ್ವಕ್ರಿಯೆಗಳಲ್ಲಿ ಅತಿಶಯ ಸಾಮಥ್ರ್ಯವುಳ್ಳವನು

36. ಈಶ್ವರಃ = ನಿರವಧಿಕವಾದ, ನಿರುಪಾಧಿಕವಾದ, ಅನ್ವರ್ಥವಾದ ಐಶ್ವರ್ಯ ಉಳ್ಳವನು, ಸರ್ವೇಶ್ವರನು

37. ಸ್ವಯಂಭೂಃ = ಸ್ವಯಂ ಸತ್ತೆಯುಳ್ಳವನು, ತಾನಾಗಿಯೇ ಇರುವವನು/ಪ್ರಕಟವಾಗುವವನು

38. ಶಂಭುಃ = (ಭಕ್ತರಿಗೆ) ಸುಖವನ್ನು ಭಾವಿಸುವವನು

39. ಆದಿತ್ಯಃ = ಆದಿತ್ಯ ರೂಪಿಯು, ಆಡಿತ್ಯ ಮಂಡಲದಲ್ಲಿರುವ ಪುರುಷನು

40. ಪುಷ್ಕರಾಕ್ಷಃ = ಕಮಲದಂತೆ ಕಣ್ಣುಗಳಿರುವವನು, ಹೃದಯ ಪುಷ್ಕರವನ್ನು ಅಕ್ಷವನ್ನಾಗಿ ಹೊಂದಿ ಶರೀರ ಯಂತ್ರವನ್ನು ನಡೆಸುವವನು

41. ಮಹಾಸ್ವನಃ = ಮಹಾನಾದವುಳ್ಳವನು, ಮಹತ್ತಾದ (ಪ್ರಣವ ವೇದರೂಪವಾದ) ನಾದವುಳ್ಳವನು

42. ಅನಾದಿನಿಧನಃ = ಆದಿ-ಅಂತ್ಯ (ಹುಟ್ಟು-ಸಾವು) ಇಲ್ಲದವನು

43. ಧಾತಾ = ವಿಶ್ವವನ್ನು ಧರಿಸಿ ಪೋಷಿಸುತ್ತಿರುವವನು

44. ವಿಧಾತಾ = ವಿಶೇಷವಾಗಿ, ವಿವಿಧವಾಗಿ ಧರಿಸಿ ಪೋಷಿಸುತ್ತಿರುವವನು

45. ಧಾತುರುತ್ತಮಃ = ಬ್ರಹ್ಮನಿಗಿಂತ ಶ್ರೇಷ್ಠನಾದವನು, ಉತ್ತಮವಾದ/ಪ್ರಸನ್ನವಾದ ಧಾತುವು, ಸರ್ವಶ್ರೇಷ್ಠವಾದ ಚಿತ್/ಜ್ಞಾನ ಧಾತುವು

46. ಅಪ್ರಮೇಯಃ = ಅಳಿಯಳು ಅಸಾಧ್ಯನಾದವನು

47. ಹೃಷೀಕೇಶಃ = ಇಂದ್ರಿಯಗಳಿಗೆ ಅಧಿಪತಿಯು, ಹರ್ಷವನ್ನುಂಟು ಮಾಡುವ ಕೇಶರಾಶಿಯುಳ್ಳವನು

48. ಪದ್ಮನಾಭಃ = ಸೃಷ್ಟಿ-ಸೂಚಕವಾದ ಪದ್ಮವನ್ನು ನಾಭಿಯಲ್ಲಿ ಹೊಂದಿರುವವನು

49. ಅಮರಪ್ರಭುಃ = ದೇವತೆಗಳಿಗೆ ಪ್ರಭುವಾದವನು, ಅಮರನಾದ ಪ್ರಭುವು

50. ವಿಶ್ವಕರ್ಮಾ = ವಿಶ್ವರೂಪವಾದ (ರಚನೆ-ವಿನಾಶ) ಕರ್ಮವುಳ್ಳವನು, ಸರ್ವಕರ್ಮರೂಪಿಯು

51. ಮನುಃ = ಜ್ಞಾನಸ್ವರೂಪಿಯು, ಮಂತ್ರಸ್ವರೂಪಿಯು, ಮನನ ಮಾಡಲ್ಪಡಬೇಕಾದವನು, ಮನನ ಮಾಡುವವನು, ಮನು-ಪ್ರಜಾಪತಿ ರೂಪಿಯು

52. ತ್ವಷ್ಟಾ = ವಿಶ್ವವನ್ನು ಸೃಷ್ಟಿ ರೂಪದಲ್ಲಿ ಕೆತ್ತುವ, ಸಂಹಾರದಲ್ಲಿ ಕೊಚ್ಚಿ ಹಾಕುವ ಶಿಲ್ಪಿ

53. ಸ್ಥವಿಷ್ಠಃ = ಅತ್ಯಂತ ಸ್ಥೂಲನಾದವನು, ಸೂಕ್ಷ್ಮತಮದಿಂದ ಸ್ಥೂಲತಮವಾಗಿ ವಿಸ್ತಾರಗೊಳಿಸುವವನು

54. ಸ್ಥವಿರೋ ಧ್ರುವಃ = ವೃದ್ಧನು, ಸರ್ವಲೋಕ ಪಿತಾಮಹನಿಗಿಂತಲೂ ವೃದ್ಧನಾದವನು, ಸ್ಥಿರನಾದವನು, ಶಾಶ್ವತನು

55. ಅಗ್ರಾಹ್ಯಃ = ಇಂದ್ರಿಯಗಳಿಂದ ಗ್ರಹಿಸಲಾಗದವನು, ಮನಸ್ಸು ಮಾತುಗಳಿಂದ ಗ್ರಹಿಸಲಾಗದವನು

56. ಶಾಶ್ವತಃ = ಎಂದೆಂದಿಗೂ ಇರುವವನು

57. ಕೃಷ್ಣಃ = ಸದಾನಂದ ಸ್ವರೂಪಿಯು, ಎಲ್ಲವನ್ನೂ ಕರ್ಷಣೆ ಮಾಡುವವನು, ನೀಲ ವರ್ಣದವನು

58. ಲೋಹಿತಾಕ್ಷಃ = ಕ್ಷತ್ರಿಯಾವತಾರಗಳಲ್ಲಿ ಸಹಜವಾದ ಕೆಂಪಾದ ಕಣ್ಣುಗಳುಳ್ಳವನು

59. ಪ್ರತರ್ದನಃ = ಧರ್ಮ ವಿರೋಧಿಗಳನ್ನು ಚೆನ್ನಾಗಿ ಹಿಂಸಿಸುವವನು, ಪ್ರಳಯದಲ್ಲಿ ಎಲ್ಲ ಭೂತಗಳನ್ನೂ ಸಂಹರಿಸುವವನು

60. ಪ್ರಭೂತಃ = ಪುಷ್ಕಳವಾಗಿರುವವನು, ಪುಷ್ಕಳವಾದ ಗುಣೈಶ್ವರ್ಯಾದಿ ಸಂಪನ್ನನು

61. ತ್ರಿಕಕುಬ್ಧಾಮ/ತ್ರಿಕಕುದ್ಧಾಮ = ಸರ್ವದೇಶಗಳಲ್ಲಿಯೂ ಪ್ರಕಾಶ/ನಿವಾಸ/ಪ್ರಭಾವ ಉಳ್ಳವನು

62. ಪವಿತ್ರಂ = ಪಾವನ ಮಾಡುವವನು, (ಯೋಗಶಾಸ್ತ್ರದಲ್ಲಿ) ವಜ್ರದಂಡದ ಮೂಲಕ ರಕ್ಷಿಸುವವನು

63. ಮಂಗಲಂ ಪರಂ = ಪರಮ ಮಂಗಳ ಸ್ವರೂಪಿಯು, ಪರಮ ಸುಖವನ್ನು ಉಂಟುಮಾಡುವವನು

64. ಈಶಾನಃ = ಎಲ್ಲಕ್ಕೂ ಅಧಿಪತಿಯಾದವನು, ಸರ್ವ ಸಮರ್ಥನು, ಐಶ್ವರ್ಯಶಾಲಿಯು

65. ಪ್ರಾಣದಃ = ಪ್ರಾಣಗಳನ್ನು ಕೊಡುವವನು, ಬಲಪ್ರದನು, ಸಂಜೀವಕನು

66. ಪ್ರಾಣಃ = ಉಜ್ಜೀವನಕ್ಕೆ ಕಾರಣಕರ್ತ, ಪ್ರಾಣಕ್ಕೂ ಪ್ರಾಣಸ್ವರೂಪನು

67. ಜ್ಯೇಷ್ಠಃ = ಅತ್ಯಂತ ಹಿರಿಯನು, ಪ್ರಶಸ್ತತಮನಾದವನು

68. ಶ್ರೇಷ್ಠಃ = ಅತ್ಯಂತ ಪ್ರಶಂಸನಿಯನು, ಮುಖ್ಯಪ್ರಾಣನು, ಸರ್ವಕಾರಣನು, ಸರ್ವರಿಗಿಂತಲೂ ಅತಿಶಯನಾದವನು

69. ಪ್ರಜಾಪತಿಃ = ಸೃಷ್ಟಿಕರ್ತನು, ಪ್ರಜಾಪಾಲಕನು

70. ಹಿರಣ್ಯಗರ್ಭಃ = ಸೃಷ್ಟಿಗೆ ಬೀಜವಾದ ಮತ್ತು ಹಿರಣ್ಮಯವಾದ ಅಂಡದೊಳಗೆ ಇರುವವನು, ಸೃಷ್ಟಿಕರ್ತನು

71. ಭೂಗರ್ಭಃ = ಭೂಮಿಯನ್ನು ಗರ್ಭದಲ್ಲಿ ಹೊಂದಿದವನು, ಭೂಮಿಯಲ್ಲಿ/ಪ್ರಕೃತಿಯಲ್ಲಿ ಗರ್ಭವನ್ನು ಇರಿಸುವವನು

72. ಮಾಧವಃ = ಲಕ್ಷ್ಮೀಪತಿಯು, ಆನಂದಕ್ಕೆ ಕಾರಣನಾದವನು, ವೈಶಾಖ ಮಾಸಕ್ಕೆ ಅಧಿಪತಿಯು

73. ಮಧುಸೂದನಃ = ತಮೋಗುಣ ರೂಪವಾದ ಅಥವಾ ಇಂದ್ರಿಯ ಸುಖರೂಪಿಯಾದ ಮಧು-ದೈತ್ಯನನ್ನು ಸಂಹರಿಸಿದವನು

74. ಈಶ್ವರಃ = ನಿರತಿಶಯವಾದ ಐಶ್ವರ್ಯವನ್ನು ಉಳ್ಳವನು, ಸ್ವಾಮಿಯು, ಸರ್ವಶಕ್ತನು

75. ವಿಕ್ರಮೀ = ಶೌರ್ಯ-ಪರಾಕ್ರಮ ಉಳ್ಳವನು, ವಿಶೇಷವಾದ ಪಾದವಿನ್ಯಾಸ ಮಾಡಿದವನು

76. ಧನ್ವೀ = ಧನುರ್ಧಾರಿಯು, ಅಹಂಕಾರ ತತ್ತ್ವ ರೂಪವಾದ ಶಾಙ್ರ್ಗ ಧನುಸ್ಸನ್ನು ಧರಿಸಿರುವವನು

77. ಮೇಧಾವೀ = ಮೇಧಾ ಎಂಬುದು ಅಪಸ್ಮಾರಕ್ಕೆ ವಿರುದ್ಧವಾಗಿರುವ ಶಕ್ತಿಯೂ, ಪ್ರಣವಕ್ಕೆ ಆವರಣವಾದ, ಜ್ಞಾನವನ್ನು ಧರಿಸುವ ಸಾಮಥ್ರ್ಯ ರೂಪವಾದ ಶಕ್ತಿಯೂ ಆಗಿದೆ; ಅಂತಹ ಮೇಧಾಶಕ್ತಿಯಿಂದ ಸಂಪನ್ನನಾದವನು.

78. ವಿಕ್ರಮಃ = ವಿಶೇಷವಾದ ಮತ್ತು ದೀರ್ಘವಾದ ಹೆಜ್ಜೆಗಳನ್ನಿಟ್ಟವನು

79. ಕ್ರಮಃ = ಪಾದವಿನ್ಯಾಸಕ್ಕೆ ಅಧಿದೇವತೆಯಾದವನು

80. ಅನುತ್ತಮಃ = ತನಗಿಂತಲೂ ಉತ್ತಮರಿಲ್ಲದವನು

81. ದುರಾಧರ್ಷಃ = ಎದುರಿಸಲು ಅಸಾಧ್ಯನಾದವನು

82. ಕೃತಜ್ಞಃ = ಪ್ರಾಣಿಗಳ ಕರ್ಮಗಳನ್ನೂ ಕರ್ಮಫಲಗಳನ್ನೂ ತಿಳಿದವನು, ಭಕ್ತರ ಆರಾಧನೆಯನ್ನು ತಿಳಿಯುವವನು, ಅತ್ಯಲ್ಪಸೇವೆಯನ್ನೂ ಮಹತ್ತಾಗಿ ತಿಳಿಯುವವನು

83. ಕೃತಿಃ = ಕ್ರಿಯಾಶಕ್ತಿ ರೂಪನಾದವನು

84. ಆತ್ಮವಾನ್ = ಆತ್ಮದಲ್ಲಿ ನೆಲೆಗೊಂಡು ನಿರ್ಲಿಪ್ತನಾಗಿರುವವನು, ಆತ್ಮಗುಣ-ವೈಭವ-ಜ್ಞಾನ-ಕ್ರಿಯೇಚ್ಛಾ ಸಂಪನ್ನನು

85. ಸುರೇಶಃ = ದೇವತೆಗಳಿಗೆ ಒಡೆಯನು

86. ಶರಣಂ = ಆಶ್ರಯನು, ಮನೆಯಾದವನು, ರಕ್ಷಕನು

87. ಶರ್ಮ = ಪರಮಸುಖ ಸ್ವರೂಪನು

88. ವಿಶ್ವರೇತಾಃ = ವಿಶ್ವಕ್ಕೆ ಕಾರಣನಾದ, ಅನಂತವಾದ, ಅಮೋಘವಾದ ವೀರ್ಯ ಉಳ್ಳವನು

89. ಪ್ರಜಾಭವಃ = ಪ್ರಜೆಗಳ ಉತ್ಪತ್ತಿಗೆ ಕಾರಣನಾದವನು

90. ಅಹಃ = ಜ್ಞಾನಿಗಳಿಗೆ ಹಗಲಾಗಿರುವವನು, ದೇವಯಾನ ಮಾರ್ಗದ ಹಗಲಿಗೆ ಅಧಿಪತಿಯು

91. ಸಂವತ್ಸರಃ = ಕಾಲದ, ಕಾಲಾವಯವದ ಸ್ವರೂಪಿಯು

92. ವ್ಯಾಲಃ = ಶೇಷನಾಗನು, ಕುಂಡಲಿನೀ ಸ್ವರೂಪನು

93. ಪ್ರತ್ಯಯಃ = ಜ್ಞಾನಸ್ವರೂಪಿಯು, ಪ್ರಕೃತಿಗೆ ಸೇರುವ ಪ್ರತ್ಯಯರೂಪಿಯು, ಸಹಾಯಕನು, ಮಧುರವಾಗಿರುವವನು

94. ಸರ್ವದರ್ಶನಃ = ಸರ್ವಸಾಕ್ಷಿಯು, ವಿಶ್ವಾಕ್ಷನು, ಸರ್ವದೇಶ-ಸರ್ವಾವಸ್ಥೆಗಳಲ್ಲಿಯೂ ಸರ್ವವನ್ನು ನೋಡುವವನು, ಎಲ್ಲರಿಗೂ ಕಣ್ಣಾಗಿರುವವನು, ವೀಕ್ಷಣ ಶಕ್ತಿಯನ್ನು ಕೊಡುವವನು, ತನ್ನ ಮಹಿಮೆಯನ್ನು ತೋರಿಸುವವನು

95. ಅಜಃ = ಹುಟ್ಟು ಇಲ್ಲದವನು

96. ಸರ್ವೇಶ್ವರಃ = ಎಲ್ಲರಿಗೂ ಈಶ್ವರನಾದವನು

97. ಸಿದ್ಧಃ = ಸ್ವಯಂಸಿದ್ಧನು, ಸರ್ವಸಿದ್ಧಿಗಳನ್ನೂ ಸಹಜವಾಗಿ ಹೊಂದಿರುವವನು, ನಿತ್ಯಸಿದ್ಧ ಸ್ವರೂಪನು

98. ಸಿದ್ಧಿಃ = ಸರ್ವಪುರುಷಾರ್ಥ ರೂಪನು, (ನಾಶವಾಗದ) ಫಲಸ್ವರೂಪನು, ಅಣಿಮಾದಿ ಸಿದ್ಧಿಸ್ವರೂಪನು

99. ಸರ್ವಾದಿಃ = ಎಲ್ಲಕ್ಕೂ (ಸರ್ವಭೂತಗಳಿಗೂ) ಆದಿಯಾದವನು

100. ಅಚ್ಯುತಃ = ಸ್ವರೂಪದಿಂದ, ಸ್ಥಾನದಿಂದ ಮತ್ತು ಸಂಕಲ್ಪದಿಂದ ಜಾರದೆ ಸದಾ ಸ್ಥಿರನಾಗಿರುವವನು

101. ವೃಷಾಕಪಿಃ = ಧರ್ಮಸ್ವರೂಪಿಯು, ಯಜ್ಞವರಾಹಸ್ವರೂಪಿಯು, ಆದಿತ್ಯಸ್ವರೂಪಿಯು

102. ಅಮೇಯಾತ್ಮಾ = ಅಳೆಯಲಾಗದ ಸ್ವರೂಪವುಳ್ಳವನು

103. ಸರ್ವಯೋಗವಿನಿಃಸೃತಃ = ಸಮಸ್ತ ಯೋಗಗಳಿಂದಲೂ ನಿರ್ಣಯಿಸಲ್ಪಟ್ಟವನು

104. ವಸುಃ = ವಸುದೇವತಾ/ಸಂಪತ್ ಸ್ವರೂಪಿಯು

105. ವಸುಮನಾಃ = ಪ್ರಶಸ್ತಧನ (ವೇದ-ಜ್ಞಾನ-ಪರಮಾತ್ಮ) ರೂಪವಾದ ಮನಸ್ಸುಳ್ಳವನು

106. ಸತ್ಯಃ = ಅವಿಕಾರಿಯು, ಮತ್ರ್ಯ-ಅಮತ್ರ್ಯಗಳಿಗೆ ಸೇತುವೆಯಾಗಿರುವವನು

107. ಸಮಾತ್ಮಾ = ಸರ್ವಭೂತಗಳಲ್ಲಿಯೂ/ವಿಷಮವಾದ-ವಸ್ತುಗಳಲ್ಲಿ ಸಮನಾದ ಸ್ವರೂಪ, ಸ್ವಭಾವ, ಮನಸ್ಸಿರುವ ಏಕವಸ್ತುವು

108. ಸಂಮಿತಃ/ಅಸಂಮಿತಃ = ಎಲ್ಲವನ್ನೂ ಚೆನ್ನಾಗಿ ಅಳೆದವನು, ಪರಿಮಿತನಾದಂತೆ ಕಾಣುವವನು

109. ಸಮಃ = ವಿಷಮಗಳಲ್ಲಿ ಸಮವಾಗಿರುವವನು, ಪಕ್ಷಪಾತವಿಲ್ಲದವನು

110. ಅಮೋಘಃ = ವ್ಯರ್ಥವಾಗದವನು, ಸಾರ್ಥಕವೇ ಆದವನು, ಸಾರ್ಥಕವೇ ಆದ ಸಂಕಲ್ಪ-ಕಾಮ-ಗುಣಾದಿಗಳುಳ್ಳವನು

111. ಪುಂಡರೀಕಾಕ್ಷಃ = ಕಮಲದ ಕಣ್ಣುಗಳುಳ್ಳವನು, ಹೃದಯ-ಕಮಲವನ್ನೇ ಕೇಂದ್ರವಾಗಿ ಉಳ್ಳವನು, (ಪುರುಷರ) ಹೃದಯ-ಗುಹೆಯಲ್ಲಿ ವಾಸಿಸುವವನು

112. ವೃಷಕರ್ಮಾ = ಧರ್ಮರೂಪವಾದ ಕರ್ಮಗಳುಳ್ಳವನು

113. ವೃಷಾಕೃತಿಃ = ಧರ್ಮಮೂರ್ತಿಯು

114. ರುದ್ರಃ = ದುಷ್ಟರನ್ನು ಅಳಿಸುವವನು, ದುಷ್ಟಭೀಕರನು, ಪ್ರಳಯ/ಸಂಹಾರ ಕಾಲದಲ್ಲಿ ಎಲ್ಲಾ ಪ್ರಾಣಿಗಳನ್ನೂ ಅಳುವಂತೆ ಮಾಡುವವನು

115. ಬಹುಶಿರಾಃ = ಸಾವಿರಾರು ತಲೆಗಳುಳ್ಳವನು

116. ಬಭ್ರುಃ = ಬಾಲಸೂರ್ಯನ ಬಣ್ಣವುಳ್ಳವನು, ಅಗ್ನಿಸ್ವರೂಪಿಯು, ಈಶ್ವರ ರೂಪಿಯು

117. ವಿಶ್ವಯೋನಿಃ = ವಿಶ್ವಕ್ಕೆ ಕಾರಣನಾದವನು, ಬ್ರಹ್ಮ-ಸ್ವರೂಪನು

118. ಶುಚಿಶ್ರವಾಃ = ಶುದ್ಧವಾದ ಕೀರ್ತಿಯುಳ್ಳವನು, ಶುದ್ಧವೂ ಶ್ರವಣ ಮಂಗಳವೂ ಆದ ನಾಮಾದಿಗಳುಳ್ಳವನು

119. ಅಮೃತಃ = ಅಳಿವಿಲ್ಲದವನು, ನಿತ್ಯನು, ಆನಂದಸ್ವರೂಪನು

120. ಶಾಶ್ವತಸ್ಥಾಣುಃ = ನಿತ್ಯನೂ, ಸ್ಥಿರನೂ ಆಗಿರುವವನು

121. ವರಾರೋಹಃ = ಶ್ರೇಷ್ಠವಾದ ಆರೋಹಣವುಳ್ಳವನು/ವಾಹನವುಳ್ಳವನು/ತೊಡೆಗಳುಳ್ಳವನು

122. ಮಹಾತಪಾಃ = ಶ್ರೇಷ್ಠವಾದ ತಪಸ್ಸುಳ್ಳವನು, ತೀವ್ರವಾದ ತಪಸ್ಸನ್ನು ಆಚರಿಸಿದ ನಾರಾಯಣ ಮಹರ್ಷಿಯು

123. ಸರ್ವಗಃ = ಸರ್ವವ್ಯಾಪಿಯು, ಸರ್ವಜ್ಞನು

124. ಸರ್ವವಿದ್ಭಾನುಃ = ಎಲ್ಲವನ್ನೂ ಅರಿತವನು, ಎಲ್ಲವನ್ನೂ ಪಡೆದವನು

125. ವಿಷ್ವಕ್ಸೇನಃ = ಎಲ್ಲೆಲ್ಲಿಯೂ ವಿಘ್ನಗಳನ್ನು ಪರಿಹರಿಸುವ ನಾನಾ ಸೇನೆಗಳುಳ್ಳವನು, ಮಹಾಗಣಪತಿ ಸ್ವರೂಪನು, ಮಹಾಸೇನಾನಿಯು

126. ಜನಾರ್ದನಃ = (ಮಹಾಪ್ರಳಯ ಕಾಲದಲ್ಲಿ) ಎಲ್ಲ ಪ್ರಾಣಿಗಳನ್ನೂ ಸಂಹರಿಸುವವನು, ದುಷ್ಟಶತ್ರುಗಳನ್ನು ಸಂಹರಿಸುವವನು

127. ವೇದಃ = ಜ್ಞಾನಸ್ವರೂಪಿಯು, ವೇದವಾಙ್ಮಯ ಸ್ವರೂಪನು

128. ವೇದವಿತ್ = ವೇದಗಳನ್ನೂ ವೇದಾರ್ಥವನ್ನೂ ಯಥಾವತ್ತಾಗಿ ಬಲ್ಲವನು

129. ಅವ್ಯಂಗಃ = ಯಾವ ನ್ಯೂನತೆಯೂ ಇಲ್ಲದ ಅಂಗಗಳುಳ್ಳವನು, ಪರಿಪೂರ್ಣನು

130. ವೇದಾಂಗಃ = ವೇದವನ್ನು ಕಣ್ಣಾಗಿ/ಅವಯವಗಳನ್ನಾಗಿ ಉಳ್ಳವನು, ಶಿಕ್ಷಾ-ವ್ಯಾಕರಣಾದಿ ವೇದಾಂಗ ರೂಪಿಯಾದವನು

131. ವೇದವಿತ್ = ವೇದಗಳನ್ನು ಬಲ್ಲವನು/ಪಡೆದವನು, ವಿಚಾರ ಮಾಡುವವನು

132. ಕವಿಃ = ಜ್ಞಾನಿಯು, ಸೃಷ್ಟಿಯನ್ನು ಹೆಣೆದವನು, ಕ್ರಾಂತದೃಷ್ಟಿ ಉಳ್ಳವನು

133. ಲೋಕಾಧ್ಯಕ್ಷಃ = ಸರ್ವಲೋಕಗಳಿಗೂ/ಪ್ರಕೃತಿಯ-ಸೃಷ್ಟಿಕಾರ್ಯಕ್ಕೆ ಅಧ್ಯಕ್ಷನಾದವನು, ಸರ್ವಸಾಕ್ಷಿಯು

134. ಸುರಾಧ್ಯಕ್ಷಃ = ದೇವತೆಗಳಿಗೆ ಸ್ವಾಮಿಯು, ಅವರ ವ್ಯಾಪಾರಗಳಿಗೆ ಅಧ್ಯಕ್ಷನಾಗಿರುವವನು

135. ಧರ್ಮಾಧ್ಯಕ್ಷಃ = ಎಲ್ಲ ಧರ್ಮಗಳಿಗೂ ಸ್ವಾಮಿಯಾದವನು, ಸರ್ವ ಧರ್ಮಗಳನ್ನು ಮತ್ತವುಗಳ ಫಲಗಳನ್ನೂ ನೋಡುವವನು

136. ಕೃತಾಕೃತಃ = ಕಾರ್ಯ-ಕಾರಣ ಸ್ವರೂಪನು, ಕರ್ಮ-ಮೋಕ್ಷ ಸ್ವರೂಪನು

137. ಚತುರಾತ್ಮಾ = ವಾಸುದೇವ-ಸಂಕರ್ಷಣ-ಪ್ರದ್ಯುಮ್ನ-ಅನಿರುದ್ಧ ಎಂಬ ನಾಲ್ಕು ಮೂರ್ತಿಗಳುಳ್ಳವನು

138. ಚತುವ್ರ್ಯೂಹಃ = ವಾಸುದೇವ-ಸಂಕರ್ಷಣ-ಪ್ರದ್ಯುಮ್ನ-ಅನಿರುದ್ಧ ಎಂಬ ನಾಲ್ಕು ವಿಭಾಗ/ವ್ಯೂಹಗಳುಳ್ಳವನು

139. ಚತುರ್ದಂಷ್ಟ್ರಃ = ನಾಲ್ಕು ಕೋರೆ ದಾಡೆಗಳುಳ್ಳ ನರಸಿಂಹ/ವರಾಹ ಮೂರ್ತಿಯು, ಮಹಾಪುರುಷ-ಲಕ್ಷಣವಾದ ನಾಲ್ಕು ದಂತವಿಶೇಷ ಉಳ್ಳವನು

140. ಚತುರ್ಭುಜಃ = ನಾಲ್ಕು ಭುಜಗಳುಳ್ಳವನು

141. ಭ್ರಾಜಿಷ್ಣುಃ = ಪ್ರಕಾಶಿಸುವ ಸ್ವಭಾವವಿರುವವನು

142. ಭೋಜನಂ = ಪಾಲಿಸುವವನು, ಭಕ್ತರಿಗೆ ಭೋಗ್ಯನಾಗಿರುವವನು

143. ಭೋಕ್ತಾ = ಆನಂದಾನುಭವದಲ್ಲಿ ಇರುವವನು, ಮಹಾಪ್ರಳಯ ಕಾಲದಲ್ಲಿ ಎಲ್ಲವನ್ನೂ ಭಕ್ಷಿಸುವವನು, ಭಕ್ತರು ಸಮರ್ಪಿಸಿದ ನೈವೇದ್ಯವನ್ನು ಭುಂಜಿಸುವವನು, ಅತಿಥಿ ಸ್ವರೂಪಿಯು

144. ಸಹಿಷ್ಣುಃ = ಸಹನಶೀಲನು

145. ಜಗದಾದಿಜಃ = ಜಗತ್ತಿಗೆ ಆದಿಯಲ್ಲಿ ಸೃಷ್ಟಿಕರ್ತನಾಗಿ ತಾನಾಗಿಯೇ ಪ್ರಕಟಗೊಂಡವನು

146. ಅನಘಃ = ದೋಷರಹಿತನು

147. ವಿಜಯಃ = ಎಲ್ಲವನ್ನೂ ವಿಶೇಷವಾಗಿ ಜಯಿಸಿದವನು, ಸರ್ವೋತ್ತಮನು

148. ಜೇತಾ = ಎಲ್ಲ ಪ್ರಾಣಿಗಳನ್ನು ಜಯಿಸುವವನು

149. ವಿಶ್ವಯೋನಿಃ = ವಿಶ್ವಕ್ಕೆ ಕಾರಣನಾದವನು

150. ಪುನರ್ವಸುಃ = ಮತ್ತೆ-ಮತ್ತೆ ಧನವನ್ನು ಅನುಗ್ರಹಿಸುವವನು, ನಿತ್ಯನಾಗಿ ಎಲ್ಲರಿಗೂ ನಿವಾಸ ಭೂಮಿಯಾಗಿರುವವನು, ಎಲ್ಲರಲ್ಲೂ ಅಂತರ್ಯಾಮಿಯಾಗಿರುವವನು, ಪುನರ್ವಸು ನಕ್ಷತ್ರ ದೇವತಾಸ್ವರೂಪನು

151. ಉಪೇಂದ್ರಃ = ಇಂದ್ರನಿಗೆ ತಮ್ಮನಾಗಿ ಅವತಾರ ಮಾಡಿದವನು

152. ವಾಮನಃ = ವಾಮನಾವತಾರವನ್ನು ಮಾಡಿದವನು, ಪ್ರಾಣಾಪಾನಗಳ ಮಧ್ಯದಲ್ಲಿ ಹೃದಯದಲ್ಲಿ ಅಂಗುಷ್ಠ ಮಾತ್ರನಾಗಿ ಗೋಚರಿಸುವ ಪರಂಜ್ಯೋತಿಯು

153. ಪ್ರಾಂಶುಃ = ಹೆಚ್ಚು ಉನ್ನತನಾದವನು, ತ್ರಿವಿಕ್ರಮನು

154. ಅಮೋಘಃ = ವ್ಯರ್ಥವಾಗದೇ ಇರುವ (ಸಾರ್ಥಕವೇ ಆದ) ಕಾಮ-ಸಂಕಲ್ಪ-ಕೃತಿಗಳುಳ್ಳವನು

155. ಶುಚಿಃ = ಸರ್ವಾತ್ಮನಾ ಪವಿತ್ರನು, ಸ್ತುತಿಸುವವರನ್ನು ಪವಿತ್ರಗೊಳಿಸುವವನು

156. ಊರ್ಜಿತಃ = ಅತ್ಯಧಿಕವಾದ ಬಲ-ಪ್ರಾಣಗಳುಳ್ಳವನು

157. ಅತೀಂದ್ರಃ = ಇಂದ್ರನನ್ನು ಜ್ಞಾನೈಶ್ವರ್ಯಾದಿಗಳಿಂದ ಮತ್ತು ಸ್ಥಾನ-ಮಾನಾದಿಗಳಿಂದ ಮೀರಿಸಿದವನು

158. ಸಂಗ್ರಹಃ = ಎಲ್ಲವನ್ನೂ ಉಪಸಂಹಾರ ಮಾಡಿಕೊಳ್ಳುವವನು, ಸೂಕ್ಷ್ಮ ರೂಪದಲ್ಲಿರುವವನು

159. ಸರ್ಗಃ = ಸೃಷ್ಟಿಗೆ ಕಾರಣನಾದವನು, ಸೃಷ್ಟಿ ರೂಪನಾದವನು

160. ಧೃತಾತ್ಮಾ = ಧೀರನು, ಆತ್ಮಸಂಯಮ ಉಳ್ಳವನು, ಅವಿಕಾರ ಸ್ವರೂಪಿಯು, ಚೇತನಗಳನ್ನು ಧರಿಸಿ ಪೋಷಿಸುತ್ತಿರುವವನು, ಧಾರಣಾಯೋಗದಿಂದ ಧರಿಸಲ್ಪಟ್ಟವನು

161. ನಿಯಮಃ = ಜೀವ-ಜಗತ್ತುಗಳನ್ನು ನಿಯಮಿಸುವವನು, ನಿಯಮ ಯೋಗವುಳ್ಳವನು

162. ಯಮಃ = ಯಮಧರ್ಮ ಸ್ವರೂಪಿಯು, ಯಮಯೋಗವುಳ್ಳವವನು, ಯಮನನ್ನು ನಿಯಮಿಸುವವನು, ಅಂತರ್ಯಾಮಿಯಾಗಿ ಎಲ್ಲವನ್ನು ನಡೆಸುವವನು

163. ವೇದ್ಯಃ = ತಿಳಿಯಲು ಯೋಗ್ಯನಾದವನು

164. ವೈದ್ಯಃ = ವಿದ್ವಾಂಸನು, ಆಧಿ-ವ್ಯಾಧಿ ಚಿಕಿತ್ಸಕನು, ಭವರೋಗ ವೈದ್ಯನು

165. ಸದಾಯೋಗೀ = ಸದಾ ಜಾಗರೂಕನಾಗಿರುವವನು, ಯಾವಾಗಲೂ ಯೋಗದಲ್ಲಿರುವವನು, ಯೋಗದಿಂದ ಎಂದಿಗೂ ಚ್ಯುತನಾಗದವನು

166. ವೀರಹಾ = ವೀರರಾದ ಅಸುರರನ್ನು ಸಂಹರಿಸುವವನು

167. ಮಾಧವಃ = ರಮಾಪತಿಯು, ಬ್ರಹ್ಮವಿದ್ಯೆಗೆ ಸ್ವಾಮಿಯು

168. ಮಧುಃ = (ಜೇನಿನಂತೆ) ಮಧುರ/ಮನನೀಯ ಸ್ವರೂಪನು

169. ಅತೀಂದ್ರಿಯಃ = ಇಂದ್ರಿಯಗಳಿಗೆ ವಿಷಯನಾಗದವನು

170. ಮಹಾಮಾಯಃ = ಮಹಾಮಾಯಾವಿಯು, ಮಾಯಾವಿಗಳನ್ನೂ ಮಾಯೆಯಲ್ಲಿ ಸಿಲುಕಿಸುವ ದೊಡ್ಡ ಮಾಯಾವಿಯು

171. ಮಹೋತ್ಸಾಹಃ = ಜಗತ್ತಿನ ವ್ಯಾಪಾರಗಳಲ್ಲಿ, ಭಕ್ತರಕ್ಷಣೆಯಲ್ಲಿ ಅಮಿತವಾದ/ಮಹತ್ತಾದ ಉತ್ಸಾಹವುಳ್ಳವನು

172. ಮಹಾಬಲಃ = ಅತ್ಯಂತ ಬಲಿಷ್ಠನಾದವನು

173. ಮಹಾಬುದ್ಧಿಃ = ಮಹಾಬುದ್ಧಿಶಾಲಿಯಾದವನು

174. ಮಹಾವೀರ್ಯಃ = ಅಮೋಘವಾದ, ಅತ್ಯಧಿಕವಾದ ಮತ್ತು ಸದಾ ಅವಿಕಾರಿಯಾದ ವೀರ್ಯವುಳ್ಳವನು

175. ಮಹಾಶಕ್ತಿಃ = ಮಹಾಶಕ್ತಿಶಾಲಿಯು, ಮಹಾ ಸಾಮಥ್ರ್ಯವುಳ್ಳವನು

176. ಮಹಾದ್ಯುತಿಃ = ಪರಂಜ್ಯೋತಿಃ ಸ್ವರೂಪನು

177. ಅನಿರ್ದೇಶ್ಯವಪುಃ = ನಿರ್ದೇಶಮಾಡಲು ಅಸಾಧ್ಯವಾದ ಸ್ವರೂಪವುಳ್ಳವನು (ಇತರರಿಗೆ ನಿರ್ದೇಶಿಸಲಾಗದ ರೂಪದವನು)

178. ಶ್ರೀಮಾನ್ = (ಐಶ್ವರ್ಯ ರೂಪವಾದ) ಶ್ರೀ ಸಂಪನ್ನನು

179. ಅಮೇಯಾತ್ಮಾ = ಅಳೆಯಲಾಗದ ಸ್ವರೂಪವುಳ್ಳವನು

180. ಮಹಾದ್ರಿಧೃಕ್ = ಮಹಾಪರ್ವತವನ್ನು ಧರಿಸಿದವನು

181. ಮಹೇಷ್ವಾಸಃ = ಮಹತ್ತಾದ ಧನುಸ್ಸುಳ್ಳವನು, ಬಾಣಗಳ ಎಸೆಯುವಿಕೆಯುಳ್ಳವನು, ದೊಡ್ಡ ಬಿಲ್ಲುಗಾರನು

182. ಮಹೀಭರ್ತಾ = ಭೂಮಿಗೆ ಪತಿಯಾದವನು, ಭೂಮಿಯನ್ನು ಧರಿಸಿದ ಯಜ್ಞವರಾಹನು

183. ಶ್ರೀನಿವಾಸಃ = ಲಕ್ಷ್ಮಿಗೆ ವಾಸಸ್ಥಾನನಾದವನು

184. ಸತಾಂ ಗತಿಃ = (ಬ್ರಹ್ಮದ ಅಸ್ತಿತ್ವವನ್ನರಿತ) ಸಂತರಿಗೆ ಗತಿಯಾದವನು, ಅವರ ಪುರುಷಾರ್ಥಗಳ ಸಾಧನೆಗೆ ಕಾರಣಕರ್ತನು

185. ಅನಿರುದ್ಧಃ = ತನ್ನ ವ್ಯಾಪಾರಗಳಲ್ಲಿ ಯಾರಿಂದಲೂ ತಡೆಯಲಾಗದವನು, ಅವ್ಯಕ್ತದಿಂದ ವ್ಯಕ್ತನಾದವನು, ಮಹಾತ್ಮನು

186. ಸುರಾನಂದಃ = ದೇವತೆಗಳಿಗೆ ಆನಂದವನ್ನುಂಟು ಮಾಡುವವನು

187. ಗೋವಿಂದಃ = ಗೋವುಗಳನ್ನು/ಪಶುಗಳನ್ನು/ದೇವ-ವಾಣಿಗಳನ್ನು/ಭೂಮಿಯನ್ನು ಪಡೆದವನು

188. ಗೋವಿದಾಂ ಪತಿಃ = ವೇದವಾಣಿಯನ್ನು ಪಡೆದವರಿಗೆ (ವೇದಜ್ಞರಿಗೆ) ಒಡೆಯನಾದವನು, ಗೋಪಾಲರಿಗೆ ಒಡೆಯನು

189. ಮರೀಚಿಃ = ಕಿರಣಗಳುಳ್ಳವನು, ತೇಜಸ್ವಿಯು

190. ದಮನಃ = ದುಷ್ಟರನ್ನು ದಮನ ಮಾಡುವವನು

191. ಹಂಸಃ = ಪ್ರಾಣ/ಸೂರ್ಯ/ಶುದ್ಧಾತ್ಮ/ಪ್ರಣವ ಸ್ವರೂಪಿಯು, ಜ್ಞಾನಪ್ರದವಾದ ಹಂಸಾವತಾರವನ್ನು ಮಾಡಿದವನು, ಹಂಸಮಹಾಮಂತ್ರ ರೂಪಿಯು

192. ಸುಪರ್ಣಃ = ಸುಂದರವಾದ ರೆಕ್ಕೆಗಳುಳ್ಳ ಗರುಡಸ್ವರೂಪನು, ವೇದಪುರುಷ ಸ್ವರೂಪಿಯು

193. ಭುಜಗೋತ್ತಮಃ = ಸರ್ವಶ್ರೇಷ್ಠನಾದ ಶೇಷನಾಗಸ್ವರೂಪಿಯು, ಶುದ್ಧ ಕುಂಡಲಿನೀ ಸ್ವರೂಪಿಯು

194. ಹಿರಣ್ಯನಾಭಃ = ಹಿರಣ್ಯದಂತೆ ಕಂಗೊಳಿಸುವ ಹಿತರಮಣೀಯವಾದ ನಾಭಿಯುಳ್ಳವನು, ಬ್ರಹ್ಮನನ್ನು ನಾಭಿಯಲ್ಲಿ ಹೊಂದಿದವನು

195. ಸುತಪಾಃ = ಶ್ರೇಷ್ಠ ತಪಸ್ವಿಯು

196. ಪದ್ಮನಾಭಃ = ಸೃಷ್ಟಿ ಮೂಲವಾದ ಕಮಲವನ್ನೇ ನಾಭಿಯಲ್ಲಿ ಹೊಂದಿರುವವನು, ಪದ್ಮ ಸದೃಶವಾದ ನಾಭಿಯುಳ್ಳವನು

197. ಪ್ರಜಾಪತಿಃ = ಸೃಷ್ಟಿಕರ್ತನು, ಪ್ರಜೆಗಳಿಗೆ ಪತಿಯಾದವನು

198. ಅಮೃತ್ಯುಃ = ಮೃತ್ಯುವಿಲ್ಲದವನು, ಮೃತ್ಯುವಿನಿಂದ ರಹಿತನಾದವನು, ಮರಣವಿಲ್ಲದವನು, ಶಾಶ್ವತನು

199. ಸರ್ವದೃಕ್ = ಎಲ್ಲವನ್ನೂ ನೋಡುವವನು, ಪ್ರಾಣಿಗಳು ಮಾಡಿದ ಪಾಪ-ಪುಣ್ಯ ಕರ್ಮಗಳನ್ನು ನೋಡುವವನು, ಎಲ್ಲ ಕಡೆಗಳಲ್ಲಿಯೂ ದೃಷ್ಟಿ ಇರುವವನು

200. ಸಿಂಹಃ = ನೃಸಿಂಹ ಸ್ವರೂಪನು, ಶ್ರೇಷ್ಠನು

201. ಸಂಧಾತಾ = (ಭಕ್ತರನ್ನು) ಚೆನ್ನಾಗಿ ಧಾರಣೆ ಮಾಡುವವನು/ಪೋಷಿಸುವವನು, ಸೇರಿಸುವವನು

202. ಸಂಧಿಮಾನ್ = ಪರಸ್ಪರವಾಗಿ ಸಂಧಿ ಮಾಡುವ ಕೌಶಲ್ಯವನ್ನು ಹೊಂದಿರುವವನು, ಕುಶಲನಾದ ರಾಯಭಾರಿಯು

203. ಸ್ಥಿರಃ = ಸುಸ್ಥಿರನಾಗಿರುವವನು

204. ಅಜಃ = ಹುಟ್ಟಿಲ್ಲದವನು

205. ದುರ್ಮರ್ಷಣಃ = ಸಹಿಸಿಕೊಳ್ಳಲು ಅಶಕ್ಯನಾದವನು

206. ಶಾಸ್ತಾ = ಶಾಸನ ಮಾಡುವವನು, ಶಾಸ್ತಾ ದೇವತಾ ಸ್ವರೂಪನು

207. ವಿಶ್ರುತಾತ್ಮಾ = ವಿಖ್ಯಾತವಾದ ಸ್ವರೂಪವುಳ್ಳವನು

208. ಸುರಾರಿಹಾ = ದೇವತೆಗಳ ಶತ್ರುಗಳನ್ನು ಸಂಹರಿಸುವವನು

209. ಗುರುಃ = ದೇವಪುರೋಹಿತ ಬೃಹಸ್ಪತಿ ಸ್ವರೂಪನು, ಪಿತೃ ಸ್ವರೂಪನು, ಅಜ್ಞಾನವನ್ನು ಹೋಗಲಾಡಿಸುವ ಉಪದೇಶಕನು, ತ್ರಿಮೂರ್ತಿ ಸ್ವರೂಪನು

210. ಗುರುತಮಃ = ಎಲ್ಲ ಗುರುಗಳಿಗಿಂತಲೂ ಶ್ರೇಷ್ಠನಾದ ಗುರುವು, ಮಹಾಗುರುವು

211. ಧಾಮ = ಸಕಲ ಚರಾಚರಗಳಿಗೂ ಆಶ್ರಯನಾದವನು, ಪ್ರಭೆಯಿಂದ ಯುಕ್ತನಾದವನು, ಮಹಾಮಹಿಮನು, ಪ್ರತಾಪಶಾಲಿಯು

212. ಸತ್ಯಃ = ಸತ್ಯಸ್ವರೂಪನು, ಸತ್ತ್ವಗುಣ ಸಂಪನ್ನನು

213. ಸತ್ಯಪರಾಕ್ರಮಃ = ಸತ್ಯಯುಕ್ತವಾದ ಪರಾಕ್ರಮವುಳ್ಳವನು, ಸತ್ಯವೇ ಪರಾಕ್ರಮವಾಗಿರುವವನು

214. ನಿಮಿಷಃ = (ಧ್ಯಾನಯೋಗದಲ್ಲಿ) ಕಣ್ಣುಗಳನ್ನು ಮುಚ್ಚಿ ಕೊಂಡಿರುವವನು, ಕಾಲಸ್ವರೂಪನು

215. ಅನಿಮಿಷಃ = ರೆಪ್ಪೆಗಳನ್ನು ಮುಚ್ಚದೇ ಇರುವವನು, ಮತ್ಸ್ಯ ಮೂರ್ತಿಯು

216. ಸ್ರಗ್ವೀ = ಮಾಲೆಯನ್ನು ಧರಿಸಿದವನು

217. ವಾಚಸ್ಪತಿರುದಾರಧೀಃ = (ಭಕ್ತರಲ್ಲಿ) ಔದಾರ್ಯ ಬುದ್ಧಿಯುಳ್ಳವನು, ವಿಶಾಲವಾದ ಬುದ್ಧಿಯುಳ್ಳವನು, ವಾಣಿಗೆ/ವೇದಗಳಿಗೆ ಒಡೆಯನು

218. ಅಗ್ರಣೀಃ = ಮುಂದಕ್ಕೆ ಒಯ್ಯುವವನು, ಅಗ್ನಿಸ್ವರೂಪನು, ಪ್ರಭುವು, ಶ್ರೇಷ್ಠನಾದವನು

219. ಗ್ರಾಮಣೀಃ = ಜನಸಮೂಹದ ಪ್ರೇರಕನು, ಪಂಚಭೂತಗಳನ್ನು ಅವುಗಳ ಕಾರ್ಯಗಳಲ್ಲಿ ನಡೆಸುವವನು

220. ಶ್ರೀಮಾನ್ = ಶ್ರೀಮಂತನು, ಲಕ್ಷ್ಮೀಯುಕ್ತನಾದವನು

221. ನ್ಯಾಯಃ = ನ್ಯಾಯ ಸ್ವರೂಪನು, ಧರ್ಮಮೂರ್ತಿಯು, ಯುಕ್ತವಾದುದನ್ನು ಮಾಡುವವನು

222. ನೇತಾ = ನಾಯಕನು

223. ಸಮೀರಣಃ = ವಾಯುದೇವತಾ ಸ್ವರೂಪನು

224. ಸಹಸ್ರಮೂರ್ಧಾ = ಸಾವಿರ ತಲೆಗಳಿರುವವನು, ಅನಂತ ಶಿರಸ್ಸಿನವನು

225. ವಿಶ್ವಾತ್ಮಾ = ವಿಶ್ವದೇಹಿಯು, ವಿಶ್ವಕ್ಕೆಲ್ಲಾ ಆತ್ಮನಾಗಿರುವವನು, ಸರ್ವರಿಗೂ ಆತ್ಮನಾಗಿರುವವನು

226. ಸಹಸ್ರಾಕ್ಷಃ = ಸಾವಿರ ಕಣ್ಣುಗಳುಳ್ಳವನು

227. ಸಹಸ್ರಪಾತ್ = ಸಾವಿರ ಕಾಲುಗಳುಳ್ಳವನು

228. ಆವರ್ತನಃ = (ಕಾಲಚಕ್ರವನ್ನು) ತಿರುಗಿಸುವವನು

229. ನಿವೃತ್ತಾತ್ಮಾ = ಅದೃಶ್ಯವಾದ ಆತ್ಮವುಳ್ಳವನು

230. ಸಂವೃತಃ = ಆವೃತನಾಗಿರುವವನು

231. ಸಂಪ್ರಮರ್ದನಃ = (ಶತ್ರುಗಳನ್ನು) ಚೆನ್ನಾಗಿ ಮರ್ದನ ಮಾಡುವವನು

232. ಅಹಃಸಂವರ್ತಕಃ = ಹಗಲು-ರಾತ್ರಿಗಳನ್ನು ಪರಿವರ್ತಿಸುವವನು

233. ವಹ್ನಿಃ = ಅಗ್ನಿ ಸ್ವರೂಪನು

234. ಅನಿಲಃ = ವಾಯು ಸ್ವರೂಪನು

235. ಧರಣೀಧರಃ = ಭೂಮಿಯನ್ನು ಧರಿಸಿರುವವನು, ಶೇಷನಾಗನು

236. ಸುಪ್ರಸಾದಃ = ಚೆನ್ನಾಗಿ ಅನುಗ್ರಹಿಸುವವನು

237. ಪ್ರಸನ್ನಾತ್ಮಾ = ಪ್ರಸನ್ನವಾದ ಮನಸ್ಸಿನಿಂದ ಕೂಡಿರುವವನು

238. ವಿಶ್ವಧೃಕ್ = ಪ್ರಪಂಚವನ್ನು ಸೃಷ್ಟಿಸುವವನು, ಜಗತ್ತನ್ನು ಧರಿಸಿರುವವನು

239. ವಿಶ್ವಭುಕ್ = ವಿಶ್ವವನ್ನು ಉಪಭುಂಜಿಸುವವನು, (ಪ್ರಳಯಕಾಲದಲ್ಲಿ) ವಿಶ್ವವನ್ನು ನುಂಗುವವನು

240. ವಿಭುಃ = ವ್ಯಾಪ್ತನಾದವನು

241. ಸತ್ಕರ್ತಾ = ಒಳ್ಳೆಯದನ್ನು ಮಾಡುವವನು, ವಿಶ್ವವನ್ನು ನಿರ್ಮಿಸಿದವನು

242. ಸತ್ಕೃತಃ = ಎಲ್ಲರಿಂದಲೂ ಪೂಜಿತನಾದವನು

243. ಸಾಧುಃ = ಸತ್ಪುರುಷನು, ಇಷ್ಟಾರ್ಥವನ್ನು ಸಾಧಿಸುವವನು

244. ಜಹ್ನುಃ = ಜಹ್ನು ಮಹರ್ಷಿ ರೂಪಿಯಾದವನು, ದುಷ್ಟರನ್ನು ತೊರೆಯುವವನು, ಅಜ್ಞಾನಿಗಳ ದೆಸೆಯಿಂದ ತನ್ನನ್ನು ಮುಚ್ಚಿಟ್ಟುಕೊಳ್ಳುವವನು

245. ನಾರಾಯಣಃ = ನರರಿಗೆಲ್ಲರಿಗೂ ಏಕಮಾತ್ರಗತಿಯೂ/ಮಾರ್ಗವೂ ಆಗಿರುವವನು, ಸೃಷ್ಟಿಮೂಲವಾದ ಜಲವನ್ನು ಆಶ್ರಯವನ್ನಾಗಿ ಹೊಂದಿರುವವನು

246. ನರಃ = ನರನೆಂಬ ಮಹರ್ಷಿಯ ರೂಪದಲ್ಲಿ ಅವತಾರ ಮಾಡಿದವನು, ಎಲ್ಲರಿಗೂ ಜೀವಭೂತನಾದವನು

247. ಅಸಂಖ್ಯೇಯಃ = ಎಣಿಸಲಾಗದ ಗುಣ ವಿಭೂತಿಗಳುಳ್ಳವನು

248. ಅಪ್ರಮೇಯಾತ್ಮಾ = ಅಳೆಯಲು ಅಸಾಧ್ಯವಾದ ಸ್ವರೂಪವುಳ್ಳವನು

249. ವಿಶಿಷ್ಟಃ = ಎಲ್ಲಕ್ಕಿಂತಲೂ ಅತಿಶಯವಾಗಿರುವವನು, ಎಲ್ಲದರೊಡನೆ ಕೂಡಿಕೊಂಡಿರುವವನು

250. ಶಿಷ್ಟಕೃತ್ = ಶಿಷ್ಟಜನರನ್ನು ಉಂಟು ಮಾಡುವವನು, ಒಳ್ಳೆಯ ಕಾರ್ಯಗಳನ್ನು ಮಾಡುವವನು

251. ಶುಚಿಃ = ಶುದ್ಧನು

252. ಸಿದ್ಧಾರ್ಥಃ = ಸಿದ್ಧವಾದ ಪ್ರಯೋಜನವುಳ್ಳವನು

253. ಸಿದ್ಧ ಸಂಕಲ್ಪಃ = ನಿರರ್ಥಕವಾಗದೇ ಸಿದ್ಧವಾಗುವ ಸಂಕಲ್ಪವುಳ್ಳವನು

254. ಸಿದ್ಧಿದಃ = ಭಕ್ತರಿಗೆ ಎಲ್ಲ ಪುರುಷಾರ್ಥಗಳನ್ನು ಈಡೇರಿಸುವವನು, ಅಣಿಮಾದಿ ಸಿದ್ಧಿಗಳನ್ನು ಅನುಗ್ರಹಿಸುವವನು

255. ಸಿದ್ಧಿಸಾಧನಃ = ಸಿದ್ಧಿಗಳನ್ನು ಸಾಧಿಸುವವನು

256. ವೃಷಾಹೀ = ಧರ್ಮಪ್ರಕಾಶವುಳ್ಳವನು, ಧರ್ಮವನ್ನೇ ಹಗಲಾಗಿ ಉಳ್ಳವನು

257. ವೃಷಭಃ = ಧರ್ಮ ವೃಷಭ ಸ್ವರೂಪಿಯಾದವನು, ಇಷ್ಟಾರ್ಥಗಳನ್ನು ವರ್ಷಿಸುವವನು, ವೃಷಭರೂಪಿಯಾಗಿ ತ್ರಿಪುರಾಸುರ ಸಂಹಾರದಲ್ಲಿ ನೆರವು ನೀಡುವವನು

258. ವಿಷ್ಣುಃ = ಬೃಹತ್ತಾಗಿರುವವನು, ವ್ಯಾಪಕನು, ಎಲ್ಲವನ್ನೂ ಪ್ರವೇಶಿಸಿರುವವನು

259. ವೃಷಪರ್ವಾ = ತನ್ನ ಬಳಿಗೆ ಆರೋಹಣ ಮಾದಿಸುವ ಧರ್ಮರೂಪವಾದ ಸೋಪಾನಗಳುಳ್ಳವನು

260. ವೃಷೋದರಃ = ಧರ್ಮರೂಪವಾದ ಉದರವುಳ್ಳವನು

261. ವರ್ಧನಃ = ವೃದ್ಧಿಪಡಿಸುವವನು

262. ವರ್ಧಮಾನಃ = ವೃದ್ಧಿ ಹೊಂದುತ್ತಿರುವವನು

263. ವಿವಿಕ್ತಃ = ಸರ್ವೋತ್ತಮನಾಗಿ ನಿರ್ಲಿಪ್ತನಾಗಿ ಎಲ್ಲಕ್ಕಿಂತಲೂ ಪ್ರತ್ಯೇಕವಾಗಿರುವವನು

264. ಶ್ರುತಿಸಾಗರಃ = ವೇದಗಳಿಗೆ ಸಮುದ್ರದಂತಿರುವವನು

265. ಸುಭುಜಃ = ಶೋಭನವಾದ ಭುಜಗಳುಳ್ಳವನು

266. ದುರ್ಧರಃ = ಧರಿಸಲು ಅಶಕ್ಯನಾದವನು, ಧರಿಸಲು ಅಸಾಧ್ಯವಾದ ವೀರ್ಯ/ತೇಜಸ್ಸು/ಶೌರ್ಯ.. ಉಳ್ಳವನು.

267. ವಾಗ್ಮೀ = ಪ್ರಶಸ್ತವಾದ/ವೇದಮಯವಾದ ವಾಣಿಯುಳ್ಳವನು, ಪ್ರಿಯವೂ/ಹಿತವೂ ಆದ ವಾಣಿಯುಳ್ಳವನು

268. ಮಹೇಂದ್ರಃ = ಪರಮೈಶ್ವರ್ಯ ಸಂಪನ್ನನಾಗಿ ಈಶ್ವರರಿಗೂ ಈಶ್ವರನಾದವನು

269. ವಸುದಃ = ಸಂಪತ್ತುಗಳನ್ನು ಅನುಗ್ರಹಿಸುವವನು, ವಸು ದೇವತಾ ಪದವಿಯನ್ನು ಕೊಡುವವನು

270. ವಸುಃ = ಅಸಾಧಾರಣವಾದ ಸಂಪತ್ತು ಆಗಿರುವವನು, ವಸು ದೇವತಾ ರೂಪಿಯು

271. ನೈಕರೂಪಃ = ಅನೇಕ ರೂಪಗಳುಳ್ಳವನು, ವಿಶ್ವರೂಪಿಯು

272. ಬೃಹದ್ರೂಪಃ = ವೃದ್ಧಿ ಹೊಂದಿದ ದೊಡ್ಡ ರೂಪವುಳ್ಳವನು, ಬ್ರಹ್ಮನು

273. ಶಿಪಿವಿಷ್ಟಃ = ಪಶುಗಳಲ್ಲಿ ಪ್ರವೇಶ ಮಾಡಿ ನೆಲೆಸುವವನು, ಕಿರಣಗಳಲ್ಲಿ ಇರುವವನು, ನಿರಾಕಾರ ರೂಪದಿಂದ ಸಮಸ್ತ ಜಗತ್ತನ್ನು ವ್ಯಾಪಿಸಿರುವವನು

274. ಪ್ರಕಾಶನಃ = ಎಲ್ಲವನ್ನೂ ಪ್ರಕಾಶಪಡಿಸುವವನು

275. ಓಜಸ್ತೇಜೋದ್ಯುತಿಧರಃ = ಬಲ-ತೇಜಸ್ಸು (ಶೌರ್ಯಾದಿ ಗುಣಗಳು, ಕೀರ್ತಿ) ಮತ್ತು ದೀಪ್ತಿಗಳುಳ್ಳವನು

276. ಪ್ರಕಾಶಾತ್ಮಾ = ಪ್ರಕಾಶ ಸ್ವರೂಪನು

277. ಪ್ರತಾಪನಃ = ಬೆಳಗಿಸುವವನು, ದುಷ್ಟರನ್ನು ಸಂತಾಪಗೊಳಿಸುವವನು

278. ಋದ್ಧಃ = ಅಭಿವೃದ್ಧಿ ಹೊಂದಿದವನು

279. ಸ್ಪಷ್ಟಾಕ್ಷರಃ = ಅಕ್ಷರವನ್ನು/ಪ್ರಣವವನ್ನು ಸ್ಪಷ್ಟಪಡಿಸಿದವನು

280. ಮಂತ್ರಃ = ಗುಪ್ತವಾಗಿರುವವನು, ಮನನ ಮಾಡಿದವರನ್ನು ರಕ್ಷಿಸುವವನು, ಮಹಾಮಂತ್ರವರ್ಣ ಸ್ವರೂಪಿಯು, ಮಂತ್ರ ಪ್ರತಿಪಾದ್ಯನು

281. ಚಂದ್ರಾಂಶುಃ = ಚಂದ್ರನಂತೆ ಆಹ್ಲಾದವನ್ನುಂಟು ಮಾಡುವವನು, ಅಮೃತಮಯವಾದ ಕಿರಣಗಳುಳ್ಳವನು, ಚಂದ್ರಕಿರಣರೂಪಿಯು

282. ಭಾಸ್ಕರದ್ಯುತಿಃ = ಸೂರ್ಯಪ್ರಕಾಶನು, ಸೂರ್ಯನಂತೆ ಸ್ವಯಂಪ್ರಕಾಶನಾಗಿ ಎಲ್ಲರಿಗೂ ಪ್ರಕಾಶವನ್ನು ನೀಡುವವನು, ಸಂತಾಪಕರವಾದ ತೇಜಸ್ಸುಳ್ಳವನು

283. ಅಮೃತಾಂಶೂದ್ಭವಃ = ಅಮೃತ ಕಿರಣನ ಉತ್ಪತ್ತಿಗೆ ಕಾರಣನಾದವನು, ಅಮೃತ ಕಿರಣಗಳಿಂದ ಧನ್ವಂತರಿಯ ರೂಪದಲ್ಲಿ ಆವಿರ್ಭವಿಸಿದವನು

284. ಭಾನುಃ = ಪ್ರಕಾಶಮಾನನಾದ ಸೂರ್ಯ ರೂಪಿಯು

285. ಶಶಬಿಂದುಃ = ಸೋಮ ರೂಪಿಯಾದವನು

286. ಸುರೇಶ್ವರಃ = ದೇವತೆಗಳಿಗೆ ಈಶ್ವರನಾದವನು

287. ಔಷಧಂ = ಆಧಿ-ವ್ಯಾಧಿಗಳನ್ನು ನಿವಾರಿಸುವ ಔಷಧರೂಪಿಯಾದವನು

288. ಜಗತಃ ಸೇತುಃ = ಜಗತ್ತಿಗೆ ಸೇತುವೆಯಂತೆ ಇರುವವನು

289. ಸತ್ಯಧರ್ಮಪರಾಕ್ರಮಃ = ಸತ್ಯವಾದ/ಸತ್ಯಕ್ಕೆ-ಅನುಗುಣವಾದ ಧರ್ಮ ಮತ್ತು ಪರಾಕ್ರಮವುಳ್ಳವನು

290. ಭೂತಭವ್ಯಭವನ್ನಾಥಃ = ಭೂತ-ಭವಿಷ್ಯತ್-ವರ್ತಮಾನಗಳಿಗೆ ಒಡೆಯನಾದವನು

291. ಪವನಃ = ಬೀಸುವ ಧರ್ಮವುಳ್ಳ ವಾಯು ಸ್ವರೂಪಿಯು

292. ಪಾವನಃ = ಪವಿತ್ರಗೊಳಿಸುವವನು

293. ಅನಲಃ = ಅಗ್ನಿ/ಜೀವ ಸ್ವರೂಪಿಯು, ಭಕ್ತರಿಗೆ ಸದಾ ಉಪಕರಿಸುವ, ಕೊನೆಯಿಲ್ಲದ ಅನುಗ್ರಹ-ಮಹಿಮೆಯುಳ್ಳವನು

294. ಕಾಮಹಾ = ದುಷ್ಟವಾದ ಕಾಮವನ್ನು ಧ್ವಂಸಮಾಡುವವನು, ಮನ್ಮಥನನ್ನು ಧ್ವಂಸ ಮಾಡಿದ ರುದ್ರ ಸ್ವರೂಪಿಯು

295. ಕಾಮಕೃತ್ = ಭಕ್ತರ ಒಳ್ಳೆಯ ಇಷ್ಟಾರ್ಥಗಳನ್ನು ನೆರವೇರಿಸುವವನು

296. ಕಾಂತಃ = ಹೆಚ್ಚು ಆಕರ್ಷಕನಾದವನು

297. ಕಾಮಃ = ಮನ್ಮಥ ಸ್ವರೂಪಿಯು, ಅತ್ಯಂತ ಕಮನೀಯನಾದವನು

298. ಕಾಮಪ್ರದಃ = ಯೋಗ್ಯವಾದ ಇಷ್ಟಾರ್ಥಗಳನ್ನು ಕೊಡುವವನು

299. ಪ್ರಭುಃ = ಅತ್ಯಂತ ಸಮರ್ಥನು, ಪ್ರಭಾವಶಾಲಿಯು

300. ಯುಗಾದಿಕೃತ್ = ಯುಗಗಳನ್ನು ಪ್ರಾರಂಭಿಸಿದವನು, ಯುಗವೇ ಮುಂತಾದ ಕಾಲಭೇದವನ್ನು ಉಂಟುಮಾಡುವವನು
