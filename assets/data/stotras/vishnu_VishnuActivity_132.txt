301. ಯುಗಾವರ್ತಃ = ಯುಗಗಳು ಮತ್ತೆ ಮತ್ತೆ ಉರುಳಿ ಬರುವಂತೆ ಮಾಡುವವನು

302. ನೈಕಮಾಯಃ = ಅನಂತ ಮಾಯೆಗಳುಳ್ಳವನು

303. ಮಹಾಶನಃ = ಮಹಾಭಕ್ಷಕನು, ಮಹಾಪ್ರಳಯದಲ್ಲಿ ವಿಶ್ವವನ್ನೇ ಭಕ್ಷಿಸುವವನು

304. ಅದೃಶ್ಯಃ = ಚರ್ಮ-ಚಕ್ಷುಗಳಿಗೆ, ಬುದ್ಧಿ-ಇಂದ್ರಿಯಗಳಿಗೆ ಗೋಚರನಾಗದವನು

305. ವ್ಯಕ್ತರೂಪಃ = ಸ್ಥೂಲ ಜಗದ್ರೂಪದಿಂದ ಮತ್ತು ಅವತಾರ ರೂಪಗಳಿಂದ ವ್ಯಕ್ತನಾಗುವವನು, ಯೋಗಿಗಳಿಗೆ ವ್ಯಕ್ತವಾಗಿರುವ ಸ್ವರೂಪವುಳ್ಳವನು

306. ಸಹಸ್ರಜಿತ್ = ಸಾವಿರಾರು ಜಯಗಳನ್ನು ಗಳಿಸಿದವನು

307. ಅನಂತಜಿತ್ = ಅನಂತ ರೀತಿಗಳಲ್ಲಿ, ಅನಂತ ವಸ್ತುಗಳ ಮೇಲೆ ಅನಂತಕಾಲಗಳಲ್ಲಿ ಜಯಗಳಿಸುವವನು

308. ಇಷ್ಟಃ = ಪರಮಾನಂದ ಸ್ವರೂಪನಾದ ಆತ್ಮನಾಗಿರುವುದರಿಂದ ಎಲ್ಲರಿಗೂ ಪ್ರಿಯನಾದವನು, ಯಜ್ಞದಲ್ಲಿ ಪೂಜಿತನು

309. ಅವಿಶಿಷ್ಟಃ = ಎಲ್ಲರಿಗೂ ಅಂತರ್ಯಾಮಿಯಾದ್ದರಿಂದ ಯಾವ ವಿಶೇಷವೂ ಇಲ್ಲದವನು

310. ಶಿಷ್ಟೇಷ್ಟಃ = ಸಜ್ಜನರಿಗೆ ಪ್ರಿಯತಮನಾದವನು, ಸಜ್ಜನರಿಂದ ಪೂಜಿಸಲ್ಪಡುವವನು, ಸಜ್ಜನರಲ್ಲಿ ಇಷ್ಟವುಳ್ಳವನು

311. ಶಿಖಂಡೀ = ನವಿಲುಗರಿಯ ಭೂಷಣವುಳ್ಳವನು, ಜ್ಞಾನಶಿಖೆಯುಳ್ಳವನು

312. ನಹುಷಃ = (ಸಂಸಾರಪಾಶದಿಂದ) ಎಲ್ಲರನ್ನೂ ಕಟ್ಟಿಹಾಕುವವನು, ಎಲ್ಲರಿಗೂ ಏಕಸೂತ್ರನಾಗಿರುವವನು

313. ವೃಷಃ = ಧರ್ಮರೂಪಿಯಾದವನು, ಇಷ್ಟಾರ್ಥಗಳನ್ನು ಮಳೆಗರೆಯುವವನು, ಅಮೃತಸೇಚನೆ ಮಾಡುವವನು

314. ಕ್ರೋಧಹಾ = ಕ್ರೋಧರೂಪವಾದ ಶತ್ರುಗಳನ್ನು ಧ್ವಂಸ ಮಾಡುವವನು

315. ಕ್ರೋಧಕೃತ್(ಕರ್ತಾ) = ರಜೋಗುಣದ ಮೂಲಕ ಕ್ರೋಧವನ್ನುಂಟು ಮಾಡುವವನು, ದುಷ್ಟರಲ್ಲಿ ಕೋಪಗೊಳ್ಳುವವನು, ಜಗತ್ತನ್ನು ರಚಿಸಿದವನು, ಜಗತ್ತೆಂಬ ಕರ್ಮಕ್ಕೆ ಕರ್ತೃ

316. ವಿಶ್ವಬಾಹುಃ = ಸರ್ವಬಾಹುವು, ಎಲ್ಲೆಲ್ಲಿಯೂ ಬಾಹುಗಳುಳ್ಳವನು, ವಿಶ್ವಕ್ಕೆ ಬಾಹುಗಳಂತೆ ಆಲಂಬನವಾಗಿರುವವನು

317. ಮಹೀಧರಃ = ಋತ-ಸತ್ಯಗಳಿಂದ/ಯಜ್ಞವರಾಹ-ಸ್ವರೂಪಿಯಾಗಿ/ಶೇಷನಾಗ-ರೂಪಿಯಾಗಿ ಭೂಮಿಯನ್ನು ಧರಿಸಿರುವವನು

318. ಅಚ್ಯುತಃ = ಸ್ಥಾನ, ಸಂಕಲ್ಪ ಮುಂತಾದವುಗಳಿಂದ ಜಾರದವನು

319. ಪ್ರಥಿತಃ = ಪ್ರಸಿದ್ಧನಾದವನು

320. ಪ್ರಾಣಃ = ಪ್ರಾಣ ರೂಪಿಯಾದವನು

321. ಪ್ರಾಣದಃ = ಬಲವನ್ನು ಕೊಡುವವನು

322. ವಾಸವಾನುಜಃ = ಇಂದ್ರನಿಗೆ ತಮ್ಮನಾದ ಉಪೇಂದ್ರನು

323. ಅಪಾಂನಿಧಿಃ = ಸಮುದ್ರ ರೂಪಿಯಾದವನು

324. ಅಧಿಷ್ಠಾನಂ = ಎಲ್ಲಕ್ಕೂ ಆಶ್ರಯನಾದವನು

325. ಅಪ್ರಮತ್ತಃ = ಸಾವಧಾನ/ಜಾಗರೂಕ-ನಾಗಿರುವವನು

326. ಪ್ರತಿಷ್ಠಿತಃ = ತನ್ನ ಮಹಿಮೆಯಲ್ಲಿ ನೆಲೆಗೊಂಡವನು

327. ಸ್ಕಂದಃ = ಷಣ್ಮುಖ ರೂಪಿಯಾದವನು

328. ಸ್ಕಂದಧರಃ = ಷಣ್ಮುಖನನ್ನು ಧರಿಸಿ ಪೋಷಿಸುವವನು

329. ಧುರ್ಯಃ = ಭಾರವನ್ನು ಹೊತ್ತಿರುವವನು

330. ವರದಃ = ವರಗಳನ್ನು ದಯಪಾಲಿಸುವವನು

331. ವಾಯುವಾಹನಃ = ಪ್ರಾಣವಾಯುಗಳನ್ನೂ, ಸಪ್ತ ಮರುತ್ತುಗಳನ್ನೂ ನಡೆಸುವವನು, ವಾಯುಪುತ್ರನನ್ನು/ಹನುಮಂತನನ್ನು ವಾಹನವನ್ನಾಗಿ ಹೊಂದಿದವನು

332. ವಾಸುದೇವಃ = ಸರ್ವಭೂತ ನಿವಾಸಿಯಾದ ದೇವನು, ಜೀವಗಳಿಗೆ ಸ್ವಾಮಿಯು, ವಸುದೇವನ ಮಗನಾಗಿ ಅವತರಿಸಿದವನು

333. ಬೃಹದ್ಭಾನುಃ = ದೊಡ್ಡ ಕಿರಣಗುಳಳ್ಳವನು, ಮಹಾ ಸೂರ್ಯ ರೂಪಿಯು

334. ಆದಿದೇವಃ = ಮೊದಲನೆಯ ದೇವತೆಯು, ಎಲ್ಲಕ್ಕೂ ಆದಿ ಕಾರಣನಾದ ದೇವನು

335. ಪುರಂದರಃ = ತ್ರಿಪುರಾಸುರ ವಿಧ್ವಂಸಕನು,  ಇಂದ್ರ ರೂಪಿಯಾದವನು

336. ಅಶೋಕಃ = ಶೋಕರಹಿತನಾದವನು

337. ತಾರಣಃ = (ಸಂಸಾರ ಸಮುದ್ರವನ್ನು) ದಾಟಿಸುವವನು, ಎಲ್ಲ ದುರ್ಗಗಳನ್ನೂ ದಾಟಿಸುವವನು

338. ತಾರಃ = ಉಚ್ಚನಾದವನು, ಪ್ರಣವಸ್ವರೂಪಿಯು

339. ಶೂರಃ = ಶೂರನು

340. ಶೌರಿಃ = ಶೂರನಾದ ವಸುದೇವನ ಮಗನು, ಶೂರವಂಶದಲ್ಲಿ ಹುಟ್ಟಿದವನು

341. ಜನೇಶ್ವರಃ = ಎಲ್ಲ ಜಂತುಗಳಿಗೂ ಈಶ್ವರನಾದವನು

342. ಅನುಕೂಲಃ = (ಭಕ್ತರಿಗೆ) ಅನುಕೂಲನಾಗಿರುವವನು, ಸರ್ವಸುಹೃತ್ತಾದ್ದರಿಂದ ಎಲ್ಲರಿಗೂ ಅನುಕೂಲನಾದವನು

343. ಶತಾವರ್ತಃ = ಶಿಷ್ಟರಕ್ಷಣೆಗಾಗಿ ನೂರಾರು ಬಾರಿ ಅವತಾರ ಮಾಡುವವನು, ನೂರಾರು ಮಾಯಾವಿಲಾಸಗಳುಳ್ಳವನು, ಐಶ್ವರ್ಯ ವಿಲಾಸವುಳ್ಳವನು, ಅನುಗ್ರಹ ವಿಲಾಸವುಳ್ಳವನು, ನೂರಾರು ಸುಳಿಗಳುಳ್ಳ ಮಾಯೆಯುಳ್ಳವನು

344. ಪದ್ಮೀ = ಕೈಯಲ್ಲಿ ಲೀಲಾಪದ್ಮವುಳ್ಳವನು, ಶರೀರದಲ್ಲಿ ಮೂಲಾಧಾರಾದಿ ಪದ್ಮಗಳಲ್ಲಿ ವಾಶ ಮಾಡುವವನು

345. ಪದ್ಮನಿಭೇಕ್ಷಣಃ = ಕಮಲನೇತ್ರನು

346. ಪದ್ಮನಾಭಃ = ಸೃಷ್ಟಿಪದ್ಮವನ್ನು ನಾಭಿಯಂತೆ ಹೊಂದಿರುವವನು, ಪದ್ಮಸದೃಶವಾದ ಸುಳಿ-ನಾಭಿಯುಳ್ಳವನು

347. ಅರವಿಂದಾಕ್ಷಃ = ಕಮಲ ಸದೃಶವಾದ ಕಣ್ಣುಗಳುಳ್ಳವನು

348. ಪದ್ಮಗರ್ಭಃ = ಹೃದಯಪದ್ಮದ ಮಧ್ಯದಲ್ಲಿರುವವನು, ಮೂಲಾಧಾರಾದಿ ಪದ್ಮಗಳಲ್ಲಿ ಉಪಾಸನೆ ಮಾಡಲ್ಪಡುವವನು

349. ಶರೀರಭೃತ್ = ಶರೀರಿಯಾಗಿ ಅವತಾರ ಮಾಡುವವನು, ಶರೀರಗಳನ್ನು ಪೋಷಿಸುವವನು

350. ಮಹದ್ರ್ಧಿಃ = ವಿಭೂತಿಯುಳ್ಳವನು, ದೊಡ್ಡ ಅಭಿವೃದ್ಧಿಯುಳ್ಳವನು

351. ಋದ್ಧಃ = ಪ್ರಪಂಚ ರೂಪದಿಂದ ಬೆಳವಣಿಗೆಯನ್ನು ಹೊಂದಿರುವವನು, ಜ್ಞಾನಾದಿ ಗುಣಗಳಿಂದ ಸಮೃದ್ಧನಾದವನು, ತ್ರಿವಿಕ್ರಮ ರೂಪದಿಂದ ವೃದ್ಧಿ ಹೊಂದಿದವನು

352. ವೃದ್ಧಾತ್ಮಾ = ವಿಸ್ತಾರಗೊಂಡ ರೂಪವುಳ್ಳವನು, ಪುರಾತನವಾದ ಆತ್ಮವುಳ್ಳವನು, ಜ್ಞಾನವೃದ್ಧರಿಗೆ ಆತ್ಮರೂಪನು, ಜ್ಞಾನದಿಂದ ಪಕ್ವವಾದ ಆತ್ಮನು

353. ಮಹಾಕ್ಷಃ = ದೊಡ್ಡ ಕಣ್ಣುಗಳುಳ್ಳವನು, ದೊಡ್ಡ ಇಂದ್ರಿಯಗಳುಳ್ಳವನು, ಶರೀರ/ವಿಶ್ವ ರಥದ ದೊಡ್ಡ ಅಚ್ಚಿನ ರೂಪದಲ್ಲಿರುವವನು

354. ಗರುಡಧ್ವಜಃ = ಗರುಡನನ್ನು ಧ್ವಜದ ಚಿಹ್ನೆಯನ್ನಾಗಿ ಹೊಂದಿರುವವನು

355. ಅತುಲಃ = ಅನುಪಮನಾದವನು

356. ಶರಭಃ = ಭೀಕರ ಶರಭ ರೂಪಿಯಾದವನು

357. ಭೀಮಃ = ಸರ್ವಭಯಂಕರನು

358. ಸಮಯಜ್ಞಃ = ಕಾಲಜ್ಞನು, ಸಿದ್ಧಾಂತವನ್ನು/ಆಚಾರವನ್ನು/ಶಾಸ್ತ್ರಗಳನ್ನು ಬಲ್ಲವನು

359. ಹವಿರ್ಹರಿಃ = ಯಜ್ಞದ ಹವಿಸ್ಸುಗಳನ್ನು ಸ್ವೀಕರಿಸುವವನು, ಯಜ್ಞದ ಹವಿಸ್ಸಿನ ರೂಪದಲ್ಲಿರುವವನು, ಎಲ್ಲವನ್ನೂ ಸಂಹರಿಸುವವನು, ಎಲ್ಲರ ಮನಸ್ಸುಗಳನ್ನು-ಪಾಪಗಳನ್ನೂ ಅಪಹರಿಸುವವನು, ಹರಿದ್ವರ್ಣದವನು

360. ಸರ್ವಲಕ್ಷಣಲಕ್ಷಣ್ಯಃ = ಸಮಸ್ತ ಶುಭಲಕ್ಷಣಗಳಿಂದ ಕೂಡಿರುವ ಲಕ್ಷಣವಂತನು, ಜಗತ್ಕರ್ತೃತ್ವವೇ ಮುಂದಾ ಲಕ್ಷಣಗಳಿಂದ ಲಕ್ಷಿಸಲ್ಪಡುವವನು, ಲಕ್ಷಿಸಲು ಯೋಗ್ಯನಾದವನು

361. ಲಕ್ಷ್ಮೀವಾನ್ = (ಸಮಸ್ತ ಸನ್ಮಂಗಳಗಳಿಗೂ ದೇವತೆಯಾದ, ಜಗನ್ಮಾತೆಯಾದ) ಲಕ್ಷ್ಮಿಯ ನಿತ್ಯಯೋಗವುಳ್ಳವನು

362. ಸಮಿತಿಂಜಯಃ = ಎಲ್ಲ ಯುದ್ಧಗಳಲ್ಲಿಯೂ ಜಯಗಳಿಸುವವನು, ಸಭೆಗಳಲ್ಲಿ ಜಯಗಳಿಸುವವನು

363. ವಿಕ್ಷರಃ = ಅಳಿವಿಲ್ಲದವನು, ಸೋರದೇ ನಿರ್ವಿಕಾರನಾಗಿರುವವನು

364. ರೋಹಿತಃ = ಕೆಂಬಣ್ಣದವನು, ರುದ್ರಮೂರ್ತಿ ಸುದರ್ಶನ ಮೂರ್ತಿ ಮುಂತಾದ ರಕ್ತವರ್ಣದ ರೂಪಗಳಿಂದ ದರ್ಶನವನ್ನೀಯುವವನು

365. ಮಾರ್ಗಃ = (ಪುರುಷಾರ್ಥ ಪ್ರಾಪ್ತಿಗೆ) ದಾರಿಯಾಗಿರತಕ್ಕವನು, ಉಪಾಯನು

366. ಹೇತುಃ = ಸರ್ವಜಗತ್ಕಾರಣನು, ಜಗತ್ತಿನ ಉಪಾದಾನ ನಿಮಿತ್ತಗಳೆ ಮುಂತಾದ ಸರ್ವಕಾರಣವೂ ಆಗಿರುವವನು, ಎಲ್ಲ ವ್ಯಾಪಾರಗಳಿಗೂ ಮೂಲಕಾರಣನಾದವನು

367. ದಾಮೋದರಃ = ಲೋಕಗಳನ್ನು ಉದರದಲ್ಲಿ ಹೊಂದಿದವನು, ಇಂದ್ರಿಯನಿಗ್ರಹದಿಂದ ಕೂಡಿದ ಅಂತರಂಗವುಳ್ಳವನು, ಕೃಷ್ಣಾವತಾರದಲ್ಲಿ ಯಶೋದೆಯಿಂದ ಉದರಕ್ಕೆ ಕಟ್ಟಲ್ಪಟ್ಟ ಹಗ್ಗವುಳ್ಳವನು

368. ಸಹಃ = ಎಲ್ಲವನ್ನೂ ಭಕ್ತರ ಅಪರಾಧಗಳನ್ನೂ ಸಹಿಸುವವನು, ಶತ್ರುಗಳ ಆಕ್ರಮಣವನ್ನು ಸಹಿಸುವವನು

369. ಮಹೀಧರಃ = ಮಹಾಕೂರ್ಮನು, ಆದಿವರಾಹನು, ಶೇಷನಾಗ ರೂಪದಲ್ಲಿ ಭೂಮಿಯನ್ನು ಹೊತ್ತಿರುವವನು, ಪರ್ವತ ರೂಪಿಯಾದವನು

370. ಮಹಾಭಾಗಃ = ಮಹಾಭಾಗ್ಯಶಾಲಿಯು

371. ವೇಗವಾನ್ = ತೀವ್ರಗತಿಯುಳ್ಳವನು, ಮನಸ್ಸಿಗಿಂತಲೂ ಮಹಾವೇಗಶಾಲಿಯು

372. ಅಮಿತಾಶನಃ = ಮಹಾಭಕ್ಷನು, ಪ್ರಳಯಕಾಲದಲ್ಲಿ ವಿಶ್ವವನ್ನು ಭಕ್ಷಿಸುವವನು, ಬ್ರಹ್ಮ-ಕ್ಷತ್ರ ಮತ್ತು ಮೃತ್ಯುಗಳನ್ನು ತಿಂದು ಹಾಕುವವನು, ಅಪರಿಮಿತವಾದ ಆನಂದವೆಂಬ ಅನ್ನವನ್ನು ಭುಂಜಿಸುವವನು

373. ಉದ್ಭವಃ = ಜಗತ್ತಿಅನ್ ಉತ್ಪತ್ತಿಗೆ ಕಾರಣಕರ್ತನು, ಅವತಾರ ರೂಪದಲ್ಲಿ ಕಾಣಿಸಿಕೊಳ್ಳುವವನು

374. ಕ್ಷೋಭಣಃ = (ಪ್ರಳಯ/ಸೃಷ್ಟಿ ಕಾಲದಲ್ಲಿ) ಕ್ಷೋಭೆಯನ್ನುಂಟು ಮಾಡುವವನು

375. ದೇವಃ = ಸ್ವಾಮಿಯು, ಪ್ರಕಾಶ ಸ್ವರೂಪನು, ವಿಶ್ವಕ್ರೀಡೆಯುಳ್ಳವನು

376. ಶ್ರೀಗರ್ಭಃ = ಶ್ರೀದೇವಿಗೆ/ವಿಭೂತಿಗಳಿಗೆ ಗರ್ಭವಾಗಿರುವವನು

377. ಪರಮೇಶ್ವರಃ = ಈಶ್ವರರಿಗೂ ಮಹೇಶ್ವರನಾದವನು, ಪರಮನಾದ ಈಶ್ವರನು

378. ಕರಣಂ = ಪುರುಷಾರ್ಥಗಳ ಪ್ರಾಪ್ತಿಗೆ ಅಥವಾ ಜಗತ್ತಿನ ಉತ್ಪತ್ತಿ ಮುಂತಾದವುಗಳಿಗೆ ಶ್ರೇಷ್ಠತಮವಾದ ಸಾಧನನಾದವನು, ಸಾಧನತಮನು

379. ಕಾರಣಂ = ಜಗತ್ತಿಗೆ ಕಾರಣನಾಗಿರುವವನು

380. ಕರ್ತಾ = ಸ್ವತಂತ್ರನಾದ ಕರ್ತೃವು, ಬಂಧನವನ್ನು ಕತ್ತರಿಸುವವನು

381. ವಿಕರ್ತಾ = ವಿವಿಧವಾಗಿ/ವಿಶೇಷವಾಗಿ/ವಿಕಾರವಾಗಿ ಮಾಡುವವನು, (ತಾನು ಮಾಡಿದ್ದನ್ನು ತಾನೇ) ಧ್ವಂಸ ಮಾಡುವವನು

382. ಗಹನಃ = ತಿಳಿಯಲು ಅಸಾಧ್ಯನಾಗಿರುವವನು

383. ಗುಹಃ = ಗೂಢವಾಗಿರುವವನು, ಮುಚ್ಚಿಕೊಂಡಿರುವವನು, ಮುಚ್ಚುವವನು

384. ವ್ಯವಸಾಯಃ = ನಿಶ್ಚಯರೂಪಿಯು, ಪ್ರಯತ್ನಶೀಲನು, ಜ್ಞಾನಮಾತ್ರ ಸ್ವರೂಪನು

385. ವ್ಯವಸ್ಥಾನಃ = ಯಾವನಲ್ಲಿ ಎಲ್ಲವೂ ವ್ಯವಸ್ಥಿತವಾಗಿರುವುದೋ ಅವನು

386. ಸಂಸ್ಥಾನಃ = ಶ್ರೇಷ್ಠವಾದ ನೆಲೆಯಾಗಿರುವವನು, ಎಲ್ಲಕ್ಕೂ ಲಯಸ್ಥಾನನಾಗಿರುವವನು

387. ಸ್ಥಾನದಃ = ಪ್ರಾಣಿಗಳಿಗೆ ಕರ್ಮಾನುಗುಣವಾದ ಸ್ಥಾನಗಳನ್ನು ಕೊಡುವವನು

388. ಧ್ರುವಃ = ಸ್ಥಿರನಾದವನು, ಅಳಿವಿಲ್ಲದವನು, ಧ್ರುವ ನಕ್ಷತ್ರ ರೂಪಿಯು

389. ಪರದ್ರ್ಧಿಃ = ಶ್ರೇಷ್ಠವಾದ ಸಂಪತ್ತುಳ್ಳವನು

390. ಪರಮಸ್ಪಷ್ಟಃ = ಜ್ಞಾನಿಗಳಿಗೆ ಅತ್ಯಂತ ಸ್ಪಷ್ಟನಾಗಿರುವವನು, ಶ್ರೇಷ್ಠನೂ ಸ್ಪಷ್ಟನೂ ಆಗಿರುವವನು

391. ತುಷ್ಟಃ = ಪರಮ ಸಂತೋಷವುಳ್ಳವನು

392. ಪುಷ್ಟಃ = ನಿತ್ಯಪುಷ್ಟಿ ಉಳ್ಳವನು, ಪೂರ್ಣನಾದವನು

393. ಶುಭೇಕ್ಷಣಃ = ಶುಭಕರವಾದ ದರ್ಶನ/ಕಣ್ಣುಗಳನ್ನು ಉಳ್ಳವನು

394. ರಾಮಃ = ಅಭಿರಾಮನಾದವನು, ಯೋಗಿಗಳಲ್ಲಿ ರಮಿಸುವವನು, ಯೋಗಿಗಳು ಯಾವನಲ್ಲಿ ರಮಿಸುತ್ತಾರೋ ಅವನು, ಶ್ರೀರಾಮ ಸ್ವರೂಪಿಯು, ವಿಶ್ವೇಶ್ವರನು

395. ವಿರಾಮಃ = ಎಲ್ಲ ಕರ್ಮಗಳನ್ನೂ ನಾಶಹೊಂದಿಸುವವನು, ವಿಶ್ರಾಂತಿ ಧಾಮನು

396. ವಿರತಃ = (ವಿಷಯಗಳಲ್ಲಿ) ಆಸಕ್ತಿ ಇಲ್ಲದವನು, ನಿರ್ಲಿಪ್ತನು

397. ಮಾರ್ಗಃ = ಪರಮ ಶ್ರೇಯಸ್ಸಿಗೆ ಏಕಮಾತ್ರ ದಾರಿಯಾಗಿರುವವನು

398. ನೇಯಃ = ಯೋಗಮಾರ್ಗದಲ್ಲಿ ಸದಾ ತನ್ನೊಡನೆ ಒಯ್ಯಲು ಯೋಗ್ಯನಾದವನು

399. ನಯಃ = ನೀತಿ ರೂಪನಾದವನು, ನಾಯಕನು

400. ಆನಯಃ = ನೀತಿ ನಿಯಮಗಳಿಲ್ಲದವನು, ನಾಯಕನಿಲ್ಲದವನು

401. ವೀರಃ = ಪರಾಕ್ರಮಶಾಲಿಯು, ಸತ್ಯ ಪರಾಕ್ರಮನು

402. ಶಕ್ತಿಮತಾಂ ಶ್ರೇಷ್ಠಃ = ಶಕ್ತಿನಿಧಿಯಾದುದರಿಂದ ದೇವತೆಗಳೇ ಮುಂತಾದವರಿಗಿಂತಲೂ ಅತಿಶಯವಾದ ಶಕ್ತಿಯುಳ್ಳವನು

403. ಧರ್ಮಃ = ಧರ್ಮ ರೂಪಿಯಾದವನು, ಸಿದ್ಧಧರ್ಮ ಸ್ವರೂಪಿಯು, ಎಲ್ಲವನ್ನೂ (ಎಲ್ಲ ಭೂತಗಳನ್ನೂ) ಧರಿಸಿದವನು

404. ಧರ್ಮವಿದುತ್ತಮಃ = ಧರ್ಮವನ್ನು ಬಲ್ಲವರಲ್ಲಿ ಶ್ರೇಷ್ಠನಾದವನು

405. ವೈಕುಂಠಃ = ಕುಂಠಿತನಾಗದೇ ಇರುವವನು, ತಡೆಯಿಲ್ಲದ ಗತಿಯುಳ್ಳವನು, ವೈಕುಂಠವೆಂಬ ಹೆಸರಿನ ದಿವ್ಯಧಾಮವುಳ್ಳವನು

406. ಪುರುಷಃ = ಪುರದಲ್ಲಿ ಶಯನಿಸುತ್ತಿರುವ ಆತ್ಮಚೈತನ್ಯ ಸ್ವರೂಪನು, ಪೂರ್ಣನಾಗಿರುವವನು

407. ಪ್ರಾಣಃ = ಸಂಜೀವಕನು, ಪ್ರಾಣ ರೂಪಿಯಾಗಿ ವ್ಯಾಪಾರ ಮಾಡುವವನು

408. ಪ್ರಾಣದಃ = ಪ್ರಾಣಗಳನ್ನು ಅನುಗ್ರಹಿಸುವವನು

409. ಪ್ರಣವಃ = ಓಂಕಾರ ಸ್ವರೂಪಿಯು, ಪ್ರಾಣಗಳನ್ನು ನಮಸ್ಕಾರ ಮಾಡಿಸುವವನು

410. ಪೃಥುಃ = ವಿಸ್ತಾರವಾಗಿರುವವನು

411. ಹಿರಣ್ಯಗರ್ಭಃ = ಹಿರಣ್ಮಯವಾದ ಅಂಡದ ಉತ್ಪತ್ತಿಗೆ ಕಾರಣನಾದವನು, ಸೃಷ್ಟಿಕರ್ತನು, ವೇದಗಳಿಂದ ಸ್ತುತಿಸಲ್ಪಡುವವನು, ಯೋಗಿಗಳಿಂದ ನಿತ್ಯವೂ ಪೂಜಿಸಲ್ಪಡುವವನು

412. ಶತ್ರುಘ್ನಃ = (ಧರ್ಮಕ್ಕೆ ವಿರೋಧಿಗಳಾದ) ಶತ್ರುಗಳನ್ನು ಸಂಹರಿಸುವವನು

413. ವ್ಯಾಪ್ತಃ = ಎಲ್ಲವನ್ನೂ ವ್ಯಾಪಿಸಿರುವವನು

414. ವಾಯುಃ = ವಾಯು ದೇವತಾ ಸ್ವರೂಪಿಯು

415. ಅಧೋಕ್ಷಜಃ = ಇಂದ್ರಿಯಗಳನ್ನು ಕೆಳಕ್ಕೆ ತಳ್ಳಿ ಯೋಗದಲ್ಲಿರುವವನು, ಪೃಥ್ವೀ ಮತ್ತು ಅಂತರಿಕ್ಷಗಳನ್ನು ಧರಿಸಿರುವವನು

416. ಋತುಃ = ಕಾಲವಿಭಾಗವಾದ ಋತು ರೂಪಿಯಾದವನು, ಕಾಲರೂಪಿಯು

417. ಸುದರ್ಶನಃ = ಶೋಭನವಾದ ದರ್ಶನ ಉಳ್ಳವನು, ಸುದರ್ಶನಚಕ್ರ ರೂಪಿಯು

418. ಕಾಲಃ = ಎಲ್ಲರನ್ನೂ ಗಣಿಸುವ/ಒಂದುಗೂಡಿಸುವ ಕಾಲರೂಪಿಯು, ಮೃತ್ಯು ಸ್ವರೂಪಿಯು

419. ಪರಮೇಷ್ಠೀ = ಶ್ರೇಷ್ಠತಮವಾದ ತನ್ನ ಸ್ಥಾನದಲ್ಲಿ ನೆಲೆಗೊಂಡಿರುವವನು, ಬ್ರಹ್ಮಸ್ವರೂಪಿಯು

420. ಪರಿಗ್ರಹಃ = ಜ್ಞಾನಿಗಳಿಂದ ಸ್ವೀಕರಿಸಲ್ಪಡುವವನು, ಎಲ್ಲ ಕಡೆಗಳಲ್ಲಿಯೂ ಗ್ರಹಿಸಲ್ಪಡುವವನು

421. ಉಗ್ರಃ = ಪ್ರಚಂಡನಾದವನು, ದುಷ್ಟರಿಗೆ ಭೀಕರನಾದವನು

422. ಸಂವತ್ಸರಃ = ಕಾಲ/ಕಾಲಾವಯವ ರೂಪಿಯು, ಕಾಲ ಪ್ರವರ್ತಕನು

423. ದಕ್ಷಃ = ಸಮರ್ಥನಾದವನು, ದಕ್ಷ ಪ್ರಜಾಪತಿ ಸ್ವರೂಪನು

424. ವಿಶ್ರಾಮಃ = ಶ್ರಮರಹಿತನು, ಸಂಸಾರ ಸಂತಪ್ತರಿಗೆ ವಿಶ್ರಾಂತಿ ಧಾಮನಾಗಿರುವವನು

425. ವಿಶ್ವದಕ್ಷಿಣಃ = ಎಲ್ಲರಿಗೂ ಅನುಕೂಲನಾಗಿರುವವನು (ಉದಾರಿಯಾಗಿರುವವನು), ಎಲ್ಲರಿಗಿಂತಲೂ ಉದಾರನೂ ಸರಳನೂ ಆಗಿರುವವನು, ಇಡೀ ವಿಶ್ವವನ್ನೇ ದಕ್ಷಿಣೆಯ ರೂಪದಲ್ಲಿ ಪಡೆದವನು

426. ವಿಸ್ತಾರಃ = ಜಗತ್ತಿನ ರೂಪದಲ್ಲಿ ವಿಸ್ತಾರಗೊಂಡವನು, ಜಗತ್ತನ್ನು ವಿಸ್ತಾರಪಡಿಸುವವನು

427. ಸ್ಥಾವರಸ್ಥಾಣುಃ = ಸ್ಥಿತಿಶೀಲನಾದವನು, ನಿಶ್ಚಲವಾದ ಕಂಬದಂತೆ ಇರುವವನು, ರುದ್ರ ರೂಪಿಯು

428. ಪ್ರಮಾಣಂ = ಶ್ರೇಷ್ಠವಾದ ಅಳತೆಯು, ವ್ಯವಸ್ಥಾಪಕನು, ಅನುಭವ ಜ್ಞಾನ/ರೂಪ-ದಲ್ಲಿ ಪ್ರಮಾಣನು

429. ಬೀಜಮವ್ಯಯಂ = ಜಗತ್ತಿಗೆ ಅವಿನಾಶಿಯಾದ ಕಾರಣ ಸ್ವರೂಪಿಯು

430. ಅರ್ಥಃ = ಪದಾರ್ಥ, ಪ್ರಯೋಜನ, ಧನ, ನಿವೃತ್ತಿ ಮತ್ತು ಎಲ್ಲ ಶಬ್ದಗಳಿಗೂ ಅಭಿಧೇಯಮಾದವನು

431. ಅನರ್ಥಃ = ಅವಾಪ್ತಸಮಸ್ತ ಕಾಮನು, ಎಲ್ಲ ಪ್ರಯೋಜನಗಳೂ ಸ್ವತಃ ಸಿದ್ಧವಾಗಿರುವುದರಿಂದ ಬೇರೆ ಯಾವ ಪ್ರಯೋಜನವಿಲ್ಲದವನು, ಯಾವ ಪ್ರಯೋಜನದ ಅಪೇಕ್ಷೆಯಿಲ್ಲದವನು

432. ಮಹಾಕೋಶಃ = ಜ್ಞಾನಾನಂದಾದಿಗಳಿಗೆ ದೊಡ್ಡ ಭಂಡಾರವಾಗಿರುವವನು, ಅನ್ನಮಯವೇ ಮುಂತಾದ ದೊಡ್ಡ ಕೋಶಗಳುಳ್ಳವನು

433. ಮಹಾಭೋಗಃ = ಪರಮಸುಖ ಭೋಗವುಳ್ಳವನು

434. ಮಹಾಧನಃ = ಯೋಗಿಗಳಿಗೆ ಪರಮಧನವಾದವನು

435. ಅನಿರ್ವಿಣ್ಣಃ = ಪೂರ್ಣಕಾಮನು ಆನಂದಮಯನೂ ಆಗಿರುವುದರಿಂದ ನಿರ್ವೇದ/ಖೇದ ಇಲ್ಲದವನು

436. ಸ್ಥವಿಷ್ಠಃ = ಸ್ಥೂಲತಮನಾದವನು

437. ಅಭೂಃ = ಜನ್ಮ ರಹಿತನು

438. ಧರ್ಮಯೂಪಃ = ಧರ್ಮಕ್ಕೆ ಯಜ್ಞಸ್ತಂಭನಾದವನು

439. ಮಹಾಮಖಃ = ಶ್ರೇಷ್ಠ ಯಜ್ಞಸ್ವರೂಪಿಯು

440. ನಕ್ಷತ್ರನೇಮಿಃ = ನಕ್ಷತ್ರಗಳನ್ನೇ ರಥದ ಪಟ್ಟಿಯನ್ನಾಗಿ ಹೊಂದಿರುವವನು

441. ನಕ್ಷತ್ರೀ = ನಕ್ಷತ್ರವುಳ್ಳವನು, ನಕ್ಷತ್ರಗಳ ಅಧಿಪತಿಯು

442. ಕ್ಷಮಃ = ಸಮರ್ಥನಾದವನು, ಕ್ಷಮಾಗುಣವುಳ್ಳವನು

443. ಕ್ಷಾಮಃ = ಸ್ಥೂಲ-ಸೂಕ್ಷ್ಮಗಳನ್ನು ಬಿಟ್ಟು ತೆಳ್ಳಗಿರುವವನು

444. ಸಮೀಹನಃ = ಒಳ್ಳೆಯ ಚೇಷ್ಟೆಯುಳ್ಳವನು

445. ಯಜ್ಞಃ = ಯಜ್ಞರೂಪಿಯು, ಯಜ್ಞವೆಂಬ ಹೆಸರಿನಿಂದ ಅವತಾರ ಮಾಡಿದವನು

446. ಇಜ್ಯಃ = ಪೂಜಿಸಲ್ಪಡಬೇಕಾದ ದೇವನು

447. ಮಹೇಜ್ಯಃ = ಪೂಜಿಸಲ್ಪಡಬೇಕಾದವರಲ್ಲಿ, ಪೂಜೆಯುಳ್ಳವನು

448. ಕ್ರತುಃ = ಕ್ರತುವೆಂಬ ಯಜ್ಞ ರೂಪಿಯಾದವನು

449. ಸತ್ರಂ = ಸತ್ರಯಜ್ಞ ರೂಪಿಯಾದವನು

450. ಸತಾಂಗತಿಃ = ಸಂತರಿಗೆ/ಬ್ರಹ್ಮಜ್ಞಾನಿಗಳಿಗೆ ಏಕಮಾತ್ರ ಗತಿಯಾದವನು

451. ಸರ್ವದರ್ಶೀ = ಸರ್ವಸಾಕ್ಷಿಯಾದವನು, ಎಲ್ಲರನ್ನೂ ಯಾವಾಗಲೂ ಎಲ್ಲೆಲ್ಲೂ ನೋಡುತ್ತಿರುವವನು, ಎಲ್ಲರ ಪುಣ್ಯ-ಪಾಪ ಕರ್ಮಗಳನ್ನೂ ನೋಡುತ್ತಿರುವವನು

452. ವಿಮುಕ್ತಾತ್ಮಾ/ನಿವೃತ್ತಾತ್ಮಾ = ಸ್ವಾಭಾವಿಕವಾಗಿಯೇ ಪರಿಪೂರ್ಣನಾಗಿರುವ ಪರಮಾತ್ಮನು, ವಿಷಯಗಳಿಂದ ಪೂರ್ಣ ನಿವೃತ್ತಿಯನ್ನು ಹೊಂದಿರುವ ವೈರ್ಗಾಗ್ಯಯುಕ್ತ ಆತ್ಮನು

453. ಸರ್ವಜ್ಞಃ = ಸ್ವಾಭಾವಿಕವಾಗಿಯೇ ಯಾವಾಗಲೂ ಎಲ್ಲವನ್ನೂ ತಿಳಿದವನು

454. ಜ್ಞಾನಮುತ್ತಮಂ = ಸರ್ವೋತ್ತಮ ಜ್ಞಾನರೂಪಿಯಾದವನು

455. ಸುವ್ರತಃ = ಶುಭವಾದ ವ್ರತವುಳ್ಳವನು

456. ಸುಮುಖಃ = ಪ್ರಸನ್ನವೂ, ಪ್ರಶಾಂತವೂ, ಸುಂದರವೂ ಆದ ಯೋಗಿಧ್ಯೇಯವಾದ ಮುಖವುಳ್ಳವನು

457. ಸೂಕ್ಷ್ಮಃ = ಸೂಕ್ಷ್ಮನಾದವನು

458. ಸುಘೋಷಃ = ಶ್ರೇಷ್ಠವಾದ ಪ್ರಣವ ಘೋಷವುಳ್ಳವನು, ವೇದಘೋಷವುಳ್ಳವನು, ಯೋಗಿಗಳ ಅಂತಃಶ್ರವಣಕ್ಕೆ ಗೋಚರವಾದ ದಶವಿಧ ನಾದಗಳುಳ್ಳವನು, ಶ್ರೇಷ್ಠವಾದ ಪಾಂಚಜನ್ಯ ಘೋಷವುಳ್ಳವನು

459. ಸುಖದಃ = ಸುಖವನ್ನು ದಯಪಾಲಿಸುವವನು

460. ಸುಹೃತ್ = ಎಲ್ಲ ಜೀವಿಗಳಿಗೂ ಸ್ವಾಭಾವಿಕವಾಗಿ ಮಿತ್ರನಾಗಿರುವವನು

461. ಮನೋಹರಃ = (ತನ್ನ ಸೌಂದರ್ಯ ಕಲ್ಯಾಣಾದಿ ಗುಣಗಳಿಂದ ಎಲ್ಲರ) ಮನಸ್ಸನ್ನು ಅಪಹರಿಸುವವನು

462. ಜಿತಕ್ರೋಧಃ = ಸ್ವಾಭಾವಿಕವಾಗಿಯೇ ಕ್ರೋಧವನ್ನು ಜಯಿಸಿದವನು

463. ವೀರಬಾಹುಃ = ಪರಾಕ್ರಮಯುಕ್ತವಾದ ಬಾಹುಗಳುಳ್ಳವನು

464. ವಿದಾರಣಃ = ಅಜ್ಞಾನವನ್ನೂ ಮತ್ತು ಧರ್ಮದ್ವೇಷಿಗಳನ್ನೂ ವಿಶೇಷವಾಗಿ ಸೀಳಿ ಹಾಕಿದವನು, ವೀರಬಾಹುವಾದ ನೃಸಿಂಹರೂಪಿಯಾಗಿ ಹಿರಣ್ಯಕಶಿಪುವಿನ ಎದೆಯನ್ನು ಸೀಳಿ ಹಾಕಿದವನು

465. ಸ್ವಾಪನಃ = ಜೀವಿಗಳಿಗೆ ನಿದ್ರೆಯನ್ನು ಉಂಟುಮಾಡುವವನು, ಪ್ರಳಯಾನಂತರ ಸೃಷ್ಟಿಯವರೆಗೂ ಜೀವಿಗಳನ್ನು ನಿದ್ರಿಸುವಂತೆ ಮಾಡುವವನು, ಜೀವಿಗಳು ಅಜ್ಞಾನದಲ್ಲಿ ನಿದ್ರಿಸುವಂತೆ ಮಾಡುವವನು

466. ಸ್ವವಶಃ = ಸ್ವತಂತ್ರನಾದವನು, ಯಾವ ವಿಷಯದಲ್ಲಿಯೂ ಪರಾಧೀನತೆಯಿಲ್ಲದವನು

467. ವ್ಯಾಪೀ = ಸರ್ವವ್ಯಾಪಕನು

468. ನೈಕಾತ್ಮಾ = ಅನಂತರೂಪ ಉಳ್ಳವನು

469. ನೈಕಕರ್ಮಕೃತ್ = ಅನಂತ ಕರ್ಮಗಳುಳ್ಳವನು

470. ವತ್ಸರಃ = ಸಂವತ್ಸರ ರೂಪಿಯಾದವನು, ಕರುಗಳನ್ನು ರಕ್ಷಿಸುವವನು

471. ವತ್ಸಲಃ = ಸ್ನೇಹವುಳ್ಳವನು, ಹಸುವು ತನ್ನ ಕರುವಿನಲ್ಲಿ ಹೇಗೋ ಹಾಗೆ ಶರಣಾಗತರಲ್ಲಿ ಸ್ನೇಹವುಳ್ಳವನು

472. ವತ್ಸೀ = ಜೀವಗಳನ್ನು ಕರುಗಳ ರೂಪದಲ್ಲಿ ಹೊಂದಿರುವವನು, ಶ್ರೀವತ್ಸ ಲಾಂಛನವುಳ್ಳವನು

473. ರತ್ನಗರ್ಭಃ = ರತ್ನ ಗರ್ಭನು, ಸಾಲಿಗ್ರಾಮ ಮೂರ್ತಿಯು, ಸಮುದ್ರ ರೂಪಿಯಾದವನು, ಅನಂತವಾದ ಧನ ಸಂಪತ್ತುಳ್ಳವನು

474. ಧನೇಶ್ವರಃ = ಧನಗಳಿಗೆ ಸ್ವಾಮಿಯಾದವನು, ಕುಬೇರ ರೂಪಿಯು

475. ಧರ್ಮಗುಪ್ = ಧರ್ಮವನ್ನು ರಕ್ಷಿಸುವವನು

476. ಧರ್ಮಕೃತ್ = ಧರ್ಮವನ್ನು ಆಚರಿಸುವವನು, ಧರ್ಮ ನಿಯಮಗಳನ್ನು ಮಾಡಿದವನು

477. ಧರ್ಮೀ = ಧರ್ಮಗಳನ್ನು ಧರಿಸಿರುವವನು

478. ಸತ್ = ಸತ್ಯ ಸ್ವರೂಪನು, ಸತ್ತಾಮಾತ್ರನು, ವಿಕಾರ ರಹಿತನು, ವ್ಯಕ್ತರೂಪಿಯು

479. ಅಸತ್ = ವಿಕಾರಹೊಂದುವ ಕಾರ್ಯ ಜಗತ್ತನ್ನು ಶರೀರವನ್ನಾಗಿ ಹೊಂದಿರುವವನು, ಅವ್ಯಕ್ತರೂಪಿಯು

480. ಕ್ಷರಂ = ನಾಶ/ವಿಕಾರ ಹೊಂದುವ ಸರ್ವಭೂತ ರೂಪಿಯಾದವನು

481. ಅಕ್ಷರಂ = ವಿಕಾರ ರಹಿತನಾದ ಪರಬ್ರಹ್ಮನು, ಕೂಟಸ್ಥನಾದ ಜೀವ ಅಥವಾ ಮೂಲ ಪ್ರಕೃತಿ ಸ್ವರೂಪನು

482. ಅವಿಜ್ಞಾತಾ = ವಿಜ್ಞಾನ (ವಿವಿಧ/ಸವಿಕಾರ ಜ್ಞಾನ) ಇಲ್ಲದವನು

483. ಸಹಸ್ರಾಂಶುಃ = ಸಹಸ್ರ ಕಿರಣನಾದ ಆದಿತ್ಯ ರೂಪನು

484. ವಿಧಾತಾ = ಸೃಷ್ಟಿಕರ್ತನು, ವಿವಿಧವಾಗಿ/ವಿಶೇಷವಾಗಿ ಧಾರಣ-ಪೋಷಣಗಳನ್ನು ಮಾಡುವವನು

485. ಕೃತಲಕ್ಷಣಃ = ಲೋಕ ಮತ್ತು ಶಾಸ್ತ್ರಗಳಿಗೆ ಸಂಬಂಧಿಸಿದ ಲಕ್ಷಣಗಳನ್ನು ಮಾಡಿರುವವನು

486. ಗಭಸ್ತಿನೇಮಿಃ = ಕಿರಣಗಳ ಪಟ್ಟಿಯುಳ್ಳವನು, ಸೂರ್ಯ ರೂಪಿಯು, ಸಹಸ್ರಾರು ಸುದರ್ಶನಚಕ್ರ ರೂಪಿಯು

487. ಸತ್ತ್ವಸ್ಥಃ = ಸತ್ತ್ವಗುಣದಲ್ಲಿ ಇರುವವನು

488. ಸಿಂಹಃ = ನರಸಿಂಹರೂಪಿಯು, ಶ್ರೇಷ್ಠನು, ಪರಾಕ್ರಮಿಯು

489. ಭೂತಮಹೇಶ್ವರಃ = ಎಲ್ಲ ಪ್ರಾಣಿಗಳಿಗೂ ಸತ್ಯವಾಗಿ ಪರಮೇಶ್ವರನಾಗಿರುವವನು

490. ಆದಿದೇವಃ = ಎಲ್ಲ ಪ್ರಾಣಿಗಳಿಗೂ ಆದಿಯಾಗಿರುವ ಸ್ವಾಮಿಯು

491. ಮಹಾದೇವಃ = ದೊಡ್ಡ ದೇವನು, ಪೂಜ್ಯನಾದ ದೇವನು

492. ದೇವೇಶಃ = ದೇವತೆಗಳಿಗೆ ಅಧಿಪತಿಯಾದವನು

493. ದೇವಭೃದ್ಗುರುಃ = ದೇವತೆಗಳನ್ನು ಭರಿಸುವವನು ಮತ್ತು ಎಲ್ಲ ವಿದ್ಯೆಗಳಿಗೂ ಆದಿಗುರು ಆಗಿರುವವನು

494. ಉತ್ತರಃ = ಎಲ್ಲರಿಗಿಂತಲೂ ಶ್ರೇಷ್ಠನಾದವನು, ಎಲ್ಲವನ್ನೂ ದಾಟಿದವನು/ದಾಟಿಸುವವನು

495. ಗೋಪತಿಃ = ಧೇನುಗಳಿಗೆ/ಭೂಮಿಗೆ/ವಾಣಿಗೆ ಸ್ವಾಮಿಯು, ಸೂರ್ಯರೂಪಿಯು

496. ಗೋಪ್ತಾ = ಸಂರಕ್ಷಕನು

497. ಜ್ಞಾನಗಮ್ಯಃ = ಜ್ಞಾನದಿಂದ ಹೊಂದಲ್ಪಡತಕ್ಕವನು

498. ಪುರಾತನಃ = ಅತ್ಯಂತ ಪ್ರಾಚೀನನಾದವನು

499. ಶರೀರಭೂತಭೃತ್ = ಶರೀರರೂಪನಾಗಿ ಎಲ್ಲ ಭೂತಗಳನ್ನೂ ಧರಿಸುವವನು, ಶರೀರಕ್ಕೆ ಕಾರಣನಾದ ಪಂಚಮಹಾ ಭೂತಗಳನ್ನು ಧರಿಸಿರುವವನು

500. ಭೋಕ್ತಾ = ಭಕ್ತರು ಸಮರ್ಪಿಸಿದ ಹವ್ಯ-ಕವ್ಯಾದಿಗಳನ್ನು ಭೋಗಿಸುವವನು, ಪರಮಾನಂದವನ್ನು ಅನುಭವಿಸುವವನು, ಸರ್ವ ಭಕ್ಷಕನು

501. ಕಪೀಂದ್ರಃ = ಕಪಿ ರೂಪದಲ್ಲಿ ತಾಳಿದ್ದ ದೇವತೆಗಳಿಗೆ ಸ್ವಾಮಿಯಾದವನು

502. ಭೂರಿದಕ್ಷಿಣಃ = ನಾನಾ ಯಜ್ಞಗಳಲ್ಲಿ ಪುಷ್ಕಳವಾದ ದಕ್ಷಿಣೆಗಳನ್ನು ಪಡೆಯುವವನು, ಯಜ್ಞಗಳನ್ನು ಆಚರಿಸಿದಾಗ ಪುಷ್ಕಳವಾದ ದಕ್ಷಿಣೆಗಳನ್ನು ಕೊಡುವವನು

503. ಸೋಮಪಃ = ಸೋಮಯಜ್ಞದಲ್ಲಿ ಅರ್ಪಿಸಲ್ಪಡುವ ಸೋಮವನ್ನು ದೇವತೆಯಾಗಿ ಸ್ವೀಕರಿಸುವವನು, ಸೋಮಪಾನ ಮಾದುವ ಯಜಮಾನ ರೂಪಿಯಾದವನು

504. ಅಮೃತಪಃ = ಪರಮಾನಂದಾಮೃತವನ್ನು ಪಾನ ಮಾಡುತ್ತಿರುವವನು

505. ಸೋಮಃ = ಓಷಧಿಗಳನ್ನು ಪೋಷಿಸುವ ಸೋಮದೇವತಾ ರೂಪನು, ಉಮಾ ಸಹಿತನಾದ ಶಿವ ಸ್ವರೂಪಿಯು, ಸೋಮ/ಅಮೃತ ಸ್ವರೂಪಿಯು, ಓಂಕಾರ ಸಹಿತನಾದವನು

506. ಪುರುಜಿತ್ = ಬಹಳ ಪ್ರಮಾಣದಲ್ಲಿ (ಬಹು-ಮಂದಿಯನ್ನು) ಗೆಲ್ಲುವವನು

507. ಪುರುಸತ್ತಮಃ = ಬಹುರೂಪಿಯೂ, ಶ್ರೇಷ್ಠನೂ ಆಗಿರುವವನು

508. ವಿನಯಃ = ಗುರು-ಹಿರಿಯರಲ್ಲಿ ವಿನಯಶಾಲಿಯಾದವನು, ಅನುಶಾಸನ ರೂಪನಾದವನು, ದಂಡ ರೂಪಿಯು, ದುಷ್ಟರನ್ನು ದಂಡಿಸುವವನು

509. ಜಯಃ = ಜಯ ರೂಪಿಯಾದವನು, ಎಲ್ಲರನ್ನೂ ಜಯಿಸುವವನು

510. ಸತ್ಯಸಂಧಃ = ಸತ್ಯವಾದ ಸಂಕಲ್ಪ ಮತ್ತು ಪ್ರತಿಜ್ಞೆಗಳುಳ್ಳವನು

511. ದಾಶಾರ್ಹಃ = ದಶಾರ್ಹ ಕುಲದಲ್ಲಿ ಹುಟ್ಟಿದವನು, ಶ್ರೀಕೃಷ್ಣ ರೂಪಿಯು

512. ಸಾತ್ವತಾಂ ಪತಿಃ = ಸಾತ್ವತ ವಂಶದವರಾದ ಯಾದವರಿಗೆ, ಸಾತ್ವತ ಪಂಥದವರಿಗೆ ಸ್ವಾಮಿಯಾದವನು

513. ಜೀವಃ = ಚೈತನ್ಯ ರೂಪಿಯಾದವನು, ಪ್ರಾಣಧಾರಕನು

514. ವಿನಯಿತಾ ಸಾಕ್ಷೀ = ವಿನಯವನ್ನುಂಟು ಮಾಡುವವನು, ಎಲ್ಲವನ್ನೂ ಸಾಕ್ಷಾತ್ತಾಗಿ ನೋಡುವವನು

515. ಮುಕುಂದಃ = ಮುಕ್ತಿಯನ್ನು ಕೊಡುವವನು

516. ಅಮಿತವಿಕ್ರಮಃ = ಅಳೆಯಲಾಗದ ಶೌರ್ಯವುಳ್ಳವನು, ಮೂರು ದೊಡ್ಡ ಪಾದವಿನ್ಯಾಸಗಳುಳ್ಳವನು

517. ಅಂಭೋನಿಧಿಃ = ಸಾಗರ ಸ್ವರೂಪಿಯಾದವನು, ಸಾಗರದಂತೆ ಇರುವವನು

518. ಅನಂತಾತ್ಮಾ = ಪರಿಚ್ಛೇದರಹಿತವಾದ ಸ್ವರೂಪ ಉಳ್ಳವನು

519. ಮಹೋದಧಿಶಯಃ = ಪ್ರಳಯ ಸಮುದ್ರದಲ್ಲಿ ಪವಡಿಸಿರುವವನು, ಪಾಲ್ಗಡಲಿನಲ್ಲಿ ಪವಡಿಸಿರುವವನು

520. ಅಂತಕಃ = ಮಹಾ ಪ್ರಳಯ ಕಾಲದಲ್ಲಿ ಸರ್ವಭೂತಗಳನ್ನೂ ಸಂಹರಿಸುವವನು, ಶತ್ರು ಸಂಹಾರಕನು

521. ಅಜಃ = ಹುಟ್ಟಿಲ್ಲದವನು, ಬ್ರಹ್ಮರೂಪಿಯು

522. ಮಹಾರ್ಹಃ = ಅತ್ಯಂತ ಯೋಗ್ಯನಾದವನು, ಪೂಜೆಗೆ ಯೋಗ್ಯನಾದವನು

523. ಸ್ವಾಭಾವ್ಯಃ = ಸ್ವಾಭಾವಿಕನಾದವನು, ನಿತ್ಯ ಸಿದ್ಧನು

524. ಜಿತಾಮಿತ್ರಃ = ಶತ್ರುಗಳನ್ನು ಜಯಿಸಿದವನು

525. ಪ್ರಮೋದನಃ = ಪರಮಾನಂದವನ್ನು ಉಂಟುಮಾಡುವವನು

526. ಆನಂದಃ = ಆನಂದ ಸ್ವರೂಪಿಯು

527. ನಂದನಃ = ಆನಂದಗೊಳಿಸುವವನು

528. ನಂದಃ = ಸಮೃದ್ಧನಾದವನು

529. ಸತ್ಯಧರ್ಮಾ = ಸತ್ಯವಾದ ಧರ್ಮ/ನಿಯಮ-ಗಳುಳ್ಳವನು

530. ತ್ರಿವಿಕ್ರಮಃ = ಮೂರು ಲೋಕಗಳನ್ನೂ ಅಳೆದ ಮೂರು ಮಹತ್ತಾದ ಪಾದವಿನ್ಯಾಸಗಳುಳ್ಳವನು

531. ಮಹರ್ಷಿಃ ಕಪಿಲಾಚಾರ್ಯಃ = ನಾರಾಯಣ ಮಹರ್ಷಿಯು, ಸಾಂಖ್ಯಾಚಾರ್ಯನಾದ ಕಪಿಲ ಮಹರ್ಷಿಯು

532. ಕೃತಜ್ಞಃ = ಭಕ್ತರು ಮಾಡಿದ ಸೇವೆಯನ್ನು ತಿಳಿಯುವವನು, ಎಲ್ಲರ ಕರ್ಮಗಳನ್ನೂ ತಿಳಿದವನು

533. ಮೇದಿನೀಪತಿಃ = ಭೂದೇವಿಗೆ ಪತಿಯಾದವನು

534. ತ್ರಿಪದಃ = ಮೂರು ಹೆಜ್ಜೆಗಳುಳ್ಳವನು

535. ತ್ರಿದಶಾಧ್ಯಕ್ಷಃ = ದೇವತೆಗಳಿಗೆ ಅಧ್ಯಕ್ಷ-ಸ್ವಾಮಿಯಾದವನು, ಜಾಗ್ರತ್-ಸ್ವಪ್ನ-ಸುಷುಪ್ತಿಗಳೆಂಬ ಮೂರು ಸ್ಥಿತಿಗಳಿಗೆ ಅಧ್ಯಕ್ಷನಾದವನು

536. ಮಹಾಶೃಂಗಃ = ಮತ್ಸ್ಯ-ವರಾಹ-ಪ್ರಣವ ರೂಪಗಳಲ್ಲಿ ದೊಡ್ಡ ಶೃಂಗವುಳ್ಳವನು

537. ಕೃತಾಂತಕೃತ್ = ಸೃಷ್ಟಿಸಲ್ಪಡುವ ಜಗತ್ತನ್ನು ಧ್ವಂಸ ಮಾಡುವವನು, ಮೃತ್ಯುವನ್ನು ಕೊಡುವವನು, ಮೃತ್ಯುವಿಗೂ ಮೃತ್ಯು ಆದವನು

538. ಮಹಾವರಾಹಃ = ಮಹಾವರಾಹ ಸ್ವಾಮಿಯು

539. ಗೋವಿಂದಃ = ಗೋವುಗಳಿಂದ/ವೇದವಾಣಿಗಳಿಂದ ಹೊಂದಲ್ಪಡುವವನು, ವೇದವಾಣಿಗಳನ್ನು ತಿಳಿದವನು, ಹಸುಗಳನ್ನು ಪಡೆದವನು

540. ಸುಷೇಣಃ = ಶ್ರೇಷ್ಠವಾದ ಸೇನೆಗಳುಳ್ಳವನು, ವಿಷ್ವಕ್ಸೇನನು, ಗಣಪತಿಯು

541. ಕನಕಾಂಗದೀ = ಸ್ವರ್ಣಮಯವಾದ ತೋಳ್ಬಳೆಗಳುಳ್ಳವನು

542. ಗುಹ್ಯಃ = ಬುದ್ಧಿ ಗುಹೆಯಲ್ಲಿ ನಿಗೂಢನಾಗಿರುವವನು, ರಹಸ್ಯ ವಿದ್ಯೆಯಿಂದ ತಿಳಿಯಲ್ಪಡುವವನು

543. ಗಭೀರಃ = ಗಂಭೀರನಾದವನು, ಆಕಾಶದಲ್ಲಿ ಅಡಗಿರುವವನು

544. ಗಹನಃ = ಪ್ರವೇಶಿಸಲು ಕಷ್ಟಸಾಧ್ಯನಾಗಿರುವವನು

545. ಗುಪ್ತಃ = ಗೂಢನಾಗಿರುವವನು

546. ಚಕ್ರಗದಾಧರಃ = ತತ್ತ್ವರೂಪವಾದ ಸುದರ್ಶನ ಚಕ್ರವನ್ನೂ ಮತ್ತು ಕೌಮೋದಕೀ ಎಂಬ ಗದೆಯನ್ನೂ ಧರಿಸಿರುವವನು

547. ವೇಧಾಃ = ಸೃಷ್ಟಿಕರ್ತನು, ವಿಧಾನ ಮಾಡುವವನು

548. ಸ್ವಾಂಗಃ = ಸೃಷ್ಟಿ-ಸ್ಥಿತಿ-ಲಯಾದಿ ವ್ಯಾಪಾರಗಳನ್ನು ಮಾಡುವಾಗ ತನಗೆ ತಾನೇ ಅಂಗವಾಗಿರುವವನು, ಬೇರೆ ಯಾವ ಸಹಕಾರಿಗಳೂ ಇಲ್ಲದವನು

549. ಅಜಿತಃ = ಸರ್ವೋತ್ಕೃಷ್ಟನಾದವನು, ಯಾರಿಂದಲೂ ಗೆಲ್ಲಲು ಆಗದವನು

550. ಕೃಷ್ಣಃ = ನೀಲವರ್ಣವುಳ್ಳವನು, ಎಲ್ಲರನ್ನೂ ಆಕರ್ಷಣೆ ಮಾಡುವವನು, ಎಲ್ಲರನ್ನೂ ಉಪಸಂಹಾರ ಮಾಡುವವನು, ಸದಾನಂದ ಸ್ವರೂಪಿಯು, ಕೃಷ್ಣದ್ವೈಪಾಯನ ವ್ಯಾಸರೂಪಿಯು

551. ದೃಢಃ = ಚ್ಯುತಿಯಿಲ್ಲದವನು, ಬಲಶಾಲಿಯು

552. ಸಂಕರ್ಷಣೋಽಚ್ಯುತಃ = ಎಲ್ಲ ಪ್ರಾಣಿಗಳನ್ನೂ ಸಂಹಾರ ಕಾಲದಲ್ಲಿ ತನ್ನಲ್ಲಿಗೇ ಎಳೆದುಕೊಳ್ಳುವವನು ಮತ್ತು ಸ್ವರೂಪ-ಸ್ಥಾನ-ಸಂಕಲ್ಪಾದಿಗಳಲ್ಲಿ ಚ್ಯುತಿ ಇಲ್ಲದವನು

553. ವರುಣಃ = ಆವರಿಸಿರುವವನು, ಜಲಕ್ಕೂ-ಅಪಾನಕ್ಕೂ-ರಾತ್ರಿಗೂ ಅಧಿದೇವತೆಯಾದ ವರುಣ ರೂಪಿಯು

554. ವಾರುಣಃ = ವರುಣನ ಮಗನು, ಅಗಸ್ತ್ಯ/ವಸಿಷ್ಠ ರೂಪಿಯು

555. ವೃಕ್ಷಃ = ವೃಕ್ಷ ಸದೃಶನು, ವೃಕ್ಷದಂತೆ ಅಚಲನಾಗಿರುವವನು/ಆಶ್ರಯನಾಗಿರುವವನು, ಇಷ್ಟಾರ್ಥಗಳನ್ನು ಪಾಲಿಸುವವನು, ಮಹಾ ವೃಕ್ಷ ಸ್ವರೂಪನು

556. ಪುಷ್ಕರಾಕ್ಷಃ = ಕಮಲ ಸದೃಶವಾದ ಕಣ್ಣುಗಳುಳ್ಳವನು, ಹೃದಯ ಕಮಲದಲ್ಲಿ ಪ್ರಕಾಶಿಸುವವನು

557. ಮಹಾಮನಾಃ = ದೊಡ್ಡ ಮನಸ್ಸುಳ್ಳವನು, ಮಹಾತ್ಮನು, ಎಲ್ಲವನ್ನೂ ವ್ಯಾಪಿಸಿ ರಕ್ಷಿಸುವ ಉದಾರಮನಸ್ಸಿನವನು

558. ಭಗವಾನ್ = ಷಡ್ಗುಣೈಶ್ವರ್ಯ (ಜ್ಞಾನ, ವೈರಾಗ್ಯ, ಬಲ, ಐಶ್ವರ್ಯ, ಶಕ್ತಿ, ತೇಜಸ್ಸು) ಸಂಪನ್ನನು

559. ಭಗಹಾ = ದೈತ್ಯರ ಐಶ್ವರ್ಯವನ್ನು ನಾಶ ಮಾಡುವವನು, ಭೌತಿಕವಾದ ಐಶ್ವರ್ಯವನ್ನು ನಾಶ ಮಾಡುವವನು, ಸಂಹಾರ ಸಮಯದಲ್ಲಿ ಎಲ್ಲರ ಐಶ್ವರ್ಯವನ್ನೂ ನಾಶ ಮಾಡುವವನು

560. ಆನಂದೀ/ನಂದೀ = ಪರಮಾನಂದವುಳ್ಳವನು, ಸರ್ವ ಸಂಪತ್ತುಗಳಿಂದ ಪೂರ್ಣನಾದವನು

561. ವನಮಾಲೀ = ಭೂತಸೂಕ್ಷ್ಮಗಳ ಮಾಲೆಯಾದ ವೈಜಯಂತೀ ವನಮಾಲೆಯನ್ನು ಧರಿಸಿರುವವನು

562. ಹಲಾಯುಧಃ = ನೇಗಿಲನ್ನು ಆಯುಧವನ್ನಾಗಿ ಧರಿಸಿರುವ ಬಲರಾಮ ಸ್ವರೂಪನು

563. ಆದಿತ್ಯಃ = ಸೂರ್ಯ ರೂಪಿಯು, ಅದಿತಿಗೆ ಪುತ್ರನಾಗಿ ಜನಿಸಿದವನು

564. ಜ್ಯೋತಿರಾದಿತ್ಯಃ = ಜ್ಯೋತಿರ್ಮಯನಾದ ಆದಿತ್ಯರೂಪಿಯು, ಆದಿತ್ಯ ಮಂಡಲದಲ್ಲಿರುವ ಜ್ಯೋತಿರ್ಮಯ ಪುರುಷನು

565. ಸಹಿಷ್ಣುಃ = ಸಹನಶೀಲನು, ದ್ವಂದ್ವಗಳನ್ನು ಸಹಿಸುವ ನಾರಾಯಣ ಮುನಿಸ್ವರೂಪನು, ಭಕ್ತರ ಅಪರಾಧಗಳನ್ನು ಸಹಿಸುವವನು, ಶತ್ರುಗಳ ಆಕ್ರಮಣವನ್ನು ಸಹಿಸುವವನು

566. ಗತಿಸತ್ತಮಃ = ಶ್ರೇಷ್ಠತಮವಾದ ಗತಿಯು

567. ಸುಧನ್ವಾ = ಶ್ರೇಷ್ಠವಾದ ಧನುಸ್ಸುಳ್ಳವನು, ಅಹಂಕಾರ ತತ್ತ್ವರೂಪವಾದ ಶಾಙ್ರ್ಗ ಧನುಸ್ಸುಳ್ಳವನು

568. ಖಂಡಪರಶುಃ = ರುದ್ರ ರೂಪಿಯು, ಪರಶುವಿನಿಂದ ಖಂಡನೆಯನ್ನು ಮಾಡಿದ ನರಮಹರ್ಷಿ ಸ್ವರೂಪಿಯು, ಶತ್ರುಗಳನ್ನು ಖಂಡಿಸಿದ ಗಂಡುಗೊಡಲಿಯುಳ್ಳ ಪರಶುರಾಮ ಸ್ವರೂಪಿಯು

569. ದಾರುಣಃ = ದುಷ್ಟರಿಗೆ ಭಯಂಕರನಾದವನು, ಶತ್ರುಗಳನ್ನು ಸೀಳಿ ಹಾಕುವವನು

570. ದ್ರವಿಣಪ್ರದಃ = ಐಶ್ವರ್ಯವನ್ನು ಅನುಗ್ರಹಿಸುವವನು

571. ದಿವಃಸ್ಪೃಕ್/ದಿವಿಸ್ಪೃಕ್ = ಪ್ರಕಾಶಮಾನವಾದ ಸ್ಥಾನವನ್ನು ಮುಟ್ಟಿರುವವನು, ಸ್ವರ್ಗವನ್ನು ಮುಟ್ಟಿದವನು

572. ಸರ್ವದೃಗ್ವ್ಯಾಸಃ = ಸರ್ವಜ್ಞನಾದ ವ್ಯಾಸ ಸ್ವರೂಪಿಯು

573. ವಾಚಸ್ಪತಿರಯೋನಿಜಃ/ ವಾಚಸ್ಪತಿಃ = ವಾಣಿಗೆ ಅಧಿಪತಿಯಾದ ಹಯಗ್ರೀವ ಸ್ವರೂಪಿಯು, ಬೃಹಸ್ಪತಿ ರೂಪಿಯು, ಸರ್ವಕ್ಕೂ ಕಾರಣನಾಗಿ ತಾಯಿಯಿಂದ ಹುಟ್ಟದವನು, ವಾಣಿಗೆ ಅಧಿಪತಿಯಾಗಿ ಭಗವಂತನ ಮುಖದಿಂದ ಸಂಜನಿಸಿದ ಸಾರಸ್ವತ ವ್ಯಾಸರೂಪಿಯು

574. ತ್ರಿಸಾಮಾ = ಮೂರು ಸಾಮಶ್ರುತಿಗಳುಳ್ಳವನು, ಮೂರು ಸಾಮಗಳಿಂದ ಸ್ತುತಿಸಲ್ಪಟ್ಟವನು

575. ಸಾಮಗಃ = ಸಾಮವನ್ನು ಗಾನ ಮಾಡುವವನು

576. ಸಾಮ = ವೇದಗಳಲ್ಲಿ ಪರಮಾತ್ಮನ ಮಹಾ ವಿಭೂತಿಯಾದ ಸಾಮವೇದ ಸ್ವರೂಪಿಯು

577. ನಿರ್ವಾಣಂ = ಪರಮಾನಂದ ಮೋಕ್ಷ ಸ್ವರೂಪಿಯು, ಆವರಣವಿಲ್ಲದವನು, ಆನಂದದಾಯಕನು

578. ಭೇಷಜಂ = ಸಂಸಾರ ರೋಗಕ್ಕೆ ಒಳ್ಳೆಯ ಔಷಧವಾಗಿರುವವನು

579. ಭಿಷಕ್ = ಭವರೋಗ ವೈದ್ಯನು, ವೈದ್ಯನಾಥೇಶ್ವರನು, ಧನ್ವಂತರಿಯು

580. ಸಂನ್ಯಾಸಕೃತ್ = ಸಂನ್ಯಾಸಾಶ್ರಮವನ್ನು ಮತ್ತು ಸಂನ್ಯಾಸ ಭಾವವನ್ನು ಉಂಟುಮಾಡುವವನು

581. ಶಮಃ = ಇಂದ್ರಿಯ ನಿಗ್ರಹವುಳ್ಳವನು

582. ಶಾಂತಃ = ಶಮಭಾವ ಉಳ್ಳವನು, ವಿಷಯಸುಖಾಸಕ್ತಿ ಇಲ್ಲದವನು

583. ನಿಷ್ಠಾ = ಎಲ್ಲಕ್ಕೂ ನೆಲೆಯಾದವನು, ವ್ರತನಿಷ್ಠಾ ಸ್ವರೂಪಿಯು

584. ಶಾಂತಿಃ = ಎಲ್ಲ ದೋಷಗಳ ನಿವೃತ್ತಿ ಸ್ವರೂಪಿಯು

585. ಪರಾಯಣಂ = ಪರಮಗತಿಯು, ಶ್ರೇಷ್ಠವಾದ ನಿವಾಸಸ್ಥಾನವುಳ್ಳವನು

586. ಶುಭಾಂಗಃ = ಶುಭವಾದ ಅವಯವಗಳುಳ್ಳವನು

587. ಶಾಂತಿದಃ = ಪರಮಾನಂದ ರೂಪವಾದ ನಿತ್ಯಶಾಂತಿಯನ್ನು ಕೊಡುವವನು

588. ಸ್ರಷ್ಟಾ = ಜಗತ್ತನ್ನು ಸೃಷ್ಟಿ ಮಾಡುವವನು

589. ಕುಮುದಃ = ಭೂಮಿಗೆ ಆನಂದವನ್ನುಂಟು ಮಾಡುವವನು, ಭೂಮಿಯಲ್ಲಿ ಆನಂದಿಸುವವನು, ಚಂದ್ರರೂಪಿಯು, ಕುಮುದಪುಷ್ಪದಂತೆ ಸ್ವಚ್ಛವಾಗಿರುವವನು

590. ಕುವಲೇಶಯಃ = ಜಲದಲ್ಲಿ ಶಯನಿಸಿರುವವನು

591. ಗೋಹಿತಃ = ಹಸುಗಳಿಗೂ, ವೇದವಾಣಿಗೂ, ಭೂಮಿಗೂ ಹಿತನಾದವನು

592. ಗೋಪತಿಃ = ವಾಣಿಗೆ, ಭೂಮಿಗೆ ಮತ್ತು ಹಸುಗಳಿಗೆ ಪತಿಯಾದವನು

593. ಗೋಪ್ತಾ = ಜಗತ್ತನ್ನು ರಕ್ಷಿಸುವವನು, ತನ್ನನ್ನು ಮರೆಮಾಚಿಕೊಂಡಿರುವವನು

594. ವೃಷಭಾಕ್ಷಃ = ಧರ್ಮ ದೃಷ್ಟಿಯುಳ್ಳವನು, ಇಷ್ಟಾರ್ಥಗಳನ್ನು ವರ್ಷಿಸುವ ಕಣ್ಣುಗಳುಳ್ಳವನು, ವೇದಗಳೇ ಕಣ್ಣುಗಳಾಗಿರುವವನು

595. ವೃಷಪ್ರಿಯಃ = ಧರ್ಮ ಪ್ರಿಯನಾದವನು

596. ಅನಿವರ್ತೀ = ಮುಕ್ತಾತ್ಮರನ್ನು ಹಿಂದಿರುಗಿಸದವನು, ಯುದ್ಧದಲ್ಲಿ ಹಿಮ್ಮೆಟ್ಟದವನು, ತನ್ನ ಮಾತಿನಿಂದ ಹಿಮ್ಮೆಟ್ಟದವನು, ಧರ್ಮದಿಂದ ಹಿಮ್ಮೆಟ್ಟದವನು

597. ನಿವೃತ್ತಾತ್ಮಾ = ವಿಷಯಗಳಿಂದ ನಿವೃತ್ತವಾದ ಮನಸ್ಸುಳ್ಳವನು, ನಿವೃತ್ತಿ ಧರ್ಮದಲ್ಲಿ ನಿಷ್ಠೆಯುಳ್ಳ ಮನಸ್ಸುಳ್ಳವನು

598. ಸಂಕ್ಷೇಪ್ತಾ = ಉಪಸಂಹಾರ ಮಾಡಿಕೊಳ್ಳುವವನು, ವಿಸ್ತಾರವಾದ ಜಗತ್ತನ್ನು ಸಂಹಾರ ಸಮಯದಲ್ಲಿ ತನ್ನಲ್ಲಿಯೇ ಸಂಕ್ಷೇಪಿಸಿಕೊಳ್ಳುವವನು

599. ಕ್ಷೇಮಕೃತ್ = ಕ್ಷೇಮವನ್ನುಂಟು ಮಾಡುವವನು, ಲಬ್ಧವಾದ ವಸ್ತುಗಳನ್ನು ಸಂರಕ್ಷಿಸುವವನು

600. ಶಿವಃ = ಶುದ್ಧಮಂಗಳ ಸ್ವರೂಪಿಯು, ಮಂಗಳವನ್ನುಂಟು ಮಾಡುವವನು
